## Cara Memuat Gambar DjVu?
Memuat file DjVu sangat mudah dengan Aspose.Imaging. Gunakan kelas `Image` untuk membuka dokumen dalam blok try‑with‑resources, yang secara otomatis melepaskan handle file setelah pemrosesan. Berikan path ke file .djvu, dan pustaka akan mengurai halamannya, membuatnya dapat diakses untuk manipulasi lebih lanjut.

`Image` adalah kelas dalam Aspose.Imaging yang mewakili file gambar dan menyediakan metode untuk memuat serta memanipulasi data gambar.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.djvu.DjvuImage;

String dataDir = "YOUR_DOCUMENT_DIRECTORY/Sample.djvu";
try (DjvuImage image = (DjvuImage) Image.load(dataDir)) {
    // The image is now loaded and ready for further processing.
}
```

- **`dataDir`**: Path ke file DjVu Anda. Ubah `YOUR_DOCUMENT_DIRECTORY` ke path direktori yang sebenarnya.  
- **`try‑with‑resources`**: Secara otomatis mengelola pelepasan sumber daya setelah pemuatan.

## Cara Mengonfigurasi Opsi Ekspor PNG?
Untuk menghasilkan PNG grayscale, buat objek `PngOptions` dan atur `colorType`-nya ke `PngColorType.Grayscale`. Konfigurasi ini memastikan gambar output menggunakan satu saluran intensitas, mengurangi ukuran file sambil mempertahankan detail visual. Opsi tambahan seperti tingkat kompresi juga dapat disesuaikan sesuai kebutuhan.

`PngOptions` mendefinisikan pengaturan untuk menyimpan gambar dalam format PNG, seperti tipe warna, tingkat kompresi, dan interlacing.  
`PngColorType` mengenumerasi format warna yang didukung untuk file PNG, termasuk Grayscale, Truecolor, dan Indexed.

```java
import com.aspose.imaging.imageoptions.PngOptions;
import com.aspose.imaging.fileformats.png.PngColorType;

PngOptions exportOptions = new PngOptions();
exportOptions.setColorType(PngColorType.Grayscale);
```

- **`PngColorType.Grayscale`**: Mengonversi gambar ke grayscale, ideal untuk keperluan arsip atau mengurangi ukuran file.

## Cara Mendefinisikan Area Ekspor pada Halaman DjVu?
Tentukan wilayah persegi panjang untuk diekstrak dari halaman DjVu dengan mengatur properti `exportArea` pada `DjvuMultiPageOptions`. Persegi panjang didefinisikan dalam piksel dan menentukan bagian halaman yang akan dirender dan disimpan, memungkinkan Anda fokus pada area tertentu yang diinginkan.

`DjvuMultiPageOptions` menentukan opsi untuk mengekspor halaman dari dokumen DjVu, termasuk indeks halaman dan area ekspor.

```java
import com.aspose.imaging.Rectangle;

Rectangle exportArea = new Rectangle(0, 0, 500, 500);
int exportPageIndex = 2; // Index starts from zero.
```

- **`exportArea`**: Menentukan dimensi persegi panjang untuk dipotong. Modifikasi nilai ini sesuai kebutuhan Anda.

## Cara Mengatur Opsi Ekspor Multi‑Halaman?
Ketika bekerja dengan file DjVu multi‑halaman, gunakan `DjvuMultiPageOptions` untuk memilih halaman yang diinginkan melalui properti `pageIndex` dan secara opsional menggabungkannya dengan `exportArea`. Pendekatan ini memungkinkan Anda mengekspor wilayah tertentu dari halaman mana pun tanpa memproses seluruh dokumen, mengoptimalkan kinerja untuk file besar.

```java
import com.aspose.imaging.fileformats.djvu.DjvuMultiPageOptions;

exportOptions.setMultiPageOptions(new DjvuMultiPageOptions(exportPageIndex, exportArea));
```

- **`DjvuMultiPageOptions`**: Menggabungkan indeks halaman dan area untuk mengekspor bagian spesifik dari file DjVu multi‑halaman.

## Cara Menyimpan Gambar?
Setelah mengonfigurasi opsi ekspor, panggil metode `save` pada instance `Image`, memberikan path file target dan objek `PngOptions`. Pastikan direktori output ada dan memiliki izin menulis; pustaka kemudian akan menulis PNG grayscale ke disk.

`save` adalah metode dari kelas `Image` yang menulis data gambar ke file yang ditentukan menggunakan opsi format yang diberikan.

```java
import com.aspose.imaging.Image;

String outputPath = "YOUR_OUTPUT_DIRECTORY/ConvertSpecificPortionOfDjVuPage_out.png";
image.save(outputPath, exportOptions);
```

- **`outputPath`**: Tentukan lokasi tempat Anda ingin menyimpan file PNG. Pastikan direktori ada sebelum menjalankan.
