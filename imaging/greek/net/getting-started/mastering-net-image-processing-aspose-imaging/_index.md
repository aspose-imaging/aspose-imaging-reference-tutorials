## Οδηγός Εφαρμογής
### Φόρτωση και επεξεργασία εικόνας
Αυτή η ενότητα περιγράφει λεπτομερώς τον τρόπο φόρτωσης μιας εικόνας και ανάκτησης της ημερομηνίας τελευταίας τροποποίησής της χρησιμοποιώντας το Aspose.Imaging. Αυτή η λειτουργία είναι απαραίτητη για εφαρμογές που πρέπει να παρακολουθούν αλλαγές ή να ενημερώνουν εικόνες με βάση τα μεταδεδομένα τους.

#### Βήμα 1: Ρύθμιση καταλόγων
Αρχικά, ορίστε τους καταλόγους εισόδου και εξόδου όπου αποθηκεύονται οι εικόνες σας:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση εικόνας
Χρήση `Image.Load` για να διαβάσει ένα αρχείο εικόνας. Αυτή η μέθοδος επιστρέφει ένα γενικό `Image` αντικείμενο, το οποίο μπορείτε να μετατρέψετε σε ένα `RasterImage` για πιο συγκεκριμένες λειτουργίες:

```csharp
using (RasterImage image = (RasterImage)Image.Load(dataDir + "/aspose-logo.jpg"))
{
    // Η λογική επεξεργασίας εικόνας εφαρμόζεται εδώ.
}
```

#### Βήμα 3: Ανάκτηση ημερομηνίας τελευταίας τροποποίησης
Για να λάβετε την ημερομηνία τελευταίας τροποποίησης ενός αρχείου εικόνας, χρησιμοποιήστε το `GetModifyDate` μέθοδος. Αυτή η μέθοδος μπορεί να λάβει υπόψη τα μεταδεδομένα XMP εάν χρειάζεται:

```csharp
string modifyDateFile = image.GetModifyDate(true).ToString();
string modifyDateXmp = image.GetModifyDate(false).ToString();

Console.WriteLine("Last modified date from file: " + modifyDateFile);
Console.WriteLine("Last modified date from XMP metadata: " + modifyDateXmp);
```

**Εξήγηση**: 
- `true` στο `GetModifyDate` Η μέθοδος λαμβάνει υπόψη τα μεταδεδομένα του συστήματος αρχείων.
- `false` Ανακτά ημερομηνίες από τα μεταδεδομένα XMP της εικόνας, εάν είναι διαθέσιμα.
