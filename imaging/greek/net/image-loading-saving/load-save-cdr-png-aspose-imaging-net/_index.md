## Οδηγός Εφαρμογής

### Δυνατότητα: Φόρτωση και αποθήκευση εικόνας ως PNG

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο CDR χρησιμοποιώντας το Aspose.Imaging και να το αποθηκεύσετε ως PNG, εφαρμόζοντας συγκεκριμένες επιλογές rasterization.

#### Βήμα 1: Ορισμός διαδρομών και φόρτωση της εικόνας

Αρχικά, ρυθμίστε τις διαδρομές εισόδου και εξόδου. Αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογο εγγράφων σας.

```csharp
using System.IO;
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Cdr;
using Aspose.Imaging.ImageOptions;

public class ImageLoadingAndSaving
{
    public static void Run()
    {
        string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "CDR");
        string inputFileName = Path.Combine(dataDir, "test2.cdr");

        using (var image = (CdrImage)Image.Load(inputFileName))
        {
            // Η εικόνα φορτώθηκε με επιτυχία.
        }
    }
}
```
*Γιατί*: Το `Image.Load` Η μέθοδος χρησιμοποιείται για τη φόρτωση του αρχείου CDR στη μνήμη για περαιτέρω επεξεργασία. 

#### Βήμα 2: Αποθήκευση ως PNG με επιλογές ραστεροποίησης

Στη συνέχεια, διαμορφώστε και αποθηκεύστε την εικόνα ως PNG χρησιμοποιώντας συγκεκριμένες επιλογές ραστεροποίησης.

```csharp
string outputFileName = Path.Combine(dataDir, "result.png");

image.Save(outputFileName, new PngOptions()
{
    VectorRasterizationOptions = new CdrRasterizationOptions
    {
        Positioning = PositioningTypes.Relative
    }
});
```
*Γιατί*: `PngOptions` επιτρέπει την προσαρμογή της εξόδου PNG. Το `VectorRasterizationOptions` βεβαιωθείτε ότι η εικόνα διατηρεί τις διανυσματικές της ιδιότητες κατά την αποθήκευση.

### Χαρακτηριστικό: Διαγραφή αρχείου

#### Επισκόπηση
Μάθετε πώς να διαγράφετε ένα αρχείο χρησιμοποιώντας τον χώρο ονομάτων System.IO του .NET, διασφαλίζοντας ότι η εφαρμογή σας διαχειρίζεται τους πόρους αποτελεσματικά.

#### Βήμα 1: Ορισμός διαδρομών και διαγραφή του αρχείου

Ορίστε τη διαδρομή για το αρχείο που θέλετε να διαγράψετε:

```csharp
public class FileDeletion
{
    public static void Run()
    {
        string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "CDR");
        string outputFileName = Path.Combine(dataDir, "result.png");

        if (File.Exists(outputFileName))
        {
            File.Delete(outputFileName);
        }
    }
}
```
*Γιατί*Ελέγχετε πάντα την ύπαρξη του αρχείου πριν επιχειρήσετε τη διαγραφή του, για να αποφύγετε εξαιρέσεις.
