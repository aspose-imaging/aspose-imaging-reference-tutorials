## Οδηγός Εφαρμογής

Θα καλύψουμε δύο κύριες λειτουργίες: τη φόρτωση μιας εικόνας και την εξαγωγή της σε μορφή WebP.

### Φόρτωση εικόνας

**Επισκόπηση:** Αυτή η ενότητα δείχνει πώς να φορτώσετε μια εικόνα από έναν κατάλογο χρησιμοποιώντας το Aspose.Imaging για .NET. Είναι απαραίτητο για την προετοιμασία εικόνων για περαιτέρω επεξεργασία ή χειρισμό.

#### Βήμα 1: Ορίστε τη διαδρομή καταλόγου σας
Αρχικά, ορίστε τη διαδρομή όπου θα αποθηκευτούν οι εικόνες σας:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση της εικόνας
Φόρτωση εικόνας από τον καθορισμένο κατάλογο χρησιμοποιώντας το `Image.Load` μέθοδος:
```csharp
using (Image image = Image.Load(dataDir + "/SampleImage1.bmp"))
{
    // Σε αυτό το σημείο, η «εικόνα» φορτώνεται και είναι έτοιμη για χειρισμό.
}
```
**Εξήγηση:** Ο `Image.Load()` Η μέθοδος ανοίγει μια ροή αρχείων από τη δεδομένη διαδρομή, επιτρέποντάς σας να εργαστείτε με τα δεδομένα εικόνας απευθείας στη μνήμη.

### Εξαγωγή σε μορφή WebP

**Επισκόπηση:** Με το Aspose.Imaging, η εξαγωγή εικόνων σε σύγχρονες μορφές όπως το WebP είναι απλή. Αυτή η λειτουργία επιτρέπει σημαντική μείωση του μεγέθους διατηρώντας παράλληλα την οπτική πιστότητα.

#### Βήμα 1: Ορισμός επιλογών εξαγωγής
Διαμορφώστε τις ρυθμίσεις εξαγωγής χρησιμοποιώντας `WebPOptions`:
```csharp
WebPOptions options = new WebPOptions
{
    Quality = 50,
    Lossless = false // Χρησιμοποιήστε συμπίεση με απώλειες.
};
```
**Εξήγηση:** Προσαρμόστε την ποιότητα για να εξισορροπήσετε την καθαρότητα της εικόνας και το μέγεθος του αρχείου. Ρύθμιση `Lossless` να `false` επιτρέπει συμπίεση με απώλειες, η οποία συνήθως έχει ως αποτέλεσμα μικρότερα αρχεία.

#### Βήμα 2: Αποθήκευση ως WebP
Εξαγάγετε την εικόνα που έχετε φορτώσει χρησιμοποιώντας τις καθορισμένες επιλογές:
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/ExportToWebP_out.webp";
image.Save(outputPath, options);
```
**Εξήγηση:** Ο `Save` Η μέθοδος γράφει την εικόνα στην καθορισμένη διαδρομή στην επιθυμητή μορφή.
