## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή πλαισίων GIF σε TIFF

Αυτός ο οδηγός δείχνει πώς να φορτώσετε μια εικόνα GIF, να επαναλάβετε τα καρέ της και να μετατρέψετε κάθε καρέ σε ξεχωριστό αρχείο TIFF.

#### Βήμα 1: Φόρτωση της εικόνας GIF

Καθορίστε τον κατάλογο όπου είναι αποθηκευμένο το GIF σας:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου εγγράφων σας
```

Φορτώστε την εικόνα GIF δημιουργώντας ένα `Image` αντικείμενο:

```csharp
Image objImage = Image.Load(Path.Combine(dataDir, "asposelogo.gif"));
```

#### Βήμα 2: Μεταφορά σε GifImage και Επανάληψη μέσω Πλαισίων

Αποκτήστε πρόσβαση στα καρέ στο GIF μεταδίδοντας την εικόνα που έχετε φορτώσει σε ένα `GifImage`:

```csharp
using (GifImage gif = (GifImage)objImage)
{
    for (int i = 0; i < gif.Frames.Length; i++)
    {
        Frame frame = gif.Frames[i];
        
        if (frame == null) continue;

        // Ο κώδικας για τη μετατροπή κάθε καρέ θα βρίσκεται εδώ
    }
}
```

#### Βήμα 3: Μετατροπή κάθε καρέ σε TIFF

Για κάθε έγκυρο `Frame`, εξαγάγετε το πλαίσιο και αποθηκεύστε το ως αρχείο TIFF:

```csharp
string tiffOutputPath = Path.Combine(dataDir, $"frame_{i}.tiff");
frame.Save(tiffOutputPath, new TiffOptions(TiffExpectedFormat.Default));
```
