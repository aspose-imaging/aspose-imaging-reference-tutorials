## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Ανάγνωση αρχείων
Αυτή η λειτουργία δείχνει πώς να διαβάσετε αρχεία από έναν καθορισμένο κατάλογο χρησιμοποιώντας Java. 

#### Επισκόπηση
Η ανάγνωση αρχείων είναι το πρώτο βήμα για την επεξεργασία δεδομένων που είναι αποθηκευμένα σε δίσκο. Αυτή η λειτουργικότητα είναι κρίσιμη για εφαρμογές που πρέπει να απορροφούν πληροφορίες, όπως προγράμματα ανάγνωσης αρχείων καταγραφής ή προγράμματα φόρτωσης ρυθμίσεων.

#### Βήμα προς βήμα εφαρμογή
##### Βήμα 1: Ορίστε τη διαδρομή αρχείου εισόδου
```java
String filePath = "@YOUR_DOCUMENT_DIRECTORY/input.txt";
```
Καθορίστε τον κατάλογο και το όνομα του αρχείου σας χρησιμοποιώντας ένα σύμβολο κράτησης θέσης για να διασφαλίσετε ευελιξία.

##### Βήμα 2: Δημιουργήστε ένα `File` Αντικείμενο
```java
File inputFile = new File(filePath);
```
Αυτό το αντικείμενο αντιπροσωπεύει το αρχείο που σκοπεύετε να διαβάσετε, επιτρέποντας στα βοηθητικά προγράμματα εισόδου/εξόδου της Java να αλληλεπιδράσουν με αυτό.

##### Βήμα 3: Διαβάστε όλες τις γραμμές
```java
List<String> lines = Files.readAllLines(inputFile.toPath(), StandardCharsets.UTF_8);
```
Ο `Files.readAllLines()` Η μέθοδος διαβάζει όλο το περιεχόμενο σε μια λίστα συμβολοσειρών, χρησιμοποιώντας κωδικοποίηση UTF-8 για συμβατότητα.

##### Βήμα 4: Επεξεργασία κάθε γραμμής
```java
for (String line : lines) {
    System.out.println(line); // Εκτύπωση κάθε γραμμής στην κονσόλα
}
```

### Χαρακτηριστικό 2: Εγγραφή αρχείων
Αυτή η λειτουργία επιδεικνύει την εγγραφή κειμένου σε αρχεία σε Java, κάτι που είναι απαραίτητο για την καταγραφή, την εξαγωγή δεδομένων και πολλά άλλα.

#### Επισκόπηση
Η σύνταξη αρχείων επιτρέπει στην εφαρμογή σας να εξάγει δεδομένα, όπως αναφορές ή αρχεία καταγραφής, τα οποία μπορεί να είναι απαραίτητα για τον εντοπισμό σφαλμάτων και την ανάλυση δεδομένων.

#### Βήμα προς βήμα εφαρμογή
##### Βήμα 1: Ορίστε τη διαδρομή αρχείου εξόδου
```java
String outputPath = "@YOUR_OUTPUT_DIRECTORY/output.txt";
```
Ορίστε τη διαδρομή όπου θέλετε να γραφτεί το αρχείο σας. Και πάλι, χρησιμοποιήστε ένα σύμβολο κράτησης θέσης για ευελιξία.

##### Βήμα 2: Δημιουργήστε ένα `BufferedWriter`
```java
try (BufferedWriter writer = Files.newBufferedWriter(Paths.get(outputPath), StandardCharsets.UTF_8)) {
    // Γράψτε γραμμές στο αρχείο
}
```
`BufferedWriter` Χρησιμοποιείται για την αποτελεσματική εγγραφή δεδομένων κειμένου, τα οποία αποθηκεύονται προσωρινά στη μνήμη πριν από την εκκαθάριση στον δίσκο.

##### Βήμα 3: Γράψτε γραμμές κειμένου
```java
writer.write("Hello, World!\n");
writer.write("This is a sample output.\n");
```
