## Οδηγός Εφαρμογής

Αυτή η ενότητα χωρίζεται σε δύο λειτουργίες: αποθήκευση εικόνων GIF ως TIFF με επιλογές πλήρους καρέ και χωρίς επιλογές πλήρους καρέ.

### Λειτουργία 1: Φόρτωση και αποθήκευση εικόνας GIF ως TIFF με πλήρες καρέ

Αυτή η λειτουργία εστιάζει στη μετατροπή μιας κινούμενης εικόνας GIF σε μορφή TIFF πλήρους καρέ, διατηρώντας ολόκληρη την ακολουθία κινουμένων σχεδίων σε ένα αρχείο.

#### Βήμα προς βήμα εφαρμογή

**Φόρτωση της εικόνας GIF**

Ξεκινήστε φορτώνοντας την εικόνα GIF στη μνήμη χρησιμοποιώντας το Aspose.Imaging. `Image` κλάση. Αυτό θέτει τα θεμέλια για τυχόν επόμενες λειτουργίες.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string fileName = "Animation.gif";
string inputFilePath = Path.Combine(dataDir, fileName);
```

**Αποθήκευση ως TIFF με πλήρες καρέ**

Ρυθμίστε τις επιλογές αποθήκευσης ώστε να περιλαμβάνουν δυνατότητες πλήρους καρέ και καθορίστε ποια καρέ της κινούμενης εικόνας θέλετε να μετατρέψετε χρησιμοποιώντας `MultiPageOptions`.

```csharp
string outputFilePath = inputFilePath + "_FullFrame.tif";

using (Image image = Image.Load(inputFilePath))
{
    image.Save(outputFilePath, new TiffOptions(TiffExpectedFormat.TiffDeflateRgb) { 
        MultiPageOptions = new MultiPageOptions(new IntRange(2, 5)), // Καθορισμός πλαισίων
        FullFrame = true // Ενεργοποίηση αποθήκευσης πλήρους καρέ
    });
}
```

**Εκκαθάριση**

Μετά την επεξεργασία, φροντίστε να αφαιρέσετε το αρχείο εξόδου εάν δεν το χρειάζεστε πλέον.

```csharp
File.Delete(outputFilePath);
```

### Λειτουργία 2: Φόρτωση και αποθήκευση εικόνας GIF ως TIFF χωρίς πλήρες καρέ

Σε ορισμένες περιπτώσεις, ίσως προτιμάτε να αποθηκεύσετε μόνο συγκεκριμένα τμήματα μιας εικόνας. Αυτή η λειτουργία σάς επιτρέπει να το κάνετε αυτό ακριβώς αποθηκεύοντας ένα GIF ως TIFF που δεν είναι πλήρους καρέ.

#### Βήμα προς βήμα εφαρμογή

**Φόρτωση της εικόνας GIF**

Όπως και πριν, ξεκινήστε φορτώνοντας το GIF σας στη μνήμη.

```csharp
string outputFilePath1 = inputFilePath + "_NonFullFrame.tif";
```

**Αποθήκευση ως TIFF χωρίς πλήρες καρέ**

Κατά τη διαμόρφωση `TiffOptions`, παραλείψτε τη ρύθμιση πλήρους καρέ για να αποθηκεύσετε μόνο συγκεκριμένα καρέ.

```csharp
using (Image image = Image.Load(inputFilePath))
{
    image.Save(outputFilePath1, new TiffOptions(TiffExpectedFormat.TiffDeflateRgb) { 
        MultiPageOptions = new MultiPageOptions(new IntRange(2, 5)) // Καθορισμός πλαισίων
    });
}
```

**Εκκαθάριση**

Αφαιρέστε το αρχείο εξόδου όπως απαιτείται.

```csharp
File.Delete(outputFilePath1);
```
