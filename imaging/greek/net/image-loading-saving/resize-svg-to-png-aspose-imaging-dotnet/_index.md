## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: την αλλαγή μεγέθους μιας εικόνας SVG και την αποθήκευσή της ως αρχείο PNG. 

### Χαρακτηριστικό 1: Αλλαγή μεγέθους εικόνας SVG

#### Επισκόπηση

Η αλλαγή μεγέθους είναι ζωτικής σημασίας όταν χρειάζεται να προσαρμόσετε τις διαστάσεις ενός SVG για διαφορετικές απαιτήσεις εμφάνισης. Το Aspose.Imaging κάνει αυτή την εργασία απλή.

#### Βήματα:

**Βήμα 1: Φόρτωση του SVG σας**

Ξεκινήστε φορτώνοντας την εικόνα SVG χρησιμοποιώντας το `Image.Load` μέθοδος. Βεβαιωθείτε ότι η διαδρομή του αρχείου σας δείχνει προς το σημείο όπου είναι αποθηκευμένο το SVG σας.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (SvgImage image = (SvgImage)Image.Load(dataDir + "/aspose_logo.Svg"))
{
    // Συνεχίστε με την αλλαγή μεγέθους...
}
```

**Βήμα 2: Αλλαγή μεγέθους εικόνας**

Αλλάξτε το μέγεθος καθορίζοντας νέες διαστάσεις. Εδώ, πολλαπλασιάζουμε το αρχικό πλάτος και ύψος με παράγοντες για να επιτύχουμε το επιθυμητό μέγεθος.
```csharp
// Αυξήστε την κλίμακα του πλάτους της εικόνας κατά 10 και του ύψους της κατά 15.
image.Resize(image.Width * 10, image.Height * 15);
```

**Βήμα 3: Αποθήκευση της εικόνας με το νέο μέγεθος**

Αφού αλλάξετε το μέγεθος, αποθηκεύστε την εικόνα σας. Αυτό το παράδειγμα την αποθηκεύει σε μορφή PNG σε έναν καθορισμένο κατάλογο εξόδου.
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/Logotype_10_15_out.png";
image.Save(outputPath, new PngOptions()
{
    VectorRasterizationOptions = new SvgRasterizationOptions()
});
```

### Δυνατότητα 2: Αποθήκευση SVG ως PNG

#### Επισκόπηση

Η μετατροπή αρχείων SVG στην ευρέως υποστηριζόμενη μορφή PNG μπορεί να βελτιώσει τη συμβατότητα. Το Aspose.Imaging απλοποιεί αυτήν τη διαδικασία μετατροπής.

#### Βήματα:

**Βήμα 1: Ορισμός επιλογών PNG**

Ρυθμίστε το δικό σας `PngOptions` αντικείμενο, καθορίζοντας ρυθμίσεις ραστεροποίησης για τη διαδικασία μετατροπής.
```csharp
PngOptions pngOptions = new PngOptions()
{
    VectorRasterizationOptions = new SvgRasterizationOptions()
};
```

**Βήμα 2: Αποθήκευση ως PNG**

Τέλος, χρησιμοποιήστε αυτές τις επιλογές για να αποθηκεύσετε την εικόνα SVG ως αρχείο PNG.
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/Logotype_out.png";
image.Save(outputPath, pngOptions);
```
