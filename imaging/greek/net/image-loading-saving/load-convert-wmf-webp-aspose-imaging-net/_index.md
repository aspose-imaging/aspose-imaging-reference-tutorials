## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Φόρτωση εικόνας WMF

**Επισκόπηση**Αυτή η λειτουργία επιδεικνύει τη φόρτωση μιας εικόνας μετα-αρχείου των Windows (WMF) χρησιμοποιώντας το Aspose.Imaging για .NET.

#### Οδηγίες βήμα προς βήμα

##### Φόρτωση υπάρχουσας εικόνας WMF

Αρχικά, καθορίστε τον κατάλογο όπου αποθηκεύονται τα αρχεία WMF:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

Φορτώστε το αρχείο WMF και εμφανίστε τις διαστάσεις του για να επιβεβαιώσετε ότι έχει φορτωθεί σωστά:

```csharp
Image image = Image.Load(dataDir + "/input.wmf");
Console.WriteLine($"Image Dimensions: Width={image.Width}, Height={image.Height}");
image.Dispose();  // Πάντα να απορρίπτετε τα υλικά μετά τη χρήση
```

### Λειτουργία 2: Δημιουργία και ρύθμιση παραμέτρων επιλογών ραστεροποίησης για εικόνα WMF

**Επισκόπηση**Μάθετε πώς να ρυθμίσετε τις επιλογές ραστεροποίησης ειδικά για τη μετατροπή μιας εικόνας WMF.

#### Οδηγίες βήμα προς βήμα

##### Υπολογισμός αναλογίας διαστάσεων

Αρχικά, υπολογίστε την αναλογία διαστάσεων για να διατηρήσετε τις αναλογίες της εικόνας κατά τη μετατροπή:

```csharp
double k = (image.Width * 1.00) / image.Height;
```

##### Ορισμός επιλογών ραστεροποίησης

Δημιουργήστε μια παρουσία του `WmfRasterizationOptions` και διαμορφώστε τις ιδιότητές του:

```csharp
WmfRasterizationOptions wmfRasterization = new WmfRasterizationOptions
{
    BackgroundColor = Color.WhiteSmoke,
    PageWidth = 400,
    PageHeight = (int)Math.Round(400 / k),
    BorderX = 5,
    BorderY = 10
};

Console.WriteLine($"Rasterization Options: Width={wmfRasterization.PageWidth}, Height={wmfRasterization.PageHeight}");
```

### Λειτουργία 3: Ρύθμιση παραμέτρων επιλογών μετατροπής WebP και αποθήκευση εικόνας

**Επισκόπηση**: Ρύθμιση της μετατροπής μιας εικόνας σε μορφή WebP χρησιμοποιώντας συγκεκριμένες επιλογές ραστεροποίησης.

#### Οδηγίες βήμα προς βήμα

##### Ρύθμιση επιλογών WebP για μετατροπή

Αρχικά, καθορίστε τον κατάλογο εξόδου σας:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

Ρύθμιση παραμέτρων `WebPOptions` και φορτώστε ξανά την εικόνα WMF για μετατροπή:

```csharp
ImageOptionsBase webpOptions = new WebPOptions();
webpOptions.VectorRasterizationOptions = wmfRasterization;

using (Image imageToConvert = Image.Load(dataDir + "/input.wmf"))
{
    imageToConvert.Save(outputDir + "/ConvertedWebP_out.webp", webpOptions);
}

Console.WriteLine("WMF Image Converted to WebP format.");
```
