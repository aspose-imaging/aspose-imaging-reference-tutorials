## Οδηγός Εφαρμογής

### Ρύθμιση BmpOptions
Το πρώτο βήμα είναι η διαμόρφωση των επιλογών BMP για να καθορίσετε τον τρόπο αποθήκευσης της εικόνας σας και να ορίσετε τα χαρακτηριστικά της, όπως τα bit ανά pixel.

#### Βήμα 1: Ορισμός του καταλόγου εγγράφων
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή καταλόγου σας
```

#### Βήμα 2: Ρύθμιση παραμέτρων BmpOptions
```csharp
BmpOptions bmpOptions = new BmpOptions();
bmpOptions.BitsPerPixel = 24; // Ορίστε σε 24 bit ανά pixel για υψηλό βάθος χρώματος
bmpOptions.Source = new FileCreateSource(dataDir + "/CreatingAnImageBySettingPath_out.bmp", false);
```
**Εξήγηση:**
- **BitsPerPixel**Καθορίζει το βάθος χρώματος της εικόνας σας. Μια συνηθισμένη ρύθμιση είναι το 24, το οποίο υποστηρίζει εκατομμύρια χρώματα.
- **FileCreateSource**: Διαχειρίζεται τη δημιουργία αρχείων και καθορίζει εάν είναι προσωρινή.

### Δημιουργία και αποθήκευση εικόνας με το BmpOptions
Τώρα που έχετε ρυθμίσει `BmpOptions`, ας δημιουργήσουμε και ας αποθηκεύσουμε μια εικόνα BMP χρησιμοποιώντας αυτές τις διαμορφώσεις.

#### Βήμα 1: Ορισμός καταλόγου εξόδου
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή καταλόγου σας
```

#### Βήμα 2: Δημιουργήστε την εικόνα
```csharp
using (Image image = Image.Create(bmpOptions, 500, 500)) // Καθορίστε διαστάσεις (πλάτος x ύψος)
{
    image.Save(outputDir + "/CreatingAnImageBySettingPath1_out.bmp"); // Αποθήκευση του αρχείου BMP
}
```
**Εξήγηση:**
- **Δημιουργία μεθόδου**: Δημιουργεί μια νέα εικόνα με καθορισμένες διαστάσεις και επιλογές.
- **Μέθοδος Αποθήκευσης**: Εγγράφει την εικόνα στον δίσκο, χρησιμοποιώντας τις διαμορφωμένες `BmpOptions`.
