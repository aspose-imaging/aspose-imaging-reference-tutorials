### Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει τη φόρτωση ενός εγγράφου DjVu και την εξαγωγή συγκεκριμένων σελίδων ως αρχεία BMP.

#### Φόρτωση και εξαγωγή συγκεκριμένων σελίδων

Εξαγάγετε συγκεκριμένες σελίδες από ένα αρχείο DjVu και αποθηκεύστε τις ως εικόνες BMP.

##### Βήμα 1: Φόρτωση του εγγράφου DjVu
```csharp
using Aspose.Imaging.FileFormats.Djvu;

// Ορίστε τη διαδρομή του εγγράφου σας
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Άνοιγμα της εικόνας DjVu
using (DjvuImage djvuImage = (DjvuImage)Image.Load(Path.Combine(dataDir, "your_document.djvu")))
{
    // Συνέχεια με την εξαγωγή σελίδων...
}
```
**Εξήγηση:** 
- `DjvuImage` Χρησιμοποιείται για τη φόρτωση και τη διαχείριση αρχείων DjVu. 
- Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` και `"your_document.djvu"` με την πραγματική διαδρομή του αρχείου σας.

##### Βήμα 2: Εξαγωγή συγκεκριμένων σελίδων ως BMP
```csharp
using Aspose.Imaging.ImageOptions;

// Καθορίστε τις σελίδες που θέλετε να εξαγάγετε (π.χ., πρώτη και τρίτη σελίδα)
int[] pagesToExport = { 0, 2 };

foreach (var pageIndex in pagesToExport)
{
    // Δημιουργήστε μια νέα εικόνα για κάθε καθορισμένη σελίδα
    using (Image pageImage = djvuImage.Pages[pageIndex])
    {
        // Ορισμός επιλογών BMP
        BmpOptions bmpOptions = new BmpOptions();
        
        // Ορίστε τις επιθυμητές διαμορφώσεις για την εξαγωγή BMP
        bmpOptions.BitsPerPixel = 24; // Παράδειγμα διαμόρφωσης

        // Αποθήκευση της σελίδας ως αρχείο BMP
        string outputFileName = $"output_page_{pageIndex + 1}.bmp";
        pageImage.Save(Path.Combine(dataDir, outputFileName), bmpOptions);
    }
}
```
**Εξήγηση:**
- `pagesToExport` είναι ένας πίνακας ευρετηρίων σελίδων για εξαγωγή.
- Ο βρόχος επαναλαμβάνεται σε κάθε καθορισμένη σελίδα και την αποθηκεύει ως αρχείο BMP.
