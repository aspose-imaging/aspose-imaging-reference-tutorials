## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα εξερευνήσουμε τη διαδικασία δημιουργίας και επεξεργασίας εικόνων BMP χρησιμοποιώντας το Aspose.Imaging για .NET.

### Χαρακτηριστικό: Δημιουργία και Επεξεργασία Εικόνας

#### Επισκόπηση

Η δημιουργία μιας εικόνας BMP με συγκεκριμένες ρυθμίσεις σάς επιτρέπει να προσαρμόσετε τα γραφικά σύμφωνα με τις απαιτήσεις σας. Αυτό το σεμινάριο δείχνει πώς να χρησιμοποιήσετε το Aspose.Imaging για να δημιουργήσετε ένα αρχείο εικόνας, να ορίσετε τις τιμές pixel του και να το αποθηκεύσετε αποτελεσματικά.

#### Βήμα 1: Ρύθμιση του έργου σας και δημιουργία της εικόνας

Αρχικά, βεβαιωθείτε ότι έχετε καθορίσει διαδρομές για τον κατάλογο εγγράφων και τον κατάλογο εξόδου:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputPath = @"YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε μια διαδρομή για το αρχείο εικόνας σας.
string imageDataPath = System.IO.Path.Combine(documentDirectory, "sample.bmp");
```

#### Βήμα 2: Δημιουργία της εικόνας BMP με συγκεκριμένες επιλογές

Θα ξεκινήσουμε δημιουργώντας μια παρουσία του `BmpOptions` και ρυθμίζοντάς το να χρησιμοποιεί 32 bit ανά pixel:

```csharp
using (System.IO.FileStream fileStream = System.IO.File.Create(imageDataPath))
{
    using (BmpOptions bmpOptions = new BmpOptions())
    {
        bmpOptions.BitsPerPixel = 32;
        bmpOptions.Source = new StreamSource(fileStream);
        
        // Δημιουργήστε μια εικόνα με τις καθορισμένες διαστάσεις.
        using (RasterImage image = (RasterImage)Image.Create(bmpOptions, 10, 10))
        {
            Color[] pixels = new Color[4];
            for (int i = 0; i < 4; ++i)
            {
                // Ορίστε το χρώμα των pixel χρησιμοποιώντας τιμές ARGB.
                pixels[i] = Color.FromArgb(40, 30, 20, 10);
            }

            // Αποθηκεύστε τα καθορισμένα pixel σε μια ορθογώνια περιοχή στην εικόνα.
            image.SavePixels(new Rectangle(0, 0, 2, 2), pixels);

            // Αποθηκεύστε την επεξεργασμένη εικόνα στη διαδρομή εξόδου.
            string processedImagePath = System.IO.Path.Combine(outputPath, "processed_image.bmp");
            image.Save(processedImagePath);
        }
    }
}
```

#### Εξήγηση

- **Επιλογές Bmp**: Ρυθμίζει τις παραμέτρους του αρχείου BMP, όπως το βάθος bit. Εδώ, ορίζουμε `BitsPerPixel` έως 32 για πιο πλούσια απεικόνιση χρωμάτων.
  
- **Πηγή ροής**Χρησιμοποιείται ως πηγή δεδομένων pixel, επιτρέποντάς μας να εργαζόμαστε απευθείας με ροές.

- **Μέθοδος SavePixels**Αυτή η μέθοδος είναι κρίσιμη για τον ορισμό συγκεκριμένων pixel μέσα σε ένα καθορισμένο ορθογώνιο στην εικόνα σας.
