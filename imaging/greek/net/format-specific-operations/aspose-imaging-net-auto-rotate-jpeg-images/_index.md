## Οδηγός Εφαρμογής

### Αυτόματη διόρθωση προσανατολισμού εικόνων JPEG

#### Επισκόπηση

Αυτή η λειτουργία επιτρέπει στην εφαρμογή σας να περιστρέφει αυτόματα τις εικόνες με βάση τα δεδομένα προσανατολισμού EXIF. Αυτό σημαίνει ότι οι χρήστες δεν θα χρειάζεται να προσαρμόζουν χειροκίνητα τους προσανατολισμούς των εικόνων—ένα συνηθισμένο σημείο απογοήτευσης στις εργασίες επεξεργασίας εικόνας.

#### Βήμα προς βήμα εφαρμογή

**1. Φορτώστε την εικόνα**

Αρχικά, φορτώστε την εικόνα JPEG από ένα αρχείο ή ροή:

```csharp
using Aspose.Imaging.FileFormats.Jpeg;
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή καταλόγου εγγράφου σας

// Φόρτωση της εικόνας Jpeg
using (JpegImage image = (JpegImage)Image.Load(dataDir + "aspose-logo.jpg"))
{
    // Συνέχεια με αυτόματη περιστροφή
}
```

**2. Αυτόματη περιστροφή εικόνας**

Χρησιμοποιήστε το `AutoRotate` μέθοδος για την προσαρμογή του προσανατολισμού:

```csharp
// Εκτέλεση αυτόματης εναλλαγής με βάση δεδομένα EXIF
image.AutoRotate();
```
Αυτή η συνάρτηση διαβάζει αυτόματα την ετικέτα προσανατολισμού EXIF και εφαρμόζει τον απαραίτητο μετασχηματισμό.

**3. Αποθηκεύστε την περιστρεφόμενη εικόνα**

Τέλος, αποθηκεύστε την επεξεργασμένη εικόνα σας σε έναν καθορισμένο κατάλογο:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του καταλόγου εξόδου σας
// Αποθήκευση της περιστραμμένης εικόνας
image.Save(outputDir + "aspose-logo_out.jpg");
```

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Λείπουν μεταδεδομένα EXIF**Βεβαιωθείτε ότι οι εικόνες έχουν δεδομένα EXIF. Εάν όχι, σκεφτείτε να ορίσετε την προεπιλεγμένη λογική προσανατολισμού.
- **Προβλήματα διαδρομής αρχείου**: Ελέγξτε ξανά τις διαδρομές καταλόγου σας για να αποφύγετε `FileNotFoundException`.

### Φόρτωση και αποθήκευση εικόνας JPEG

#### Επισκόπηση

Αυτή η λειτουργία δείχνει πόσο εύκολα μπορείτε να φορτώσετε μια εικόνα JPEG από τον δίσκο και να την αποθηκεύσετε ξανά, καθιστώντας την ιδανική για βασικές λειτουργίες αρχείων.

#### Βήμα προς βήμα εφαρμογή

**1. Φόρτωση της εικόνας**

Ξεκινήστε φορτώνοντας μια υπάρχουσα εικόνα JPEG:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή καταλόγου εγγράφου σας
using (Image image = Image.Load(dataDir + "aspose-logo.jpg"))
{
    // Έτοιμο για αποθήκευση ή περαιτέρω επεξεργασία
}
```

**2. Αποθήκευση της εικόνας**

Μετά από οποιεσδήποτε τροποποιήσεις, αποθηκεύστε την εικόνα ξανά στον δίσκο:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του καταλόγου εξόδου σας
// Αποθήκευση της φορτωμένης εικόνας
image.Save(outputDir + "aspose-logo_copy.jpg");
```
