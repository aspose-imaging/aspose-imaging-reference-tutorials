## Οδηγός Εφαρμογής

### Φόρτωση εικόνας PNG με το Aspose.Imaging

**Επισκόπηση:**
Η φόρτωση μιας εικόνας είναι το πρώτο βήμα για οποιονδήποτε χειρισμό. Αυτή η ενότητα θα σας δείξει πώς να φορτώσετε εύκολα ένα αρχείο PNG χρησιμοποιώντας το Aspose.Imaging.

#### Βήμα 1: Φόρτωση εικόνας
```csharp
using System;
using Aspose.Imaging;

namespace CSharp.ModifyingAndConvertingImages.PNG
{
    public class LoadPngImage
    {
        private string dataDir = "YOUR_DOCUMENT_DIRECTORY/template.png";

        public void Run()
        {
            using (RasterImage image = (RasterImage)Image.Load(dataDir))
            {
                // Η εικόνα έχει πλέον φορτωθεί και είναι έτοιμη για χειρισμό.
            }
        }
    }
}
```
**Εξήγηση:**
- `Image.Load`: Ανοίγει το καθορισμένο αρχείο PNG, μετατρέποντάς το σε `RasterImage` για περαιτέρω χειρισμούς.

### Αποθήκευση εικόνας PNG με επιλογές συμπίεσης

**Επισκόπηση:**
Μόλις φορτωθεί η εικόνα σας, η αποθήκευσή της με συγκεκριμένες ρυθμίσεις, όπως το επίπεδο συμπίεσης και ο τύπος χρώματος, μπορεί να βελτιστοποιήσει την απόδοση και την ποιότητα.

#### Βήμα 2: Ρύθμιση παραμέτρων και αποθήκευση της εικόνας
```csharp
using System;
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Png;

namespace CSharp.ModifyingAndConvertingImages.PNG
{
    public class SavePngWithCompressionOptions
    {
        private string dataDir = "YOUR_DOCUMENT_DIRECTORY/template.png";
        private string outputDir = "YOUR_OUTPUT_DIRECTORY/result.png";

        public void Run()
        {
            using (RasterImage image = (RasterImage)Image.Load(dataDir))
            {
                PngOptions options = new PngOptions
                {
                    CompressionLevel = 8, // Μέτριο επίπεδο συμπίεσης.
                    ColorType = PngColorType.IndexedColor,
                    Palette = ColorPaletteHelper.GetCloseTransparentImagePalette(image, 256),
                    FilterType = PngFilterType.Avg,
                };

                image.Save(outputDir, options);
            }
        }
    }
}
```
**Εξήγηση:**
- **Επίπεδο συμπίεσης**: Ρύθμιση αυτού σε `8` προσφέρει ισορροπία μεταξύ μεγέθους αρχείου και ποιότητας.
- **Τύπος χρώματος και παλέτα**Η χρήση χρωμάτων από ευρετήριο με διαφάνεια βοηθά στη μείωση του μεγέθους του αρχείου, διατηρώντας παράλληλα την οπτική πιστότητα.
- **Τύπος φίλτρου**Το φίλτρο μέσης τιμής μπορεί να βοηθήσει στην ελαχιστοποίηση του μεγέθους του αρχείου χωρίς σημαντική απώλεια στην ποιότητα της εικόνας.

### Διαγραφή αρχείου

**Επισκόπηση:**
Μερικές φορές, χρειάζεται να καταργήσετε επεξεργασμένα αρχεία από το σύστημά σας. Αυτή η ενότητα δείχνει πώς να διαγράψετε ένα PNG εξόδου χρησιμοποιώντας .NET. `System.IO.File` μεθόδους.

#### Βήμα 3: Διαγραφή της εικόνας εξόδου
```csharp
using System;
using System.IO;

namespace CSharp.ModifyingAndConvertingImages.PNG
{
    public class DeleteFile
    {
        private string outputPath = "YOUR_OUTPUT_DIRECTORY/result.png";

        public void Run()
        {
            if (File.Exists(outputPath))
            {
                File.Delete(outputPath);
            }
        }
    }
}
```
**Εξήγηση:**
- **Αρχείο.Υπάρχει & Αρχείο.Διαγραφή**Αυτές οι μέθοδοι ελέγχουν την ύπαρξη του αρχείου και το διαγράφουν, διασφαλίζοντας ότι ο κατάλογός σας παραμένει καθαρός.
