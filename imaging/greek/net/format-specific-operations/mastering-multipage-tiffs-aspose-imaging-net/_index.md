### Φόρτωση πολλαπλών εικόνων (H2)
**Επισκόπηση:**
Η φόρτωση πολλαπλών εικόνων στη μνήμη είναι ένα βασικό βήμα σε πολλές εργασίες επεξεργασίας εικόνας. Αυτή η λειτουργία επιτρέπει την εκτέλεση μαζικών εργασιών, διευκολύνοντας τον ταυτόχρονο χειρισμό πολλαπλών αρχείων.

#### Βήμα 1: Ορισμός διαδρομών καταλόγου
```csharp
string baseFolder = "YOUR_DOCUMENT_DIRECTORY";
```
Αυτό ορίζει τη διαδρομή καταλόγου όπου βρίσκονται οι εικόνες εισόδου σας.

#### Βήμα 2: Λίστα αρχείων και φόρτωση εικόνων
```csharp
string[] files = new string[]
{
    "33266.tif", "Animation.gif", "elephant.png",
    "MultiPage.cdr"
};

List<Image> images = new List<Image>();
foreach (var file in files)
{
    string filePath = Path.Combine(baseFolder, file);
    images.Add(Image.Load(filePath));
}
```
Εδώ, δημιουργούμε μια λίστα με διαδρομές εικόνας και τις φορτώνουμε χρησιμοποιώντας `Image.Load()`Αυτή η μέθοδος διαβάζει αποτελεσματικά κάθε αρχείο στη μνήμη.

### Δημιουργία εικόνας TIFF πολλαπλών σελίδων (H2)
**Επισκόπηση:**
Η δημιουργία ενός πολυσέλιδου TIFF περιλαμβάνει τη συγκέντρωση πολλαπλών εικόνων μίας σελίδας σε ένα αρχείο. Είναι ιδιαίτερα χρήσιμη για την αρχειοθέτηση εγγράφων ή τη μείωση του αριθμού των αρχείων προς διαχείριση.

#### Βήμα 3: Συνδυασμός εικόνων
```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "MultipageImageCreateTest.tif");

using (Image image = Image.Create(images.ToArray(), true))
{
    image.Save(outputFilePath, new TiffOptions(TiffExpectedFormat.TiffJpegRgb));
}
```
Αυτό το απόσπασμα κώδικα δημιουργεί ένα αρχείο TIFF πολλαπλών σελίδων χρησιμοποιώντας τις εικόνες που έχουν φορτωθεί και το αποθηκεύει σε μορφή JPEG RGB.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές αρχείων έχουν καθοριστεί σωστά για να αποφύγετε `FileNotFoundException`.
- Επαληθεύστε ότι έχετε δικαιώματα εγγραφής για τον κατάλογο εξόδου.
- Χειρισμός εξαιρέσεων κατά τη φόρτωση εικόνων με κατεστραμμένα αρχεία.
