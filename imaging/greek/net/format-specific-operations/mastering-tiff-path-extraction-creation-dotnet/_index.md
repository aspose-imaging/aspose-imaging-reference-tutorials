### Εξαγωγή διαδρομών από εικόνες TIFF

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στην εξαγωγή διαδρομών που είναι αποθηκευμένες ως πόροι μέσα σε μια εικόνα TIFF, κάτι που είναι χρήσιμο για διάφορες εργασίες επεξεργασίας και επεξεργασίας.

**Βήμα 1: Φόρτωση της εικόνας**
```csharp
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Tiff;

var documentDirectory = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Sample.tif");

// Φορτώστε την εικόνα TIFF από την καθορισμένη διαδρομή.
using (var image = (TiffImage)Image.Load(documentDirectory))
{
    // Συνεχίστε με την εξαγωγή διαδρομών...
}
```

**Βήμα 2: Εξαγωγή διαδρομών**
```csharp
foreach (var path in image.ActiveFrame.PathResources)
{
    Console.WriteLine(path.Name); // Εξάγετε το όνομα κάθε διαδρομής
}

// Αποθηκεύστε την έξοδο εάν είναι απαραίτητο
image.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "SampleWithPaths.psd"), new PsdOptions());
```

**Εξήγηση:**
- `ActiveFrame.PathResources`: Πρόσβαση σε διαδρομές εντός του ενεργού πλαισίου.
- `PsdOptions()`: Εξασφαλίζει συμβατότητα κατά την αποθήκευση σε μορφή PSD.

### Δημιουργία διαδρομής αποκοπής σε TIFF

#### Επισκόπηση
Σε αυτήν την ενότητα, θα δημιουργήσουμε και θα προσθέσουμε μια διαδρομή αποκοπής σε μια εικόνα TIFF. Αυτό είναι κρίσιμο για συγκεκριμένες ροές εργασίας σχεδίασης ή επεξεργασίας.

**Βήμα 1: Φόρτωση της εικόνας**
```csharp
using Aspose.Imaging.FileFormats.Tiff;
using Aspose.Imaging.FileFormats.Tiff.Enums;

var documentDirectory = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Sample.tif");

// Φορτώστε την εικόνα TIFF από την καθορισμένη διαδρομή.
using (var image = (TiffImage)Image.Load(documentDirectory))
{
    // Συνεχίστε να δημιουργείτε μια νέα διαδρομή...
}
```

**Βήμα 2: Δημιουργία και αντιστοίχιση διαδρομής αποκοπής**
```csharp
var newPathResource = new PathResource
{
    BlockId = 2000, // Σύμφωνα με τις προδιαγραφές του Photoshop
    Name = "My Clipping Path",
    Records = CreateRecords(
        0.2f, 0.2f, 0.8f, 0.2f, 
        0.8f, 0.8f, 0.2f, 0.8f)
};

// Αντιστοιχίστε τον νέο πόρο διαδρομής στο ενεργό πλαίσιο.
image.ActiveFrame.PathResources = new List<PathResource> { newPathResource };

// Αποθήκευση με την προσθήκη της διαδρομής αποκοπής
image.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "ImageWithPath.tif"));
```

**Βοηθητικές Μέθοδοι:**
```csharp
private static List<VectorPathRecord> CreateRecords(params float[] coordinates)
{
    var records = CreateBezierRecords(coordinates);
    records.Insert(0, new LengthRecord { IsOpen = false, RecordCount = (ushort)records.Count });
    return records;
}

private static List<VectorPathRecord> CreateBezierRecords(float[] coordinates)
{
    return CoordinatesToPoints(coordinates).Select(CreateBezierRecord).ToList();
}

private static IEnumerable<PointF> CoordinatesToPoints(float[] coordinates)
{
    for (var index = 0; index < coordinates.Length; index += 2)
        yield return new PointF(coordinates[index], coordinates[index + 1]);
}

private static VectorPathRecord CreateBezierRecord(PointF point)
{
    return new BezierKnotRecord { PathPoints = new[] { point, point, point } };
}
```

**Εξήγηση:**
- **Αναγνωριστικό μπλοκ**: Μοναδικό αναγνωριστικό σύμφωνα με τις προδιαγραφές του Photoshop.
- **Εγγραφή μήκους**: Υποδεικνύει το κλείσιμο της διαδρομής και τον αριθμό των εγγραφών.
