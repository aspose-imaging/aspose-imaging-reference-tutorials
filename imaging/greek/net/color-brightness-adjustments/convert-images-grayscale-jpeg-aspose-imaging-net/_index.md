## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στη μετατροπή εικόνων σε διαφορετικές λειτουργίες χρώματος JPEG, εστιάζοντας στη μετατροπή σε κλίμακα του γκρι.

### Μετατροπή σε κλίμακα του γκρι με επιλογές Jpeg

Μετατρέψτε την εικόνα σας σε διάφορες λειτουργίες χρώματος JPEG χρησιμοποιώντας συγκεκριμένες επιλογές JPEG. Θα επικεντρωθούμε στη μετατροπή σε κλίμακα του γκρι:

#### Βήμα 1: Ορισμός καταλόγων και λειτουργιών χρώματος

Ρυθμίστε καταλόγους και καθορίστε τις λειτουργίες χρώματος JPEG στις οποίες θέλετε να μετατρέψετε εικόνες:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

JpegCompressionColorMode[] colorTypes = new JpegCompressionColorMode[]
{
    JpegCompressionColorMode.Grayscale,
};
```
#### Βήμα 2: Αρχικοποίηση επιλογών JPEG

Ρυθμίστε τις παραμέτρους των επιλογών JPEG για να ελέγξετε την επεξεργασία εικόνας:
```csharp
JpegOptions options = new JpegOptions();
options.BitsPerChannel = 12; // Προσαρμόστε τα bit ανά κανάλι για ποιότητα εικόνας
```
Ο `BitsPerChannel` Η παράμετρος καθορίζει το βάθος χρώματος κάθε καναλιού, επηρεάζοντας τόσο την ποιότητα όσο και το μέγεθος του αρχείου.

#### Βήμα 3: Μετατροπή εικόνων

Επαναλάβετε τους τύπους χρωμάτων για να μετατρέψετε και να αποθηκεύσετε εικόνες:
```csharp
string[] sourceFileNames = { "Grayscale.jpg" };

for (int i = 0; i < colorTypes.Length; ++i)
{
    options.ColorType = colorTypes[i];
    string fileName = $"{outputDir}/{colorTypes[i].ToString()}_12-bit.jpg";

    using (Image image = Image.Load($"{dataDir}/{sourceFileNames[i]}"))
    {
        image.Save(fileName, options);
    }
}
```
Αυτός ο βρόχος επαναλαμβάνεται σε κάθε καθορισμένη λειτουργία χρώματος JPEG και αποθηκεύει τις εικόνες που έχουν μετατραπεί με μοναδικά ονόματα αρχείων.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι ο κατάλογος πηγής σας περιέχει τα αρχεία εικόνας που προορίζονται για μετατροπή.
- Επαληθεύστε ότι `outputDir` είναι εγγράψιμο ή χειρίζεται τα δικαιώματα ανάλογα.
- Εάν αντιμετωπίζετε προβλήματα μνήμης, εξετάστε το ενδεχόμενο επεξεργασίας εικόνων σε μικρότερες παρτίδες ή αύξησης των πόρων του συστήματος.
