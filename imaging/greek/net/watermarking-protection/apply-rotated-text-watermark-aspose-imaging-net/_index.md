Είτε κοινοποιείτε φωτογραφίες στα μέσα κοινωνικής δικτύωσης είτε τις διανέμετε σε επαγγελματικά χαρτοφυλάκια, η προσθήκη ενός περιστρεφόμενου υδατογραφήματος κειμένου μπορεί να κάνει τη διαφορά. **Aspose.Imaging για .NET**, αυτή η εργασία γίνεται απρόσκοπτη και αποτελεσματική. ## Οδηγός Εφαρμογής
Θα αναλύσουμε τη διαδικασία σε λογικά τμήματα με βάση τα βασικά χαρακτηριστικά.

### Φόρτωση εικόνας
#### Επισκόπηση
Η φόρτωση μιας εικόνας είναι το πρώτο βήμα σε οποιαδήποτε εργασία χειρισμού εικόνας. Δείτε πώς μπορείτε να το κάνετε χρησιμοποιώντας το Aspose.Imaging.

**Βήμα 1**: Φορτώστε το αρχείο εικόνας σας.
```csharp
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SampleTiff1.tiff");
if (!File.Exists(dataDir))
{
    throw new FileNotFoundException("Image file not found at the specified path.");
}

using (Image image = Image.Load(dataDir))
{
    // Η εικόνα έχει πλέον φορτωθεί και είναι έτοιμη για χειρισμό
}
```

### Δημιουργία αντικειμένου γραφικών για χειρισμό εικόνας
#### Επισκόπηση
ΕΝΑ `Graphics` Το αντικείμενο σάς επιτρέπει να εκτελείτε λειτουργίες σχεδίασης σε μια εικόνα.

**Βήμα 2**: Αρχικοποίηση ενός `Graphics` αντικείμενο.
```csharp
Image image = Image.Load(Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SampleTiff1.tiff"));
Graphics graphics = new Graphics(image);
SizeF imageSize = graphics.Image.Size;
// Έτοιμο για σχεδίαση
```

### Εφαρμογή κειμένου με μετασχηματισμό σε μια εικόνα
#### Επισκόπηση
Η προσθήκη ενός περιστρεφόμενου υδατογραφήματος κειμένου περιλαμβάνει τη ρύθμιση γραμματοσειρών, πινέλων και μετασχηματισμών.

**Βήμα 3**: Ρύθμιση της γραμματοσειράς, του πινέλου και του μετασχηματισμού.
```csharp
Font font = new Font("Times New Roman", 20, FontStyle.Bold);
SolidBrush brush = new SolidBrush { Color = Color.Red, Opacity = 0 };
StringFormat format = new StringFormat
{
    Alignment = StringAlignment.Center,
    FormatFlags = StringFormatFlags.MeasureTrailingSpaces
};
Matrix matrix = new Matrix();
matrix.Translate(imageSize.Width / 2, imageSize.Height / 2);
matrix.Rotate(-45.0f);
graphics.Transform = matrix;
```

**Βήμα 4**: Σχεδιάστε το περιστρεφόμενο κείμενο.
```csharp
string theString = "45 Degree Rotated Text";
graphics.DrawString(theString, font, brush, 0, 0, format);
```

#### Επισκόπηση
Τέλος, αποθηκεύστε την τροποποιημένη εικόνα σας στο δίσκο.

**Βήμα 5**: Αποθηκεύστε την εικόνα με τις αλλαγές που έχουν εφαρμοστεί.
```csharp
string outputDir = Path.Combine("YOUR_OUTPUT_DIRECTORY", "AddDiagonalWatermarkToImage_out.jpg");

```
