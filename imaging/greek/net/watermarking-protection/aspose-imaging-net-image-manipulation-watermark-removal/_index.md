## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή εικόνας
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε μια εικόνα PNG από έναν κατάλογο χρησιμοποιώντας το Aspose.Imaging και να την αποθηκεύσετε σε άλλη τοποθεσία.

#### Βήμα 1: Φόρτωση της εικόνας
Ξεκινήστε καθορίζοντας το έγγραφό σας και τους καταλόγους εξόδου. Στη συνέχεια, χρησιμοποιήστε `Image.Load()` για να φορτώσετε το αρχείο PNG σας.

```csharp
using Aspose.Imaging;
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
var imagePath = Path.Combine(dataDir, "ball.png");
Image image = Image.Load(imagePath);
var pngImage = (PngImage)image; // Πρωταγωνιστές για συγκεκριμένες λειτουργίες
```

#### Βήμα 2: Αποθήκευση της εικόνας
Μόλις φορτωθεί, μπορείτε να το αποθηκεύσετε σε έναν κατάλογο εξόδου.

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
pngImage.Save(Path.Combine(outputDir, "loaded_image.png"));
```

### Δημιουργία και χρήση μάσκας διαδρομής γραφικών
Η δημιουργία μασκών διαδρομής γραφικών επιτρέπει περίπλοκους χειρισμούς εικόνας, όπως η προσθήκη σχημάτων ή εφέ.

#### Βήμα 1: Αρχικοποίηση του GraphicsPath
Δημιουργήστε ένα νέο `GraphicsPath` αντικείμενο για να ορίσετε τη μάσκα σας.

```csharp
using Aspose.Imaging.MagicWand.ImageMasks;

GraphicsPath mask = new GraphicsPath();
var firstFigure = new Figure();
```

#### Βήμα 2: Προσθήκη σχημάτων
Προσθέστε σχήματα όπως ελλείψεις στο σχήμα σας, τα οποία θα αποτελούν μέρος της μάσκας.

```csharp
firstFigure.AddShape(new EllipseShape(new RectangleF(350, 170, 570 - 350, 400 - 170)));
masks.AddFigure(firstFigure);
```

### Αφαίρεση υδατογραφήματος από μια εικόνα
Αυτή η λειτουργία δείχνει πώς να αφαιρέσετε ανεπιθύμητα υδατογραφήματα χρησιμοποιώντας τα εργαλεία αφαίρεσης υδατογραφημάτων του Aspose.Imaging.

#### Βήμα 1: Διαμόρφωση επιλογών
Στήνω `ContentAwareFillWatermarkOptions` με τη μάσκα γραφικών σας και ορίστε τις προσπάθειες ζωγραφικής.

```csharp
using Aspose.Imaging.Watermark;

var options = new ContentAwareFillWatermarkOptions(mask)
{
    MaxPaintingAttempts = 1 // Μέγιστος αριθμός προσπαθειών για την αφαίρεση υδατογραφήματος
};
```

#### Βήμα 2: Αφαίρεση του υδατογραφήματος
Χρήση `WatermarkRemover` για να εφαρμόσετε τη διαμόρφωσή σας και να αφαιρέσετε το υδατογράφημα.

```csharp
var result = WatermarkRemover.PaintOver(pngImage, options);
result.Save(Path.Combine(outputDir, "watermark_removed.png"));
File.Delete(Path.Combine(dataDir, "ball.png")); // Καθαρίστε το αρχικό αρχείο, εάν είναι απαραίτητο
```
