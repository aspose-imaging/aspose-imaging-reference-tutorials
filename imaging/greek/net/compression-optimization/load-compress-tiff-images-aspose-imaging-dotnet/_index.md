## Οδηγός Εφαρμογής

### Φόρτωση και εμφάνιση εικόνας TIFF

Η φόρτωση μιας εικόνας TIFF είναι απλή με το Aspose.Imaging. Δείτε πώς μπορείτε να το κάνετε:

#### Βήμα 1: Ορισμός διαδρομών καταλόγου

Ξεκινήστε ορίζοντας τις διαδρομές καταλόγου τόσο για τα αρχεία εισόδου όσο και για τα αρχεία εξόδου.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση της εικόνας

Χρησιμοποιήστε το `Image.Load` μέθοδος για να φορτώσετε ένα αρχείο TIFF από την καθορισμένη διαδρομή σας.

```csharp
using Aspose.Imaging;
using System;

// Φόρτωση της εικόνας
Image image = Image.Load(dataDir + "/SampleTiff1.tiff");
```

Με αυτά τα βήματα, έχετε φορτώσει με επιτυχία μια εικόνα TIFF για χειρισμό ή αποθήκευση.

### Συμπίεση εικόνων TIFF με συμπίεση Adobe Deflate

Τώρα, ας συμπιέσουμε μια εικόνα TIFF χρησιμοποιώντας τη συμπίεση Adobe Deflate για να μειώσουμε το μέγεθος του αρχείου διατηρώντας παράλληλα την ποιότητα.

#### Βήμα 3: Ρύθμιση παραμέτρων του TiffOptions

Δημιουργήστε μια παρουσία του `TiffOptions` και διαμορφώστε το ώστε να χρησιμοποιεί συμπίεση Adobe Deflate.

```csharp
using Aspose.Imaging.FileFormats.Tiff;
using Aspose.Imaging.ImageOptions;

// Ρύθμιση παραμέτρων εξόδου για τη συμπιεσμένη εικόνα
TiffOptions outputSettings = new TiffOptions(TiffExpectedFormat.Default);
outputSettings.BitsPerSample = new ushort[] { 4 };
outputSettings.Compression = TiffCompressions.AdobeDeflate;
outputSettings.Photometric = TiffPhotometrics.Palette;

// Ρύθμιση παλέτας κλίμακας του γκρι για την εικόνα
outputSettings.Palette = ColorPaletteHelper.Create4BitGrayscale(false);

// Αποθηκεύστε το συμπιεσμένο TIFF στον κατάλογο εξόδου
image.Save("YOUR_OUTPUT_DIRECTORY/CompressedSampleTiff.tiff", outputSettings);
```

Αυτό το απόσπασμα κώδικα δημιουργεί μια παλέτα κλίμακας του γκρι 4 bit και εφαρμόζει συμπίεση Adobe Deflate, μειώνοντας αποτελεσματικά το μέγεθος του αρχείου χωρίς σημαντική απώλεια στην ποιότητα της εικόνας.
