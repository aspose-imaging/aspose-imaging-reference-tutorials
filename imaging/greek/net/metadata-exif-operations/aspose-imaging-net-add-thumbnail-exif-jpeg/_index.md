## Οδηγός Εφαρμογής

### Προσθήκη μικρογραφίας σε τμήμα EXIF

Αυτή η λειτουργία σάς επιτρέπει να ενσωματώσετε μια μικρογραφία απευθείας στα δεδομένα EXIF των αρχείων JPEG.

#### Βήμα 1: Προετοιμασία καταλόγων

Ορίστε διαδρομές για τους καταλόγους εισόδου και εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```

#### Βήμα 2: Δημιουργία εικόνας μικρογραφίας

Δημιουργήστε μια νέα εικόνα JPEG που θα λειτουργεί ως μικρογραφία:
```csharp
JpegImage thumbnailImage = new JpegImage(100, 100);
```

#### Βήμα 3: Προετοιμασία κύριας εικόνας με δεδομένα EXIF

Δημιουργήστε μια άλλη εικόνα JPEG και αρχικοποιήστε τα δεδομένα EXIF της:
```csharp
JpegImage image = new JpegImage(1000, 1000);
image.ExifData = new JpegExifData();
image.ExifData.Thumbnail = thumbnailImage;
```

#### Βήμα 4: Αποθήκευση της εικόνας

Αποθηκεύστε την τροποποιημένη εικόνα με μια ενσωματωμένη μικρογραφία στον κατάλογο εξόδου σας:
```csharp
string outputPath = Path.Combine(outputDirectory, "thumbnail_out.jpg");
image.Save(outputPath);
```

### Χειρισμός ροής αρχείων χρησιμοποιώντας το MemoryStream

Χειρισμός εικόνων χρησιμοποιώντας ένα `MemoryStream` μπορεί να είναι χρήσιμο για επεξεργασία στη μνήμη χωρίς εγγραφή σε δίσκο.

#### Βήμα 1: Αρχικοποίηση MemoryStream

Δημιουργήστε μια παρουσία του `MemoryStream`:
```csharp
using (MemoryStream stream = new MemoryStream())
{
    JpegImage jpegImage = new JpegImage(500, 500);
    
    // Οι λειτουργίες στο jpegImage μπορούν να εκτελεστούν εδώ
    
    jpegImage.Save(stream);
}
```

Αυτό το παράδειγμα δείχνει την αποθήκευση μιας εικόνας JPEG σε μια ροή μνήμης.
