## Οδηγός Εφαρμογής

### Αυτόματη κάλυψη κοπής γραφήματος με υπολογισμό περιγράμματος

#### Επισκόπηση

Αυτή η λειτουργία χρησιμοποιεί τη μέθοδο Graph Cut για να υπολογίζει αυτόματα τις γραμμές για την τμηματοποίηση της εικόνας, παρέχοντας έναν απρόσκοπτο τρόπο απομόνωσης και χειρισμού συγκεκριμένων τμημάτων μιας εικόνας.

#### Βήμα προς βήμα εφαρμογή

**1. Φορτώστε την εικόνα εισόδου σας**

Ξεκινήστε φορτώνοντας την εικόνα σας από έναν κατάλογο. Αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική σας διαδρομή:

```csharp
string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "input.jpg");
using (RasterImage image = (RasterImage)Image.Load(inputFile))
```

**2. Διαμόρφωση επιλογών αποκοπής γραφήματος**

Ρυθμίστε το `AutoMaskingGraphCutOptions` για να καθορίσετε πώς θα πρέπει να γίνεται η τμηματοποίηση:

```csharp
AutoMaskingGraphCutOptions options = new AutoMaskingGraphCutOptions
{
    CalculateDefaultStrokes = true,
    FeatheringRadius = (Math.Max(image.Width, image.Height) / 500) + 1,
    Method = SegmentationMethod.GraphCut,
    Decompose = false,
    ExportOptions = new PngOptions()
    {
        ColorType = PngColorType.TruecolorWithAlpha,
        Source = new FileCreateSource("tempFile")
    },
    BackgroundReplacementColor = System.Drawing.Color.Transparent
};
```

**3. Εκτελέστε τμηματοποίηση εικόνας**

Εκτελέστε τη διαδικασία τμηματοποίησης και αποθηκεύστε την έξοδο:

```csharp
MaskingResult results = new ImageMasking(image).Decompose(options);

using (RasterImage resultImage = (RasterImage)results[1].GetImage())
{
    string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.png");
    resultImage.Save(outputPath, new PngOptions() { ColorType = PngColorType.TruecolorWithAlpha });
}
```

**4. Καθαρισμός**

Μετά την επεξεργασία, διαγράψτε τυχόν προσωρινά αρχεία:

```csharp
File.Delete(Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.png"));
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Συνηθισμένο πρόβλημα**Βεβαιωθείτε ότι η διαδρομή της εικόνας εισόδου είναι σωστή για να αποφύγετε `FileNotFoundException`.
- **Καθυστέρηση απόδοσης**Βελτιστοποιήστε προσαρμόζοντας το `FeatheringRadius` με βάση τις συγκεκριμένες διαστάσεις της εικόνας σας.
