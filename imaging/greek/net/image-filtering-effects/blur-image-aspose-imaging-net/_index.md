## Οδηγός Υλοποίησης: Θάμπωμα Εικόνας με Φίλτρο Γκαουσιανού

Ας αναλύσουμε βήμα προς βήμα τον τρόπο εφαρμογής αυτής της λειτουργικότητας:

### Φόρτωση της εικόνας

Ξεκινήστε καθορίζοντας τους καταλόγους για τις εικόνες προέλευσης και εξόδου. Βεβαιωθείτε ότι έχετε ένα αρχείο με το όνομα `asposelogo.gif` στον κατάλογο εγγράφων σας.

```csharp
using Aspose.Imaging;
using Aspose.Imaging.ImageFilters.FilterOptions;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

using (Image image = Image.Load(dataDir + "/asposelogo.gif"))
{
    // Τα βήματα μετατροπής και επεξεργασίας ακολουθούν εδώ
}
```

### Μετατροπή σε RasterImage

Για να εφαρμόσετε φίλτρα, μετατρέψτε την εικόνα που φορτώθηκε σε `RasterImage`.

```csharp
RasterImage rasterImage = (RasterImage)image;
// Συνέχεια με τις λειτουργίες φιλτραρίσματος
```

### Εφαρμογή θολώματος Gauss

Χρησιμοποιήστε το `GaussianBlurFilterOptions` για να θολώσετε την εικόνα σας. Εδώ, εφαρμόζεται ακτίνα 5x5 σε ολόκληρα τα όρια της εικόνας.

```csharp
rasterImage.Filter(rasterImage.Bounds, new GaussianBlurFilterOptions(5, 5));
```

**Εξήγηση**: 
- Ο **ακτίνα** (5, 5) καθορίζει την ισχύ του εφέ θολώματος. Μεγαλύτερες τιμές δημιουργούν πιο έντονα θολώματα.
- **Όρια**: Εξασφαλίζει ότι το φίλτρο εφαρμόζεται σε ολόκληρη την εικόνα.

### Αποθήκευση της θολής εικόνας

Μετά την επεξεργασία, αποθηκεύστε την θολή εικόνα στον καθορισμένο κατάλογο εξόδου.

```csharp
rasterImage.Save(outputDir + "/BlurAnImage_out.gif");
```
