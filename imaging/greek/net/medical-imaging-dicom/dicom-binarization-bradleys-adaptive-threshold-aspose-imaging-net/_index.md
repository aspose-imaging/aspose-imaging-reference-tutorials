## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Δυαδοποίηση με το Προσαρμοστικό Κατώφλι του Bradley
**Επισκόπηση:**
Αυτή η λειτουργία μετατρέπει μια εικόνα DICOM σε δυαδική μορφή χρησιμοποιώντας το προσαρμοστικό όριο του Bradley, ενισχύοντας την τοπική αντίθεση και βελτιώνοντας τα αποτελέσματα της ανάλυσης.

#### Βήμα 1: Ανοίξτε το αρχείο DICOM
Αρχικά, ανοίξτε το αρχείο DICOM καθορίζοντας τη διαδρομή του. Αντικαταστήστε `'YOUR_DOCUMENT_DIRECTORY'` με τον πραγματικό κατάλογο του εγγράφου σας.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string inputFile = Path.Combine(dataDir, "image.dcm");

using (var fileStream = new FileStream(inputFile, FileMode.Open, FileAccess.Read))
{
    // Συνέχεια στο Βήμα 2
}
```
*Γιατί;*Το άνοιγμα του αρχείου σε ροή επιτρέπει την αποτελεσματική ανάγνωση και επεξεργασία χωρίς να φορτωθεί ολόκληρο το περιεχόμενο στη μνήμη.

#### Βήμα 2: Φόρτωση της εικόνας από τη ροή χρησιμοποιώντας την κλάση DicomImage
Φορτώστε την εικόνα χρησιμοποιώντας `DicomImage`, το οποίο παρέχει μεθόδους ειδικές για αρχεία DICOM.

```csharp
using (DicomImage image = new DicomImage(fileStream))
{
    // Προχωρήστε στο Βήμα 3
}
```
*Γιατί;*Αυτό το βήμα προετοιμάζει τα δεδομένα DICOM για επεξεργασία, επιτρέποντας διάφορους μετασχηματισμούς και αναλύσεις.

#### Βήμα 3: Εφαρμογή του Προσαρμοστικού Κατωφλίου του Bradley για δυαδοποίηση της εικόνας
Η δυαδοποίηση βελτιώνει την αντίθεση της εικόνας ορίζοντας τα pixel πάνω από ένα όριο σε λευκό και εκείνα κάτω από ένα όριο σε μαύρο. Η παράμετρος `10` τελειοποιεί αυτή τη διαδικασία.

```csharp
image.BinarizeBradley(10);
```
*Γιατί;*Η μέθοδος του Bradley προσαρμόζεται στις τοπικές διακυμάνσεις της φωτεινότητας, καθιστώντας την ιδανική για ιατρικές εικόνες με ανομοιόμορφο φωτισμό.

#### Βήμα 4: Αποθηκεύστε την προκύπτουσα δυαδική εικόνα σε μορφή BMP
Τέλος, αποθηκεύστε την επεξεργασμένη εικόνα σας. Αντικαταστήστε την `'YOUR_OUTPUT_DIRECTORY'` με το πού θέλετε να αποθηκευτεί η έξοδος.

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDir, "BinarizationWithBradleysAdaptiveThreshold_out.bmp");
image.Save(outputFile, new BmpOptions());
```
*Γιατί;*Η μορφή BMP υποστηρίζεται ευρέως και παρέχει έναν απλό τρόπο απεικόνισης των δυαδικών αποτελεσμάτων.
