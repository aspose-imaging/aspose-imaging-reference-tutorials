## Οδηγός υλοποίησης: Περιστροφή εικόνας DICOM
Ας εμβαθύνουμε στην περιστροφή μιας εικόνας DICOM χρησιμοποιώντας το Aspose.Imaging .NET. Αυτή η ενότητα έχει δομηθεί για να σας καθοδηγήσει σε κάθε βήμα μεθοδικά.

### Φόρτωση και προετοιμασία του αρχείου DICOM
**Επισκόπηση:**
Ξεκινήστε φορτώνοντας το αρχείο DICOM από τον κατάλογό του και, στη συνέχεια, δημιουργήστε μια παρουσία του `DicomImage` να χειραγωγήσουν την εικόνα.

#### Βήμα 1: Ρύθμιση καταλόγων και άνοιγμα ροής αρχείων
Αρχικά, ορίστε διαδρομές για τους καταλόγους προέλευσης και εξόδου:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```
Στη συνέχεια, ανοίξτε μια ροή αρχείων για να διαβάσετε το αρχείο DICOM:
```csharp
using (var fileStream = new FileStream(dataDir + "/file.dcm", FileMode.Open, FileAccess.Read))
{
    // Συνεχίστε με την επεξεργασία εικόνας εδώ.
}
```

#### Βήμα 2: Δημιουργία και περιστροφή του DicomImage
Δημιουργήστε μια παρουσία του `DicomImage` χρησιμοποιώντας τη ροή αρχείων που φορτώθηκε:
```csharp
using (DicomImage image = new DicomImage(fileStream))
{
    // Περιστρέψτε την εικόνα DICOM κατά οποιαδήποτε επιθυμητή γωνία.
    image.Rotate(10);
```
Ο `Rotate` Η μέθοδος λαμβάνει μια γωνία σε μοίρες, επιτρέποντάς σας να περιστρέψετε την εικόνα σας δεξιόστροφα. Προσαρμόστε αυτήν την τιμή όπως απαιτείται.

#### Βήμα 3: Αποθήκευση της περιστρεφόμενης εικόνας
Τέλος, αποθηκεύστε την περιστραμμένη εικόνα σε νέα μορφή αρχείου:
```csharp
    // Αποθηκεύστε την περιστραμμένη εικόνα ως αρχείο BMP.
    image.Save(outputDir + "/RotatingDICOMImage_out.bmp", new BmpOptions());
}
```
Εδώ, χρησιμοποιούμε `BmpOptions` για να καθορίσετε τη μορφή εξόδου. Μπορείτε να επιλέξετε άλλες μορφές που υποστηρίζονται από το Aspose.Imaging, εάν θέλετε.
