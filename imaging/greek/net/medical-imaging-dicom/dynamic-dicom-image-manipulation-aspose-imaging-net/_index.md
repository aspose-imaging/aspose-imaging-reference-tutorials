# Δυναμική Επεξεργασία Εικόνας DICOM με Aspose.Imaging .NET

Ωστόσο, η βελτίωση αυτών των εικόνων ή η προσθήκη σχολίων μπορεί να είναι δύσκολη με τη χρήση παραδοσιακών μεθόδων. - Τεχνικές για την αποθήκευση δεδομένων pixel σε πολλαπλές σελίδες DICOM.
- Βήματα για την αποθήκευση ενός αρχείου DICOM πολλαπλών σελίδων με βελτιωμένες λειτουργίες.

Ας εμβαθύνουμε στη διαδικασία και ας εξερευνήσουμε πώς αυτές οι λειτουργίες μπορούν να εφαρμοστούν απρόσκοπτα στα έργα .NET σας.

### Σχεδίαση σε μια εικόνα Dicom

Η σχεδίαση διανυσματικών γραφικών όπως ορθογώνια και ελλείψεις σε εικόνες DICOM μπορεί να είναι ζωτικής σημασίας για την επισήμανση συγκεκριμένων περιοχών στην ιατρική διαγνωστική. Δείτε πώς μπορείτε να το πετύχετε αυτό με το Aspose.Imaging:

#### Επισκόπηση
Θα δημιουργήσουμε μια παρουσία του `DicomImage`, αρχικοποιήστε το γραφικό αντικείμενο και σχεδιάστε σχήματα πάνω σε αυτό.

#### Βήματα:

##### 1. Δημιουργήστε μια νέα παρουσία DicomImage

Ξεκινήστε αρχικοποιώντας την εικόνα DICOM:
```csharp
using (DicomImage image = (DicomImage)Image.Create(
    new DicomOptions() { Source = new StreamSource(new MemoryStream()) },
    100, 100))
{
    // Ο κώδικας σχεδίασης θα τοποθετηθεί εδώ
}
```

##### 2. Αρχικοποίηση του αντικειμένου γραφικών

Ο `Graphics` Το αντικείμενο σάς επιτρέπει να σχεδιάζετε πάνω στην εικόνα:
```csharp
Graphics graphics = new Graphics(image);
```

##### 3. Σχεδιάστε σχήματα

Δείτε πώς μπορείτε να σχεδιάσετε ορθογώνια και ελλείψεις με διαφορετικά χρώματα:
- **Ορθογώνιο παραλληλόγραμμο** καλύπτοντας όλα τα όρια:
  
  ```csharp
graphics.FillRectangle(new SolidBrush(Color.BlueViolet), image.Bounds);
```

- **Aqua Rectangle** at a specific location:
  
  ```csharp
graphics.FillRectangle(new SolidBrush(Color.Aqua), 10, 20, 50, 20);
```

- **Πορτοκαλί Έλλειψη** κεντραρισμένο σε ένα σημείο:
  
  ```csharp
graphics.FillEllipse(new SolidBrush(Color.Orange), 30, 50, 70, 30);
```

### Saving Pixel Data to DicomPage Instances

To save drawn image pixels across multiple DICOM pages:

#### Overview
We'll load pixel data from the initial page and replicate it across new pages, adjusting brightness as needed.

#### Steps:

##### 1. Load ARGB32 Pixel Data

First, extract pixel data from the image:
```csharp
int[] pixels = image.LoadArgb32Pixels(image.Bounds);
```

##### 2. Προσθήκη νέων σελίδων και προσαρμογή φωτεινότητας

Κάντε επανάληψη για να προσθέσετε σελίδες και να τροποποιήσετε τη φωτεινότητά τους:
```csharp
for (int i = 1; i < 5; i++)
{
    DicomPage page = image.AddPage();
    page.SaveArgb32Pixels(page.Bounds, pixels);
    page.AdjustBrightness(i * 30);
}
```

Ομοίως, εισαγάγετε σελίδες στην αρχή και προσαρμόστε τη φωτεινότητά τους αντίστροφα:
```csharp
for (int i = 1; i < 5; i++)
{
    DicomPage page = image.InsertPage(0);
    page.SaveArgb32Pixels(page.Bounds, pixels);
    page.AdjustBrightness(-i * 30);
}
```
