### Φόρτωση και αποθήκευση εικόνας DICOM
#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει τη φόρτωση μιας εικόνας DICOM από ένα αρχείο και την αποθήκευσή της ως PNG. Απλοποιεί τον χειρισμό ιατρικών εικόνων για περαιτέρω επεξεργασία ή εμφάνιση σε εφαρμογές που δεν είναι DICOM.
#### Φόρτωση εικόνας DICOM
Αρχικά, πρέπει να φορτώσετε την εικόνα DICOM χρησιμοποιώντας το `Image.Load` μέθοδος:

```csharp
using Aspose.Imaging;
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
var inputPath = Path.Combine(dataDir, "input.dcm");

// Φορτώστε την εικόνα DICOM από την καθορισμένη διαδρομή.
using (var image = Image.Load(inputPath))
{
    // Προχωρήστε στην επεξεργασία ή την αποθήκευση, όπως απαιτείται.
}
```
**Εξήγηση:**  
- `Image.Load` χρησιμοποιείται για το άνοιγμα και τη φόρτωση του αρχείου DICOM. Το αντικείμενο εικόνας που φορτώθηκε μπορεί στη συνέχεια να χειριστεί ή να αποθηκευτεί.
#### Αποθήκευση ως PNG
Μετά τη φόρτωση, μπορείτε να αποθηκεύσετε την εικόνα σε διαφορετική μορφή, όπως PNG:

```csharp
string outputPath = Path.Combine(dataDir, "output.png");
image.Save(outputPath);
```
**Εξήγηση:**  
- `image.Save` Η μέθοδος γράφει την εικόνα που φορτώθηκε σε ένα αρχείο. Εδώ, αποθηκεύει την εικόνα DICOM ως PNG.
#### Καθαρισμός
Προαιρετικά, διαγράψτε το αποθηκευμένο αρχείο PNG εάν δεν το χρειάζεστε πλέον:

```csharp
File.Delete(Path.Combine(dataDir, "output.png"));
```
