## Οδηγός Εφαρμογής

Αυτός ο οδηγός χωρίζεται σε ενότητες ανά χαρακτηριστικό, διασφαλίζοντας ότι θα αποκτήσετε ολοκληρωμένη γνώση για κάθε πτυχή.

### Ρύθμιση διάρκειας καρέ GIF

#### Επισκόπηση
Η ρύθμιση της διάρκειας καρέ επιτρέπει τον έλεγχο της διάρκειας εμφάνισης κάθε καρέ στο κινούμενο GIF σας. Αυτό μπορεί να είναι κρίσιμο για τον συγχρονισμό των κινούμενων εικόνων με άλλα μέσα ή την επίτευξη των επιθυμητών οπτικών εφέ.

#### Βήματα Υλοποίησης

**1. Φορτώστε την εικόνα GIF**
Ξεκινήστε φορτώνοντας την εικόνα GIF από έναν καθορισμένο κατάλογο:
```csharp
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "gif_images");
string filepath = Path.Combine(dataDir, "example.gif");

using (GifImage image = (GifImage)Image.Load(filepath))
{
    // Περαιτέρω επεξεργασία...
}
```

**2. Ορισμός διάρκειας καρέ**
Ορίστε τη διάρκεια για όλα τα καρέ σε 2000 χιλιοστά του δευτερολέπτου και προσαρμόστε τους χρονισμούς μεμονωμένων καρέ:
```csharp
image.SetFrameTime(2000); // Ορίζει τον προεπιλεγμένο χρόνο καρέ

// Προσαρμόστε τη διάρκεια του πρώτου καρέ
((GifFrameBlock)image.Pages[0]).FrameTime = 200; // Ορίζει συγκεκριμένο χρόνο καρέ
```

**Εξήγηση:**
- `SetFrameTime(2000)`: Ρυθμίζει όλα τα καρέ ώστε να εμφανίζονται για 2000 χιλιοστά του δευτερολέπτου από προεπιλογή.
- `((GifFrameBlock)image.Pages[0]).FrameTime = 200;`: Προσαρμόζει τη διάρκεια του πρώτου καρέ, δείχνοντας πώς μπορείτε να στοχεύσετε συγκεκριμένα καρέ.

### Ρύθμιση αριθμού επανάληψης GIF

#### Επισκόπηση
Ο έλεγχος του αριθμού των επαναλήψεων καθορίζει πόσες φορές θα αναπαραχθεί το GIF σας. Αυτή η λειτουργία είναι χρήσιμη για τη δημιουργία κινούμενων εικόνων που πρέπει να επαναληφθούν για έναν καθορισμένο αριθμό φορών.

#### Βήματα Υλοποίησης

**1. Φορτώστε και αποθηκεύστε το GIF**
Φορτώστε την εικόνα και αποθηκεύστε την με έναν καθορισμένο αριθμό βρόχων:
```csharp
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.gif");

using (GifImage image = (GifImage)Image.Load(filepath))
{
    // Ορισμός αριθμού βρόχων σε 4 φορές
    image.Save(outputPath, new GifOptions() { LoopsCount = 4 });
}
```

**Εξήγηση:**
- `LoopsCount = 4`: Ρυθμίζει το GIF ώστε να αναπαράγεται τέσσερις φορές πριν σταματήσει.

### Διαγραφή αρχείου εξόδου

#### Επισκόπηση
Ο καθαρισμός μετά την επεξεργασία διασφαλίζει ότι ο κατάλογος εξόδου σας παραμένει οργανωμένος, αφαιρώντας τα περιττά αρχεία.

#### Βήματα Υλοποίησης

**1. Διαγραφή καθορισμένου αρχείου**
Ελέγξτε την ύπαρξη αρχείου και διαγράψτε το, εάν είναι απαραίτητο:
```csharp
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.gif");

if (File.Exists(outputPath))
{
    File.Delete(outputPath);
}
```
