## Τι είναι το “convert JPEG to TIFF”;
Η μετατροπή JPEG σε TIFF περιλαμβάνει την αποκωδικοποίηση ενός αρχείου JPEG, την προαιρετική του μετασχηματισμό (π.χ., αλλαγή μεγέθους ή αλλαγή βάθους χρώματος) και στη συνέχεια την κωδικοποίηση του αποτελέσματος ως αρχείο TIFF. Το TIFF υποστηρίζει πολλαπλά καρέ, ασυμπίεστη συμπίεση και μεταδεδομένα που δεν διαθέτει το JPEG, καθιστώντας το ιδανικό για αρχειοθέτηση και σενάρια πολλαπλών σελίδων.

## Πώς να Μετατρέψετε JPEG σε TIFF και να Προσθέσετε Πολλαπλά Καρέ

Ας χωρίσουμε την υλοποίηση σε διαχειρίσιμα τμήματα.

### Δημιουργία και Διαμόρφωση TiffOptions για Εικόνα TIFF

#### Επισκόπηση
Η δημιουργία ενός αντικειμένου `TiffOptions` σας επιτρέπει να ορίσετε ρυθμίσεις όπως η συμπίεση και η φωτομετρική ερμηνεία, που είναι ουσιώδεις για την προσαρμογή των αρχείων TIFF.

#### Υλοποίηση Βήμα‑Βήμα

**1. Εισαγωγή Απαραίτητων Namespaces**  
Βεβαιωθείτε ότι έχετε συμπεριλάβει αυτά τα namespaces στο αρχείο σας:

```csharp
using Aspose.Imaging.FileFormats.Tiff;
using Aspose.Imaging.FileFormats.Tiff.Enums;
using Aspose.Imaging.ImageOptions;
```

**2. Διαμόρφωση TiffOptions**  
Ρυθμίστε το αντικείμενο `TiffOptions` με συγκεκριμένες ρυθμίσεις για μια ασπρόμαυρη εικόνα χρησιμοποιώντας συμπίεση CCITT Fax Group 3.

```csharp
// Create TiffOptions with default settings
TiffOptions outputSettings = new TiffOptions(TiffExpectedFormat.Default);

// Set bits per sample to 1 (black and white)
outputSettings.BitsPerSample = new ushort[] { 1 };

// Use CCITT Fax Group 3 compression
outputSettings.Compression = TiffCompressions.CcittFax3;

// Define photometric interpretation as MinIsWhite
outputSettings.Photometric = TiffPhotometrics.MinIsWhite;

// Set source to an empty stream for creating new TIFF from scratch
outputSettings.Source = new Aspose.Imaging.Sources.StreamSource(new System.IO.MemoryStream());
```

### Προσθήκη Καρέ στο TiffImage και Αποθήκευση

#### Επισκόπηση
Η προσθήκη καρέ σε μια εικόνα TIFF περιλαμβάνει την αντιγραφή των pixel των αλλαγμένων JPEG σε κάθε καρέ και την αποθήκευση του πλήρους πολυ‑καρέ TIFF.

**1. Αρχικοποίηση του Παραδείγματος TiffImage**

```csharp
using (TiffImage tiffImage = (TiffImage)Aspose.Imaging.Image.Create(outputSettings, newWidth, newHeight))
{
    int index = 0; // Frame index tracker
    
    foreach (var file in Directory.GetFiles(dataDir, "*.jpg"))
    {
        using (Aspose.Imaging.RasterImage ri = (Aspose.Imaging.RasterImage)Aspose.Imaging.Image.Load(file))
        {
            ri.Resize(newWidth, newHeight, Aspose.Imaging.ResizeType.NearestNeighbourResample);
            
            TiffFrame frame = tiffImage.ActiveFrame;
            if (index > 0)
            {
                // Create a new frame for each subsequent image
                frame = new TiffFrame(new TiffOptions(outputSettings), newWidth, newHeight);
            }
            
            // Copy pixels from the resized JPEG into the TIFF frame
            frame.SavePixels(frame.Bounds, ri.LoadPixels(ri.Bounds));
            if (index > 0)
            {
                tiffImage.AddFrame(frame); // Add to TIFF image only after the first frame
            }
            index++;
        }
    }
    
    tiffImage.Save(path); // Save the final TIFF with all frames
}
```

**Ε: Μπορώ να μετατρέψω JPEG σε TIFF χωρίς απώλεια ποιότητας;**

**Ε: Πόσα καρέ μπορεί να περιέχει ένα αρχείο TIFF;**  
Α: Πρακτικά απεριόριστο· το όριο καθορίζεται από το μέγεθος του αρχείου και τη διαθέσιμη μνήμη.

**Ε: Είναι το παραγόμενο TIFF συμβατό με κοινούς προβολείς;**  
Α: Το παράδειγμα χρησιμοποιεί την τυπική συμπίεση CCITT Fax Group 3, η οποία υποστηρίζεται ευρέως από τους περισσότερους προβολείς και επεξεργαστές TIFF.

**Ε: Τι γίνεται αν θέλω να προσθέσω διαφορετική συμπίεση για έγχρωμες εικόνες;**  
Α: Αντικαταστήστε το `TiffCompressions.CcittFax3` με `TiffCompressions.Lzw` ή `TiffCompressions.Jpeg` και προσαρμόστε το `BitsPerSample` ανάλογα.
