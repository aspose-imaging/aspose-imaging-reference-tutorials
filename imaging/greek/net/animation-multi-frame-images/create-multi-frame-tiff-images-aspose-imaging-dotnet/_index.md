### Δημιουργία και ρύθμιση παραμέτρων TiffOptions για εικόνα TIFF

#### Επισκόπηση
Δημιουργώντας ένα `TiffOptions` Το αντικείμενο σάς επιτρέπει να ορίσετε ρυθμίσεις όπως η συμπίεση και η φωτομετρική ερμηνεία, απαραίτητες για την προσαρμογή των αρχείων TIFF σας.

#### Βήμα προς βήμα εφαρμογή

**1. Εισαγωγή απαραίτητων χώρων ονομάτων**
Βεβαιωθείτε ότι έχετε συμπεριλάβει αυτούς τους χώρους ονομάτων στο αρχείο σας:

```csharp
using Aspose.Imaging.FileFormats.Tiff;
using Aspose.Imaging.FileFormats.Tiff.Enums;
using Aspose.Imaging.ImageOptions;
```

**2. Ρύθμιση παραμέτρων TiffOptions**
Ρυθμίστε το `TiffOptions` αντικείμενο με συγκεκριμένες διαμορφώσεις για ασπρόμαυρη εικόνα χρησιμοποιώντας συμπίεση CCITT Fax Group 3.

```csharp
// Δημιουργήστε το TiffOptions με τις προεπιλεγμένες ρυθμίσεις
TiffOptions outputSettings = new TiffOptions(TiffExpectedFormat.Default);

// Ορισμός bit ανά δείγμα σε 1 (ασπρόμαυρο)
outputSettings.BitsPerSample = new ushort[] { 1 };

// Χρήση συμπίεσης CCITT Fax Group 3
outputSettings.Compression = TiffCompressions.CcittFax3;

// Ορίστε τη φωτομετρική ερμηνεία ως MinIsWhite
outputSettings.Photometric = TiffPhotometrics.MinIsWhite;

// Ορισμός πηγής σε μια κενή ροή για δημιουργία νέου TIFF από την αρχή
outputSettings.Source = new Aspose.Imaging.Sources.StreamSource(new System.IO.MemoryStream());
```

```csharp
string path = "@YOUR_OUTPUT_DIRECTORY\\AddFramesToTIFFImage_out.tif";
```

**2. Δημιουργήστε μια παρουσία TiffImage**
Αρχικοποίηση του `TiffImage` με καθορισμένες διαστάσεις και ρυθμίσεις εξόδου.

```csharp
using (TiffImage tiffImage = (TiffImage)Aspose.Imaging.Image.Create(outputSettings, newWidth, newHeight))

    // Η λογική για την προσθήκη πλαισίων θα προστεθεί εδώ.
```

### Φόρτωση αρχείων JPEG από τον κατάλογο και αλλαγή μεγέθους τους

#### Επισκόπηση
Η φόρτωση εικόνων JPEG, η αλλαγή μεγέθους τους και η προετοιμασία για συμπερίληψή τους σε ένα αρχείο TIFF βελτιστοποιείται με το Aspose.Imaging.

**1. Φόρτωση εικόνων JPEG**

```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY"; // Κατάλογος που περιέχει εικόνες εισόδου

foreach (var file in Directory.GetFiles(dataDir, "*.jpg"))
{
    using (Aspose.Imaging.RasterImage ri = (Aspose.Imaging.RasterImage)Aspose.Imaging.Image.Load(file))
    {
        // Αλλαγή μεγέθους εικόνας ώστε να ταιριάζει με τις διαστάσεις του πλαισίου TIFF
        ri.Resize(newWidth, newHeight, Aspose.Imaging.ResizeType.NearestNeighbourResample);
        
        // Εδώ θα προστεθεί επιπλέον λογική για τον χειρισμό πολλαπλών πλαισίων.
    }
}
```

### Προσθήκη πλαισίου στο TiffImage και αποθήκευσή του

#### Επισκόπηση
Η προσθήκη καρέ σε μια εικόνα TIFF περιλαμβάνει την αντιγραφή εικονοστοιχείων JPEG με αλλαγμένο μέγεθος σε κάθε καρέ και την αποθήκευση ολόκληρου του TIFF πολλαπλών καρέ.

**1. Αρχικοποίηση της παρουσίας TiffImage**

```csharp
using (TiffImage tiffImage = (TiffImage)Aspose.Imaging.Image.Create(outputSettings, newWidth, newHeight))
{
    int index = 0; // Παρακολούθηση ευρετηρίου καρέ
    
    foreach (var file in Directory.GetFiles(dataDir, "*.jpg"))
    {
        using (Aspose.Imaging.RasterImage ri = (Aspose.Imaging.RasterImage)Aspose.Imaging.Image.Load(file))
        {
            ri.Resize(newWidth, newHeight, Aspose.Imaging.ResizeType.NearestNeighbourResample);
            
            TiffFrame frame = tiffImage.ActiveFrame;
            if (index > 0)
            {
                // Δημιουργήστε ένα νέο πλαίσιο για κάθε επόμενη εικόνα
                frame = new TiffFrame(new TiffOptions(outputSettings), newWidth, newHeight);
            }
            
            // Αντιγραφή pixel από το JPEG με το νέο μέγεθος στο πλαίσιο TIFF
            frame.SavePixels(frame.Bounds, ri.LoadPixels(ri.Bounds));
            if (index > 0)
            {
                tiffImage.AddFrame(frame); // Προσθήκη σε εικόνα TIFF μόνο μετά το πρώτο καρέ
            }
            index++;
        }
    }
    
    tiffImage.Save(path); // Αποθηκεύστε το τελικό TIFF με όλα τα καρέ
}
```
