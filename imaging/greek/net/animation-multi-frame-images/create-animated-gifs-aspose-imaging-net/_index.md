### Οδηγός Εφαρμογής

#### Επισκόπηση της λειτουργίας
Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε GIF συνδυάζοντας πολλά καρέ, καθιστώντας την ιδανική για κινούμενα σχέδια ή ακολουθίες εικόνων.

##### Βήμα 1: Ορισμός καταλόγων
Ξεκινήστε καθορίζοντας τους καταλόγους εγγράφων και εξόδου:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

##### Βήμα 2: Φόρτωση πλαισίων
Φορτώστε όλα τα καρέ εικόνας που θέλετε να συμπεριλάβετε στο GIF σας:

```csharp
private static IEnumerable<RasterImage> LoadFrames(string directory)
{
    foreach (var filePath in Directory.GetFiles(directory))
    {
        yield return (RasterImage)Image.Load(filePath);
    }
}
```
Εδώ, επαναλαμβάνουμε αρχεία σε έναν καθορισμένο κατάλογο και τα φορτώνουμε ως `RasterImage` αντικείμενα.

##### Βήμα 3: Δημιουργήστε το GIF
Αρχικοποιήστε το GIF σας με το πρώτο καρέ και, στη συνέχεια, προσθέστε επιπλέον καρέ:

```csharp
var frames = LoadFrames(Path.Combine(dataDir, "Animation frames")).ToArray();

using (var image = new GifImage(new GifFrameBlock(frames[0])))
{
    for (var index = 1; index < frames.Length; index++)
    {
        image.AddPage(frames[index]);
    }
    
    image.Save(Path.Combine(outputDir, "Multipage.gif"));
}
```
Αυτό το απόσπασμα φορτώνει κάθε καρέ και το προσθέτει διαδοχικά στο GIF. `GifImage` Ο κατασκευαστής αρχικοποιεί το GIF χρησιμοποιώντας το πρώτο καρέ.
