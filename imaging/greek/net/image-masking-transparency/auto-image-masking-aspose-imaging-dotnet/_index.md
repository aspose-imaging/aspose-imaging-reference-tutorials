### Αυτόματη μάσκα εικόνας με μέθοδο αποκοπής γραφήματος
#### Επισκόπηση
Η αυτόματη απόκρυψη εικόνας σάς επιτρέπει να διαχωρίζετε αυτόματα το προσκήνιο από το φόντο χρησιμοποιώντας προηγμένους αλγόριθμους όπως η μέθοδος Graph Cut. Αυτή η λειτουργία απλοποιεί τις πολύπλοκες εργασίες που εμπλέκονται στη χειροκίνητη απόκρυψη.
#### Βήμα προς βήμα εφαρμογή
1. **Φόρτωση εικόνας**
   Ξεκινήστε φορτώνοντας την εικόνα που θέλετε να αποκρύψετε:
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   string sourceFileName = Path.Combine(dataDir, "Colored by Faith_small.png");

   using (RasterImage image = (RasterImage)Image.Load(sourceFileName))
   {
       // Περαιτέρω βήματα επεξεργασίας...
   }
   ```
2. **Ρύθμιση παραμέτρων επιλογών μάσκας**
   Ρυθμίστε τις απαραίτητες επιλογές κάλυψης:
   ```csharp
   AutoMaskingArgs maskingArgs = new AutoMaskingArgs();

   MaskingOptions maskingOptions = new MaskingOptions()
   {
       Method = SegmentationMethod.GraphCut,
       Args = maskingArgs,
       Decompose = false,
       ExportOptions = new PngOptions() 
       {
           ColorType = PngColorType.TruecolorWithAlpha,
           Source = new StreamSource(new MemoryStream())
       }
   };
   ```
3. **Εφαρμογή μάσκας**
   Εκτελέστε τη λειτουργία κάλυψης και αποθηκεύστε το αποτέλεσμα:
   ```csharp
   using (MaskingResult maskingResults = new ImageMasking(image).Decompose(maskingOptions))
   {
       string outputFileName = Path.Combine(dataDir, "Colored by Faith_small_auto.png");
       
       using (Image resultImage = maskingResults[1].GetImage())
       {
           resultImage.Save(outputFileName);
       }
   }
   ```
#### Βασικές επιλογές διαμόρφωσης
- **Μέθοδος αποκοπής γραφήματος:** Χρησιμοποιεί μια μέθοδο τμηματοποίησης κατάλληλη για τον ακριβή διαχωρισμό προσκηνίου-φόντου.
- **Επιλογές εξαγωγής:** Ρυθμίζει τον τρόπο αποθήκευσης της εικόνας εξόδου, καθορίζοντας τον τύπο χρώματος και την πηγή.
### Συμπλήρωση σημείων εισόδου για ορίσματα κάλυψης
#### Επισκόπηση
Τα σημεία εισόδου βοηθούν στη βελτίωση της κάλυψης, παρέχοντας πρόσθετα δεδομένα σχετικά με τις περιοχές ενδιαφέροντος σε μια εικόνα. Αυτή η λειτουργία επιτρέπει την προσαρμογή της διαδικασίας δημιουργίας μάσκας με προκαθορισμένα ορθογώνια ή σημεία αντικειμένων.
#### Βήμα προς βήμα εφαρμογή
1. **Αποσειριοποίηση δεδομένων**
   Φόρτωση δεδομένων σημείου εισόδου από ένα σειριοποιημένο αρχείο:
   ```csharp
   private static void FillInputPoints(string filePath, AutoMaskingArgs autoMaskingArgs)
   {
       using (Stream stream = File.OpenRead(filePath))
       {
           BinaryFormatter serializer = new BinaryFormatter();
           
           bool hasObjectRectangles = (bool)serializer.Deserialize(stream);
           bool hasObjectPoints = (bool)serializer.Deserialize(stream);

           if (hasObjectRectangles)
           {
               autoMaskingArgs.ObjectsRectangles = ((System.Drawing.Rectangle[])serializer.Deserialize(stream))
                   .Select(rect => new Aspose.Imaging.Rectangle(rect.X, rect.Y, rect.Width, rect.Height))
                   .ToArray();
           }

           if (hasObjectPoints)
           {
               autoMaskingArgs.ObjectsPoints = ((System.Drawing.Point[][])serializer.Deserialize(stream))
                   .Select(pointArray => pointArray.Select(point => new Aspose.Imaging.Point(point.X, point.Y)).ToArray())
                   .ToArray();
           }
       }
   }
   ```
#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η μορφή του αρχείου δεδομένων ταιριάζει με αυτό που αναμένει η αποσειριοποίηση.
- Επαληθεύστε ότι οι διαδρομές προς τα σειριοποιημένα αρχεία είναι σωστές και προσβάσιμες.
