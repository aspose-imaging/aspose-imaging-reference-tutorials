## Οδηγός Εφαρμογής
### Φόρτωση εικόνων με προσαρμοσμένες πηγές γραμματοσειράς
Αυτή η λειτουργία σάς επιτρέπει να φορτώνετε εικόνες χρησιμοποιώντας προσαρμοσμένες γραμματοσειρές που έχετε ορίσει εσείς. Δείτε πώς μπορείτε να την εφαρμόσετε:

#### Βήμα 1: Ορίστε τους καταλόγους εισόδου και εξόδου
```csharp
string inputPath = "YOUR_DOCUMENT_DIRECTORY";
string outputPath = "YOUR_OUTPUT_DIRECTORY";
string fileName = "missing-font.emf"; // Παράδειγμα ονόματος αρχείου
string fontPath = Path.Combine(inputPath, "Fonts");
```

#### Βήμα 2: Ρύθμιση παραμέτρων προσαρμοσμένης πηγής γραμματοσειράς
Δημιουργήστε μια μέθοδο για τη φόρτωση προσαρμοσμένων γραμματοσειρών και την ενσωμάτωσή της με το Aspose.Imaging:
```csharp
var loadOptions = new Aspose.Imaging.LoadOptions();
loadOptions.AddCustomFontSource(GetFontSource, fontPath);
```

#### Βήμα 3: Φόρτωση της εικόνας χρησιμοποιώντας προσαρμοσμένες γραμματοσειρές
Χρησιμοποιήστε τις διαμορφωμένες επιλογές για να φορτώσετε την εικόνα σας:
```csharp
using (var img = Image.Load(Path.Combine(inputPath, fileName), loadOptions))
{
    // Λήψη προεπιλεγμένων επιλογών διανυσματικής ραστεροποίησης με συγκεκριμένο χρώμα φόντου και διαστάσεις.
    var vectorRasterizationOptions = img.GetDefaultOptions(new object[] { Color.White, img.Width, img.Height }).VectorRasterizationOptions;

    // Ορίστε υποδείξεις απόδοσης για κείμενο και λειτουργία εξομάλυνσης για την έξοδο εικόνας.
    vectorRasterizationOptions.TextRenderingHint = TextRenderingHint.SingleBitPerPixel;
    vectorRasterizationOptions.SmoothingMode = SmoothingMode.None;

    // Αποθηκεύστε την επεξεργασμένη εικόνα στην καθορισμένη διαδρομή εξόδου με προσαρμοσμένες επιλογές ραστεροποίησης.
    img.Save(Path.Combine(outputPath, fileName + ".png"), new PngOptions { VectorRasterizationOptions = vectorRasterizationOptions });
}
```

#### Βήμα 4: Ορισμός παρόχου προσαρμοσμένης πηγής γραμματοσειράς
Δημιουργήστε μια συνάρτηση που καθορίζει την πηγή της γραμματοσειράς σας:
```csharp
public static CustomFontData[] GetFontSource(params object[] args)
{
    string fontsPath = "";
    if (args.Length > 0) { fontsPath = args[0].ToString(); }

    var customFontData = new List<CustomFontData>();

    // Επαναλάβετε κάθε αρχείο γραμματοσειράς στον καθορισμένο κατάλογο.
    foreach (var font in Directory.GetFiles(fontsPath))
    {
        string fontName = Path.GetFileNameWithoutExtension(font);
        byte[] fontData = File.ReadAllBytes(font);
        customFontData.Add(new CustomFontData(fontName, fontData));
    }

    return customFontData.ToArray();
}
```
