## Οδηγός Εφαρμογής
Θα εξερευνήσουμε δύο κύριες λειτουργίες: τη μετατροπή πόρων διαδρομής σε `GraphicsPath` και δημιουργώντας νέες διαδρομές για να οριστούν ως πόροι σε εικόνες TIFF.

### Μετατροπή πόρων διαδρομής από μια εικόνα TIFF σε ένα αντικείμενο GraphicsPath
Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε δεδομένα διανυσματικών γραφικών που είναι ενσωματωμένα σε ένα αρχείο TIFF για περαιτέρω επεξεργασία ή απόδοση.

#### Βήμα 1: Φόρτωση της εικόνας TIFF
Φορτώστε την εικόνα-στόχο σας χρησιμοποιώντας `Image.Load()`, καθορίζοντας τον κατάλογο όπου βρίσκεται το TIFF σας.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (var image = (TiffImage)Image.Load(Path.Combine(dataDir, "Bottle.tif")))
{
    // Συνεχίστε στη μετατροπή διαδρομών
}
```

#### Βήμα 2: Μετατροπή PathResources σε GraphicsPath
Χρήση `PathResourceConverter.ToGraphicsPath()` για να μετατρέψετε τους πόρους διαδρομής σε ένα σχεδιασμένο γραφικό αντικείμενο.
```csharp
var graphicsPath = PathResourceConverter.ToGraphicsPath(image.ActiveFrame.PathResources.ToArray(), image.ActiveFrame.Size);
```
Αυτή η μέθοδος μετατρέπει τις ενσωματωμένες διαδρομές διανυσμάτων σε μορφή που μπορεί εύκολα να χειριστεί ή να αποδοθεί χρησιμοποιώντας τυπικά εργαλεία σχεδίασης .NET.

#### Βήμα 3: Σχεδίαση χρησιμοποιώντας το GraphicsPath
Δημιουργήστε ένα `Graphics` αντικείμενο από την εικόνα TIFF σας και χρησιμοποιήστε το για να σχεδιάσετε με τη διαδρομή που έχει μετατραπεί.
```csharp
var graphics = new Graphics(image);
graphics.DrawPath(new Pen(Color.Red, 10), graphicsPath);
```
Εδώ, χρησιμοποιούμε ένα κόκκινο στυλό για εικονογράφηση.

#### Βήμα 4: Αποθήκευση της τροποποιημένης εικόνας
Αποθηκεύστε τις αλλαγές σας σε έναν κατάλογο εξόδου.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
image.Save(Path.Combine(outputDir, "BottleWithRedBorder.tif"));
```

### Δημιουργήστε GraphicsPath και ορίστε ως πόρους διαδρομής σε μια εικόνα TIFF
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε νέα διανυσματικά γραφικά και να τα ενσωματώσετε σε ένα αρχείο TIFF.

#### Βήμα 1: Φόρτωση της εικόνας TIFF
Φορτώστε την εικόνα-στόχο σας όπως πριν.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (var image = (TiffImage)Image.Load(Path.Combine(dataDir, "Bottle.tif")))
{
    // Προετοιμασία για δημιουργία και προσθήκη διαδρομών
}
```

#### Βήμα 2: Δημιουργήστε ένα σχήμα Bezier
Χρησιμοποιήστε βοηθητικές μεθόδους για να δημιουργήσετε σύνθετα σχήματα όπως καμπύλες Bezier.
```csharp
var figure = new Figure();
figure.AddShape(CreateBezierShape(100f, 100f, 500f, 100f, 500f, 1000f, 100f, 1000f));
```

#### Βήμα 3: Μετατροπή GraphicsPath σε PathResources
Μετατρέψτε την προσαρμοσμένη διαδρομή γραφικών σας και ορίστε την ως τους πόρους διαδρομής της εικόνας.
```csharp
var graphicsPath = new GraphicsPath();
graphicsPath.AddFigure(figure);
var pathResource = PathResourceConverter.FromGraphicsPath(graphicsPath, image.Size);
image.ActiveFrame.PathResources = new List<PathResource>(pathResource);
```

#### Βήμα 4: Αποθήκευση της τροποποιημένης εικόνας
Αποθηκεύστε το ενημερωμένο αρχείο TIFF με τις νέες προστιθέμενες διαδρομές διανυσμάτων.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
image.Save(Path.Combine(outputDir, "BottleWithRectanglePath.tif"));
```
