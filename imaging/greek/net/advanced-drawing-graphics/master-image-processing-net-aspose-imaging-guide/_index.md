## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στην υλοποίηση διαφόρων λειτουργιών χρησιμοποιώντας το Aspose.Imaging για .NET.

### Φόρτωση και κανονικοποίηση μιας εικόνας

#### Επισκόπηση
Η φόρτωση μιας εικόνας στην εφαρμογή σας είναι το πρώτο βήμα για την επεξεργασία της. Μετά τη φόρτωση, η ομαλοποίηση του ιστογράμματός της εξασφαλίζει καλύτερη κατανομή χρωμάτων.

#### Βήμα 1: Φόρτωση εικόνας JPEG
Για να φορτώσετε μια εικόνα, καθορίστε τη διαδρομή προς το αρχείο σας:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/sample.jpg";
using (RasterImage image = (RasterImage)Image.Load(dataDir))
{
    // Συνέχεια στην επεξεργασία...
}
```

#### Βήμα 2: Κανονικοποίηση του Ιστογράμματος
Η κανονικοποίηση ρυθμίζει την ισορροπία χρωμάτων μιας εικόνας, κάνοντάς την να φαίνεται πιο ζωντανή.

```csharp
// Κανονικοποιήστε το ιστόγραμμα για βελτιωμένη κατανομή χρωμάτων
image.NormalizeHistogram();
```

### Ρύθμιση αντίθεσης εικόνας
Η ρύθμιση της αντίθεσης μπορεί να κάνει μια εικόνα να ξεχωρίζει αυξάνοντας το οπτικό της βάθος. Δείτε πώς μπορείτε να το κάνετε:

#### Βήμα 1: Αποθήκευση της κανονικοποιημένης εικόνας
Πριν από την προσαρμογή, αποθηκεύστε την κανονικοποιημένη εικόνα:

```csharp
string outputFilePath = "YOUR_OUTPUT_DIRECTORY" + "/result.png";
image.Save(outputFilePath);
```

#### Βήμα 2: Προσαρμόστε την αντίθεση και αποθηκεύστε ξανά
Αυξήστε ή μειώστε την αντίθεση χρησιμοποιώντας `AdjustContrast`:

```csharp
// Αύξηση αντίθεσης κατά 30 μονάδες
image.AdjustContrast(30);

// Αποθήκευση της προσαρμοσμένης εικόνας σε άλλο αρχείο
string outputFilePath2 = "YOUR_OUTPUT_DIRECTORY" + "/result2.png";
image.Save(outputFilePath2);
```

### Εκκαθάριση: Διαγραφή αποθηκευμένων αρχείων
Μετά την επεξεργασία, καθαρίστε διαγράφοντας προσωρινά αρχεία:

```csharp
File.Delete(outputFilePath);
File.Delete(outputFilePath2);
```
