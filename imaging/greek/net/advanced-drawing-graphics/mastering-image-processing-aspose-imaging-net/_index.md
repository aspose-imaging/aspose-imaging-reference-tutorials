## Οδηγός Εφαρμογής

### Μετατροπή εικόνας

**Επισκόπηση:**
Η μετατροπή εικόνων είναι απαραίτητη για τη μετατροπή εικόνων σε διαφορετικές μορφές, βελτιστοποιώντας τες για διάφορες πλατφόρμες ή εφαρμογές.

#### Βήμα προς βήμα διαδικασία:

**1. Φορτώστε την εικόνα**
```csharp
using (var image = Image.Load("input.jpg"))
{
    // Συνέχεια στην επεξεργασία
}
```
*Γιατί;* Η φόρτωση της εικόνας την αρχικοποιεί στη μνήμη για περαιτέρω χειρισμό.

**2. Μετατροπή και Αποθήκευση**
```csharp
image.Save("output.png", new PngOptions());
```
*Παράμετροι & Σκοπός:* Ο `Save` Η μέθοδος αποθηκεύει την εικόνα σε μια καθορισμένη μορφή, μετατρέποντάς την εδώ σε PNG με τις προεπιλεγμένες επιλογές.

### Επεξεργασία εικόνας

**Επισκόπηση:**
Χειριστείτε εικόνες αλλάζοντας το μέγεθος, περικόπτοντας ή εφαρμόζοντας φίλτρα.

#### Βήμα προς βήμα διαδικασία:

**1. Αλλαγή μεγέθους εικόνας**
```csharp
using (var image = Image.Load("input.jpg"))
{
    image.Resize(200, 200);
    image.Save("resized.jpg");
}
```
*Εξήγηση:* Αυτό το απόσπασμα αλλάζει το μέγεθος της εικόνας σε 200x200 pixel.

### Συμβουλές αντιμετώπισης προβλημάτων

- **Συνηθισμένο πρόβλημα:** Ενδέχεται να προκύψουν σφάλματα "Το αρχείο δεν βρέθηκε" εάν οι διαδρομές είναι εσφαλμένες.
- **Διάλυμα:** Ελέγξτε ξανά τις διαδρομές των αρχείων και βεβαιωθείτε ότι οι εικόνες υπάρχουν στον καθορισμένο κατάλογο.
