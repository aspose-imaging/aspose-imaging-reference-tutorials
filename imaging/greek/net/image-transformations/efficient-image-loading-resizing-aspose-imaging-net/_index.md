## Οδηγός Εφαρμογής
Θα αναλύσουμε την υλοποίηση σε λογικά τμήματα με βάση τα χαρακτηριστικά. Ας ξεκινήσουμε με τη φόρτωση μιας εικόνας από τον δίσκο και στη συνέχεια ας προχωρήσουμε στην αναλογική αλλαγή μεγέθους της.

### Φόρτωση εικόνας από δίσκο
Η αποτελεσματική φόρτωση εικόνων είναι ζωτικής σημασίας για την απόδοση, ειδικά όταν πρόκειται για μεγάλα αρχεία ή πολλά αιτήματα.

#### Βήμα 1: Φόρτωση της εικόνας
Ξεκινήστε ρυθμίζοντας τη διαδρομή του έργου σας και φορτώνοντας την εικόνα χρησιμοποιώντας το Aspose.Imaging:

```csharp
using System;
using Aspose.Imaging;

// Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/aspose-logo.jpg";

// Φόρτωση εικόνας από δίσκο
Image image = Image.Load(dataDir);
if (!image.IsCached)
{
    // Αποθήκευση της εικόνας στην προσωρινή μνήμη για περαιτέρω επεξεργασία
    image.CacheData();
}

// Αποθηκεύστε την εικόνα που φορτώθηκε για να επαληθεύσετε την επιτυχή φόρτωση (προαιρετικό)
string outputDir = "YOUR_OUTPUT_DIRECTORY";
image.Save(outputDir + "/LoadedImage_out.png");
```

- **Εξήγηση:**
  - `Image.Load(dataDir)`: Φορτώνει το καθορισμένο αρχείο εικόνας.
  - `IsCached` Ο έλεγχος διασφαλίζει ότι η εικόνα αποθηκεύεται προσωρινά στη μνήμη για αποτελεσματική πρόσβαση και χειρισμό.

### Αλλαγή μεγέθους εικόνας αναλογικά κατά πλάτος
Η αλλαγή μεγέθους εικόνων διατηρώντας παράλληλα την αναλογία διαστάσεων βοηθά στη διατήρηση της οπτικής ποιότητας. Ας επικεντρωθούμε στην αλλαγή μεγέθους κατά πλάτος.

#### Βήμα 2: Ορισμός νέου πλάτους και αλλαγή μεγέθους

```csharp
using System;
using Aspose.Imaging;

string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/aspose-logo.jpg";

// Φόρτωση της εικόνας για αλλαγή μεγέθους
Image imageWidth = Image.Load(dataDir);
if (!imageWidth.IsCached)
{
    // Αποθήκευση της εικόνας στην προσωρινή μνήμη πριν από την αλλαγή μεγέθους
    imageWidth.CacheData();
}

// Ορισμός νέου πλάτους και αλλαγή μεγέθους τύπου
int newWidth = imageWidth.Width / 2;

// Αλλαγή μεγέθους πλάτους εικόνας αναλογικά χρησιμοποιώντας την αναδειγματοληψία Lanczos
imageWidth.ResizeWidthProportionally(newWidth, ResizeType.LanczosResample);

// Αποθήκευση της εικόνας με το νέο μέγεθος
string outputDirWidth = "YOUR_OUTPUT_DIRECTORY";
imageWidth.Save(outputDirWidth + "/ResizeWidth_out.png");
```

- **Εξήγηση:**
  - `ResizeWidthProportionally`: Προσαρμόζει το πλάτος διατηρώντας παράλληλα την αναλογία διαστάσεων.
  - `ResizeType.LanczosResample` παρέχει υψηλής ποιότητας αλλαγή μεγέθους ελαχιστοποιώντας τα τεχνουργήματα.

### Αλλαγή μεγέθους εικόνας αναλογικά κατά ύψος
Ομοίως, μπορούμε να αλλάξουμε το μέγεθος των εικόνων με βάση το ύψος. Αυτή η μέθοδος είναι χρήσιμη όταν οι κάθετες διαστάσεις χρειάζονται προσαρμογή.

#### Βήμα 3: Ορισμός νέου ύψους και αλλαγή μεγέθους

```csharp
using System;
using Aspose.Imaging;

string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/aspose-logo.jpg";

// Φόρτωση της εικόνας για αλλαγή μεγέθους
Image imageHeight = Image.Load(dataDir);
if (!imageHeight.IsCached)
{
    // Αποθήκευση της εικόνας στην προσωρινή μνήμη πριν από την αλλαγή μεγέθους
    imageHeight.CacheData();
}

// Ορισμός νέου ύψους και αλλαγής μεγέθους τύπου
int newHeight = imageHeight.Height / 2;

// Αλλαγή μεγέθους ύψους εικόνας αναλογικά χρησιμοποιώντας την αναδειγματοληψία Nearest Neighbor
imageHeight.ResizeHeightProportionally(newHeight, ResizeType.NearestNeighbourResample);

// Αποθήκευση της εικόνας με το νέο μέγεθος
string outputDirHeight = "YOUR_OUTPUT_DIRECTORY";
imageHeight.Save(outputDirHeight + "/ResizeHeight_out.png");
```

- **Εξήγηση:**
  - `ResizeHeightProportionally`: Ρυθμίζει το ύψος διατηρώντας παράλληλα την αναλογία διαστάσεων.
  - `ResizeType.NearestNeighbourResample` είναι μια ταχύτερη μέθοδος, κατάλληλη για απλές εργασίες αλλαγής μεγέθους.
