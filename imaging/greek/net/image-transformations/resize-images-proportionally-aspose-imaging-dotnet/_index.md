### Αναλογική αλλαγή μεγέθους εικόνας
Η αναλογική αλλαγή μεγέθους των εικόνων διασφαλίζει ότι η αναλογία διαστάσεων παραμένει άθικτη, αποφεύγοντας οποιαδήποτε παραμόρφωση στα οπτικά σας στοιχεία. Αυτή η λειτουργία είναι απαραίτητη για τη διατήρηση της ποιότητας της εικόνας σε διαφορετικές πλατφόρμες και συσκευές.
#### Βήμα 1: Φόρτωση της εικόνας
Ξεκινήστε φορτώνοντας την εικόνα της οποίας το μέγεθος θέλετε να αλλάξετε:
```csharp
using (Image image = Image.Load("YOUR_DOCUMENT_DIRECTORY/aspose-logo.jpg"))
{
    // Περαιτέρω επεξεργασία...
}
```
**Εξήγηση**Χρησιμοποιούμε `Image.Load()` για να διαβάσετε ένα αρχείο εικόνας από τον καθορισμένο κατάλογο.
#### Βήμα 2: Αποθήκευση δεδομένων εικόνας στην προσωρινή μνήμη
Η προσωρινή αποθήκευση των δεδομένων εικόνας βελτιώνει την απόδοση κατά την επεξεργασία:
```csharp
if (!image.IsCached)
{
    image.CacheData();
}
```
**Γιατί;**Αυτό το βήμα διασφαλίζει ότι όλα τα απαραίτητα δεδομένα εικόνας φορτώνονται στη μνήμη, βελτιστοποιώντας τις επόμενες λειτουργίες.
#### Βήμα 3: Υπολογισμός Νέων Διαστάσεων
Προσδιορίστε τις νέες διαστάσεις διατηρώντας παράλληλα την αναλογία διαστάσεων:
```csharp
int newWidth = image.Width / 2;
image.ResizeWidthProportionally(newWidth);

int newHeight = image.Height / 2;
image.ResizeHeightProportionally(newHeight);
```
**Εξήγηση**Εδώ, μειώνουμε στο μισό τόσο το πλάτος όσο και το ύψος για να διατηρήσουμε την αναλογικότητα.
#### Βήμα 4: Αποθήκευση της εικόνας με το νέο μέγεθος
Τέλος, αποθηκεύστε την εικόνα με το νέο μέγεθος με ένα νέο όνομα αρχείου:
```csharp
image.Save("YOUR_OUTPUT_DIRECTORY/SimpleResizeImageProportionally_out.png");
```
Αυτό το βήμα εγγράφει την επεξεργασμένη εικόνα σας στον κατάλογο εξόδου.
