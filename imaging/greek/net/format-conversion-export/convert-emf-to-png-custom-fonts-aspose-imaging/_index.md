## Οδηγός Εφαρμογής
Θα αναλύσουμε την υλοποίηση σε δύο βασικά χαρακτηριστικά: τη φόρτωση και αποθήκευση εικόνων EMF και τη χρήση προσαρμοσμένων γραμματοσειρών.

### Χαρακτηριστικό 1: Φόρτωση και αποθήκευση εικόνας EMF ως PNG με προεπιλεγμένες γραμματοσειρές
#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα υπάρχον αρχείο EMF, να ρυθμίσετε τις επιλογές rasterization και να το αποθηκεύσετε ως εικόνα PNG χρησιμοποιώντας τις προεπιλεγμένες ρυθμίσεις γραμματοσειράς.

##### Βήματα:

**Βήμα 1: Ρύθμιση επιλογών ραστεροποίησης**
```csharp
using Aspose.Imaging.FileFormats.Emf;
using Aspose.Imaging.ImageOptions;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή καταλόγου εγγράφου σας

// Δημιουργήστε μια παρουσία επιλογών ραστεροποίησης για εικόνες EMF
EmfRasterizationOptions emfRasterizationOptions = new EmfRasterizationOptions();
emfRasterizationOptions.BackgroundColor = System.Drawing.Color.WhiteSmoke;
```

**Βήμα 2: Ρύθμιση παραμέτρων επιλογών PNG**
```csharp
// Ρύθμιση επιλογών PNG χρησιμοποιώντας τις ρυθμίσεις rasterization
PngOptions pngOptions = new PngOptions();
pngOptions.VectorRasterizationOptions = emfRasterizationOptions;
```

**Βήμα 3: Φόρτωση και προσωρινή αποθήκευση δεδομένων εικόνας EMF**
```csharp
using (EmfImage image = (EmfImage)Aspose.Imaging.Image.Load(dataDir + "Picture1.emf"))
{
    // Αποθήκευση προσωρινά των δεδομένων της εικόνας που φορτώθηκε
    image.CacheData();
    
    // Ορισμός διαστάσεων εξόδου για την εικόνα PNG
    pngOptions.VectorRasterizationOptions.PageWidth = 300;
    pngOptions.VectorRasterizationOptions.PageHeight = 350;

    // Επαναφορά ρυθμίσεων γραμματοσειράς στις προεπιλογές
    Aspose.Imaging.Fonts.FontSettings.Reset();

    // Αποθηκεύστε την εικόνα EMF ως PNG με προεπιλεγμένες γραμματοσειρές
    string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του καταλόγου εξόδου σας
    image.Save(outputDir + "Picture1_default_fonts_out.png", pngOptions);
}
```

### Λειτουργία 2: Καθορισμός φακέλου προσαρμοσμένων γραμματοσειρών
#### Επισκόπηση
Αυτή η ενότητα επεκτείνει τη λειτουργικότητα ώστε να περιλαμβάνει προσαρμοσμένες πηγές γραμματοσειρών, ενισχύοντας την ευελιξία στην απόδοση κειμένου.

##### Βήματα:

**Βήμα 1: Προετοιμασία επιλογών ραστεροποίησης και PNG**
```csharp
// Δημιουργήστε μια παρουσία επιλογών ραστεροποίησης για εικόνες EMF
EmfRasterizationOptions emfRasterizationOptions = new EmfRasterizationOptions();
emfRasterizationOptions.BackgroundColor = System.Drawing.Color.WhiteSmoke;

// Ρύθμιση επιλογών PNG χρησιμοποιώντας τις ρυθμίσεις rasterization
PngOptions pngOptions = new PngOptions();
pngOptions.VectorRasterizationOptions = emfRasterizationOptions;
```

**Βήμα 2: Φόρτωση εικόνας EMF και δεδομένων προσωρινής μνήμης**
```csharp
using (EmfImage image = (EmfImage)Aspose.Imaging.Image.Load(dataDir + "Picture1.emf"))
{
    // Αποθήκευση προσωρινά των δεδομένων της εικόνας που φορτώθηκε
    image.CacheData();
    
    // Ορισμός διαστάσεων εξόδου για την εικόνα PNG
    pngOptions.VectorRasterizationOptions.PageWidth = 300;
    pngOptions.VectorRasterizationOptions.PageHeight = 350;

    // Αποκτήστε προεπιλεγμένους φακέλους γραμματοσειρών και προσθέστε έναν προσαρμοσμένο
    List<string> fonts = new List<string>(FontSettings.GetDefaultFontsFolders());
    fonts.Add(dataDir + "arialAndTimesAndCourierRegular.xml");

    // Αντιστοιχίστε τη λίστα φακέλων γραμματοσειρών στις ρυθμίσεις γραμματοσειρών
    FontSettings.SetFontsFolders(fonts.ToArray(), true);

    // Αποθηκεύστε την εικόνα EMF ως PNG χρησιμοποιώντας προσαρμοσμένες γραμματοσειρές
    string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του καταλόγου εξόδου σας
    image.Save(outputDir + "Picture1_with_my_fonts_out.png", pngOptions);
}
```
