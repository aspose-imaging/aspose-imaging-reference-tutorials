## Οδηγός Εφαρμογής

### Δυνατότητα 1: Αποθήκευση εικόνας JPEG σε μορφή CMYK χωρίς απώλειες
#### Επισκόπηση
Αποθηκεύστε το αρχείο JPEG χρησιμοποιώντας συμπίεση χωρίς απώλειες με τον χρωματικό χώρο CMYK, ιδανικό για εκτυπώσεις υψηλής ποιότητας.
#### Βήμα προς βήμα εφαρμογή
**1. Ορίστε τη διαδρομή εικόνας πηγής**
Καθορίστε τη διαδρομή προς την εικόνα JPEG πηγής σας:
```csharp
string sourceImagePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "056.jpg");
```
**2. Φόρτωση και διαμόρφωση της εικόνας**
Φορτώστε το αρχείο JPEG και ρυθμίστε τις επιλογές για συμπίεση CMYK χωρίς απώλειες:
```csharp
using (MemoryStream jpegStream = new MemoryStream())
{
    using (JpegImage image = (JpegImage)Image.Load(sourceImagePath))
    {
        JpegOptions options = new JpegOptions();
        
        // Ρύθμιση παραμέτρων τύπου χρώματος σε CMYK για συμπίεση χωρίς απώλειες
        options.ColorType = JpegCompressionColorMode.Cmyk;
        options.CompressionType = JpegCompressionMode.Lossless;
        
        // Αποθηκεύστε την εικόνα με αυτές τις ρυθμίσεις
        image.Save(jpegStream, options);
    }
}
```
Αυτός ο κώδικας διαμορφώνει `ColorType` σε CMYK και χρησιμοποιεί συμπίεση χωρίς απώλειες για τη διατήρηση της ποιότητας.
### Χαρακτηριστικό 2: Μετατροπή εικόνας JPEG από μορφή CMYK χωρίς απώλειες σε μορφή PNG
#### Επισκόπηση
Μόλις η εικόνα σας είναι σε μορφή CMYK χωρίς απώλειες, ίσως θελήσετε να τη μετατρέψετε για χρήση στο διαδίκτυο ή σε άλλες εφαρμογές. Αυτή η λειτουργία δείχνει πώς να το κάνετε αυτό χρησιμοποιώντας το Aspose.Imaging.
#### Βήμα προς βήμα εφαρμογή
**1. Φόρτωση της εικόνας από τη ροή μνήμης**
Για να ξεκινήσετε τη μετατροπή, φορτώστε την εικόνα JPEG από τη ροή μνήμης:
```csharp
using (MemoryStream jpegStream = new MemoryStream())
{
    // Βεβαιωθείτε ότι το JPEG σας έχει φορτωθεί εδώ.
}
```
**2. Μετατροπή σε μορφή PNG και αποθήκευση**
Χρησιμοποιήστε την υποστήριξη μορφής του Aspose.Imaging για μετατροπή και αποθήκευση ως αρχείο PNG:
```csharp
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "056_cmyk.png");
using (JpegImage image = (JpegImage)Image.Load(jpegStream))
{
    // Μετατρέψτε την εικόνα JPEG CMYK σε αρχείο PNG
    image.Save(outputPath, new PngOptions());
}
```
Αυτή η μετατροπή διατηρεί την ακεραιότητα των χρωμάτων κατά την αλλαγή των μορφών.
