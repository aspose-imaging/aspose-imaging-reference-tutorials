## Οδηγός Εφαρμογής

### Φόρτωση εικόνας CAD

#### Επισκόπηση
Η φόρτωση ενός αρχείου CAD στην εφαρμογή .NET είναι απλή με το Aspose.Imaging. Αυτή η λειτουργία σάς επιτρέπει να έχετε πρόσβαση και να χειρίζεστε το περιεχόμενο αρχείων όπως `.cdr`.

#### Βήμα προς βήμα εφαρμογή
**Φόρτωση του αρχείου CAD**
```csharp
using Aspose.Imaging;
using System.IO;

string inputFileName = "YOUR_DOCUMENT_DIRECTORY/MultiPage.cdr"; // Αντικατάσταση με τη διαδρομή σας

// Φόρτωση της εικόνας σε ένα αντικείμενο Aspose.Imaging CdrImage
Aspose.Imaging.FileFormats.Cdr.CdrImage image = (Aspose.Imaging.FileFormats.Cdr.CdrImage)Image.Load(inputFileName);

image.Dispose(); // Καθαρισμός πόρων μετά τη φόρτωση
```
**Εξήγηση**: 
- `Image.Load` διαβάζει το αρχείο CAD σας, επιστρέφοντας ένα `CdrImage` αντικείμενο για περαιτέρω χειρισμό.
- Να θυμάστε πάντα να καλείτε `.Dispose()` για να ελευθερωθεί η μνήμη.

### Εξαγωγή εικόνας πολλαπλών σελίδων σε PSD με συγχώνευση επιπέδων

#### Επισκόπηση
Η εξαγωγή εικόνων CAD πολλαπλών σελίδων ως PSD μονής στρώσης είναι ζωτικής σημασίας για την απλοποίηση σύνθετων σχεδίων. Αυτή η λειτουργία συγχωνεύει όλα τα επίπεδα σε ένα, καθιστώντας την εικόνα πιο διαχειρίσιμη.

#### Βήμα προς βήμα εφαρμογή
**Ρύθμιση παραμέτρων και αποθήκευση ως PSD**
```csharp
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Psd;
using Aspose.Imaging.ImageOptions;

string outputFileName = "YOUR_OUTPUT_DIRECTORY/MultiPageOut.psd"; // Αντικατάσταση με τη διαδρομή σας

Aspose.Imaging.FileFormats.Cdr.CdrImage image = (Aspose.Imaging.FileFormats.Cdr.CdrImage)Image.Load("YOUR_DOCUMENT_DIRECTORY/MultiPage.cdr");
try
{
    ImageOptionsBase options = new PsdOptions();

    // Συγχώνευση επιπέδων σε ένα στο αρχείο PSD
    options.MultiPageOptions = new MultiPageOptions { MergeLayers = true };

    // Ορίστε επιλογές ραστεροποίησης για καλύτερη ποιότητα εικόνας
    options.VectorRasterizationOptions = image.GetDefaultOptions(new object[] { Color.White, image.Width, image.Height }).VectorRasterizationOptions;
    options.VectorRasterizationOptions.TextRenderingHint = TextRenderingHint.SingleBitPerPixel;
    options.VectorRasterizationOptions.SmoothingMode = SmoothingMode.None;

    // Αποθηκεύστε το CDR ως αρχείο PSD με συγχωνευμένα επίπεδα
    image.Save(outputFileName, options);
}
finally
{
    image.Dispose();
}
```
**Εξήγηση**: 
- `PsdOptions` Ρυθμίζει τον τρόπο αποθήκευσης των εικόνων CAD ως PSD.
- `MultiPageOptions.MergeLayers = true` διασφαλίζει ότι όλα τα επίπεδα από το αρχείο προέλευσης συνδυάζονται σε ένα.

```csharp
{
    File.Delete(outputFilePath); // Διαγράψτε το αρχείο εάν υπάρχει
}
```
