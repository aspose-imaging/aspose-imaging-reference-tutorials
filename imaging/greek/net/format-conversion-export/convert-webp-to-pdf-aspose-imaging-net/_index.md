## Οδηγός Εφαρμογής

### Μετατροπή WebP σε PDF

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε αποτελεσματικά ένα αρχείο εικόνας WebP σε έγγραφο PDF. Δείτε πώς:

#### Φόρτωση της εικόνας WebP
```csharp
using System;
using System.IO;
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Pdf;
using Aspose.Imaging.ImageOptions;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string outputPath = @"YOUR_OUTPUT_DIRECTORY";

// Φορτώστε την εικόνα WebP από τον καθορισμένο κατάλογο.
string inputFile = Path.Combine(dataDir, "Animation.webp");
```

#### Επιλογές δημιουργίας PDF
```csharp
using (Aspose.Imaging.FileFormats.Webp.WebPImage image = 
    (Aspose.Imaging.FileFormats.Webp.WebPImage)Image.Load(inputFile))
{
    // Ρυθμίστε τις παραμέτρους του PDF εξόδου.
    PdfOptions options = new PdfOptions();
    options.PdfDocumentInfo = new PdfDocumentInfo();
}
```

#### Αποθήκευση ως PDF
```csharp
// Ορίστε τη διαδρομή του αρχείου εξόδου και αποθηκεύστε την εικόνα ως PDF.
string outputFile = Path.Combine(outputPath, "Animation.pdf");
image.Save(outputFile, options);
```
**Εξήγηση:** Αυτό το απόσπασμα κώδικα φορτώνει μια εικόνα WebP, ρυθμίζει τις επιλογές μετατροπής PDF και την αποθηκεύει ως έγγραφο PDF. Προσαρμογή `dataDir` και `outputPath` για να ταιριάζει με τους καταλόγους σας.

### Αφαίρεση δημιουργημένου PDF

Μετά τη μετατροπή, ίσως θελήσετε να καθαρίσετε τα προσωρινά αρχεία:

#### Διαγραφή του PDF
```csharp
string outputFile = Path.Combine(outputPath, "Animation.pdf");

// Βεβαιωθείτε ότι το αρχείο υπάρχει πριν από τη διαγραφή.
if (File.Exists(outputFile))
{
    File.Delete(outputFile);
}
```
**Εξήγηση:** Αυτός ο κώδικας ελέγχει για την ύπαρξη του δημιουργημένου PDF και το διαγράφει για να ελευθερώσει χώρο.
