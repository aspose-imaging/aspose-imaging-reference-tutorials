## Οδηγός Εφαρμογής

### Μετατροπή JPEG σε DICOM

Αυτή η λειτουργία δείχνει πώς να μετατρέψετε μια εικόνα JPEG μίας σελίδας σε μορφή DICOM.

#### Βήμα 1: Φόρτωση της εικόνας JPEG

Καθορίστε τον κατάλογο και το όνομα αρχείου του JPEG πηγής σας:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string fileName = "sample.jpg"; // Καθορίστε εδώ το όνομα του αρχείου JPEG πηγής σας
```

Φορτώστε το JPEG χρησιμοποιώντας το Aspose.Imaging `Image` τάξη:
```csharp
using (var image = Image.Load(Path.Combine(dataDir, fileName)))
{
    // Συνέχεια αποθήκευσης σε μορφή DICOM
}
```

#### Βήμα 2: Αποθήκευση ως DICOM

Χρήση `DicomOptions` για να μετατρέψετε και να αποθηκεύσετε την εικόνα σας ως αρχείο DICOM:
```csharp
string outputFileNameSingleDcm = Path.Combine(dataDir, "output.dcm");
image.Save(outputFileNameSingleDcm, new DicomOptions());
```

### Μετατροπή πολυσέλιδου TIFF σε DICOM

Στη συνέχεια, μετατρέψτε μια εικόνα TIFF πολλαπλών σελίδων σε μορφή αρχείου DICOM.

#### Βήμα 1: Φόρτωση της εικόνας TIFF πολλαπλών σελίδων

Προσδιορίστε το αρχείο TIFF πηγής σας:
```csharp
string inputFileNameMultipage = Path.Combine(dataDir, "multipage.tif");
```

Φόρτωσή του χρησιμοποιώντας το Aspose.Imaging `Image` τάξη:
```csharp
using (var image = Image.Load(inputFileNameMultipage))
{
    // Προχωρήστε στην αποθήκευση σε μορφή DICOM
}
```

#### Βήμα 2: Αποθήκευση ως DICOM πολλαπλών σελίδων

Παρόμοια με τη μετατροπή JPEG, χρησιμοποιήστε `DicomOptions` για εξοικονόμηση:
```csharp
string outputFileNameMultipageDcm = Path.Combine(dataDir, "outputMultipage.dcm");
image.Save(outputFileNameMultipageDcm, new DicomOptions());
```
