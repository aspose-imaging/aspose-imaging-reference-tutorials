## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Μετατροπή SVG σε PNG χρησιμοποιώντας το Aspose.Imaging για .NET

Αυτή η λειτουργία δείχνει πώς να μετατρέψετε ένα αρχείο SVG σε μορφή PNG χρησιμοποιώντας τις επιλογές rasterization που είναι διαθέσιμες στο Aspose.Imaging.

**Επισκόπηση:**
Θα φορτώσουμε μια εικόνα SVG, θα διαμορφώσουμε τις ρυθμίσεις rasterization και θα την αποθηκεύσουμε ως αρχείο PNG. Αυτή η μέθοδος διασφαλίζει μετατροπή υψηλής ποιότητας διατηρώντας παράλληλα την πιστότητα της εικόνας.

**Βήματα:**
1. **Φόρτωση του αρχείου SVG:**
   - Χρήση `Image.Load()` για να διαβάσετε το έγγραφο SVG σας.
2. **Ρύθμιση παραμέτρων επιλογών ραστεροποίησης:**
   - Στήνω `SvgRasterizationOptions` για να ορίσετε πώς θα πρέπει να ραστεροποιηθεί το SVG, συμπεριλαμβανομένου του μεγέθους σελίδας και άλλων παραμέτρων.
3. **Ορισμός συγκεκριμένων επιλογών PNG:**
   - Συνδέστε αυτές τις ρυθμίσεις ραστεροποίησης με `PngOptions`, διασφαλίζοντας ότι η μετατροπή χρησιμοποιεί τις καθορισμένες ρυθμίσεις σας.
4. **Εκτέλεση μετατροπής και αποθήκευση:**
   - Αποθηκεύστε την ραστεροποιημένη εικόνα σε μια ροή ή αρχείο χρησιμοποιώντας το `Save()` μέθοδος.

**Απόσπασμα κώδικα:**
```csharp
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Svg;
using Aspose.Imaging.ImageOptions;
using System.IO;

public static void RasterizeSvgToPng()
{
    string svgFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "asposenet_220_src02.svg");
    using (MemoryStream drawnImageStream = new MemoryStream())
    {
        using (SvgImage svgImage = (SvgImage)Image.Load(svgFilePath))
        {
            SvgRasterizationOptions rasterizationOptions = new SvgRasterizationOptions();
            rasterizationOptions.PageSize = svgImage.Size;

            PngOptions saveOptions = new PngOptions();
            saveOptions.VectorRasterizationOptions = rasterizationOptions;

            svgImage.Save(drawnImageStream, saveOptions);
        }
    }
}
```

**Εξήγηση:**
- **Εικόνα Svg:** Αντιπροσωπεύει το αρχείο SVG που έχει φορτωθεί στη μνήμη.
- **Επιλογές SvgRasterization & PngOptions:** Ρυθμίστε τον τρόπο μετατροπής και αποθήκευσης της εικόνας.

### Χαρακτηριστικό 2: Σχεδίαση σε ραστεροποιημένη εικόνα και αποθήκευση ως SVG

Βελτιώστε τις εικόνες PNG σχεδιάζοντας επιπλέον γραφικά πάνω σε αυτές και, στη συνέχεια, αποθηκεύστε αυτές τις βελτιώσεις ως SVG.

**Επισκόπηση:**
Φόρτωση ενός ραστεροποιημένου PNG από μια ροή, σχεδίαση νέων γραφικών χρησιμοποιώντας `SvgGraphics2D`και τέλος να το μετατρέψετε ξανά σε μορφή SVG.

**Βήματα:**
1. **Προετοιμάστε το περιβάλλον σας:**
   - Φορτώστε το αρχικό SVG και αποθηκεύστε την ραστεροποιημένη μορφή του σε μια ροή μνήμης.
2. **Ρύθμιση γραφικών για σχέδιο:**
   - Αρχικοποίηση `SvgGraphics2D` με την εικόνα ράστερ σας για να προετοιμαστείτε για λειτουργίες σχεδίασης.
3. **Υπολογισμός διαστάσεων και θέσης:**
   - Προσδιορίστε πού στην επιφάνεια SVG θέλετε να σχεδιάσετε.
4. **Σχεδίαση και αποθήκευση:**
   - Σχεδιάστε πάνω στο SVG και αποθηκεύστε το ξανά ως νέο αρχείο χρησιμοποιώντας `EndRecording()`.

**Απόσπασμα κώδικα:**
```csharp
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Svg;
using Aspose.Imaging.FileFormats.Svg.Graphics;
using System.IO;

public static void DrawOnRasterizedImageAndSaveAsSvg()
{
    string svgInputPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "asposenet_220_src02.svg");
    string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "asposenet_220_src02.DrawVectorImage.svg");

    using (MemoryStream drawnImageStream = new MemoryStream())
    {
        using (SvgImage svgImage = (SvgImage)Image.Load(svgInputPath))
        {
            SvgRasterizationOptions rasterizationOptions = new SvgRasterizationOptions();
            rasterizationOptions.PageSize = svgImage.Size;

            PngOptions saveOptions = new PngOptions();
            saveOptions.VectorRasterizationOptions = rasterizationOptions;

            svgImage.Save(drawnImageStream, saveOptions);

            drawnImageStream.Seek(0, SeekOrigin.Begin);
            using (RasterImage imageToDraw = (RasterImage)Image.Load(drawnImageStream))
            {
                SvgGraphics2D graphics = new SvgGraphics2D(svgImage);

                int width = imageToDraw.Width / 2;
                int height = imageToDraw.Height / 2;
                Point origin = new Point((svgImage.Width - width) / 2, (svgImage.Height - height) / 2);
                Size size = new Size(width, height);

                graphics.DrawImage(imageToDraw, origin, size);

                using (SvgImage resultImage = graphics.EndRecording())
                {
                    resultImage.Save(outputPath);
                }
            }
        }
    }
}
```

**Εξήγηση:**
- **SvgGraphics2D:** Παρέχει μεθόδους σχεδίασης στον καμβά SVG.
- **Εικόνα ράστερ:** Αντιπροσωπεύει την φορτωμένη εικόνα PNG έτοιμη για χειρισμό.
