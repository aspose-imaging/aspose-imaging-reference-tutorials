## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Συμπίεση JPEG - Μετατροπή PNG σε JPEG-LS

#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει τη μετατροπή μιας εικόνας PNG με 8 bit ανά δείγμα σε μορφή JPEG-LS χρησιμοποιώντας μόνο 2 bit ανά δείγμα, μειώνοντας σημαντικά το μέγεθος του αρχείου διατηρώντας παράλληλα αποδεκτή ποιότητα.

#### Βήμα προς βήμα εφαρμογή

##### Βήμα 1: Ορισμός διαδρομών και ρυθμίσεων αρχείων

Ορίστε διαδρομές για το αρχείο PNG προέλευσης και το αρχείο JPEG-LS προορισμού. Ορίστε τα bits ανά κανάλι (bpp) σε 2 για επίδειξη συμπίεσης:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
int bpp = 2; // Ρύθμιση bit ανά κανάλι
string originPngFileName = System.IO.Path.Combine(dataDir, "lena_16g_lin.png");
string outputJpegFileName = System.IO.Path.Combine("YOUR_OUTPUT_DIRECTORY", "output_image.jpegls");
```

##### Βήμα 2: Φόρτωση και μετατροπή της εικόνας

Φορτώστε την εικόνα PNG, διαμορφώστε τις επιλογές JPEG-LS και αποθηκεύστε το αρχείο που έχει μετατραπεί:

```csharp
using (var image = Image.Load(originPngFileName))
{
    var jpegOptions = new JpegLsOptions()
    {
        BitsPerSample = bpp,
    };

    image.Save(outputJpegFileName, jpegOptions);
}
```

**Επεξήγηση παραμέτρων:**
- `BitsPerSample`: Καθορίζει το βάθος χρώματος για κάθε κανάλι στη μορφή JPEG-LS.
- `image.Load()`: Ανοίγει και φορτώνει μια εικόνα από μια διαδρομή αρχείου.
- `image.Save()`Αποθηκεύει την επεξεργασμένη εικόνα στην καθορισμένη διαδρομή εξόδου.

##### Συμβουλές αντιμετώπισης προβλημάτων
- **Το αρχείο δεν βρέθηκε:** Βεβαιωθείτε ότι το αρχείο PNG πηγής σας υπάρχει στην καθορισμένη διαδρομή.
- **Προβλήματα δικαιωμάτων:** Επαληθεύστε ότι η εφαρμογή σας έχει δικαιώματα ανάγνωσης/εγγραφής στους καταλόγους με τους οποίους εργάζεστε.
