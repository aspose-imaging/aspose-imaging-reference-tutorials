## Οδηγός Εφαρμογής

Ας αναλύσουμε βήμα προς βήμα τη διαδικασία υλοποίησης για τη μετατροπή μιας εικόνας JPEG σε CMYK JPEG-LS χρησιμοποιώντας το Aspose.Imaging για .NET.

### Φόρτωση και μετατροπή εικόνων

**1. Φόρτωση εικόνας JPEG:**

Ξεκινήστε φορτώνοντας ένα υπάρχον αρχείο JPEG. Αυτό αποτελεί τη βάση της διαδικασίας μετατροπής μας.

```csharp
using System.IO;
using Aspose.Imaging.FileFormats.Jpeg;
using Aspose.Imaging.ImageOptions;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
MemoryStream jpegLsStream = new MemoryStream();
try
{
    // Φόρτωση εικόνας JPEG από αρχείο.
    using (JpegImage image = (JpegImage)Image.Load("056.jpg"))
```

**2. Διαμόρφωση επιλογών JPEG:**

Διαμορφώστε τις επιλογές για μετατροπή ορίζοντας τον τύπο χρώματος σε CMYK και καθορίζοντας JPEG-LS ως τον τύπο συμπίεσης.

```csharp
    JpegOptions options = new JpegOptions();
    
    // Ορίστε τον τύπο χρώματος σε CMYK.
    options.ColorType = JpegCompressionColorMode.Cmyk;

    // Χρησιμοποιήστε συμπίεση JPEG-LS.
    options.CompressionType = JpegCompressionMode.JpegLs;

    // Δεν χρησιμοποιούνται συγκεκριμένα προφίλ RGB ή CMYK.
    options.RgbColorProfile = null;
    options.CmykColorProfile = null;

    // Αποθηκεύστε την εικόνα σε μια ροή μνήμης σε μορφή JPEG-LS.
    image.Save(jpegLsStream, options);
```

### Αποθήκευση μετατρεπόμενων εικόνων

**3. Φόρτωση από τη Ροή Μνήμης και Αποθήκευση ως PNG:**

Μετά τη μετατροπή, φορτώστε την εικόνα από τη ροή μνήμης και αποθηκεύστε την σε άλλη μορφή, όπως PNG.

```csharp
    // Επαναφέρετε τη θέση της ροής για να διασφαλίσετε τη σωστή φόρτωση.
    jpegLsStream.Position = 0;
    
    // Φορτώστε το CMYK JPEG-LS από τη ροή μνήμης.
    using (JpegImage image = (JpegImage)Image.Load(jpegLsStream))
    {
        // Αποθήκευση ως PNG.
        image.Save("YOUR_OUTPUT_DIRECTORY/056_cmyk.png", new PngOptions());
    }
}
finally
{
    // Απορρίψτε τη ροή μνήμης σε ελεύθερους πόρους.
    jpegLsStream.Dispose();
}
```
