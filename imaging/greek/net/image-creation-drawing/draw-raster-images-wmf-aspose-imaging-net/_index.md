## Οδηγός Εφαρμογής

### Φόρτωση και σχεδίαση εικόνας ράστερ σε WMF

Αυτή η ενότητα σας καθοδηγεί στη φόρτωση μιας εικόνας ράστερ και στη σχεδίαση της σε έναν καμβά WMF χρησιμοποιώντας το Aspose.Imaging για .NET. Θα καλύψουμε:

#### Βήμα 1: Ορισμός διαδρομών καταλόγου

Ξεκινήστε ορίζοντας διαδρομές προς τον κατάλογο εγγράφων και τον κατάλογο εξόδου, οι οποίες θα χρησιμοποιηθούν σε όλο τον κώδικα.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

Αντικαθιστώ `YOUR_DOCUMENT_DIRECTORY` και `YOUR_OUTPUT_DIRECTORY` με πραγματικές διαδρομές όπου αποθηκεύονται οι εικόνες σας.

#### Βήμα 2: Φόρτωση εικόνας ράστερ

Φορτώστε την εικόνα ράστερ που θέλετε να σχεδιάσετε στον καμβά WMF χρησιμοποιώντας το API του Aspose.Imaging.

```csharp
using (RasterImage imageToDraw = (RasterImage)Image.Load(dataDir + "/asposenet_220_src01.png"))
{
    // Ο κώδικας συνεχίζεται...
}
```

Βεβαιωθείτε ότι η διαδρομή του αρχείου εικόνας σας έχει καθοριστεί σωστά. Αυτό το απόσπασμα φορτώνει ένα αρχείο PNG ως εικόνα raster.

#### Βήμα 3: Φόρτωση εικόνας WMF

Στη συνέχεια, φορτώστε την εικόνα WMF που θα λειτουργήσει ως επιφάνεια σχεδίασης.

```csharp
using (WmfImage canvasImage = (WmfImage)Image.Load(dataDir + "/asposenet_222_wmf_200.wmf"))
{
    // Συνέχεια με τη ρύθμιση γραφικών...
}
```

Βεβαιωθείτε ότι το αρχείο προορισμού WMF είναι προσβάσιμο από την καθορισμένη διαδρομή.

#### Βήμα 4: Αρχικοποίηση γραφικών για σχέδιο

Αρχικοποίηση `WmfRecorderGraphics2D` για την καταγραφή σχεδίων στην εικόνα WMF που έχει φορτωθεί. Αυτό το αντικείμενο επιτρέπει λειτουργίες σχεδίασης όπως η προσθήκη εικόνων, γραμμών και σχημάτων στον καμβά.

```csharp
WmfRecorderGraphics2D graphics = WmfRecorderGraphics2D.FromWmfImage(canvasImage);
```

#### Βήμα 5: Σχεδιάστε μια εικόνα ράστερ

Χρησιμοποιήστε το `DrawImage()` Μέθοδος για να σχεδιάσετε την φορτωμένη εικόνα ράστερ στον καμβά WMF σας. Καθορίστε τα ορθογώνια προέλευσης και προορισμού, επιλέγοντας μονάδες pixel για ακρίβεια σχεδίασης.

```csharp
graphics.DrawImage(
    imageToDraw,
    new Rectangle(67, 67, canvasImage.Width, canvasImage.Height),
    new Rectangle(0, 0, imageToDraw.Width, imageToDraw.Height),
    GraphicsUnit.Pixel
);
```

Αυτό τοποθετεί την εικόνα ράστερ στον καμβά WMF και την τεντώνει ώστε να ταιριάζει εντός καθορισμένων ορίων.

#### Βήμα 6: Αποθηκεύστε την εικόνα που προκύπτει

Τέλος, τερματίστε τη διαδικασία εγγραφής και αποθηκεύστε το τροποποιημένο αρχείο WMF με την εικόνα ράστερ που σχεδιάσατε.

```csharp
using (WmfImage resultImage = graphics.EndRecording())
{
    resultImage.Save(outputDir + "/asposenet_222_wmf_200.DrawImage.wmf");
}
```

Αυτό εξάγει ένα νέο αρχείο WMF με την ενσωματωμένη εικόνα ράστερ στον καθορισμένο κατάλογο εξόδου σας.
