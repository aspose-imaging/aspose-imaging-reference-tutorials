### Δημιουργία εικόνας χρησιμοποιώντας BmpOptions και Stream
#### Επισκόπηση
Θα δείξουμε πώς να δημιουργήσετε ένα αρχείο BMP καθορίζοντας διάφορες ρυθμίσεις, όπως bits ανά pixel.

#### Βήμα προς βήμα εφαρμογή:
**1. Ορισμός καταλόγων:**
Ξεκινήστε ορίζοντας τους καταλόγους όπου αποθηκεύονται τα έγγραφά σας και πού θέλετε να αποθηκεύσετε την έξοδο.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**2. Ρύθμιση παραμέτρων BmpOptions:**
Δημιουργήστε μια παρουσία του `BmpOptions` και ορίστε τις ιδιότητές του, όπως bits ανά pixel για το βάθος χρώματος.
```csharp
BmpOptions imageOptions = new BmpOptions();
imageOptions.BitsPerPixel = 24; // Ρύθμιση BMP πραγματικού χρώματος
```

**3. Ορίστε μια ροή για έξοδο:**
Χρησιμοποιήστε μια ροή για να διαχειριστείτε το αρχείο εξόδου όπου θα αποθηκευτούν τα δεδομένα BMP.
```csharp
using (Stream stream = new FileStream(outputDir + "sample_out.bmp", FileMode.Create))
{
    // Προσθέστε περισσότερες λεπτομέρειες υλοποίησης εδώ...
}
```

#### Εξήγηση
- **BitsPerPixel:** Ορίζει το βάθος χρώματος της εικόνας. Η τιμή 24 χρησιμοποιείται για εικόνες πραγματικού χρώματος.
- **FileStream:** Διαχειρίζεται την εγγραφή και την ανάγνωση από αρχεία, διασφαλίζοντας την ορθή διάθεση των πόρων με `using` δήλωση.

**Συμβουλές αντιμετώπισης προβλημάτων:**

- Ελέγξτε τα δικαιώματα αρχείων εάν αντιμετωπίσετε προβλήματα πρόσβασης.
