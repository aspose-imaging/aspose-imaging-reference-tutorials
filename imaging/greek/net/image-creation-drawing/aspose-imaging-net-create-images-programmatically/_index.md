## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στη δημιουργία μιας εικόνας και στη σχεδίαση πάνω σε αυτήν με το Aspose.Imaging για .NET.

### Δημιουργία εικόνας με συγκεκριμένες επιλογές

**Επισκόπηση**Ξεκινήστε δημιουργώντας μια κενή εικόνα με καθορισμένες ιδιότητες όπως μέγεθος και βάθος χρώματος.

```csharp
using Aspose.Imaging;
using Aspose.Imaging.ImageOptions;

// Ορίστε τον κατάλογο εξόδου.
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Ρυθμίστε τις παραμέτρους του BmpOptions για τις ρυθμίσεις εικόνας.
BmpOptions imageOptions = new BmpOptions();
imageOptions.BitsPerPixel = 24; // Ορίστε το βάθος χρώματος σε 24 bit ανά pixel.

// Καθορίστε τις επιλογές διαδρομής αρχείου και προέλευσης.
string filePath = System.IO.Path.Combine(outputDir, "SampleImage_out.bmp");
imageOptions.Source = new FileCreateSource(filePath, false); // Η αντικατάσταση δεν επιτρέπεται εάν το αρχείο υπάρχει.

using (var image = Image.Create(imageOptions, 500, 500)) // Δημιουργήστε μια εικόνα 500x500 pixel.
{
    // Συνεχίστε με τις λειτουργίες σχεδίασης σε αυτήν την περίπτωση εικόνας.
}
```
**Εξήγηση**: Εδώ, διαμορφώνουμε `BmpOptions` για να ορίσετε το βάθος χρώματος και να καθορίσετε τη διαδρομή αρχείου για την αποθήκευση της εικόνας. `Image.Create` Η μέθοδος αρχικοποιεί ένα αντικείμενο εικόνας καθορισμένων διαστάσεων.

### Σχεδίαση σχημάτων και εφαρμογή εφέ

**Επισκόπηση**Σχεδιάστε σχήματα όπως ελλείψεις και εφαρμόστε εφέ διαβάθμισης σε πολύγωνα χρησιμοποιώντας το Aspose.Imaging.

```csharp
using Aspose.Imaging;
using Aspose.Imaging.Brushes;
using System.Drawing;

using (var graphics = new Graphics(image)) // Λήψη ενός γραφικού αντικειμένου.
{
    // Αλλάξτε το χρώμα φόντου της εικόνας σε λευκό.
    graphics.Clear(Color.White);

    // Δημιουργήστε ένα στυλό για να σχεδιάζετε σχήματα, ορίστε το χρώμα του σε μπλε.
    var pen = new Pen(Color.Blue);

    // Σχεδιάστε μια έλλειψη στην εικόνα.
    graphics.DrawEllipse(pen, new Rectangle(10, 10, 150, 100));

    // Εφαρμόστε ένα γραμμικό πινέλο με διαβάθμιση από κόκκινο σε λευκό υπό γωνία 45 μοιρών.
    using (var linearGradientBrush = new LinearGradientBrush(image.Bounds, Color.Red, Color.White, 45f))
    {
        // Γεμίστε ένα πολύγωνο με το εφέ διαβάθμισης.
        graphics.FillPolygon(
            linearGradientBrush,
            new[] { new Point(200, 200), new Point(400, 200), new Point(250, 350) });
    }
}
```
**Εξήγηση**Ξεκινάμε καθαρίζοντας το φόντο της εικόνας και στη συνέχεια σχεδιάζουμε μια έλλειψη. Στη συνέχεια, ένα `LinearGradientBrush` χρησιμοποιείται για να γεμίσει ένα πολύγωνο με εφέ διαβάθμισης.
