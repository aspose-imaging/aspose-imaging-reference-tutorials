## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τη διαδικασία σε διαχειρίσιμα βήματα, εστιάζοντας στη σχεδίαση ορθογωνίων σε μια εικόνα χρησιμοποιώντας το Aspose.Imaging για .NET.

### Δημιουργία και Ρύθμιση της Εικόνας

Αρχικά, ας δημιουργήσουμε μια νέα εικόνα bitmap όπου μπορούμε να σχεδιάσουμε τα ορθογώνιά μας:

```csharp
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SampleRectangle_out.bmp");

using (FileStream stream = new FileStream(dataDir, FileMode.Create))
{
    BmpOptions saveOptions = new BmpOptions();
    saveOptions.BitsPerPixel = 32; // Ορίστε τα bit ανά pixel σε 32 για εικόνες υψηλής ποιότητας
    saveOptions.Source = new StreamSource(stream);
    
    using (Image image = Image.Create(saveOptions, 100, 100)) 
    {
        Graphics graphic = new Graphics(image);
        
        // Καθαρίστε την επιφάνεια με ένα κίτρινο χρώμα φόντου
        graphic.Clear(Color.Yellow);

        // Θα σχεδιάσουμε ορθογώνια στα επόμενα βήματα.
    }
}
```

### Σχεδίαση ορθογωνίων

Θα επικεντρωθούμε τώρα στο να σχεδιάσουμε δύο ορθογώνια διαφορετικών χρωμάτων στην εικόνα μας.

#### Κόκκινο ορθογώνιο

```csharp
// Σχεδιάστε ένα κόκκινο ορθογώνιο στη θέση (30, 10) με πλάτος 40 και ύψος 80
graphic.DrawRectangle(new Pen(Color.Red), new Rectangle(30, 10, 40, 80));
```

Αυτό το απόσπασμα κώδικα δημιουργεί ένα κόκκινο περίγραμμα για το ορθογώνιο. `Pen` Η κλάση καθορίζει το χρώμα και το στυλ.

#### Μπλε γεμισμένο ορθογώνιο

```csharp
// Σχεδιάστε ένα μπλε ορθογώνιο στη θέση (10, 30) με πλάτος 80 και ύψος 40
graphic.DrawRectangle(
    new Pen(new SolidBrush(Color.Blue)),
    new Rectangle(10, 30, 80, 40)
);
```

Εδώ, χρησιμοποιούμε ένα `SolidBrush` για να γεμίσετε το ορθογώνιο με μπλε χρώμα.

### Αποθήκευση της εικόνας

Μόλις ολοκληρωθούν όλα τα σχέδια, αποθηκεύστε τις αλλαγές:

```csharp
image.Save();
```

Αυτό το βήμα εγγράφει την τελική εικόνα στο σύστημα αρχείων όπως καθορίζεται από τη διαδρομή ροής μας.
