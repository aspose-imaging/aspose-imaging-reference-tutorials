## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή SVG σε καμβά HTML5

**Επισκόπηση:**
Αυτή η ενότητα παρουσιάζει τη φόρτωση ενός αρχείου εικόνας SVG και τη μετατροπή του σε μορφή καμβά HTML5 χρησιμοποιώντας το Aspose.Imaging. Η έμφαση δίνεται στη χρήση συγκεκριμένων επιλογών ραστεροποίησης για βέλτιστα αποτελέσματα.

#### Βήμα 1: Φόρτωση του αρχείου SVG

Για να ξεκινήσετε, φορτώστε το αρχείο SVG χρησιμοποιώντας `Image.Load()`Βεβαιωθείτε ότι έχετε καθορίσει τη σωστή διαδρομή καταλόγου:

```csharp
var dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (var image = Image.Load(System.IO.Path.Combine(dataDir, "Sample.svg")))
```

*Γιατί;* Η σωστή φόρτωση της εικόνας είναι ζωτικής σημασίας για την περαιτέρω επεξεργασία της.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών ραστεροποίησης

Στη συνέχεια, διαμορφώστε το `SvgRasterizationOptions`Αυτό το βήμα σάς επιτρέπει να καθορίσετε διαστάσεις όπως το πλάτος και το ύψος της σελίδας:

```csharp
new SvgRasterizationOptions() { PageWidth = 100, PageHeight = 100 }
```

*Γιατί;* Η προσαρμογή αυτών των επιλογών διασφαλίζει ότι το SVG σας ταιριάζει τέλεια μέσα στον καμβά.

#### Βήμα 3: Αποθήκευση ως καμβά HTML5

Τέλος, αποθηκεύστε την εικόνα χρησιμοποιώντας `image.Save()` με `Html5CanvasOptions`:

```csharp
image.Save(
    System.IO.Path.Combine("YOUR_OUTPUT_DIRECTORY", "Sample.html"),
    new Html5CanvasOptions
    {
        VectorRasterizationOptions = 
            new SvgRasterizationOptions() { PageWidth = 100, PageHeight = 100 }
    });
```

*Γιατί;* Αυτό το βήμα μετατρέπει το SVG σας σε ένα στοιχείο καμβά που μπορεί να ενσωματωθεί σε ιστοσελίδες.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι οι διαδρομές καταλόγου είναι σωστές για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".
- Επαληθεύστε ότι η βιβλιοθήκη Aspose.Imaging αναφέρεται σωστά στο έργο σας.
