## Οδηγός Εφαρμογής

### Βήμα 1: Ορισμός διαδρομής καταλόγου εγγράφων

Ξεκινήστε καθορίζοντας τη διαδρομή όπου βρίσκονται τα αρχεία προέλευσης:

```csharp
string dataDir = "/path/to/your/document/directory";
```

Αυτό θέτει τις βάσεις για τη φόρτωση και αποθήκευση αρχείων στα επόμενα βήματα.

### Βήμα 2: Φόρτωση της εικόνας ράστερ

Φορτώστε την εικόνα ράστερ που θέλετε να σχεδιάσετε στον καμβά SVG:

```csharp
using (RasterImage imageToDraw = (RasterImage)Image.Load(dataDir + "/asposenet_220_src01.png"))
{
    // Συνεχίστε με τη φόρτωση του SVG και τις λειτουργίες σχεδίασης εδώ.
}
```

Αυτό το τμήμα κώδικα φορτώνει το αρχείο raster σας, προετοιμάζοντάς το για περαιτέρω χειρισμό.

### Βήμα 3: Φόρτωση της εικόνας SVG

Στη συνέχεια, φορτώστε την εικόνα SVG που θα χρησιμεύσει ως καμβάς σας:

```csharp
using (SvgImage canvasImage = (SvgImage)Image.Load(dataDir + "/asposenet_220_src02.svg"))
{
    // Δημιουργήστε μια παρουσία του SvgGraphics2D για σχεδίαση.
}
```

Αυτό το βήμα ορίζει την διανυσματική επιφάνεια στην οποία θα σχεδιάσετε.

### Βήμα 4: Δημιουργία στιγμιότυπου SvgGraphics2D

Δημιουργία στιγμιαίας εικόνας `SvgGraphics2D` για να εκτελέσετε γραφικές λειτουργίες στο SVG:

```csharp
SvgGraphics2D graphics = new SvgGraphics2D(canvasImage);
```

Εδώ, αποκτάτε πρόσβαση σε διάφορες μεθόδους σχεδίασης για τον καμβά SVG σας.

### Βήμα 5: Σχεδιάστε την εικόνα ράστερ

Σχεδιάστε την φορτωμένη εικόνα ράστερ στο SVG χρησιμοποιώντας καθορισμένα όρια:

```csharp
graphics.DrawImage(
    new Rectangle(0, 0, imageToDraw.Width, imageToDraw.Height),
    new Rectangle(67, 67, imageToDraw.Width, imageToDraw.Height),
    imageToDraw);
```

Τα ορθογώνια προέλευσης και προορισμού διασφαλίζουν ότι η εικόνα σχεδιάζεται χωρίς τέντωμα.

### Βήμα 6: Αποθήκευση του τελικού SVG

Τέλος, αποθηκεύστε το τροποποιημένο αρχείο SVG:

```csharp
using (SvgImage resultImage = graphics.EndRecording())
{
    string outputDir = "/path/to/your/output/directory";
    resultImage.Save(outputDir + "/asposenet_220_src02.DrawImage.svg");
}
```

Αυτό το βήμα ολοκληρώνει και αποθηκεύει τη συνδυασμένη εικόνα.
