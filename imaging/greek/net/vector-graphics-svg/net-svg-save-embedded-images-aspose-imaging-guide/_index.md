## Οδηγός Εφαρμογής

### Αποθήκευση SVG με ενσωματωμένες εικόνες

Αυτή η λειτουργία σάς επιτρέπει να ενσωματώνετε εικόνες απευθείας σε ένα αρχείο SVG κατά την αποθήκευση. Ακολουθήστε αυτά τα βήματα χρησιμοποιώντας το Aspose.Imaging.

#### Βήμα 1: Φόρτωση του αρχείου SVG

Ξεκινήστε φορτώνοντας το αρχείο SVG:
```csharp
using (var image = SvgImage.Load("auto.svg"))
{
    // Συνεχίστε με την ενσωμάτωση εικόνων και την αποθήκευση.
}
```
*Εξήγηση*: Ανοίγουμε `auto.svg` για επεξεργασία. Το `SvgImage` Η κλάση αντιπροσωπεύει ένα αρχείο SVG στο Aspose.Imaging.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών εικόνας

Ρυθμίστε τις απαραίτητες επιλογές για να αποθηκεύσετε το SVG σας με ενσωματωμένους πόρους:
```csharp
var svgOptions = new SvgOptions()
{
    VectorRasterizationOptions = new SvgRasterizationOptions()
    {
        BackgroundColor = Color.White,
        PageWidth = image.Width,
        PageHeight = image.Height
    }
};
```
*Εξήγηση*: Εδώ, ορίζουμε `SvgOptions` που περιλαμβάνουν ρυθμίσεις ραστεροποίησης όπως χρώμα φόντου και διαστάσεις.

#### Βήμα 3: Αποθηκεύστε το SVG με ενσωματωμένες εικόνες

Τέλος, αποθηκεύστε το αρχείο χρησιμοποιώντας τις διαμορφωμένες επιλογές σας:
```csharp
image.Save("auto_with_embedded_images.svg", svgOptions);
```
*Εξήγηση*: Αυτή η γραμμή αποθηκεύει το αρχείο SVG με όλες τις ενσωματωμένες εικόνες που περιλαμβάνονται όπως ορίζεται στο `svgOptions`.
