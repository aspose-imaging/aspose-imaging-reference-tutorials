### Ορισμός φακέλου προσαρμοσμένων γραμματοσειρών

#### Επισκόπηση

**Βήματα Υλοποίησης:**

##### Ορίστε τον κατάλογο εγγράφων και τη διαδρομή γραμματοσειράς

Αρχικά, καθορίστε πού βρίσκονται ο κατάλογος εγγράφων και τα αρχεία γραμματοσειρών.
```csharp
using Aspose.Imaging;
using System.IO;

public static void SetCustomFontsFolder()
{
    string dataDir = "YOUR_DOCUMENT_DIRECTORY/";
    FontSettings.SetFontsFolder(Path.Combine(dataDir, "fonts"));
}
```
- **Παράμετροι:** `dataDir` θα πρέπει να είναι η διαδρομή προς τον κατάλογο εγγράφων σας.
- **Σκοπός:** Αυτή η μέθοδος διασφαλίζει ότι το Aspose.Imaging χρησιμοποιεί μόνο τις γραμματοσειρές που καθορίζετε.

##### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι ο φάκελος γραμματοσειρών υπάρχει και περιέχει αρχεία .ttf ή .otf.
- Επαληθεύστε τα δικαιώματα αρχείων για την εφαρμογή, ώστε να έχει πρόσβαση σε αυτόν τον κατάλογο.

### Εξαγωγή διανυσματικών γραφικών σε PNG

#### Επισκόπηση
Μετατρέψτε διανυσματικά γραφικά (όπως ODP ή SVG) σε μορφή PNG υψηλής ποιότητας χρησιμοποιώντας τις ισχυρές λειτουργίες του Aspose.Imaging.

**Βήματα Υλοποίησης:**

##### Ορισμός προεπιλεγμένης γραμματοσειράς και φόρτωση εγγράφου

Ρυθμίστε την προεπιλεγμένη γραμματοσειρά για την απόδοση και, στη συνέχεια, φορτώστε το έγγραφό σας:
```csharp
using Aspose.Imaging;
using Aspose.Imaging.ImageOptions;
using System.IO;

public static void ExportVectorToPng(string inputFilePath, string defaultFontName, string outputFilePath)
{
    FontSettings.DefaultFontName = defaultFontName;
    
    using (Aspose.Imaging.Image document = Aspose.Imaging.Image.Load(inputFilePath))
    {
        PngOptions saveOptions = new PngOptions();
        saveOptions.VectorRasterizationOptions = new OdgRasterizationOptions()
        {
            PageWidth = 1000,
            PageHeight = 1000
        };
        
        document.Save(outputFilePath, saveOptions);
    }
    
    File.Delete(outputFilePath); // Προαιρετικά: Διαγραφή εξόδου μετά την αποθήκευση
}
```
- **Παράμετροι:**
  - `inputFilePath`: Διαδρομή προς το αρχείο διανυσματικών γραφικών.
  - `defaultFontName`: Η γραμματοσειρά που χρησιμοποιείται για την απόδοση κειμένου στην εικόνα.
  - `outputFilePath`: Πού θα αποθηκευτεί το PNG που προκύπτει.
- **Σκοπός:** Μετατρέπει διανυσματικές μορφές σε εικόνες raster διατηρώντας παράλληλα την ποιότητα και διασφαλίζοντας τη σωστή απόδοση κειμένου με συγκεκριμένες γραμματοσειρές.

##### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι τα αρχεία διανυσμάτων είναι προσβάσιμα και σωστά μορφοποιημένα.
- Προσαρμόζω `PageWidth` και `PageHeight` με βάση τις συγκεκριμένες ανάγκες σας για την κατάλληλη προσαρμογή του περιεχομένου.
