### Εξαγωγή εικόνων ράστερ σε SVG (H2)

#### Επισκόπηση:
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε μορφές εικόνας ράστερ όπως JPEG, PNG, GIF και άλλες σε SVG χρησιμοποιώντας το Aspose.Imaging για .NET. Η μετατροπή διατηρεί άψογα τα διανυσματικά γραφικά υψηλής ποιότητας.

##### Βήμα 1: Ορισμός διαδρομών και φόρτωση εικόνων (H3)
Ξεκινήστε καθορίζοντας τις διαδρομές των εικόνων σας για επεξεργασία.
```csharp
string[] paths = new string[]
{
    "@YOUR_DOCUMENT_DIRECTORY\butterfly.gif",
    "@YOUR_DOCUMENT_DIRECTORY\33715-cmyk.jpeg",
    // Προσθέστε άλλες διαδρομές εικόνας εδώ...
};
```

##### Βήμα 2: Ρύθμιση επιλογών SVG (H3)
Ρυθμίστε τις παραμέτρους για την αποθήκευση εικόνων ως SVG.
```csharp
using Aspose.Imaging.ImageOptions;
using Aspose.Imaging.FileFormats.Svg;

// Αρχικοποίηση ρυθμίσεων SvgOptions και Rasterization
SvgOptions svgOptions = new SvgOptions();
svgOptions.VectorRasterizationOptions = new SvgRasterizationOptions();
```

##### Βήμα 3: Ρύθμιση παραμέτρων διαστάσεων σελίδας (H3)
Βεβαιωθείτε ότι οι διαστάσεις SVG ταιριάζουν με την αρχική σας εικόνα.
```csharp
foreach (string path in paths)
{
    using (Image image = Image.Load(path))
    {
        svgOptions.VectorRasterizationOptions.PageWidth = image.Width;
        svgOptions.VectorRasterizationOptions.PageHeight = image.Height;

        string destPath = "YOUR_OUTPUT_DIRECTORY\" + Path.GetFileNameWithoutExtension(path) + ".svg";
        image.Save(destPath, svgOptions);
    }
}
```

##### Παράμετροι και επιλογές διαμόρφωσης:
- **Επιλογές Svg**: Διαχειρίζεται τον τρόπο αποθήκευσης του αρχείου SVG.
- **Επιλογές SvgRasterization**Καθορίζει τις ρυθμίσεις ραστεροποίησης για τη μετατροπή διανυσμάτων.

### Συμβουλές αντιμετώπισης προβλημάτων
