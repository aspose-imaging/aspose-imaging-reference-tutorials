```java

        String inputFile = "YOUR_DOCUMENT_DIRECTORY/input.png";

        try (RasterImage image = (RasterImage) Image.load(inputFile)) {
            // Σε αυτό το σημείο, η εικόνα είναι έτοιμη για περαιτέρω επεξεργασία.
        }
    }
}
```

**Εξήγηση:**
- `String inputFile`: Αντικατάσταση `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική διαδρομή καταλόγου σας για να καθορίσετε πού βρίσκεται η εικόνα εισόδου σας.
- `try-with-resources` διασφαλίζει ότι οι πόροι κλείνουν αυτόματα μετά τη χρήση.

### Χαρακτηριστικό 2: Αυτόματη κάλυψη κοπής γραφήματος

#### Επισκόπηση
Η αφαίρεση φόντου είναι μια συνηθισμένη εργασία στην επεξεργασία φωτογραφιών. Χρησιμοποιώντας τη μέθοδο Graph Cut, το Aspose.Imaging μπορεί να αυτοματοποιήσει αυτήν τη διαδικασία με εντυπωσιακή ακρίβεια.

**Βήμα προς βήμα εφαρμογή**

**Εισαγωγή απαραίτητων κλάσεων:**
```java
import com.aspose.imaging.Color;
import com.aspose.imaging.Image;
import com.aspose.imaging.RasterImage;
import com.aspose.imaging.imageoptions.PngOptions;
import com.aspose.imaging.fileformats.png.PngColorType;
import com.aspose.imaging.masking.AutoMaskingGraphCutOptions;
import com.aspose.imaging.masking.SegmentationMethod;
import com.aspose.imaging.masking.ImageMasking;
import com.aspose.imaging.masking.result.MaskingResult;
import com.aspose.imaging.sources.FileCreateSource;
```

**Ρύθμιση παραμέτρων και εκτέλεση αυτόματης μάσκας αποκοπής γραφήματος:**
```java
public class RemoveBackgroundGraphCut {
    public static void main(String[] args) {
        // Ορίστε καταλόγους εισόδου και εξόδου.
        String inputFile = "YOUR_DOCUMENT_DIRECTORY/input.png";
        String outDir = "YOUR_OUTPUT_DIRECTORY/";

        try (RasterImage image = (RasterImage) Image.load(inputFile)) {
            AutoMaskingGraphCutOptions options = new AutoMaskingGraphCutOptions();
            
            // Ενεργοποιήστε τον αυτόματο υπολογισμό της διαδρομής κατά την αποσύνθεση.
            options.setCalculateDefaultStrokes(true);

            // Ορίστε την ακτίνα φτερώματος για ομαλές μεταβάσεις στις άκρες.
            options.setFeatheringRadius((Math.max(image.getWidth(), image.getHeight()) / 500) + 1);

            // Καθορίστε τη μέθοδο τμηματοποίησης ως Graph Cut.
            options.setMethod(SegmentationMethod.GraphCut);

            // Απενεργοποιήστε την αποσύνθεση στρώσεων για να διατηρήσετε μία μόνο εικόνα εξόδου.
            options.setDecompose(false);

            // Ορίστε το χρώμα φόντου σε διαφανές για την κάλυψη.
            options.setBackgroundReplacementColor(Color.getTransparent());

            PngOptions pngOptions = new PngOptions();
            pngOptions.setColorType(PngColorType.TruecolorWithAlpha);
            pngOptions.setSource(new FileCreateSource("tempFile"));
            options.setExportOptions(pngOptions);

            MaskingResult results = new ImageMasking(image).decompose(options);

            try (RasterImage resultImage = results.get_Item(1).getImage()) {
                // Αποθηκεύστε την εικόνα με διαφανές φόντο.
                resultImage.save(outDir + "output.png", pngOptions);
            }

            results.close();
        }
    }
}
```

**Εξήγηση:**
- **`AutoMaskingGraphCutOptions`**Διαμορφώνει τις παραμέτρους του αλγορίθμου Graph Cut για βέλτιστη απόδοση και ακρίβεια.
- **Ακτίνα φτερώματος**: Προσαρμόζεται με βάση το μέγεθος της εικόνας για να εξασφαλίζονται ομαλές μεταβάσεις στις άκρες.
- **Επιλογές εξαγωγής**: Ορισμός σε PNG με κανάλι άλφα, επιτρέποντας τη διαφάνεια στην έξοδο.
