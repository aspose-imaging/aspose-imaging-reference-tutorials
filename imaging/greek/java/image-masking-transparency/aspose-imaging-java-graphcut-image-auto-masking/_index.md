## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Αυτόματη μάσκα εικόνας

Αυτή η λειτουργία επιτρέπει την αυτόματη κάλυψη εικόνων χρησιμοποιώντας τη μέθοδο τμηματοποίησης GraphCut του Aspose.Imaging. Δείτε πώς λειτουργεί:

#### Βήμα 1: Αρχικοποίηση διαδρομών και φόρτωση της εικόνας
Αρχικά, ορίστε τη διαδρομή εισόδου της εικόνας και πού θέλετε να αποθηκεύσετε την έξοδο.

```java
String sourceFileName = "YOUR_DOCUMENT_DIRECTORY/Colored by Faith_small.png";
String outputFileName = "YOUR_OUTPUT_DIRECTORY/Colored by Faith_small_auto.png";
```

Φορτώστε την εικόνα σας χρησιμοποιώντας το `Image.load()` μέθοδος:

```java
try (RasterImage image = (RasterImage) Image.load(sourceFileName)) {
    // Περαιτέρω βήματα επεξεργασίας θα γίνουν εδώ.
}
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών μάσκας

Ρυθμίστε τις επιλογές μασκαρίσματος χρησιμοποιώντας το GraphCut ως μέθοδο τμηματοποίησης.

```java
MaskingOptions maskingOptions = new MaskingOptions();
maskingOptions.setMethod(SegmentationMethod.GraphCut);  // Χρησιμοποιήστε το GraphCut για τμηματοποίηση
maskingOptions.setArgs(new AutoMaskingArgs());           // Αρχικοποίηση ορισμάτων αυτόματης απόκρυψης
```

#### Βήμα 3: Ορισμός επιλογών εξαγωγής

Ρυθμίστε τις παραμέτρους των επιλογών εξαγωγής για να χειριστείτε τη διαφάνεια, η οποία είναι κρίσιμη για την απόκρυψη των αποτελεσμάτων.

```java
PngOptions options = new PngOptions();
options.setColorType(PngColorType.TruecolorWithAlpha);   // Ενεργοποίηση καναλιού άλφα για διαφάνεια
maskingOptions.setExportOptions(options);
```

#### Βήμα 4: Αποσύνθεση και αποθήκευση της εικόνας με μάσκα

Τέλος, εφαρμόστε την μάσκα και αποθηκεύστε το αποτέλεσμα.

```java
try (MaskingResult maskingResults = new ImageMasking(image).decompose(maskingOptions)) {
    try (Image resultImage = maskingResults.get_Item(1).getImage()) {
        resultImage.save(outputFileName);
    }
}
```

### Χαρακτηριστικό: Συμπλήρωση σημείων εισόδου για αυτόματη μάσκα

Για να βελτιώσετε περαιτέρω τη διαδικασία αυτόματης απόκρυψης, μπορείτε να καθορίσετε σημεία εισόδου και ορθογώνια που καθοδηγούν την τμηματοποίηση.

```java
private static void fillInputPoints(String filePath, AutoMaskingArgs autoMaskingArgs) throws IOException {
    try (InputStream inputStream = new FileInputStream(filePath)) {
        LEIntegerReader reader = new LEIntegerReader(inputStream);
        
        // Ανάγνωση δεδομένων για τον προσδιορισμό της παρουσίας ορθογωνίων και σημείων αντικειμένων
        boolean hasObjectRectangles = inputStream.read() != 0;
        boolean hasObjectPoints = inputStream.read() != 0;

        autoMaskingArgs.setObjectsRectangles(null);
        autoMaskingArgs.setObjectsPoints(null);

        if (hasObjectRectangles) {
            int len = reader.read();
            Rectangle[] rects = new Rectangle[len];
            for (int i = 0; i < len; i++) {
                rects[i] = new Rectangle(
                    reader.read(), // Χ
                    reader.read(), // Υ
                    reader.read(), // Πλάτος
                    reader.read()  // Υψος
                );
            }
            autoMaskingArgs.setObjectsRectangles(rects);
        }

        if (hasObjectPoints) {
            int len = reader.read();
            Point[][] points = new Point[len][];
            for (int i = 0; i < len; i++) {
                int il = reader.read(); // Αριθμός πόντων
                points[i] = new Point[il];
                for (int j = 0; j < il; j++) {
                    points[i][j] = new Point(
                        reader.read(), // Χ
                        reader.read()  // Υ
                    );
                }
            }
            autoMaskingArgs.setObjectsPoints(points);
        }
    }
}
```

### Χαρακτηριστικό: LEIntegerReader

Αυτή η κλάση βοηθητικού προγράμματος βοηθά στην ανάγνωση ακεραίων σε μορφή little-endian, απαραίτητη για την επεξεργασία αρχείων εισόδου.

```java
class LEIntegerReader {
    private final InputStream stream;
    private final byte[] buffer = new byte[4];

    LEIntegerReader(InputStream stream) {
        this.stream = stream;
    }

    int read() throws IOException {
        int len = stream.read(buffer);
        if (len != 4) {
            throw new RuntimeException("Unexpected EOF");
        }
        return ((buffer[3] & 0xff) << 24) | ((buffer[2] & 0xff) << 16) |
               ((buffer[1] & 0xff) << 8) | (buffer[0] & 0xFF);
    }
}
```
