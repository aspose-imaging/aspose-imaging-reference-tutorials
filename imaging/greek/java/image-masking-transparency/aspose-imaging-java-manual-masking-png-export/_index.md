Αυτό το σεμινάριο θα σας δείξει πώς να αξιοποιήσετε τη δύναμη του **Aspose.Imaging για Java** για να αποκρύψετε χειροκίνητα μια εικόνα χρησιμοποιώντας προσαρμοσμένα σχήματα και να την εξαγάγετε ως PNG με διαφάνεια. ## Οδηγός Εφαρμογής

### Φόρτωση εικόνας

#### Επισκόπηση
Το πρώτο βήμα είναι να φορτώσετε την εικόνα σας σε ένα `RasterImage` αντικείμενο, το οποίο επιτρέπει διάφορους χειρισμούς.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.RasterImage;

String sourceFileName = "YOUR_DOCUMENT_DIRECTORY/Colored by Faith_small.png";
try (RasterImage image = (RasterImage) Image.load(sourceFileName)) {
    // Η εικόνα έχει πλέον φορτωθεί και μπορεί να υποβληθεί σε επεξεργασία.
}
```
Σε αυτό το απόσπασμα κώδικα, εισάγουμε τις απαραίτητες κλάσεις και φορτώνουμε μια εικόνα από μια καθορισμένη διαδρομή. Αυτό την προετοιμάζει για περαιτέρω επεξεργασία.

### Δημιουργία GraphicsPath για μάσκαση

#### Επισκόπηση
Στη συνέχεια, δημιουργήστε προσαρμοσμένα σχήματα για να ορίσετε τη μάσκα σας χρησιμοποιώντας `GraphicsPath`.

```java
import com.aspose.imaging.GraphicsPath;
import com.aspose.imaging.RectangleF;
import com.aspose.imaging.shapes.*;

GraphicsPath manualMask = new GraphicsPath();

Figure firstFigure = new Figure();
firstFigure.addShape(new EllipseShape(new RectangleF(100, 30, 40, 40)));
firstFigure.addShape(new RectangleShape(new RectangleF(10, 200, 50, 30)));
manualMask.addFigure(firstFigure);

GraphicsPath subPath = new GraphicsPath();
Figure secondFigure = new Figure();
secondFigure.addShape(
    new PolygonShape(new PointF[]{
        new PointF(310, 100), new PointF(350, 200), new PointF(250, 200)
    }, true));
secondFigure.addShape(new PieShape(new RectangleF(10, 10, 80, 80), 30, 120));
subPath.addFigure(secondFigure);
manualMask.addPath(subPath);
```
Αυτή η ενότητα εξηγεί πώς να ορίσετε περίπλοκα σχήματα όπως ελλείψεις, ορθογώνια, πολύγωνα και πίτες για ακριβή κάλυψη εικόνας.

### Ρύθμιση παραμέτρων επιλογών κάλυψης

#### Επισκόπηση
Ρυθμίστε τις επιλογές μασκαρίσματος για να εφαρμόσετε την προσαρμοσμένη διαδρομή γραφικών σας.

```java
import com.aspose.imaging.masking.*;
import com.aspose.imaging.masking.options.*;

MaskingOptions maskingOptions = new MaskingOptions();
maskingOptions.setMethod(SegmentationMethod.Manual);
maskingOptions.setDecompose(false);

ManualMaskingArgs argus = new ManualMaskingArgs();
argus.setMask(manualMask);
maskingOptions.setArgs(argus);
```
Εδώ, διαμορφώνουμε `MaskingOptions` για να χρησιμοποιήσετε μια μέθοδο χειροκίνητης τμηματοποίησης με την προηγουμένως δημιουργημένη διαδρομή γραφικών.

### Εξαγωγή εικόνας με μάσκα με PngOptions

#### Επισκόπηση
Τέλος, εξαγάγετε την εικόνα με μάσκα ως αρχείο PNG χρησιμοποιώντας τις επιλογές για προχωρημένους.

```java
import com.aspose.imaging.imageoptions.PngOptions;
import com.aspose.imaging.fileformats.png.PngColorType;
import com.aspose.imaging.sources.StreamSource;

String outputFileName = "YOUR_OUTPUT_DIRECTORY/Colored by Faith_small_manual.png";
PngOptions options = new PngOptions();
options.setColorType(PngColorType.TruecolorWithAlpha);
options.setSource(new StreamSource());
maskingOptions.setExportOptions(options);

try (MaskingResult maskingResults = new ImageMasking(image).decompose(maskingOptions)) {
    try (Image resultImage = maskingResults.get_Item(1).getImage()) {
        // Αποθηκεύει την εικόνα με μάσκα σε μια καθορισμένη διαδρομή εξόδου.
        resultImage.save(outputFileName);
    }
}
```
Αυτό το τμήμα επισημαίνει τον τρόπο διαμόρφωσης `PngOptions` για εξαγωγή με διαφάνεια και αποθήκευση της τελικής εικόνας.
