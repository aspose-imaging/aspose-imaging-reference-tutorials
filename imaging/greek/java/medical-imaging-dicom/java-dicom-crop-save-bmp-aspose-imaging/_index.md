## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση εικόνας DICOM

**Επισκόπηση:**  
Η φόρτωση μιας εικόνας DICOM είναι το πρώτο βήμα. Αυτή η λειτουργία δείχνει πώς να φορτώσετε μια εικόνα σε μια παρουσία του `DicomImage`.

#### Βήμα προς βήμα:

##### Εισαγωγή απαραίτητων πακέτων
```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.dicom.DicomImage;
```

##### Φόρτωση της εικόνας DICOM
Δημιουργήστε μια κλάση και μια μέθοδο για να χειριστείτε τη φόρτωση.

```java
public class LoadDicomImage {
    public static void main(String[] args) {
        String inputFile = "YOUR_DOCUMENT_DIRECTORY/image.dcm";

        try (DicomImage dicomImage = (DicomImage) Image.load(inputFile)) {
            // Η εικόνα έχει πλέον φορτωθεί και είναι έτοιμη για επεξεργασία.
        } catch (Exception e) {
            System.err.println("Error loading the DICOM image: " + e.getMessage());
        }
    }
}
```

**Εξήγηση:**  
- `Image.load()` διαβάζει το αρχείο από την καθορισμένη διαδρομή. Βεβαιωθείτε ότι η διαδρομή προς το αρχείο DICOM είναι σωστή, διαφορετικά θα αντιμετωπίσετε σφάλματα.
- Η πρόταση try-with-resources διασφαλίζει ότι `DicomImage` Το αντικείμενο κλείνει μετά τη χρήση.

### Λειτουργία 2: Περικοπή εικόνας DICOM με Shifts

**Επισκόπηση:**  
Η περικοπή περιλαμβάνει την προσαρμογή των διαστάσεων μιας εικόνας. Αυτή η λειτουργία επιδεικνύει την περικοπή χρησιμοποιώντας συγκεκριμένες τιμές μετατόπισης για κάθε πλευρά της εικόνας.

#### Βήμα προς βήμα:

##### Εισαγωγή απαραίτητων πακέτων
```java
import com.aspose.imaging.fileformats.dicom.DicomImage;
```

##### Περικοπή της εικόνας
Ορίστε μια κλάση για να εκτελέσετε τη λειτουργία περικοπής.

```java
public class CropDicomImage {
    public static void main(String[] args) {
        String inputFile = "YOUR_DOCUMENT_DIRECTORY/image.dcm";

        try (DicomImage dicomImage = (DicomImage) Image.load(inputFile)) {
            int leftShift = 10;
            int topShift = 20;
            int rightShift = 30;
            int bottomShift = 40;

            dicomImage.crop(leftShift, topShift, rightShift, bottomShift);
        } catch (Exception e) {
            System.err.println("Error cropping the DICOM image: " + e.getMessage());
        }
    }
}
```

**Εξήγηση:**  
- Ο `crop()` Η μέθοδος χρησιμοποιεί τιμές μετατόπισης για να ορίσει πόσο από κάθε πλευρά αφαιρείται. Προσαρμόστε τις τιμές αυτές ανάλογα με τις ανάγκες σας.
- Οι αρνητικές ή υπερβολικές τιμές μετατόπισης ενδέχεται να προκαλέσουν σφάλματα, επομένως βεβαιωθείτε ότι βρίσκονται εντός των διαστάσεων της εικόνας.

### Λειτουργία 3: Αποθήκευση περικομμένης εικόνας DICOM ως BMP

**Επισκόπηση:**  
Μόλις περικοπεί, μπορείτε να αποθηκεύσετε την εικόνα σε διαφορετικές μορφές, όπως BMP, για ευρύτερη συμβατότητα.

#### Βήμα προς βήμα:

##### Εισαγωγή απαραίτητων πακέτων
```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.dicom.DicomImage;
import com.aspose.imaging.imageoptions.BmpOptions;
```

##### Αποθήκευση της περικομμένης εικόνας
Δημιουργήστε μια κλάση για να χειριστείτε την αποθήκευση.

```java
public class SaveCroppedDicomAsBmp {
    public static void main(String[] args) {
        String inputFile = "YOUR_DOCUMENT_DIRECTORY/image.dcm";
        String outputFile = "YOUR_OUTPUT_DIRECTORY/CropbyShifts_out.bmp";

        try (DicomImage dicomImage = (DicomImage) Image.load(inputFile)) {
            int leftShift = 10;
            int topShift = 20;
            int rightShift = 30;
            int bottomShift = 40;

            dicomImage.crop(leftShift, topShift, rightShift, bottomShift);
            dicomImage.save(outputFile, new BmpOptions());
        } catch (Exception e) {
            System.err.println("Error saving the DICOM image: " + e.getMessage());
        }
    }
}
```

**Εξήγηση:**  
- Ο `save()` Η μέθοδος γράφει την επεξεργασμένη εικόνα σε ένα αρχείο BMP χρησιμοποιώντας `BmpOptions`.
- Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει ή χειριστείτε πιθανές εξαιρέσεις που σχετίζονται με την εγγραφή αρχείων.
