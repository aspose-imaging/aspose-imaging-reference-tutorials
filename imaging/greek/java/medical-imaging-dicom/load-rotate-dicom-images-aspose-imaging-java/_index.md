Για να ξεκινήσετε **Aspose.Imaging**, μπορείτε να το προσθέσετε ως εξάρτηση χρησιμοποιώντας το Maven ή το Gradle. Εναλλακτικά, κατεβάστε τη βιβλιοθήκη απευθείας από την επίσημη ιστοσελίδα τους.

**Maven:**
```xml
<dependency>

</dependency>
```

**Βαθμός:**


## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να φορτώνουμε μια εικόνα DICOM, να την περιστρέφουμε και να την αποθηκεύουμε ως αρχείο BMP χρησιμοποιώντας το Aspose.Imaging για Java. 

### Φόρτωση και περιστροφή εικόνας DICOM

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο εικόνας DICOM στην εφαρμογή σας, να το περιστρέψετε κατά μια καθορισμένη γωνία και να αποθηκεύσετε την εικόνα που προκύπτει σε μορφή BMP.

**1. Εισαγωγή απαιτούμενων πακέτων**

Ξεκινήστε εισάγοντας τις απαραίτητες κλάσεις από τη βιβλιοθήκη Aspose.Imaging:

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.dicom.DicomImage;
import com.aspose.imaging.imageoptions.BmpOptions;
```

**2. Ορισμός διαδρομών αρχείων**

Ορίστε τις διαδρομές για τα αρχεία εισόδου και εξόδου:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String inputFile = dataDir + "image.dcm"; // Διαδρομή προς το αρχείο εικόνας DICOM
String outputFile = "YOUR_OUTPUT_DIRECTORY" + "RotateDICOMImage_out.bmp"; // Διαδρομή για το αρχείο BMP εξόδου
```

**3. Φορτώστε την εικόνα DICOM**

Φορτώστε την εικόνα DICOM χρησιμοποιώντας το Aspose.Imaging `Image.load` μέθοδος:

```java
try (DicomImage image = (DicomImage) Image.load(inputFile)) {
    // Προχωρήστε στην περιστροφή και αποθηκεύστε την εικόνα
}
```

#### Περιστροφή της εικόνας

Περιστρέψτε την εικόνα DICOM που έχει φορτωθεί κατά μια συγκεκριμένη γωνία, όπως 10 μοίρες:

```java
image.rotate(10);
```

**4. Αποθηκεύστε την περιστρεφόμενη εικόνα**

Τέλος, αποθηκεύστε την περιστρεφόμενη εικόνα σε μορφή BMP χρησιμοποιώντας `BmpOptions`:

```java
image.save(outputFile, new BmpOptions());
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Η εικόνα δεν φορτώνει**Βεβαιωθείτε ότι η διαδρομή του αρχείου DICOM είναι σωστή και προσβάσιμη.
- **Σφάλματα περιστροφής**Επαληθεύστε ότι η γωνία που καθορίζεται για την περιστροφή είναι έγκυρη (π.χ., εντός εύρους 0-360 μοιρών).

Με αυτά τα βήματα, θα πρέπει να μπορείτε να περιστρέψετε οποιαδήποτε εικόνα DICOM χρησιμοποιώντας το Aspose.Imaging για Java.
