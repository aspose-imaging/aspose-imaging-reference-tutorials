## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε πώς να ρυθμίσουμε τη φωτεινότητα μιας εικόνας DICOM χρησιμοποιώντας το Aspose.Imaging για Java.

### Ρύθμιση φωτεινότητας μιας εικόνας DICOM

Αυτή η ενότητα σας καθοδηγεί στη φόρτωση μιας εικόνας DICOM και στη ρύθμιση της φωτεινότητάς της τροποποιώντας τις τιμές των pixel.

#### Φόρτωση της εικόνας DICOM
Αρχικά, φορτώστε το αρχείο DICOM σε μια παρουσία του `DicomImage`.

```java
// Ορίστε διαδρομές για αρχεία εισόδου και εξόδου χρησιμοποιώντας placeholders.
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
String inputFile = dataDir + "image.dcm"; // Διαδρομή προς την εικόνα DICOM σας

try (DicomImage image = (DicomImage) Image.load(inputFile)) {
    // Η εικόνα DICOM φορτώθηκε με επιτυχία
}
```

#### Προσαρμογή φωτεινότητας
Προσαρμόστε τη φωτεινότητα κατά συντελεστή 50. Οι θετικές τιμές αυξάνονται, ενώ οι αρνητικές τιμές μειώνουν τη φωτεινότητα.

```java
// Προσαρμόστε τη φωτεινότητα της εικόνας που έχει φορτωθεί.
image.adjustBrightness(50);
```

#### Αποθήκευση ως αρχείο BMP
Τέλος, αποθηκεύστε την προσαρμοσμένη εικόνα σας σε μορφή BMP χρησιμοποιώντας `BmpOptions`.

```java
String outputFile = "YOUR_OUTPUT_DIRECTORY/AdjustingBrightness_out.bmp"; // Διαδρομή αρχείου BMP εξόδου
image.save(outputFile, new BmpOptions());
```
