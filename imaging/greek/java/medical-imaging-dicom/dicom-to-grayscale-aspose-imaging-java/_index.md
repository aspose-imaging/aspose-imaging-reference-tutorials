## Οδηγός Εφαρμογής

### Αλλαγή κλίμακας γκρι σε μια εικόνα DICOM

Σε αυτήν την ενότητα, θα επικεντρωθούμε στη μετατροπή μιας εικόνας DICOM σε κλίμακα του γκρι. Το Aspose.Imaging απλοποιεί αυτήν την εργασία με το εύχρηστο API του.

#### Βήμα 1: Ορισμός διαδρομών εισόδου και εξόδου
Αρχικά, καθορίστε τις διαδρομές για το αρχείο DICOM εισόδου σας και πού θέλετε να αποθηκεύσετε το αποτέλεσμα:

```java
String inputFile = "YOUR_DOCUMENT_DIRECTORY/image.dcm"; // Αντικαταστήστε με τη διαδρομή εικόνας DICOM
String outputFile = "YOUR_OUTPUT_DIRECTORY/Grayscaling_out.bmp"; // Έξοδος αρχείου BMP σε κλίμακα του γκρι
```

#### Βήμα 2: Φόρτωση της εικόνας DICOM
Χρήση `DicomImage` για να φορτώσετε την εικόνα σας. Η πρόταση try-with-resources διασφαλίζει ότι οι πόροι διαχειρίζονται αυτόματα:

```java
try (DicomImage image = (DicomImage) Image.load(inputFile)) {
    // Συνέχεια στην επεξεργασία
}
```
Αυτό το βήμα αρχικοποιεί ένα `DicomImage` αντικείμενο, το οποίο αντιπροσωπεύει το αρχείο DICOM.

#### Βήμα 3: Μετατροπή σε κλίμακα του γκρι
Μετατρέψτε την εικόνα που έχετε φορτώσει σε κλίμακα του γκρι χρησιμοποιώντας:

```java
image.grayscale();
```

Αυτή η μέθοδος τροποποιεί τα εσωτερικά δεδομένα της εικόνας για να την αναπαραστήσει σε κλίμακα του γκρι, απορρίπτοντας τις πληροφορίες χρώματος και εστιάζοντας στις τιμές έντασης.

#### Βήμα 4: Αποθήκευση της εικόνας
Τέλος, αποθηκεύστε την τροποποιημένη εικόνα σε μορφή BMP:

```java
image.save(outputFile, new BmpOptions());
```
Ο `BmpOptions` Η κλάση χρησιμοποιείται εδώ για να καθορίσει ότι αποθηκεύουμε την έξοδο ως αρχείο bitmap. Μπορείτε να επιλέξετε άλλες μορφές που υποστηρίζονται από το Aspose.Imaging, εάν χρειάζεται.
