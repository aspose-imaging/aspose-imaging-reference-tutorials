Αυτό το σεμινάριο θα σας καθοδηγήσει στη χρήση της ισχυρής βιβλιοθήκης Aspose.Imaging για Java για να δημιουργήσετε περίπλοκα γραφικά χωρίς κόπο.

- Μέθοδοι σχεδίασης σχημάτων και μοτίβων χρησιμοποιώντας `HatchBrush`
- Προηγμένες επιλογές διαμόρφωσης για συνδέσεις γραμμών και λειτουργίες φόντου

Ας δούμε τις προϋποθέσεις πριν ξεκινήσουμε.

## Οδηγός Εφαρμογής

### Σχεδίαση γραμμών με το EmfRecorderGraphics2D

Αυτή η ενότητα καλύπτει τα βασικά της σχεδίασης γραμμών χρησιμοποιώντας `EmfRecorderGraphics2D`, επιτρέποντάς σας να προσαρμόσετε το χρώμα της γραμμής, το πάχος και τα στυλ των καπακιών.

#### Βήμα 1: Αρχικοποίηση αντικειμένου γραφικών
Δημιουργήστε μια παρουσία του `EmfRecorderGraphics2D` για να στήσετε τον καμβά σας:

```java
EmfRecorderGraphics2D graphics = new EmfRecorderGraphics2D(new Rectangle(0, 0, 1000, 1000),
        new Size(1000, 1000), new Size(100, 100));
```

#### Βήμα 2: Σχεδιάστε μια βασική γραμμή

Χρησιμοποιήστε το `Pen` κλάση για τον ορισμό ιδιοτήτων γραμμής:

```java
// Δημιουργήστε ένα στυλό με χρώμα Bisque και σχεδιάστε μια γραμμή.
Pen pen = new Pen(Color.getBisque());
graphics.drawLine(pen, 1, 1, 50, 50);
```

#### Βήμα 3: Προσαρμογή στυλ πένας

Αλλάξτε το χρώμα, το πάχος και το στυλ του καπακιού της πένας για να προσθέσετε ποικιλία:

```java
// Ρυθμίστε την πένα σε BlueViolet με πάχος 3 και στρογγυλό καπάκι.
pen = new Pen(Color.getBlueViolet(), 3);
pen.setEndCap(LineCap.Round);
graphics.drawLine(pen, 15, 5, 50, 60);

// Αλλάξτε το τελικό καπάκι σε τετράγωνο και σχεδιάστε μια άλλη γραμμή.
pen.setEndCap(LineCap.Square);
graphics.drawLine(pen, 5, 10, 50, 10);

// Χρησιμοποιήστε ένα στυλ καπακιού με επίπεδο άκρο.
pen.setEndCap(LineCap.Flat);
graphics.drawLine(pen, new Point(5, 20), new Point(50, 20));
```

### Σχεδίαση σχημάτων με HatchBrush

Εξερευνήστε τη σχεδίαση ορθογωνίων χρησιμοποιώντας `HatchBrush` για να γεμίσετε μοτίβα και να προσαρμόσετε τις λειτουργίες φόντου.

#### Βήμα 1: Δημιουργήστε ένα HatchBrush
Ορίστε το στυλ διαγράμμισης για γεμίσματα με μοτίβο:

```java
// Ρυθμίστε ένα HatchBrush με μοτίβο Cross.
HatchBrush hatchBrush = new HatchBrush();
hatchBrush.setBackgroundColor(Color.getAliceBlue());
hatchBrush.setForegroundColor(Color.getRed());
hatchBrush.setHatchStyle(HatchStyle.Cross);
```

#### Βήμα 2: Σχεδιάστε ένα ορθογώνιο με μοτίβο

Χρησιμοποιήστε το `Pen` αντικείμενο για σχεδίαση ορθογωνίων με καθορισμένα μοτίβα:

```java
pen = new Pen(hatchBrush, 7);
graphics.drawRectangle(pen, 50, 50, 20, 30);

// Ορίστε τη λειτουργία φόντου σε ΑΔΙΑΦΑΝΗ για να σχεδιάσετε μια άλλη γραμμή.
graphics.setBackgroundMode(EmfBackgroundMode.OPAQUE);
graphics.drawLine(pen, 80, 50, 80, 80);
```

### Σχεδίαση πολυγώνων και ορθογωνίων με στυλ σύνδεσης γραμμών

Αυτή η λειτουργία δείχνει πώς να σχεδιάζετε πολύγωνα και ορθογώνια χρησιμοποιώντας διαφορετικά `LineJoin` στυλ.

#### Βήμα 1: Ρύθμιση παραμέτρων πένας για πολύγωνο
Ορίστε το στυλ ένωσης γραμμών της πένας για τη σχεδίαση ενός πολυγώνου:

```java
// Ρυθμίστε την πένα σε χρώμα Aqua με τη σύνδεση MiterClipped.
pen = new Pen(new SolidBrush(Color.getAqua()), 3);
pen.setLineJoin(LineJoin.MiterClipped);
graphics.drawPolygon(pen, new Point[]{new Point(10, 20), new Point(12, 45),
        new Point(22, 48), new Point(48, 36), new Point(30, 55)});
```

#### Βήμα 2: Σχεδιάστε ορθογώνια με διαφορετικές ενώσεις γραμμών

Πειραματιστείτε με διάφορα στυλ σύνδεσης:

```java
// Αλλάξτε σε Bevel join και σχεδιάστε ένα ορθογώνιο.
pen.setLineJoin(LineJoin.Bevel);
graphics.drawRectangle(pen, 50, 10, 10, 5);

// Ορίστε την επιλογή Στρογγυλή ένωση για ένα άλλο ορθογώνιο.
pen.setLineJoin(LineJoin.Round);
graphics.drawRectangle(pen, 65, 10, 10, 5);

// Χρησιμοποιήστε την ένωση Miter για το τελικό ορθογώνιο.
pen.setLineJoin(LineJoin.Miter);
graphics.drawRectangle(pen, 80, 10, 10, 5);
```

### Οριστικοποίηση και αποθήκευση των γραφικών σας

Ολοκληρώστε την συνεδρία σχεδίασης αποθηκεύοντας τα γραφικά ως αρχείο EMF ή PDF:

```java
try (EmfImage image = graphics.endRecording()) {
    PdfOptions options = new PdfOptions();
    EmfRasterizationOptions rasterizationOptions = new EmfRasterizationOptions();
    options.setVectorRasterizationOptions(rasterizationOptions);
    rasterizationOptions.setPageSize(Size.to_SizeF(image.getSize()));
    
    String outPath = "YOUR_OUTPUT_DIRECTORY/Picture1.emf.pdf";
    image.save(outPath, options);
}
```
