## Οδηγός Εφαρμογής

Ας χωρίσουμε τη διαδικασία σε διαχειρίσιμα βήματα:

### Ορισμός ιδιοτήτων bitmap

#### Επισκόπηση
Αρχικά, πρέπει να ορίσουμε ιδιότητες bitmap, όπως bits ανά pixel. Αυτό το βήμα είναι κρίσιμο για τον καθορισμό του τρόπου με τον οποίο αποδίδονται και αποθηκεύονται οι εικόνες σας.

#### Υλοποίηση κώδικα
```java
import com.aspose.imaging.imageoptions.BmpOptions;
import java.io.ByteArrayInputStream;

// Ορίστε τον αριθμό των bit ανά pixel σε 32
try (BmpOptions bmpCreateOptions = new BmpOptions()) {
    bmpCreateOptions.setBitsPerPixel(32);

    // Ορίστε μια πηγή ροής πίνακα byte για επιλογές BMP, προσομοιώνοντας ένα μέγεθος εικόνας
    bmpCreateOptions.setSource(new com.aspose.imaging.sources.StreamSource(
        new ByteArrayInputStream(new byte[100 * 100 * 4])));
}
```

**Εξήγηση:**
- `BmpOptions`Διαμορφώνει ρυθμίσεις ειδικά για εικόνες BMP.
- `setBitsPerPixel(32)`: Ορίζει το βάθος χρώματος, επιτρέποντας πλούσια χρωματική απεικόνιση.
- `ByteArrayInputStream`: Προετοιμάζει μια ροή δεδομένων ψεύτικης εικόνας για σκοπούς επίδειξης.

### Δημιουργία και Χειρισμός Εικόνας

#### Επισκόπηση
Στη συνέχεια, θα δημιουργήσουμε μια πραγματική εικόνα και θα χρησιμοποιήσουμε τις δυνατότητες γραφικών του Aspose.Imaging για να σχεδιάσουμε πάνω σε αυτήν. Αυτή η ενότητα δείχνει τη δημιουργία μιας εικόνας, την αρχικοποίηση γραφικών αντικειμένων και τη σχεδίαση σχημάτων όπως τόξα.

#### Υλοποίηση κώδικα
```java
import com.aspose.imaging.Color;
import com.aspose.imaging.Graphics;
import com.aspose.imaging.Image;
import com.aspose.imaging.Pen;

// Αρχικοποίηση των επιλογών BMP για τη δημιουργία εικόνας
try (BmpOptions bmpCreateOptions = new BmpOptions()) {
    bmpCreateOptions.setBitsPerPixel(32);
    bmpCreateOptions.setSource(new com.aspose.imaging.sources.StreamSource(
        new ByteArrayInputStream(new byte[100 * 100 * 4])));

    // Δημιουργήστε την εικόνα με καθορισμένες διαστάσεις
    try (Image image = Image.create(bmpCreateOptions, 100, 100)) {
        Graphics graphic = new Graphics(image);
        
        // Καθαρίστε την επιφάνεια με ένα κίτρινο χρώμα
        graphic.clear(Color.getYellow());
        
        // Ορισμός ιδιοτήτων για τη σχεδίαση ενός τόξου
        int width = 100;
        int height = 200;
        int startAngle = 45;
        int sweepAngle = 270;
        Pen pen = new Pen(Color.getBlack());

        // Σχεδιάστε το τόξο στην επιφάνεια της εικόνας
        graphic.drawArc(pen, 0, 0, width, height, startAngle, sweepAngle);
        
        // Αποθηκεύστε την τελική εικόνα στην επιθυμητή θέση
        image.save("YOUR_OUTPUT_DIRECTORY/DrawingArc_out.bmp");
    }
}
```

**Εξήγηση:**
- `Graphics`: Επιτρέπει τον χειρισμό εικόνων σχεδιάζοντας σχήματα.
- `clear(Color.getYellow())`: Γεμίζει το φόντο της εικόνας με κίτρινο, προετοιμάζοντας το σκηνικό για περαιτέρω σχέδια.
- `drawArc()`: Σχεδιάζει ένα τόξο με καθορισμένες διαστάσεις και γωνίες.
