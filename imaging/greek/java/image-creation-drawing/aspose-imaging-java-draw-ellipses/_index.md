## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: σχεδίαση ελλείψεων με διακεκομμένα και συνεχή περιγράμματα.

### Χαρακτηριστικό 1: Σχεδίαση έλλειψης με διάστικτο περίγραμμα

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να σχεδιάσετε μια έλλειψη με διακεκομμένο περίγραμμα, προσθέτοντας ένα μοναδικό στυλιστικό στοιχείο στα γραφικά σας.

#### Βήματα Υλοποίησης

**Βήμα 1: Ρύθμιση παραμέτρων επιλογών BMP**

Ξεκινήστε δημιουργώντας μια παρουσία του `BmpOptions` και ορίζοντας τις ιδιότητές του:
```java
try (BmpOptions bmpCreateOptions = new BmpOptions()) {
    bmpCreateOptions.setBitsPerPixel(32);
    bmpCreateOptions.setSource(new StreamSource(new java.io.ByteArrayInputStream(new byte[100 * 100 * 4])));
```

**Βήμα 2: Δημιουργία και αρχικοποίηση εικόνας**

Χρησιμοποιήστε το `bmpCreateOptions` για να δημιουργήσετε μια παρουσία εικόνας:
```java
    try (Image image = Image.create(bmpCreateOptions, 100, 100)) {
        Graphics graphic = new Graphics(image);
        graphic.clear(Color.getYellow());
```

**Βήμα 3: Σχεδιάστε μια διακεκομμένη έλλειψη**

Ορίστε ένα στυλό για το διακεκομμένο περίγραμμα και σχεδιάστε την έλλειψη:
```java
        Pen dottedPen = new Pen(Color.getRed(), 1);
        dottedPen.setDashStyle(Pen.DashStyle.Dot);
        graphic.drawEllipse(dottedPen, new Rectangle(30, 10, 40, 80));
```

**Βήμα 4: Αποθήκευση εικόνας**

Τέλος, αποθηκεύστε την εικόνα σας στον καθορισμένο κατάλογο:
```java
        image.save("YOUR_OUTPUT_DIRECTORY/DrawingDottedEllipse_out.bmp");
    }
}
```
*Σημείωμα*: Βεβαιωθείτε `YOUR_OUTPUT_DIRECTORY` έχει οριστεί σωστά στο σημείο όπου θέλετε να αποθηκεύσετε το αρχείο.

### Χαρακτηριστικό 2: Σχεδίαση έλλειψης με συνεχές περίγραμμα

#### Επισκόπηση
Η δημιουργία ελλείψεων με συνεχή περιγράμματα προσφέρει μια πιο καθαρή εμφάνιση, ιδανική για εφαρμογές που απαιτούν σαφείς ορισμούς ορίων.

#### Βήματα Υλοποίησης

**Βήμα 1: Ρύθμιση παραμέτρων επιλογών BMP**

Όπως και πριν, ξεκινήστε ρυθμίζοντας `BmpOptions`:
```java
try (BmpOptions bmpCreateOptions = new BmpOptions()) {
    bmpCreateOptions.setBitsPerPixel(32);
    bmpCreateOptions.setSource(new StreamSource(new java.io.ByteArrayInputStream(new byte[100 * 100 * 4])));
```

**Βήμα 2: Δημιουργία και αρχικοποίηση εικόνας**

Δημιουργήστε την εικόνα χρησιμοποιώντας τις επιλογές:
```java
    try (Image image = Image.create(bmpCreateOptions, 100, 100)) {
        Graphics graphic = new Graphics(image);
        graphic.clear(Color.getYellow());
```

**Βήμα 3: Σχεδιάστε συνεχή έλλειψη**

Στήστε ένα στυλό με ένα συμπαγές πινέλο και σχεδιάστε την έλλειψη:
```java
        Pen solidPen = new Pen(new SolidBrush(Color.getBlue()), 1);
        graphic.drawEllipse(solidPen, new Rectangle(10, 30, 80, 40));
```

**Βήμα 4: Αποθήκευση εικόνας**

Αποθηκεύστε την τελική εικόνα σας:
```java
        image.save("YOUR_OUTPUT_DIRECTORY/DrawingContinuousEllipse_out.bmp");
    }
}
```
*Σημείωμα*: Προσαρμογή `YOUR_OUTPUT_DIRECTORY` όπως απαιτείται.
