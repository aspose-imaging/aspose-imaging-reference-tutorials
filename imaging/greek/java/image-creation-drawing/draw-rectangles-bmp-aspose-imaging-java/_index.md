## Οδηγός Εφαρμογής

Αυτή η ενότητα χωρίζεται σε χαρακτηριστικά, παρέχοντας λεπτομερή βήματα για κάθε μέρος της διαδικασίας υλοποίησης.

### Δημιουργία BmpOptions και Ορισμός Ιδιοτήτων

**Επισκόπηση:**
Ο `BmpOptions` Η κλάση σάς επιτρέπει να ρυθμίσετε διάφορες παραμέτρους για τη δημιουργία εικόνας BMP. Δείτε πώς μπορείτε να ορίσετε τις ιδιότητές της:

1. **Αρχικοποίηση `BmpOptions`:**

   ```java
   import com.aspose.imaging.imageoptions.BmpOptions;
   import java.io.ByteArrayInputStream;

   try (BmpOptions bmpCreateOptions = new BmpOptions()) {
       // Ορίστε τα bit ανά pixel για την εικόνα
       bmpCreateOptions.setBitsPerPixel(32);

       // Ορίστε μια πηγή χρησιμοποιώντας μια ροή εισόδου πίνακα byte
       bmpCreateOptions.setSource(new StreamSource(new ByteArrayInputStream(
           new byte[100 * 100 * 4])));
   }
   ```

   **Εξήγηση:**
   - `setBitsPerPixel(32)`Ρυθμίζει την εικόνα ώστε να χρησιμοποιεί 32 bit ανά pixel, επιτρέποντας βάθος χρώματος υψηλής ποιότητας.
   - Το μέγεθος του πίνακα byte `[100 * 100 * 4]` υπολογίζει τα απαιτούμενα byte για μια εικόνα 100x100 με 32 bit (4 byte) ανά pixel.

### Δημιουργία εικόνας και σχεδίαση ορθογωνίων

**Επισκόπηση:**
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε μια εικόνα και να σχεδιάσετε ορθογώνια χρησιμοποιώντας διαφορετικά στυλ.

1. **Δημιουργήστε μια παρουσία εικόνας:**

   ```java
   import com.aspose.imaging.Image;
   import com.aspose.imaging.Graphics;
   import com.aspose.imaging.Color;
   import com.aspose.imaging.Rectangle;
   import com.aspose.imaging.Pen;
   import com.aspose.imaging.brushes.SolidBrush;

   try (BmpOptions bmpCreateOptions = new BmpOptions()) {
       bmpCreateOptions.setBitsPerPixel(32);
       bmpCreateOptions.setSource(new StreamSource(new ByteArrayInputStream(
           new byte[100 * 100 * 4])));

       // Δημιουργήστε μια εικόνα μεγέθους 100x100 χρησιμοποιώντας τις διαμορφωμένες επιλογές
       try (Image image = Image.create(bmpCreateOptions, 100, 100)) {
           Graphics graphic = new Graphics(image);
           graphic.clear(Color.getYellow());

           // Σχεδιάστε ένα ορθογώνιο με διακεκομμένες γραμμές με κόκκινο στυλό
           Pen redPen = new Pen(Color.getRed());
           Rectangle rect1 = new Rectangle(30, 10, 40, 80);
           graphic.drawRectangle(redPen, rect1);

           // Σχεδιάστε ένα συνεχές ορθογώνιο χρησιμοποιώντας ένα μπλε συμπαγές πινέλο
           Pen bluePen = new Pen(new SolidBrush(Color.getBlue()));
           Rectangle rect2 = new Rectangle(10, 30, 80, 40);
           graphic.drawRectangle(bluePen, rect2);

           image.save("YOUR_OUTPUT_DIRECTORY/DrawingRectangle_out.bmp");
       }
   }
   ```

   **Εξήγηση:**
   - `Image.create(bmpCreateOptions, 100, 100)`: Αρχικοποιεί μια νέα εικόνα με καθορισμένες διαστάσεις και επιλογές.
   - `Graphics` αντικείμενο: Χρησιμοποιείται για σχεδίαση στην επιφάνεια της εικόνας. 
   - `drawRectangle()`: Σχεδιάζει ορθογώνια χρησιμοποιώντας συγκεκριμένα στυλό και πινέλα.
