## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία σε δύο κύριες λειτουργίες: τον ορισμό επιλογών BMP και τη δημιουργία/τροποποίηση μιας εικόνας. Ας εξερευνήσουμε κάθε λειτουργία βήμα προς βήμα.

### Λειτουργία 1: Ρύθμιση επιλογών Bmp

Αυτή η ενότητα παρουσιάζει τον τρόπο διαμόρφωσης `BmpOptions` για βελτιωμένη ποιότητα εικόνας ορίζοντας συγκεκριμένες ιδιότητες όπως bit ανά pixel.

#### Βήμα 1: Αρχικοποίηση BmpOptions

Ξεκινήστε δημιουργώντας μια νέα παρουσία του `BmpOptions`Αυτό το αντικείμενο θα μας επιτρέψει να ορίσουμε διάφορες παραμέτρους για την εικόνα bitmap:

```java
try (BmpOptions bmpCreateOptions = new BmpOptions()) {
    // Ορίστε τα bit ανά pixel σε 32 για μεγαλύτερο βάθος χρώματος.
    bmpCreateOptions.setBitsPerPixel(32);

    // Ορίστε μια ροή πηγής χρησιμοποιώντας έναν πίνακα byte εντός μνήμης.
    bmpCreateOptions.setSource(new StreamSource(
        new java.io.ByteArrayInputStream(new byte[100 * 100 * 4])));
}
```

**Εξήγηση:**

- **`setBitsPerPixel(32)`:** Αυξάνει το βάθος χρώματος, επιτρέποντας περισσότερα χρώματα και καλύτερη ποιότητα εικόνας.
- **`StreamSource`:** Καθορίζει μια πηγή στη μνήμη για τα δεδομένα bitmap.

### Χαρακτηριστικό 2: Δημιουργία και τροποποίηση εικόνας

Τώρα που έχουμε διαμορφώσει `BmpOptions`, ας δημιουργήσουμε μια νέα εικόνα και ας σχεδιάσουμε ελλείψεις πάνω της χρησιμοποιώντας τις δυνατότητες σχεδίασης του Aspose.Imaging.

#### Βήμα 1: Δημιουργήστε μια εικόνα

Χρησιμοποιώντας τις επιλογές που ορίστηκαν προηγουμένως, δημιουργήστε μια νέα εικόνα:

```java
try (Image image = Image.create(bmpCreateOptions, 100, 100)) {
    // Αρχικοποιήστε το αντικείμενο Graphics για σχεδίαση.
    Graphics graphic = new Graphics(image);

    // Διαυγές με κίτρινο φόντο.
    graphic.clear(Color.getYellow());

    // Σχεδιάστε μια διακεκομμένη κόκκινη έλλειψη και ορίστε το ορθογώνιο που την οριοθετεί.
    graphic.drawEllipse(new Pen(Color.getRed()), new Rectangle(30, 10, 40, 80));

    // Σχεδιάστε μια συμπαγή μπλε έλλειψη μέσα σε ένα άλλο ορθογώνιο.
    graphic.drawEllipse(new Pen(new com.aspose.imaging.brushes.SolidBrush(Color.getBlue())), 
        new Rectangle(10, 30, 80, 40));
    
    // Αποθηκεύστε την εικόνα στον επιθυμητό κατάλογο.
    image.save("YOUR_OUTPUT_DIRECTORY/DrawingEllipse_out.bmp");
}
```

**Εξήγηση:**

- **`Graphics`:** Παρέχει μεθόδους για τη σχεδίαση σχημάτων και κειμένου σε εικόνες.
- **`drawEllipse()`:** Σχεδιάζει ελλείψεις με συγκεκριμένα στυλό και ορθογώνια. Προσαρμόζει τα χρώματα και τα μεγέθη όπως απαιτείται.
