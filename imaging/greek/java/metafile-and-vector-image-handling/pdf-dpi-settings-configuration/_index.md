## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Φόρτωση της Εικόνας

Το πρώτο βήμα στη μετατροπή μιας εικόνας σε PDF είναι η φόρτωση της εικόνας. Χρησιμοποιήστε τον παρακάτω κώδικα για να φορτώσετε την εικόνα από τον φάκελο εγγράφων σας:

```java
String dataDir = "Your Document Directory" + "ModifyingImages/";
String fileName = "SampleTiff1.tiff";
String inputFileName = dataDir + fileName;

try (Image image = Image.load(inputFileName)) {
    // Your code for further steps will go here
}
```

Σε αυτόν τον κώδικα, αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή προς τον φάκελο εγγράφων σας και το `"SampleTiff1.tiff"` με το όνομα του αρχείου εικόνας σας.

### Βήμα 2: Διαμόρφωση Ρυθμίσεων PDF

Στη συνέχεια, πρέπει να διαμορφώσετε τις ρυθμίσεις PDF. Αυτό περιλαμβάνει τον καθορισμό του μεγέθους σελίδας και τυχόν άλλων επιλογών που θέλετε να εφαρμόσετε στο PDF. Ακολουθεί ένα παράδειγμα για το πώς να ορίσετε το μέγεθος σελίδας:

```java
PdfOptions pdfOptions = new PdfOptions();
pdfOptions.setPageSize(new SizeF(612, 792));
```

Μπορείτε να προσαρμόσετε αυτές τις επιλογές σύμφωνα με τις συγκεκριμένες απαιτήσεις σας.

### Βήμα 3: Αποθήκευση ως PDF

Τέλος, θα αποθηκεύσετε την εικόνα ως PDF. Χρησιμοποιήστε τον παρακάτω κώδικα για να το κάνετε:

```java
String outFileName = "Your Document Directory" + fileName + ".pdf";
image.save(outFileName, pdfOptions);
```

Βεβαιωθείτε ότι αντικαθιστάτε το `"Your Document Directory"` με την πραγματική διαδρομή προς τον φάκελο εγγράφων σας, και το `"SampleTiff1.tiff"` με το όνομα του αρχείου εικόνας σας.
