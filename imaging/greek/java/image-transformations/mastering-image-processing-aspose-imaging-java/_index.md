## Οδηγός Εφαρμογής

### Μετατροπή μορφής εικόνας

**Επισκόπηση:** Η μετατροπή εικόνων σε διαφορετικές μορφές είναι μια συνηθισμένη εργασία και το Aspose.Imaging την κάνει απρόσκοπτη.

#### Βήμα 1: Φόρτωση της εικόνας
```java
import com.aspose.imaging.Image;
import com.aspose.imaging.imageoptions.PngOptions;

public class ConvertToPng {
    public static void main(String[] args) {
        // Φόρτωση εικόνας από αρχείο
        try (Image image = Image.load("input.jpg")) {
            PngOptions options = new PngOptions();
            // Αποθήκευση σε μορφή PNG
            image.save("output.png", options);
        }
    }
}
```
*Εξήγηση:* Αυτό το απόσπασμα δείχνει τη φόρτωση μιας εικόνας JPEG και την αποθήκευσή της ως αρχείο PNG. `PngOptions` Η κλάση χρησιμοποιείται για να καθορίσει οποιαδήποτε πρόσθετη διαμόρφωση για το αρχείο εξόδου.

#### Βήμα 2: Αλλαγή μεγέθους εικόνων

**Επισκόπηση:** Η αλλαγή μεγέθους εικόνων διατηρώντας παράλληλα την αναλογία διαστάσεων ή εφαρμόζοντας συγκεκριμένους μετασχηματισμούς μπορεί εύκολα να επιτευχθεί.

```java
import com.aspose.imaging.ResizeType;

public class ResizeImage {
    public static void main(String[] args) {
        try (Image image = Image.load("input.jpg")) {
            // Αλλαγή μεγέθους εικόνας χρησιμοποιώντας έναν τύπο αλλαγής μεγέθους και διαστάσεις
            image.resize(800, 600, ResizeType.HighQualityResample);
            image.save("resized_output.jpg");
        }
    }
}
```
*Εξήγηση:* Αυτός ο κώδικας αλλάζει το μέγεθος μιας εικόνας σε 800x600 pixel με δειγματοληψία υψηλής ποιότητας. `ResizeType` Η παράμετρος διασφαλίζει ότι η εικόνα με το νέο μέγεθος διατηρεί την ποιότητα.
