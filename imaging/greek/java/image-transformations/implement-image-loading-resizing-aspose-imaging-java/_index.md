## Οδηγός Εφαρμογής

### Φόρτωση και αποθήκευση εικόνας με αλλαγή μεγέθους

Αυτή η λειτουργία δείχνει πώς να φορτώσετε μια εικόνα JPEG, να αλλάξετε το μέγεθός της και να την αποθηκεύσετε χρησιμοποιώντας τις βασικές λειτουργίες που παρέχονται από το Aspose.Imaging.

#### Επισκόπηση
Η φόρτωση και η αποθήκευση εικόνων είναι θεμελιώδης σε πολλές εφαρμογές. Αλλάζοντας το μέγεθος πριν από την αποθήκευση, μπορείτε να μειώσετε το μέγεθος του αρχείου διατηρώντας παράλληλα την οπτική ποιότητα για χρήση στο διαδίκτυο ή σε κινητά.

**Βήμα 1: Φόρτωση της εικόνας**

```java
import com.aspose.imaging.Image;

String dataDir = "YOUR_DOCUMENT_DIRECTORY/ModifyingImages/";
try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
    // Προχωρήστε στην αλλαγή μεγέθους και την αποθήκευση
}
```

- `Image.load`: Ανοίγει ένα αρχείο εικόνας από την καθορισμένη διαδρομή.

**Βήμα 2: Αλλαγή μεγέθους εικόνας**

```java
image.resize(300, 300);
```
- Αλλάζει το μέγεθος της φορτωμένης εικόνας σε μια σταθερή διάσταση 300x300 pixel.

**Βήμα 3: Αποθήκευση της εικόνας με το νέο μέγεθος**
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/";
image.save(outputDir + "SimpleResizing_out.jpg");
```
- `Image.save`Αποθηκεύει την εικόνα με το νέο μέγεθος σε ένα νέο αρχείο στον καθορισμένο κατάλογο.

### Φόρτωση και αλλαγή μεγέθους εικόνας με παρεμβολή Catmull-Rom

Αυτή η λειτουργία δείχνει πώς να αλλάξετε το μέγεθος εικόνων χρησιμοποιώντας συγκεκριμένους αλγόριθμους για βελτιωμένα αποτελέσματα.

#### Επισκόπηση
Οι αλγόριθμοι παρεμβολής όπως ο Catmull-Rom παρέχουν ομαλότερους μετασχηματισμούς υπολογίζοντας ενδιάμεσες τιμές pixel, μειώνοντας τα τεχνουργήματα κατά την αλλαγή μεγέθους.

**Βήμα 1: Φόρτωση της εικόνας**

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.ResizeType;

String dataDir = "YOUR_DOCUMENT_DIRECTORY/ModifyingImages/";
try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
    // Προχωρήστε στην αλλαγή μεγέθους με παρεμβολή
}
```

**Βήμα 2: Αλλαγή μεγέθους χρησιμοποιώντας την παρεμβολή Catmull-Rom**

```java
image.resize(200, 200, ResizeType.CatmullRom);
```
- `ResizeType.CatmullRom`: Καθορίζει τον αλγόριθμο που χρησιμοποιείται για την παρεμβολή κατά την αλλαγή μεγέθους.
