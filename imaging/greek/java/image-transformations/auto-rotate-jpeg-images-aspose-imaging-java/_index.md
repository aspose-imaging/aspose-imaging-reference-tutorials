## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση εικόνας JPEG

#### Επισκόπηση
Η φόρτωση μιας εικόνας είναι το πρώτο βήμα σε οποιαδήποτε εργασία επεξεργασίας εικόνας. Δείτε πώς μπορείτε να φορτώσετε μια εικόνα JPEG από έναν καθορισμένο κατάλογο.

##### Βήμα προς βήμα εφαρμογή

**H3: Εισαγωγή βιβλιοθηκών**

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.jpeg.JpegImage;
```

**H3: Φόρτωση της εικόνας**

Ορίστε τη διαδρομή του καταλόγου εγγράφου σας και φορτώστε την εικόνα:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
try (JpegImage image = (JpegImage) Image.load(dataDir + "aspose-logo.jpg")) {
    // Η εικόνα έχει φορτωθεί τώρα για χειρισμό
}
```

**Εξήγηση**: Το `Image.load()` Η μέθοδος διαβάζει το αρχείο JPEG σε ένα `JpegImage` αντικείμενο, επιτρέποντας τις επόμενες λειτουργίες.

### Χαρακτηριστικό 2: Αυτόματη περιστροφή εικόνας JPEG με βάση τα δεδομένα προσανατολισμού EXIF

#### Επισκόπηση
Η αυτόματη εναλλαγή εικόνων με βάση τα μεταδεδομένα προσανατολισμού EXIF διασφαλίζει ότι εμφανίζονται σωστά χωρίς χειροκίνητες προσαρμογές.

##### Βήμα προς βήμα εφαρμογή

**H3: Φόρτωση και αυτόματη περιστροφή της εικόνας**

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
try (JpegImage image = (JpegImage) Image.load(dataDir + "aspose-logo.jpg")) {
    // Εκτέλεση αυτόματης περιστροφής με βάση τον προσανατολισμό EXIF
    image.autoRotate();
}
```

**Εξήγηση**: Το `autoRotate()` Η μέθοδος προσαρμόζει τον προσανατολισμό της εικόνας σύμφωνα με τα ενσωματωμένα δεδομένα EXIF, βελτιστοποιώντας τη ροή εργασίας σας.

### Λειτουργία 3: Αποθήκευση εικόνας JPEG σε δίσκο

#### Επισκόπηση
Μετά την επεξεργασία μιας εικόνας, η αποθήκευσή της ξανά στον δίσκο είναι ζωτικής σημασίας για αποθήκευση ή περαιτέρω χρήση. Δείτε πώς μπορείτε να αποθηκεύσετε μια επεξεργασμένη εικόνα JPEG.

##### Βήμα προς βήμα εφαρμογή

**H3: Φόρτωση, Αυτόματη περιστροφή και αποθήκευση της εικόνας**

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";

try (JpegImage image = (JpegImage) Image.load(dataDir + "aspose-logo.jpg")) {
    // Εκτέλεση αυτόματης περιστροφής με βάση τον προσανατολισμό EXIF
    image.autoRotate();

    // Αποθήκευση της επεξεργασμένης εικόνας στο δίσκο
    image.save(outDir + "AutoCorrectOrientationOfJPEGImages_out.jpg");
}
```

**Εξήγηση**: Το `save()` Η μέθοδος γράφει το επεξεργασμένο JPEG πίσω στον καθορισμένο κατάλογο, διατηρώντας όλες τις τροποποιήσεις.
