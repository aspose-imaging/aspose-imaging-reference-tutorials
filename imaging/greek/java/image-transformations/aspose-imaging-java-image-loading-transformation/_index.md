### Οδηγός Εφαρμογής

#### Φόρτωση εικόνας με βελτιστοποίηση μεγέθους buffer

Η βελτιστοποίηση της χρήσης μνήμης είναι ζωτικής σημασίας κατά τη φόρτωση μεγάλων εικόνων και το Aspose.Imaging παρέχει έναν τρόπο καθορισμού μεγεθών buffer για αποτελεσματικό χειρισμό.

**Επισκόπηση**

Αυτή η λειτουργία βοηθά στη διαχείριση της μνήμης κατά τη φόρτωση εικόνων, ορίζοντας μια υπόδειξη μεγέθους buffer. Είναι ιδιαίτερα χρήσιμη όταν εργάζεστε με μεγάλα αρχεία TIFF ή δέσμες εικόνων.

##### Βήματα Υλοποίησης

1. **Εισαγωγή απαιτούμενων κλάσεων:**
   ```java
   import com.aspose.imaging.Image;
   import com.aspose.imaging.LoadOptions;
   ```

2. **Υπόδειξη για το μέγεθος του buffer:**
   - Χρήση `LoadOptions` για να καθορίσετε το μέγεθος του buffer.
   ```java
   String inputFileName = "YOUR_DOCUMENT_DIRECTORY/SampleTiff1.tiff";
   LoadOptions loadOptions = new LoadOptions();
   loadOptions.setBufferSizeHint(50); // Ορίζει μια υπόδειξη 50MB για το μέγεθος του buffer
   ```

3. **Φόρτωση εικόνας με μέγεθος buffer:**
   ```java
   try (Image image = Image.load(inputFileName, loadOptions)) {
       // Εκτέλεση λειτουργιών στην εικόνα που έχει φορτωθεί
   }
   ```
   - Η υπόδειξη μεγέθους buffer βοηθά στη διαχείριση της μνήμης περιορίζοντας τον αριθμό των δεδομένων που διατηρούνται στη μνήμη ταυτόχρονα.

#### Περιστροφή και αναστροφή εικόνας

Ο μετασχηματισμός εικόνων είναι απλός με τις ισχυρές μεθόδους περιστροφής και αναστροφής του Aspose.Imaging.

**Επισκόπηση**

Αυτή η λειτουργία σάς επιτρέπει να περιστρέφετε και να αναστρέφετε εύκολα εικόνες, επιτρέποντας δυναμικούς μετασχηματισμούς εικόνας για διάφορες εφαρμογές.

##### Βήματα Υλοποίησης

1. **Εισαγωγή απαιτούμενων κλάσεων:**
   ```java
   import com.aspose.imaging.Image;
   import com.aspose.imaging.RasterImage;
   import com.aspose.imaging.RotateFlipType;
   ```

2. **Φόρτωση εικόνας:**
   - Ξεκινήστε φορτώνοντας την εικόνα σας.
   ```java
   String inputFileName = "YOUR_DOCUMENT_DIRECTORY/SampleTiff1.tiff";
   try (Image image = Image.load(inputFileName)) {
       // Προχωρήστε στην περιστροφή και αναστροφή της εικόνας
   }
   ```

3. **Περιστροφή και αναστροφή:**
   - Περιστρέψτε χωρίς να γυρίσετε.
   ```java
   image.rotateFlip(RotateFlipType.Rotate90FlipNone);
   ```
   - Επιπλέον περιστροφή για `RasterImage`.
   ```java
   if (image instanceof RasterImage) {
       ((RasterImage) image).rotate(60); // Περιστροφή 60 μοίρες δεξιόστροφα
   }
   ```
