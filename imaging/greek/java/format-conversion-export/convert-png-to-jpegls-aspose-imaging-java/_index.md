## Χρήση του Aspose Imaging Java για τη Μετατροπή PNG σε JPEG‑LS (H2)

### Μετατροπή PNG σε JPEG‑LS (H2)

Αυτή η δυνατότητα σας επιτρέπει να συμπιέσετε μια εικόνα PNG 8‑bit ανά δείγμα σε πιο αποδοτική μορφή JPEG‑LS 2‑bit ανά δείγμα.

#### Επισκόπηση
Το JPEG‑LS είναι γνωστό για τις δυνατότητες lossless και near‑lossless συμπίεσης, καθιστώντας το ιδανικό για αποθήκευση εικόνων υψηλής ποιότητας με μειωμένα μεγέθη αρχείων. Αυτή η ενότητα θα σας καθοδηγήσει στη μετατροπή ενός PNG σε αυτή τη μορφή χρησιμοποιώντας το Aspose Imaging.

#### Βήματα Υλοποίησης

##### Φόρτωση της Αρχικής Εικόνας PNG (H3)
Ξεκινήστε φορτώνοντας την πηγαία εικόνα PNG:

```java
String originPngFileName = "YOUR_DOCUMENT_DIRECTORY/lena24b.png";
PngImage pngImage = (PngImage) Image.load(originPngFileName);
```
*Γιατί;*: Η φόρτωση της εικόνας σας επιτρέπει να έχετε πρόσβαση και να τη διαχειριστείτε χρησιμοποιώντας το ισχυρό API του Aspose Imaging.

##### Διαμόρφωση Επιλογών JPEG (H3)
Ρυθμίστε τις απαραίτητες επιλογές για τη μετατροπή:

```java
int bpp = 2; // Bits per sample
JpegOptions jpegOptions = new JpegOptions();
jpegOptions.setBitsPerChannel((byte) bpp);
jpegOptions.setCompressionType(JpegCompressionMode.JpegLs);
```
*Γιατί;*: Η διαμόρφωση αυτών των επιλογών καθορίζει τον τύπο συμπίεσης και τις παραμέτρους ποιότητας για την έξοδο εικόνας JPEG‑LS.

##### Αποθήκευση της Εικόνας (H3)
Τέλος, αποθηκεύστε τη μετατρεπόμενη εικόνα:

```java
String outputJpegFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.jls";
pngImage.save(outputJpegFileName, jpegOptions);
```
*Γιατί;*: Η αποθήκευση σε συγκεκριμένη μορφή ολοκληρώνει τη διαδικασία μετατροπής.

##### Διαχείριση Πόρων (H3)
Βεβαιωθείτε ότι οι πόροι απελευθερώνονται μετά την επεξεργασία:

```java
finally {
    pngImage.dispose();
}
```
*Γιατί;*: Η απελευθέρωση των αντικειμένων εικόνας βοηθά στη διαχείριση της χρήσης μνήμης αποτελεσματικά.

#### Συμβουλές Επίλυσης Προβλημάτων

- **Κοινό Πρόβλημα**: Εάν αντιμετωπίσετε `FileNotFoundException`, ελέγξτε ξανά τις διαδρομές αρχείων για τα αρχεία εισόδου και εξόδου.
- **Συμβουλή Επίλυσης**: Βεβαιωθείτε ότι το IDE σας έχει άδεια ανάγνωσης και εγγραφής στους καθορισμένους καταλόγους.

### Μετατροπή JPEG‑LS σε PNG (H2)

Αυτή η δυνατότητα σας επιτρέπει να μετατρέψετε πίσω ένα αρχείο JPEG‑LS σε μορφή PNG, διευκολύνοντας οπτικές συγκρίσεις ή περαιτέρω επεξεργασία.

#### Επισκόπηση
Μετά τη μετατροπή μιας εικόνας σε JPEG‑LS για σκοπούς συμπίεσης, μπορεί να είναι απαραίτητο να επαναφέρετε τη διαδικασία για αξιολόγηση της ποιότητας ή για άλλες χρήσεις. Αυτή η ενότητα εξηγεί πώς να το επιτύχετε με το Aspose Imaging.

#### Βήματα Υλοποίησης

##### Φόρτωση της Εικόνας JPEG‑LS (H3)
Αρχίστε φορτώνοντας το αρχείο JPEG‑LS:

```java
String inputJpegFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.jls";
JpegImage jpegImage = (JpegImage) Image.load(inputJpegFileName);
```
*Γιατί;*: Η φόρτωση της εικόνας την προετοιμάζει για μετατροπή σε άλλη μορφή.

##### Αποθήκευση ως PNG (H3)
Μετατρέψτε και αποθηκεύστε την εικόνα σε μορφή PNG:

```java
String outputPngFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.png";
jpegImage.save(outputPngFileName, new PngOptions());
```
*Γιατί;*: Αυτό το βήμα εξασφαλίζει ότι το αρχείο JPEG‑LS μετατρέπεται πίσω σε μια ευρέως συμβατή και lossless μορφή PNG.

##### Διαχείριση Πόρων (H3)
Απελευθερώστε τους πόρους εικόνας μετά την αποθήκευση:

```java
finally {
    jpegImage.dispose();
}
```
*Γιατί;*: Η σωστή απελευθέρωση αποτρέπει διαρροές μνήμης, διατηρώντας την απόδοση της εφαρμογής.
