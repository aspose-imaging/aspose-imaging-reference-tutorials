### Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε δύο κύριες λειτουργίες: εξαγωγή SVG σε PDF και αποθήκευση SVG με συγκεκριμένες επιλογές χειρισμού γραμματοσειρών.

#### Φόρτωση και εξαγωγή SVG σε PDF

**Επισκόπηση**Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο SVG σε έγγραφο PDF υψηλής ποιότητας χρησιμοποιώντας το Aspose.Imaging για Java.

1. **Προετοιμάστε το περιβάλλον σας**

   Βεβαιωθείτε ότι το έργο σας έχει τις απαραίτητες ρυθμίσεις, συμπεριλαμβανομένης της εξάρτησης Aspose.Imaging που έχει ρυθμιστεί όπως φαίνεται παραπάνω.

2. **Φόρτωση του αρχείου SVG**

   Χρησιμοποιήστε το `Image.load()` μέθοδος για να φορτώσετε το αρχείο SVG σας από έναν καθορισμένο κατάλογο. Αυτό το βήμα αρχικοποιεί το αντικείμενο εικόνας που θα χρησιμοποιηθεί για τη μετατροπή.
   
   ```java
   final Image image = Image.load(inputFile);
   ```

3. **Ρύθμιση παραμέτρων επιλογών εξαγωγής PDF**

   Στήνω `PdfOptions` με `SvgRasterizationOptions` ώστε να ταιριάζει με το μέγεθος σελίδας των διαστάσεων SVG σας.

   ```java
   new PdfOptions()
   {
{
       setVectorRasterizationOptions(new SvgRasterizationOptions() 
       {
{
           setSize(image.getWidth(), image.getHeight());
       }
});
   }
};
   ```

4. **Save as PDF**

   Use the `image.save()` method to export the loaded SVG file into a PDF document.

   ```java
   image.save(outFile, pdfOptions);
   ```

5. **Διαχείριση Πόρων**

   Πάντα να πετάτε το αντικείμενο Image μετά τη χρήση για να ελευθερώσετε πόρους.

   ```java
   finally
   {
       image.dispose();
   }
   ```

#### Αποθήκευση SVG με επιλογές χειρισμού γραμματοσειρών

**Επισκόπηση**Αυτή η λειτουργία σάς επιτρέπει να αποθηκεύσετε ένα αρχείο SVG με επιλογές για ενσωμάτωση ή ροή γραμματοσειρών, παρέχοντας ευελιξία στον τρόπο διαχείρισης των γραμματοσειρών μέσα στο έγγραφο.

1. **Αρχικοποίηση επιλογών εικόνας και ραστεροποίησης**

   Φορτώστε το SVG εισόδου σας χρησιμοποιώντας το αρχείο SVG. `Image.load()` και να στήσω `EmfRasterizationOptions` για να ορίσετε το χρώμα φόντου και το μέγεθος σελίδας.
   
   ```java
   final EmfRasterizationOptions emfRasterizationOptions = new EmfRasterizationOptions();
   emfRasterizationOptions.setBackgroundColor(Color.getWhite());
   ```

2. **Ρύθμιση παραμέτρων χειρισμού γραμματοσειρών**

   Χρησιμοποιήστε έναν μηχανισμό επανάκλησης με `SvgResourceKeeperCallback` για να αποφασίσετε εάν οι γραμματοσειρές θα πρέπει να ενσωματωθούν ή να μεταδοθούν σε ροή.

   ```java
   setCallback(new SvgResourceKeeperCallback()
   {
{
       public void onFontResourceReady(FontStoringArgs ορίσματα)
       {
           αν (χρήσηΕνσωματωμένου) 
               args.setFontStoreType(FontStoreType.Embedded);
           αλλού 
           {
               // Χειρισμός της λογικής γραμματοσειράς ροής εδώ...
           }
       }
   }
});
   ```

3. **Save the SVG File**

   Save your SVG file with these configurations.

   ```java
   image.save(outputFile, new SvgOptions()
   {
{
       setVectorRasterizationOptions(emfRasterizationOptions);
   }
});
   ```
