## Οδηγός Υλοποίησης

Παρακάτω θα βρείτε **ενότητες σε μορφή ερώτησης** που παρέχουν μια σύντομη απάντηση πρώτα, ακολουθούμενη από πιο λεπτομερή περιγραφή. Κάθε ενότητα ξεκινά με άγκυρα ορισμού για την κύρια κλάση που χρησιμοποιείται.

### Πώς να μετατρέψω μια εικόνα σε BMP χρησιμοποιώντας το Aspose Imaging Java;
Η μετατροπή σε BMP είναι απλή: φορτώστε την πηγαία εικόνα, διαμορφώστε τις ειδικές επιλογές BMP όπως το βάθος χρώματος, και αποθηκεύστε το αποτέλεσμα. Η διαδικασία διατηρεί κάθε pixel χωρίς συμπίεση, καθιστώντας το BMP ιδανικό για μη‑συμπιεστική αρχειοθέτηση ή περαιτέρω βήματα επεξεργασίας.

Η κλάση `Image` είναι το βασικό αντικείμενο του Aspose.Imaging που αντιπροσωπεύει οποιαδήποτε raster εικόνα στη μνήμη.  
Η κλάση `BmpOptions` καθορίζει ρυθμίσεις για την έξοδο BMP όπως το βάθος χρώματος και η συμπίεση.  
```java
Image image = Image.load("source.png");
BmpOptions bmpOptions = new BmpOptions();
bmpOptions.setBitsPerPixel(24);
image.save("output.bmp", bmpOptions);
```
```java
image.save("YOUR_OUTPUT_DIRECTORY/ExportToBMP_out.bmp", new BmpOptions());
```

### Πώς να μετατρέψω μια εικόνα σε JPEG χρησιμοποιώντας το Aspose Imaging Java;
Για να δημιουργήσετε ένα JPEG, δημιουργήστε ένα αντικείμενο `JpegOptions`, ορίστε το επιθυμητό επίπεδο ποιότητας (0‑100), προαιρετικά ορίστε υποδειγματοληψία και καλέστε `save`. Η συμπίεση JPEG μειώνει το μέγεθος του αρχείου διατηρώντας την οπτική πιστότητα, ειδικά όταν επιλέγετε ποιότητα μεταξύ 80‑90 για εικόνες έτοιμες για web.

Η κλάση `JpegOptions` σας επιτρέπει να ρυθμίσετε λεπτομερώς τη συμπίεση, την υποδειγματοληψία και το χρωματικό χώρο.  
```java
Image image = Image.load("source.bmp");
JpegOptions jpegOptions = new JpegOptions();
jpegOptions.setQuality(85);
image.save("output.jpg", jpegOptions);
```
```java
image.save("YOUR_OUTPUT_DIRECTORY/ExportToJPEG_out.jpeg", new JpegOptions());
```

### Πώς να μετατρέψω μια εικόνα σε PNG χρησιμοποιώντας το Aspose Imaging Java;
Η μετατροπή σε PNG χρησιμοποιεί το `PngOptions` για να διατηρήσει τη διαφάνεια και τις λεπτομέρειες χωρίς απώλειες. Μετά τη φόρτωση της πηγής, καλέστε `save` με τις επιλογές PNG· η βιβλιοθήκη επιλέγει αυτόματα το βέλτιστο φίλτρο για τη μέγιστη συμπίεση διατηρώντας την εικόνα πλήρως αμετάβλητη.

Η κλάση `PngOptions` παρέχει έλεγχο του βάθους χρώματος, του τύπου φίλτρου και της διαπλοκής (interlacing).  
```java
Image image = Image.load("source.tiff");
PngOptions pngOptions = new PngOptions();
pngOptions.setColorType(PngColorType.TruecolorWithAlpha);
image.save("output.png", pngOptions);
```
```java
image.save("YOUR_OUTPUT_DIRECTORY/ExportToPNG_out.png", new PngOptions());
```

### Πώς να μετατρέψω μια εικόνα σε TIFF χρησιμοποιώντας το Aspose Imaging Java;
Η δημιουργία ενός TIFF περιλαμβάνει τη δημιουργία ενός αντικειμένου `TiffOptions`, την επιλογή του κατάλληλου αλγορίθμου συμπίεσης (π.χ., LZW, CCITT) και την αποθήκευση. Το TIFF είναι ιδανικό για αρχειοθέτηση επειδή μπορεί να αποθηκεύσει πολλαπλές σελίδες και δεδομένα υψηλού βάθους χρώματος σε ένα μόνο αρχείο.

Η κλάση `TiffOptions` υποστηρίζει μια ευρεία γκάμα αλγορίθμων συμπίεσης και σας επιτρέπει να ορίσετε τη μορφή αρχείου που αναμένεται (π.χ., `TiffExpectedFormat`).  
```java
Image image = Image.load("source.jpeg");
TiffOptions tiffOptions = new TiffOptions(TiffExpectedFormat.TiffJpegRgb);
tiffOptions.setCompression(TiffCompression.Lzw);
image.save("output.tiff", tiffOptions);
```
```java
image.save("YOUR_OUTPUT_DIRECTORY/ExportToTIFF_out.tiff", new TiffOptions(TiffExpectedFormat.Default));
```

## Πρακτικές Εφαρμογές

Η κατανόηση των περιπτώσεων όπου η μετατροπή εικόνων διακρίνεται σας βοηθά να επιλέξετε τη σωστή μορφή:

- **Web Development:** Μετατρέψτε PNG υψηλής ανάλυσης σε βελτιστοποιημένα JPEG για μείωση του εύρους ζώνης.
- **Digital Archives:** Διατηρήστε τα πρωτότυπα ως TIFF για μη‑συμπιεστική αποθήκευση ενώ παρέχετε προεπισκοπήσεις PNG για γρήγορη προβολή.
- **Printing Services:** Δημιουργήστε JPEG υψηλής ποιότητας με ενσωματωμένα προφίλ χρώματος για ακριβή αναπαραγωγή χρώματος.
- **Medical Imaging:** Χρησιμοποιήστε multi‑page TIFF για αποθήκευση σαρώσεων τύπου DICOM χωρίς απώλεια διαγνωστικών λεπτομερειών.
- **Game Asset Pipelines:** Μετατρέψτε υφές σε BMP ή JPEG ανάλογα με τους περιορισμούς μνήμης κατά την εκτέλεση.

## Παράγοντες Απόδοσης

### Συμβουλές Βελτιστοποίησης
- **Επιλέξτε το ελάχιστο βάθος χρώματος** που απαιτείται για τη χρήση σας· χαμηλότερο βάθος μειώνει το μέγεθος του αρχείου και τον χρόνο επεξεργασίας.
- **Επαναχρησιμοποιήστε τις παρουσίες `Image`** όταν μετατρέπετε την ίδια πηγή σε πολλαπλές μορφές· αυτό αποφεύγει την περιττή αποκωδικοποίηση.
- **Μεταδώστε (stream) μεγάλα αρχεία** χρησιμοποιώντας `Image.load` με ένα `FileStream` για να διατηρήσετε τη χρήση μνήμης χαμηλή.

### Οδηγίες Χρήσης Πόρων
- Για εικόνες μεγαλύτερες από **1 GB**, ενεργοποιήστε τη **λειτουργία εξοικονόμησης μνήμης** (`ImageLoadOptions.setLoadMode(LoadMode.Stream)`) ώστε η heap να παραμένει κάτω από 200 MB.
- Χρησιμοποιήστε το **try‑with‑resources** της Java για να διασφαλίσετε ότι τα αντικείμενα `Image` απορρίπτονται αυτόματα, αποτρέποντας διαρροές μνήμης.
