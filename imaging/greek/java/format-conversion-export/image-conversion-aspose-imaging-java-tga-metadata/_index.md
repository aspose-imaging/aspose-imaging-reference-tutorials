## Οδηγός Εφαρμογής

### Μετατροπή JPEG σε TGA

Αυτή η λειτουργία επιδεικνύει τη μετατροπή μιας εικόνας JPEG σε μορφή TGA, η οποία είναι χρήσιμη για εφαρμογές που απαιτούν δεδομένα εικόνας χωρίς απώλειες.

#### Βήμα 1: Φόρτωση της εικόνας

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.RasterImage;
import com.aspose.imaging.fileformats.jpeg.JpegImage;
import com.aspose.imaging.imageoptions.TgaOptions;

String inputFile = "YOUR_DOCUMENT_DIRECTORY/test.jpg";
String output = "YOUR_OUTPUT_DIRECTORY/test_.tga";

try (RasterImage image = (JpegImage) Image.load(inputFile)) {
    // Βήμα 2: Αποθηκεύστε την εικόνα σε μορφή TGA
    image.save(output, new TgaOptions());
}
```

- **Παράμετροι**: `inputFile` και `output` είναι διαδρομές αρχείων για την ανάγνωση και την εγγραφή εικόνων.
- **Σκοπός της μεθόδου**: Το `save()` Η μέθοδος μετατρέπει και αποθηκεύει το JPEG σας ως αρχείο TGA χρησιμοποιώντας τις προεπιλεγμένες επιλογές.

### Μετατροπή PNG σε TGA

Αυτή η ενότητα σας καθοδηγεί στη μετατροπή αρχείων PNG σε μορφή TGA, διατηρώντας δεδομένα εικόνας υψηλής ποιότητας.

#### Βήμα 1: Φόρτωση της εικόνας

```java
import com.aspose.imaging.fileformats.tga.TgaImage;

String inputFile = "YOUR_DOCUMENT_DIRECTORY/test.png";
String output = "YOUR_OUTPUT_DIRECTORY/test2_.tga";

try (RasterImage image = (RasterImage) Image.load(inputFile)) {
    // Βήμα 2: Δημιουργήστε μια εικόνα TGA από το RasterImage
    try (TgaImage tgaImage = new TgaImage(image)) {
        tgaImage.save(output);
    }
}
```

- **Σκοπός της μεθόδου**: Το `TgaImage` Ο κατασκευαστής δημιουργεί μια παρουσία της εικόνας TGA, και `save()` το γράφει στον δίσκο.

### Τροποποίηση και αποθήκευση υπάρχουσας εικόνας TGA

Μάθετε πώς να τροποποιείτε μεταδεδομένα σε ένα αρχείο TGA, όπως χρονικές σημάνσεις, στοιχεία συντάκτη και άλλα, ενισχύοντας τον πλούτο δεδομένων της εικόνας σας.

#### Βήμα 1: Φόρτωση και τροποποίηση μεταδεδομένων

```java
import com.aspose.imaging.Color;
import java.util.Date;

String inputFile = "YOUR_DOCUMENT_DIRECTORY/test.tga";
String output = "YOUR_OUTPUT_DIRECTORY/test3_.tga";

try (TgaImage image = (TgaImage) Image.load(inputFile)) {
    // Βήμα 2: Ορισμός ιδιοτήτων μεταδεδομένων
    image.setDateTimeStamp(new Date());
    image.setAuthorName("John Smith");
    image.setAuthorComments("Comment");
    image.setImageId("ImageId");
    image.setJobNameOrId("Important Job");
    image.setJobTime(new Date(0, 0, 10));
    image.setTransparentColor(Color.fromArgb(123));
    image.setSoftwareId("SoftwareId");
    image.setSoftwareVersion("abc1");
    image.setSoftwareVersionLetter('a');
    image.setSoftwareVersionNumber(2);
    image.setXOrigin(1000);
    image.setYOrigin(1000);

    // Βήμα 3: Αποθήκευση τροποποιήσεων
    image.save(output);
}
```

- **Παράμετροι**Τα πεδία μεταδεδομένων όπως `setAuthorName`, `setImageId` σας επιτρέπουν να ορίσετε συγκεκριμένες πληροφορίες.
- **Βασικές επιλογές διαμόρφωσης**Προσαρμόστε τη διαφάνεια, τις πληροφορίες λογισμικού και τις λεπτομέρειες της εργασίας για ολοκληρωμένη διαχείριση δεδομένων.
