## Τι είναι το σύστημα χρωμάτων RGB;

Το μοντέλο χρωμάτων RGB ορίζει κάθε pixel με τρία στοιχεία βασισμένα στο φως — Κόκκινο, Πράσινο και Μπλε — καθένα από τα οποία κυμαίνεται από 0 έως 255. Αναμειγνύοντας αυτές τις τρεις τιμές μπορείτε να αναπαράγετε το πλήρες φάσμα χρωμάτων που αποδίδουν οι ψηφιακές οθόνες. Στην πράξη, τα περισσότερα αρχεία εικόνας αποθηκεύουν τα δεδομένα pixel σε αυτή τη μορφή, καθιστώντας την προεπιλογή για ροές εργασίας προσανατολισμένες στην οθόνη και ένα κοινό σημείο εκκίνησης πριν τη μετατροπή σε εκτυπώσιμους χώρους όπως το CMYK.

## Εισαγωγή Πακέτων

Ο χώρος ονομάτων `com.aspose.imaging` παρέχει όλες τις βασικές κλάσεις που χρειάζεστε για τη φόρτωση, μετατροπή και αποθήκευση εικόνων. Εισάγετέ τες στην κορυφή του αρχείου πηγαίου κώδικα Java ώστε ο μεταγλωττιστής να μπορεί να αναγνωρίσει τους τύπους.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.imageoptions.TiffOptions;
import com.aspose.imaging.fileformats.tiff.enums.TiffExpectedFormat;
```

## Βήμα 1: Φόρτωση Εικόνας

Για να διαβάσετε μια εικόνα TIFF σε Java, καλέστε τη στατική μέθοδο `Image.load` με τη διαδρομή του αρχείου. Αυτή η μέθοδος επιστρέφει ένα αντικείμενο `Image` που αντιπροσωπεύει ολόκληρο το raster στη μνήμη, έτοιμο για περαιτέρω επεξεργασία.

```java
String dataDir = "Your Document Directory" + "ConvertingImages/";
String sourceFilePath = "testTileDeflate.tif";
Image image = Image.load(dataDir + sourceFilePath);
```

## Βήμα 2: Εκτέλεση Επεξεργασίας Εικόνας

Η κλάση `TiffOptions` ρυθμίζει τη μορφή εξόδου και τη συμπίεση για αρχεία TIFF. Ορίζοντας τις ιδιότητες `bitsPerSample` και `compression`, μπορείτε να δημιουργήσετε ένα TIFF κωδικοποιημένο σε CMYK με συμπίεση LZW που είναι τόσο έτοιμο για εκτύπωση όσο και αποδοτικό σε μέγεθος.

```java
TiffOptions options = new TiffOptions(TiffExpectedFormat.TiffLzwCmyk);
String outputFilePath = "testTileDeflateCmyk.tif";

try {
    image.save("Your Document Directory" + outputFilePath, options);
} finally {
    image.dispose();
}
```

> **Pro tip:** Adjust the `TiffOptions` to experiment with other compression methods (e.g., `TiffLzwRgb`) or color formats based on your project needs.
