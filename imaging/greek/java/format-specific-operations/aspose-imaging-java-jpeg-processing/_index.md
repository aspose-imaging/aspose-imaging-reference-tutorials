## Οδηγός Εφαρμογής

Αυτή η ενότητα θα αναλύσει κάθε λειτουργία της φόρτωσης και αποθήκευσης εικόνων JPEG με συγκεκριμένες λειτουργίες χρώματος χρησιμοποιώντας το Aspose.Imaging.

### Λειτουργία 1: Φόρτωση και αποθήκευση εικόνας με συγκεκριμένες επιλογές JPEG

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε μια εικόνα JPEG από το σύστημά σας, να διαμορφώσετε τις ιδιότητές της και να την αποθηκεύσετε με συγκεκριμένες επιλογές, όπως bit ανά κανάλι και μετατροπή σε κλίμακα του γκρι.

##### Βήμα προς βήμα εφαρμογή:

**Βήμα 1: Ρύθμιση καταλόγων**
Ορίστε διαδρομές για εικόνες πηγής και καταλόγους εξόδου.
```java
String srcDir = "YOUR_DOCUMENT_DIRECTORY";
String outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

**Βήμα 2: Ρύθμιση παραμέτρων επιλογών JPEG**
Δημιουργήστε ένα `JpegOptions` αντίρρηση για τον ορισμό bits ανά κανάλι και άλλων διαμορφώσεων.
```java
JpegOptions options = new JpegOptions();
options.setBitsPerChannel((byte) 12); // Ορίστε τα bits ανά κανάλι σε 12
```

**Βήμα 3: Φόρτωση και αποθήκευση της εικόνας**
Φορτώστε μια εικόνα από τον κατάλογό σας, εφαρμόστε τις ρυθμίσεις χρώματος και αποθηκεύστε την με τις καθορισμένες επιλογές JPEG.
```java
Image image = Image.load(srcDir + "Rgb.jpg");
try {
    String outputPath = outputFolder + "/grayscale_12-bit.jpg";
    options.setColorType(JpegCompressionColorMode.Grayscale); // Ορισμός σε κλίμακα του γκρι
    image.save(outputPath, options);
} finally {
    image.dispose();  // Απελευθέρωση πόρων μετά την επεξεργασία
}
```

Αυτή η προσέγγιση σάς επιτρέπει να προσαρμόσετε τον τρόπο αποθήκευσης των εικόνων σας, βελτιστοποιώντας την ποιότητα και το μέγεθος του αρχείου.

### Χαρακτηριστικό 2: Επανάληψη και μετατροπή εικόνων με διαφορετικές λειτουργίες χρώματος

#### Επισκόπηση
Εξερευνήστε διάφορες λειτουργίες χρώματος JPEG εφαρμόζοντας κάθε λειτουργία στην ίδια εικόνα και αποθηκεύοντάς τες, επιδεικνύοντας την ευελιξία του Aspose.Imaging.

##### Βήμα προς βήμα εφαρμογή:

**Βήμα 1: Ορισμός τύπων χρωμάτων**
Δημιουργήστε μια σειρά από διαφορετικούς τύπους χρωμάτων που θέλετε να εφαρμόσετε.
```java
int[] colorTypes = new int[]{
    JpegCompressionColorMode.Grayscale,
    JpegCompressionColorMode.YCbCr,
    JpegCompressionColorMode.Rgb,
    JpegCompressionColorMode.Cmyk,
    JpegCompressionColorMode.Ycck
};
```

**Βήμα 2: Επανάληψη και αποθήκευση**
Επαναλάβετε τους τύπους χρωμάτων, εφαρμόστε τον καθένα στην εικόνα σας και αποθηκεύστε την με ένα μοναδικό όνομα.
```java
JpegOptions options = new JpegOptions();
options.setBitsPerChannel((byte) 12); // Ορισμός bit ανά κανάλι

for (int i = 0; i < colorTypes.length; ++i) {
    options.setColorType(colorTypes[i]); // Εφαρμογή τρέχοντος τύπου χρώματος
    String fileName = JpegCompressionColorMode.getName(JpegCompressionColorMode.class, colorTypes[i]) + "_12-bit.jpg";
    String outputPath = outputFolder + "/" + fileName;
    
    Image image = Image.load(srcDir + "Rgb.jpg");
    try {
        image.save(outputPath, options);  // Αποθήκευση με τις τρέχουσες ρυθμίσεις
    } finally {
        image.dispose();  // Απελευθέρωση πόρων για κάθε επανάληψη
    }
}
```

Αυτή η λειτουργία είναι ιδιαίτερα χρήσιμη για πειραματισμό και επιλογή βέλτιστων λειτουργιών χρώματος για διάφορες περιπτώσεις χρήσης.
