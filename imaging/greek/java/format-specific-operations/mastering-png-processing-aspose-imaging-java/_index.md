## Ρύθμιση του Aspose.Imaging για Java

Για να ξεκινήσετε να χρησιμοποιείτε το Aspose.Imaging, θα πρέπει να το προσθέσετε στο έργο σας. Δείτε πώς μπορείτε να το κάνετε αυτό χρησιμοποιώντας διαφορετικά εργαλεία δημιουργίας:

Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml` αρχείο:

```gradle
implementation 'com.aspose:aspose-imaging:25.5'
```

## Οδηγός Εφαρμογής

### Φόρτωση και αποθήκευση εικόνων PNG

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε μια εικόνα από τον δίσκο, να την επεξεργαστείτε και να αποθηκεύσετε ξανά το αποτέλεσμα. Δείτε πώς:

#### Βήμα 1: Εισαγωγή απαραίτητων κλάσεων
```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.png.PngImage;
```

#### Βήμα 2: Φόρτωση της εικόνας PNG
Καθορίστε τον κατάλογο εγγράφων σας και φορτώστε την εικόνα:
```java
String baseFolder = "YOUR_DOCUMENT_DIRECTORY/Png";
String imageFilePath = baseFolder + "/ball.png";

try (Image image = Image.load(imageFilePath)) {
    PngImage pngImage = (PngImage) image;
```

#### Βήμα 3: Αποθήκευση της επεξεργασμένης εικόνας
Μετά την επεξεργασία, αποθηκεύστε την εικόνα σε μια νέα τοποθεσία:
```java
    String outputFilePath = "YOUR_OUTPUT_DIRECTORY/result.png";
    pngImage.save(outputFilePath);
}
```

### Δημιουργία διαδρομής γραφικών και προσθήκη σχήματος

Η δημιουργία σχημάτων μπορεί να είναι απαραίτητη για τον γραφιστικό σχεδιασμό μέσα σε εικόνες:

#### Βήμα 1: Εισαγωγή απαιτούμενων κλάσεων
```java
import com.aspose.imaging.GraphicsPath;
import com.aspose.imaging.RectangleF;
import com.aspose.imaging.shapes.EllipseShape;
import com.aspose.imaging.shapes.Figure;
```

#### Βήμα 2: Δημιουργήστε μια διαδρομή γραφικών και προσθέστε ένα σχήμα έλλειψης
Ορίστε το σχήμα και προσθέστε το στη διαδρομή γραφικών σας:
```java
GraphicsPath mask = new GraphicsPath();
Figure firstFigure = new Figure();
firstFigure.addShape(new EllipseShape(new RectangleF(350, 170, 220, 230)));
mask.addFigure(firstFigure);
```

### Ρύθμιση παραμέτρων επιλογών υδατογραφήματος πλήρωσης με επίγνωση περιεχομένου

Αυτή η λειτουργία σάς επιτρέπει να διαμορφώσετε επιλογές υδατογραφήματος για απρόσκοπτη ενσωμάτωση:

#### Βήμα 1: Δημιουργήστε μια παρουσία του `ContentAwareFillWatermarkOptions`
```java
import com.aspose.imaging.watermark.options.ContentAwareFillWatermarkOptions;

GraphicsPath mask = new GraphicsPath();
ContentAwareFillWatermarkOptions options = new ContentAwareFillWatermarkOptions(mask);
options.setMaxPaintingAttempts(4);
```

### Αφαίρεση υδατογραφήματος χρησιμοποιώντας τη μέθοδο Paint Over

Η αφαίρεση υδατογραφημάτων μπορεί να είναι ζωτικής σημασίας για την επαναφορά εικόνων:

#### Βήμα 1: Ρύθμιση και φόρτωση εικόνας
```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.png.PngImage;
import com.aspose.imaging.watermark.WatermarkRemover;

String imageFilePath = "YOUR_DOCUMENT_DIRECTORY/Png/ball.png";

try (Image image = Image.load(imageFilePath)) {
    PngImage pngImage = (PngImage) image;
```

#### Βήμα 2: Ρύθμιση παραμέτρων και εφαρμογή αφαίρεσης υδατογραφήματος
Δημιουργήστε μια μάσκα, διαμορφώστε τις επιλογές και εφαρμόστε την αφαίρεση:
```java
    GraphicsPath mask = new GraphicsPath();
    Figure firstFigure = new Figure();
    firstFigure.addShape(new EllipseShape(new RectangleF(350, 170, 220, 230)));
    mask.addFigure(firstFigure);
    
    ContentAwareFillWatermarkOptions options = new ContentAwareFillWatermarkOptions(mask);
    options.setMaxPaintingAttempts(4);

    RasterImage result = WatermarkRemover.paintOver(pngImage, options);
    String outputFilePath = "YOUR_OUTPUT_DIRECTORY/result.png";
    result.save(outputFilePath);
}
```
