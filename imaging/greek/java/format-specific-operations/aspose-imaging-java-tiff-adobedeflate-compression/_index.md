## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, αναλύουμε τη διαδικασία σε διαχειρίσιμα βήματα:

### Φόρτωση εικόνας ως RasterImage

Για να ξεκινήσετε τη μετατροπή εικόνων σε TIFF, πρέπει πρώτα να τις φορτώσετε ως `RasterImage` αντικείμενα. Αυτό είναι απαραίτητο για την αποτελεσματική επεξεργασία δεδομένων raster.

```java
try (RasterImage image = (RasterImage) Image.load("YOUR_DOCUMENT_DIRECTORY/SampleTiff1.tiff")) {
    // Ο επιπλέον κώδικας θα τοποθετηθεί εδώ
}
```
*Εξήγηση:* Εδώ, ενσωματώνουμε τη λειτουργία φόρτωσης σε μια εντολή try-with-resources για την αυτόματη διαχείριση των πόρων. `Image.load` Η μέθοδος διαβάζει το αρχείο στη μνήμη.

### Ρύθμιση παραμέτρων επιλογών TIFF

Στη συνέχεια, διαμορφώστε το `TiffOptions`Εδώ ορίζετε βασικές ιδιότητες όπως τον τύπο συμπίεσης και την ανάλυση.

```java
TiffOptions options = new TiffOptions(TiffExpectedFormat.Default);
options.setBitsPerSample(new int[] { 8, 8, 8 });
options.setPhotometric(TiffPhotometrics.Rgb);
options.setXresolution(new TiffRational(72));
options.setYresolution(new TiffRational(72));
options.setResolutionUnit(TiffResolutionUnit.Inch);
options.setPlanarConfiguration(TiffPlanarConfig.Contiguous);
options.setCompression(TiffCompression.AdobeDeflate);
```
*Εξήγηση:* Ρυθμίζοντας `TiffOptions`, προσαρμόζετε τον τρόπο αποθήκευσης της εικόνας TIFF. Η συμπίεση AdobeDeflate είναι ιδιαίτερα αποτελεσματική για τη μείωση του μεγέθους του αρχείου διατηρώντας παράλληλα την ποιότητα.

### Αποθήκευση της μετατρεπόμενης εικόνας

Τέλος, αποθηκεύστε την εικόνα TIFF που έχετε μετατρέψει χρησιμοποιώντας τις διαμορφωμένες επιλογές:

```java
try (TiffFrame tiffFrame = TiffFrame.fromRasterImage(image)) {
    TiffImage tiffImage = new TiffImage(tiffFrame);
    tiffImage.save("YOUR_OUTPUT_DIRECTORY/SavingRasterImage_out.tiff", options);
}
```
*Εξήγηση:* Αυτή η ενότητα χρησιμοποιεί `TiffFrame` για την ενσωμάτωση των δεδομένων raster και στη συνέχεια την αποθήκευσή τους ως αρχείο TIFF. Το μοτίβο δοκιμής με πόρους διασφαλίζει την ορθή διαχείριση των πόρων.
