import com.aspose.imaging.imageoptions.TiffOptions;

## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: τον ορισμό επιλογών TIFF και τη δημιουργία/χειρισμό μιας εικόνας TIFF. Κάθε ενότητα έχει σχεδιαστεί για να σας καθοδηγήσει βήμα προς βήμα στη διαδικασία.

### Ορισμός ιδιοτήτων TiffOptions

Αυτή η λειτουργία δείχνει πώς να ρυθμίσετε διάφορες ιδιότητες για τη δημιουργία ενός αρχείου TIFF με τις επιθυμητές προδιαγραφές.

#### Επισκόπηση

Ρύθμιση παραμέτρων `TiffOptions` σας επιτρέπει να ορίσετε παραμέτρους όπως bit ανά δείγμα, ανάλυση και τύπο συμπίεσης, διασφαλίζοντας ότι η εικόνα TIFF σας πληροί συγκεκριμένα πρότυπα ποιότητας.

##### Ρύθμιση παραμέτρων bit ανά δείγμα

```java
// Δημιουργήστε μια παρουσία του TiffOptions
TiffOptions options = new TiffOptions(TiffExpectedFormat.Default);

// Ορισμός bit ανά δείγμα για διαμόρφωση RGB
options.setBitsPerSample(new int[] { 8, 8, 8 });
```

Αυτός ο κώδικας ορίζει το βάθος χρώματος σε 24-bit RGB, το οποίο είναι τυπικό για εικόνες υψηλής ποιότητας.

##### Ρύθμιση φωτομετρικής ερμηνείας

```java
// Χρήση φωτομετρικής ερμηνείας RGB
options.setPhotometric(TiffPhotometrics.Rgb);
```

Ο `setPhotometric` Η μέθοδος καθορίζει ότι η εικόνα χρησιμοποιεί μια παλέτα RGB.

##### Ορισμός ανάλυσης και μονάδων

```java
// Ορισμός ανάλυσης σε 72 DPI και για τους δύο άξονες X και Y
options.setXresolution(new TiffRational(72));
options.setYresolution(new TiffRational(72));

// Καθορίστε τη μονάδα ανάλυσης ως ίντσες
options.setResolutionUnit(TiffResolutionUnits.Inch);
```

Αυτές οι ρυθμίσεις διασφαλίζουν ότι η εικόνα έχει ένα σταθερό μέγεθος εμφάνισης σε διαφορετικές συσκευές.

##### Ρύθμιση συμπίεσης

```java
// Ορίστε τη συμπίεση σε AdobeDeflate για αποτελεσματική αποθήκευση
options.setCompression(TiffCompressions.AdobeDeflate);
```

Χρησιμοποιώντας `AdobeDeflate` μειώνει το μέγεθος του αρχείου χωρίς να χάνει την ποιότητα, καθιστώντας το ιδανικό για αρχειοθέτηση.

### Δημιουργία και χειρισμός ενός TiffImage

Τώρα που οι επιλογές έχουν οριστεί, ας δημιουργήσουμε μια εικόνα χρησιμοποιώντας αυτές τις διαμορφώσεις.

#### Επισκόπηση

Η δημιουργία μιας εικόνας TIFF περιλαμβάνει την αρχικοποίηση μιας `TiffImage`, ορίζοντας τα pixel του και αποθηκεύοντας το αποτέλεσμα. Αυτή η διαδικασία είναι απλή με το Aspose.Imaging.

##### Αρχικοποίηση ενός νέου TiffImage

```java
try (TiffImage tiffImage = new TiffImage(new TiffFrame(options, 100, 100))) {
    // Επαναλάβετε κάθε pixel για να το ορίσετε σε κόκκινο χρώμα
    for (int i = 0; i < 100; i++) {
        tiffImage.getActiveFrame().setPixel(i, i, Color.getRed());
    }
    
    // Αποθηκεύστε την εικόνα στον επιθυμητό κατάλογο εξόδου
    tiffImage.save("YOUR_OUTPUT_DIRECTORY" + "/CreatingTIFFImageWithCompression.tiff");
}
```

Σε αυτό το απόσπασμα, δημιουργούμε μια εικόνα TIFF 100x100 pixel και τη γεμίζουμε με κόκκινα pixel χρησιμοποιώντας τις προκαθορισμένες ρυθμίσεις μας.
