## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση και προσωρινή αποθήκευση μιας εικόνας

Η φόρτωση και η προσωρινή αποθήκευση εικόνων μπορεί να βελτιώσει σημαντικά την απόδοση μειώνοντας τις περιττές λειτουργίες φόρτωσης.

#### Επισκόπηση
Αυτή η λειτουργία διασφαλίζει ότι μόλις φορτωθεί μια εικόνα από μια διαδρομή αρχείου, ελέγχει εάν τα δεδομένα της εικόνας έχουν αποθηκευτεί στην προσωρινή μνήμη. Εάν όχι, αποθηκεύει τα δεδομένα στην προσωρινή μνήμη για μελλοντική χρήση.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.examples.Utils;

String dataDir = Utils.getSharedDataDir() + "ModifyingImages/";

try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
    // Ελέγξτε αν η εικόνα είναι αποθηκευμένη στην προσωρινή μνήμη. Αποθηκεύστε την στην προσωρινή μνήμη αν όχι.
    if (!image.isCached()) {
        image.cacheData();
    }
}
```

**Εξήγηση:**
- **`Image.load()`**: Φορτώνει μια εικόνα από μια καθορισμένη διαδρομή αρχείου.
- **`isCached()`**Ελέγχει εάν τα δεδομένα εικόνας είναι ήδη αποθηκευμένα στην προσωρινή μνήμη.
- **`cacheData()`**Αποθηκεύει στην προσωρινή μνήμη τα δεδομένα εικόνας για βελτίωση της απόδοσης σε επόμενες προσβάσεις.

### Λειτουργία 2: Αλλαγή μεγέθους εικόνας αναλογικά κατά πλάτος

Η αλλαγή μεγέθους εικόνων διατηρώντας παράλληλα την αναλογία διαστάσεων μπορεί να είναι κρίσιμη για τον responsive σχεδιασμό και τις διεπαφές χρήστη.

#### Επισκόπηση
Αυτή η λειτουργία δείχνει την αναλογική αλλαγή μεγέθους του πλάτους μιας εικόνας, διασφαλίζοντας ότι το ύψος προσαρμόζεται αυτόματα για να διατηρηθεί η αρχική αναλογία διαστάσεων.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.examples.Utils;

String dataDir = Utils.getSharedDataDir() + "ModifyingImages/";

try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
    if (!image.isCached()) {
        image.cacheData();
    }
    
    int newWidth = image.getWidth() / 2;
    image.resizeWidthProportionally(newWidth);
}
```

**Εξήγηση:**
- **`resizeWidthProportionally()`**: Προσαρμόζει το πλάτος, ενημερώνοντας αυτόματα το ύψος για να διατηρηθεί η αναλογικότητα.

### Λειτουργία 3: Αλλαγή μεγέθους εικόνας αναλογικά κατά ύψος

Αυτή η λειτουργία επιτρέπει την αλλαγή μεγέθους με βάση ένα νέο ύψος, διατηρώντας παράλληλα την αναλογία διαστάσεων.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.examples.Utils;

String dataDir = Utils.getSharedDataDir() + "ModifyingImages/";

try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
    if (!image.isCached()) {
        image.cacheData();
    }
    
    int newHeight = image.getHeight() / 2;
    image.resizeHeightProportionally(newHeight);
}
```

**Εξήγηση:**
- **`resizeHeightProportionally()`**: Προσαρμόζει το ύψος, ενημερώνοντας αυτόματα το πλάτος για να διατηρείται η αναλογικότητα.

### Λειτουργία 4: Αποθήκευση εικόνας με επιλογές

Η αποθήκευση εικόνων με συγκεκριμένες διαμορφώσεις μπορεί να είναι απαραίτητη για τη βελτιστοποίηση του μεγέθους και της ποιότητας του αρχείου.

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να αποθηκεύσετε μια εικόνα χρησιμοποιώντας `PngOptions`, επιτρέποντάς σας να καθορίσετε πρόσθετες ρυθμίσεις.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.examples.Utils;
import com.aspose.imaging.imageoptions.PngOptions;

String dataDir = Utils.getSharedDataDir() + "ModifyingImages/";

try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
    if (!image.isCached()) {
        image.cacheData();
    }
    
    String outputPath = Utils.getOutDir() + "SimpleResizeImageProportionally_out.png";
    image.save(outputPath, new PngOptions());
}
```

**Εξήγηση:**
- **`PngOptions()`**Παρέχει επιλογές για τη διαμόρφωση της μορφής PNG κατά την αποθήκευση.
