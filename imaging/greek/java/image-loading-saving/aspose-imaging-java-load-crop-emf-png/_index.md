## Οδηγός Εφαρμογής

Αυτός ο οδηγός θα σας καθοδηγήσει στην υλοποίηση δύο βασικών λειτουργιών χρησιμοποιώντας το Aspose.Imaging για Java: την περικοπή ενός αρχείου EMF και τον ορισμό επιλογών rasterization.

### Λειτουργία 1: Φόρτωση, περικοπή και αποθήκευση αρχείου EMF ως PNG

#### Επισκόπηση

Σε αυτήν την ενότητα, θα φορτώσουμε ένα αρχείο EMF, θα εφαρμόσουμε περικοπή με βάση καθορισμένες τιμές μετατόπισης και θα αποθηκεύσουμε το αποτέλεσμα ως εικόνα PNG. Αυτό είναι χρήσιμο για την προετοιμασία εικόνων για προβολή στο web ή περαιτέρω επεξεργασία.

#### Βήμα προς βήμα εφαρμογή

**Φόρτωση του αρχείου EMF**

```java
// Εισαγάγετε τα απαραίτητα πακέτα Aspose.Imaging
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.emf.MetaImage;

try (MetaImage metaImage = (MetaImage) Image.load("YOUR_DOCUMENT_DIRECTORY/Picture1.emf")) {
    // Συνεχίστε με την περικοπή και την αποθήκευση της εικόνας
}
```

**Ορισμός τιμών μετατόπισης**

```java
// Ορίστε τιμές μετατόπισης και για τις τέσσερις πλευρές της εικόνας
int leftShift = 10;
int rightShift = 10;
int topShift = 10;
int bottomShift = 10;

// Εφαρμογή περικοπής στην εικόνα με βάση τις τιμές μετατόπισης
metaImage.crop(leftShift, rightShift, topShift, bottomShift);
```

**Αποθήκευση ως PNG**

```java
import com.aspose.imaging.imageoptions.PngOptions;
import com.aspose.imaging.imageoptions.EmfRasterizationOptions;

PngOptions pngOptions = new PngOptions();
pngOptions.setVectorRasterizationOptions(new EmfRasterizationOptions() {
{
    setPageSize(metaImage.getSize());
}
});
metaImage.save("YOUR_OUTPUT_DIRECTORY/CropbyShifts_out.png", pngOptions);
```

#### Βασικές επιλογές διαμόρφωσης

- **Τιμές μετατόπισης καλλιέργειας**Προσαρμόστε τα για να περικόψετε διαφορετικά τμήματα της εικόνας σας.
- **PngOptions και ρυθμίσεις ραστεροποίησης**: Προσαρμόστε τη μορφή και την ποιότητα εξόδου.

### Λειτουργία 2: Ρύθμιση μεγέθους εικόνας και επιλογών ραστεροποίησης

#### Επισκόπηση

Αυτή η ενότητα εστιάζει στη ρύθμιση επιλογών ραστεροποίησης κατά την αποθήκευση ενός αρχείου EMF ως PNG, διασφαλίζοντας ότι το τελικό αποτέλεσμα πληροί συγκεκριμένες απαιτήσεις μεγέθους.

**Δημιουργία EmfRasterizationOptions**

```java
import com.aspose.imaging.imageoptions.EmfRasterizationOptions;
import com.aspose.imaging.Size;

// Υποθέτοντας ότι το 'metaImage' είναι ήδη φορτωμένο και διαθέσιμο
EmfRasterizationOptions rasterizationOptions = new EmfRasterizationOptions();
rasterizationOptions.setPageSize(metaImage.getSize());
```
