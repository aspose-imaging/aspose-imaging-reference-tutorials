## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε κάθε χαρακτηριστικό σε διαχειρίσιμα βήματα χρησιμοποιώντας λογικές ενότητες.

### Φόρτωση εικόνας TIFF

**Επισκόπηση:**  
Η φόρτωση μιας εικόνας TIFF πολλαπλών καρέ είναι το πρώτο σας βήμα. Αυτό θέτει τις βάσεις για την επεξεργασία μεμονωμένων καρέ αργότερα.

1. **Εισαγωγή απαιτούμενων κλάσεων:**

   ```java
   import com.aspose.imaging.Image;
   import com.aspose.imaging.fileformats.tiff.TiffImage;
   ```

2. **Φόρτωση της εικόνας TIFF:**
   
   Αντικαθιστώ `YOUR_DOCUMENT_DIRECTORY/sample.tif` με τη διαδρομή της εικόνας σας:

   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY/sample.tif";
   try (TiffImage multiImage = (TiffImage) Image.load(dataDir)) {
       // Η εικόνα TIFF έχει φορτωθεί και είναι έτοιμη για επεξεργασία.
   }
   ```

3. **Εξήγηση:**
   
   - `Image.load()` φορτώνει το καθορισμένο αρχείο TIFF σε ένα `TiffImage` αντικείμενο, το οποίο επιτρέπει περαιτέρω χειρισμό.
   - Η εντολή try-with-resources διασφαλίζει ότι οι πόροι κλείνουν αυτόματα μετά τη χρήση.

### Εξαγωγή καρέ από εικόνα TIFF

**Επισκόπηση:**  
Η εξαγωγή καρέ περιλαμβάνει την επανάληψη κάθε καρέ στην εικόνα TIFF για την προετοιμασία τους για μετατροπή.

1. **Εισαγωγή απαιτούμενων κλάσεων:**

   ```java
   import com.aspose.imaging.fileformats.tiff.TiffFrame;
   ```

2. **Επανάληψη σε Πλαίσια:**

   ```java
   try (TiffImage multiImage = (TiffImage) Image.load("YOUR_DOCUMENT_DIRECTORY/sample.tif")) {
       int frameCounter = 0;
       for (TiffFrame tiffFrame : multiImage.getFrames()) {
           multiImage.setActiveFrame(tiffFrame);
           frameCounter++;
       }
   }
   ```

3. **Εξήγηση:**

   - `getFrames()` Ανακτά όλα τα καρέ μέσα στην εικόνα TIFF.
   - `setActiveFrame()` ορίζει το τρέχον ενεργό πλαίσιο για επεξεργασία.

### Μετατροπή πλαισίου TIFF σε JPEG

**Επισκόπηση:**  
Η μετατροπή κάθε καρέ TIFF σε μορφή JPEG περιλαμβάνει τη φόρτωση δεδομένων pixel και την αποθήκευσή τους ως νέο αρχείο JPEG.

1. **Εισαγωγή απαιτούμενων κλάσεων:**

   ```java
   import com.aspose.imaging.Color;
   import com.aspose.imaging.fileformats.tiff.TiffFrame;
   import com.aspose.imaging.Image;
   import com.aspose.imaging.imageoptions.JpegOptions;
   import com.aspose.imaging.sources.FileCreateSource;
   ```

2. **Μετατροπή κάθε καρέ:**

   ```java
   try (TiffImage multiImage = (TiffImage) Image.load("YOUR_DOCUMENT_DIRECTORY/sample.tif")) {
       int frameCounter = 0;
       for (TiffFrame tiffFrame : multiImage.getFrames()) {
           multiImage.setActiveFrame(tiffFrame);
           Color[] pixels = multiImage.loadPixels(tiffFrame.getBounds());

           JpegOptions jpgCreateOptions = new JpegOptions();
           jpgCreateOptions.setSource(new FileCreateSource("YOUR_OUTPUT_DIRECTORY/Concat" + frameCounter + ".jpg", false));

           com.aspose.imaging.fileformats.jpeg.JpegImage jpgImage = 
               (com.aspose.imaging.fileformats.jpeg.JpegImage) Image.create(jpgCreateOptions, tiffFrame.getWidth(), tiffFrame.getHeight());

           try {
               jpgImage.savePixels(tiffFrame.getBounds(), pixels);
               jpgImage.save();
           } finally {
               jpgImage.close();
               jpgCreateOptions.close();
           }

           frameCounter++;
       }
   }
   ```

3. **Εξήγηση:**

   - `loadPixels()` εξάγει δεδομένα pixel από το τρέχον καρέ TIFF.
   - `JpegOptions` έχει ρυθμιστεί ώστε να καθορίζει πού και πώς θα αποθηκευτεί το αρχείο JPEG.
   - Ένα νέο `JpegImage` Το αντικείμενο δημιουργείται με διαστάσεις που ταιριάζουν με αυτές του αρχικού πλαισίου TIFF.
