## Οδηγός Εφαρμογής

### Φόρτωση και αλλαγή μεγέθους εικόνας

**Επισκόπηση:**
Το πρώτο βήμα περιλαμβάνει τη φόρτωση μιας εικόνας από ένα σύστημα αρχείων και την αλλαγή μεγέθους της χρησιμοποιώντας τη μέθοδο αναδειγματοληψίας του πλησιέστερου γείτονα.

#### Βήμα 1: Φόρτωση εικόνας
```java
import com.aspose.imaging.Image;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Image tempImage = Image.load(dataDir + "/Image1.png");
```
- **Εξήγηση**: Το `load` Η μέθοδος διαβάζει την εικόνα από τον καθορισμένο κατάλογο. Βεβαιωθείτε ότι η διαδρομή σας είναι σωστή για να αποφύγετε σφάλματα "το αρχείο δεν βρέθηκε".

#### Βήμα 2: Αλλαγή μεγέθους εικόνας
```java
int width = tempImage.getWidth();
int height = tempImage.getHeight();
tempImage.resize(width, height, ResizeType.NearestNeighbourResample);
```
- **Εξήγηση**: Αλλαγή μεγέθους με `ResizeType.NearestNeighbourResample` Διατηρεί δεδομένα pixel χωρίς παρεμβολή, διατηρώντας την ποιότητα της εικόνας σε ορισμένες περιπτώσεις.

### Δημιουργία επιλογών TIFF και πηγής

**Επισκόπηση:**
Στη συνέχεια, ορίζουμε τις απαραίτητες επιλογές για να δημιουργήσουμε ένα νέο αρχείο TIFF πολλαπλών σελίδων. Αυτό περιλαμβάνει τον καθορισμό της αναμενόμενης μορφής και της διαμόρφωσης πηγής.

#### Βήμα 1: Ρύθμιση επιλογών TIFF
```java
import com.aspose.imaging.fileformats.tiff.TiffOptions;
import com.aspose.imaging.fileformats.tiff.enums.TiffExpectedFormat;
import com.aspose.imaging.sources.FileCreateSource;

TiffOptions tiffOptions = new TiffOptions(TiffExpectedFormat.Default);
tiffOptions.setSource(new FileCreateSource("YOUR_DOCUMENT_DIRECTORY/MultiPage.tiff", false));
```
- **Εξήγηση**: `TiffOptions` ρυθμίστε τον τρόπο δημιουργίας του αρχείου TIFF. Η παράμετρος source υποδεικνύει πού θα αποθηκευτεί το αρχείο TIFF πολλαπλών σελίδων.

### Δημιουργία και αρχικοποίηση μιας εικόνας TIFF πολλαπλών σελίδων

**Επισκόπηση:**
Τώρα, δημιουργούμε μια παρουσία μιας εικόνας TIFF πολλαπλών σελίδων με συγκεκριμένες διαστάσεις.

#### Βήμα 1: Αρχικοποίηση της εικόνας TIFF
```java
import com.aspose.imaging.fileformats.tiff.TiffImage;

TiffImage tiffImage = (TiffImage) Image.create(tiffOptions, width, height);
```
- **Εξήγηση**: Το `create` Η μέθοδος αρχικοποιεί ένα νέο αρχείο TIFF πολλαπλών σελίδων με διαστάσεις που προέρχονται από την εικόνα που έχουμε αλλάξει μέγεθος.

### Επεξεργασία πολλαπλών εικόνων και προσθήκη πλαισίων στο TIFF

**Επισκόπηση:**
Αυτή η ενότητα περιλαμβάνει την προσθήκη πολλαπλών εικόνων ως καρέ στο αρχείο TIFF πολλαπλών σελίδων.

#### Βήμα 1: Φόρτωση επιπλέον εικόνων
```java
import java.io.File;
import java.io.FilenameFilter;
import com.aspose.imaging.fileformats.tiff.TiffFrame;

File di = new File(dataDir);
String[] files = di.list(new FilenameFilter() {
    @Override
    public boolean accept(File dir, String name) {
        return name.endsWith(".img");
    }
});

if (files != null) {
    int index = 0;
    for (String file : files) {
        try (Image inputImage = Image.load(dataDir + "/" + file)) {
            if (index > 0) {
                TiffFrame newframe = new TiffFrame(tiffOptions, width, height);
                tiffImage.addFrame(newframe);
            }
            TiffFrame frame = tiffImage.getFrames()[index];
            frame.savePixels(frame.getBounds(), ((RasterImage) inputImage).loadPixels(inputImage.getBounds()));
            index += 1;
        }
    }
}
```
- **Εξήγηση**: Αυτό το μπλοκ επαναλαμβάνεται πάνω από αρχεία στον κατάλογο που τελειώνουν σε `.img`, προσθέτοντας το καθένα ως νέο καρέ στην εικόνα TIFF. `savePixels` Η μέθοδος διασφαλίζει ότι τα δεδομένα pixel από κάθε εικόνα που φορτώνεται μεταφέρονται με ακρίβεια.

### Αποθήκευση της εικόνας TIFF πολλαπλών σελίδων

**Επισκόπηση:**
Τέλος, αποθηκεύστε το αρχείο TIFF πολλαπλών σελίδων για να διατηρήσετε όλες τις αλλαγές που έγιναν κατά την επεξεργασία.

#### Βήμα 1: Αποθηκεύστε την εργασία σας
```java
import com.aspose.imaging.examples.Utils;

tiffImage.save("YOUR_OUTPUT_DIRECTORY/output.tiff");
```
- **Εξήγηση**: Το `save` Η μέθοδος γράφει την πλήρη εικόνα TIFF πολλαπλών σελίδων στον καθορισμένο κατάλογο εξόδου. Βεβαιωθείτε ότι αυτή η διαδρομή είναι προσβάσιμη και εγγράψιμη.
