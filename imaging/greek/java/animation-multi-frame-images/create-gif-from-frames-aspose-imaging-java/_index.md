## Οδηγός Εφαρμογής

### Δημιουργία GIF από πολλαπλά καρέ

Η δημιουργία ενός GIF από πολλά καρέ περιλαμβάνει τη φόρτωση κάθε καρέ, τη διαμόρφωση των ρυθμίσεων GIF και την αποθήκευση του τελικού αποτελέσματος. Δείτε πώς μπορείτε να το πετύχετε αυτό:

#### Φόρτωση πλαισίων

1. **Προσδιορισμός καταλόγου πλαισίων**Βεβαιωθείτε ότι όλα τα καρέ εικόνας σας είναι αποθηκευμένα σε έναν μόνο κατάλογο.

   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY/Animation frames";
   ```

2. **Φόρτωση πλαισίων**: Χρήση `Iterable<RasterImage>` για να φορτώσετε κάθε πλαίσιο από τον κατάλογο.

   ```java
   Iterable<RasterImage> frames = loadFrames(dataDir);
   ```

#### Δημιουργία και προσθήκη πλαισίων

3. **Αρχικοποίηση εικόνας GIF**:

   Ξεκινήστε δημιουργώντας ένα νέο `GifImage` παράδειγμα με το πρώτο καρέ και, στη συνέχεια, επαναλάβετε τα επόμενα καρέ για να τα προσθέσετε.

   ```java
   GifImage image = null;

   for (RasterImage frame : frames) {
       if (image == null) {
           image = new GifImage(new GifFrameBlock(frame));
           continue;
       }
       // Προσθέστε επιπλέον πλαίσια εδώ
   }
   ```

#### Αποθήκευση του GIF

4. **Αποθήκευση της εξόδου**:

   Μόλις προστεθούν όλα τα καρέ, αποθηκεύστε το GIF σας στον καθορισμένο κατάλογο εξόδου.

   ```java
   String outDir = "YOUR_OUTPUT_DIRECTORY";
   image.save(outDir + "/output.gif");
   ```

### Επεξήγηση Βασικών Βημάτων

- **GifFrameBlock**Αυτή η κλάση ενσωματώνει μεμονωμένες ρυθμίσεις πλαισίου. Κατανοήστε τις παραμέτρους του για προσαρμοσμένες διαμορφώσεις.
- **Ποιότητα εικόνας και βελτιστοποίηση**Προσαρμόστε τις ρυθμίσεις για να εξισορροπήσετε την ποιότητα και το μέγεθος αρχείου ανάλογα με τις ανάγκες σας.
