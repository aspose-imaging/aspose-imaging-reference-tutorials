## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στη διαδικασία βήμα προς βήμα για τη μετατροπή πλαισίων TIFF σε αρχεία BMP χρησιμοποιώντας το Aspose.Imaging Java.

### Χαρακτηριστικό: Εξαγωγή και αποθήκευση πλαισίων TIFF ως BMP (H2)

#### Επισκόπηση
Θα φορτώσουμε μια εικόνα TIFF πολλαπλών καρέ, θα την επαναλάβουμε σε κάθε καρέ και θα την αποθηκεύσουμε ως μεμονωμένα αρχεία BMP. Αυτή η λειτουργία είναι ιδιαίτερα χρήσιμη για εφαρμογές που απαιτούν επεξεργασία ή αρχειοθέτηση ειδικά για κάθε καρέ.

##### Βήμα 1: Φόρτωση της εικόνας TIFF

Ξεκινήστε φορτώνοντας την εικόνα TIFF χρησιμοποιώντας το Aspose.Imaging. `TiffImage` τάξη. Βεβαιωθείτε ότι θα αντικαταστήσετε `"YOUR_DOCUMENT_DIRECTORY/sample.tiff"` με την πραγματική διαδρομή του αρχείου σας.

```java
try (TiffImage multiImage = (TiffImage) Image.load("YOUR_DOCUMENT_DIRECTORY/sample.tiff")) {
    // Προχωρήστε στην επεξεργασία κάθε πλαισίου.
}
```

##### Βήμα 2: Επανάληψη σε πλαίσια TIFF

Χρησιμοποιήστε ένα `for` βρόχος για να περάσετε από κάθε `TiffFrame`Αυτό σας επιτρέπει να χειρίζεστε τα πλαίσια ξεχωριστά.

```java
int frameCounter = 0;
for (TiffFrame tiffFrame : multiImage.getFrames()) {
    // Επεξεργαστείτε κάθε πλαίσιο.
}
```

##### Βήμα 3: Εξαγωγή και αποθήκευση κάθε καρέ ως BMP

Για κάθε καρέ, ορίστε το ως ενεργό, εξαγάγετε τα pixel του και αποθηκεύστε τα χρησιμοποιώντας `BmpOptions`.

```java
multiImage.setActiveFrame(tiffFrame);
Color[] pixels = multiImage.loadPixels(tiffFrame.getBounds());

try (BmpOptions bmpCreateOptions = new BmpOptions()) {
    bmpCreateOptions.setBitsPerPixel(24);
    String outputPath = String.format("%sConcatExtractTIFFFramesToBMP_out%d.bmp", "YOUR_OUTPUT_DIRECTORY/", frameCounter);

    bmpCreateOptions.setSource(new FileCreateSource(outputPath, false));
    try (BmpImage bmpImage = (BmpImage) Image.create(bmpCreateOptions, tiffFrame.getWidth(), tiffFrame.getHeight())) {
        bmpImage.savePixels(tiffFrame.getBounds(), pixels);
        bmpImage.save(outputPath);  // Αποθηκεύστε το αρχείο BMP.
    }
}
frameCounter++;
```
