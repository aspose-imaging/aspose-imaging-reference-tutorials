### Φόρτωση SVG και εξαγωγή ενσωματωμένων εικόνων

#### Επισκόπηση

Αυτή η λειτουργία μας επιτρέπει να φορτώσουμε ένα συγκεκριμένο αρχείο SVG και να έχουμε πρόσβαση σε οποιεσδήποτε ενσωματωμένες εικόνες ράστερ που περιέχει.

#### Βήματα Υλοποίησης

1. **Ορίστε τη διαδρομή εισόδου:**

   Ορίστε τη διαδρομή καταλόγου του αρχείου SVG στον κώδικά σας:

   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY/svg/";
   String fileName = dataDir + "test2.svg";
   ```

2. **Φόρτωση και μετάδοση της εικόνας:**

   Χρησιμοποιήστε το Aspose.Imaging για να φορτώσετε το SVG και, στη συνέχεια, να το μεταδώσετε σε ένα `VectorImage` τύπος.

   ```java
   try (Image image = Image.load(fileName)) {
       EmbeddedImage[] images = ((VectorImage)image).getEmbeddedImages();
   }
   ```

3. **Εξαγωγή ενσωματωμένων εικόνων:**

   Ο `getEmbeddedImages()` Η μέθοδος επιστρέφει έναν πίνακα ενσωματωμένων εικόνων ράστερ, οι οποίες μπορούν στη συνέχεια να υποβληθούν σε περαιτέρω επεξεργασία.

### Επανάληψη και αποθήκευση ενσωματωμένων εικόνων

#### Επισκόπηση

Αυτή η λειτουργία περιλαμβάνει την επανάληψη κάθε εξαγόμενης εικόνας, τη δημιουργία ενός μοναδικού ονόματος αρχείου και την αποθήκευση των εικόνων στην επιθυμητή τοποθεσία.

#### Βήματα Υλοποίησης

1. **Ρύθμιση διαδρομής εξόδου:**

   Ορίστε πού θέλετε να αποθηκεύσετε τις εξαγόμενες εικόνες:

   ```java
   String outputFolder = "YOUR_OUTPUT_DIRECTORY/svg/";
   ```

2. **Επανάληψη πάνω από εικόνες:**

   Βρόχος μέσα από κάθε μία `EmbeddedImage` αντικείμενο και εξαγωγή των δεδομένων εικόνας του.

   ```java
   List<String> files = new ArrayList<>();
   int i = 0;
   
   for (EmbeddedImage im : images) {
       Image imImage = im.getImage();
       String outFileName = "svg_image" + i++ + getExtension(imImage.getFileFormat());
       String outFilePath = outputFolder + outFileName;
       files.add(outFilePath);
       
       // Αποθήκευση της εικόνας
       imImage.save(outFilePath);
   }
   ```

3. **Δημιουργία επεκτάσεων αρχείων:**

   Χρησιμοποιήστε μια βοηθητική μέθοδο για να προσδιορίσετε την επέκταση αρχείου με βάση τη μορφή εικόνας.

   ```java
   private static String getExtension(long format) {
       if (format == com.aspose.imaging.FileFormat.Jpeg) {
           return ".jpg";
       } else if (format == com.aspose.imaging.FileFormat.Png) {
           return ".png";
       } else if (format == com.aspose.imaging.FileFormat.Bmp) {
           return ".bmp";
       }
       return "." + com.aspose.imaging.FileFormat.toString(com.aspose.imaging.FileFormat.class, format);
   }
   ```

### Καθαρισμός μετά την επεξεργασία εικόνας

#### Επισκόπηση

Μετά την επεξεργασία των εικόνων, είναι καλή πρακτική να καθαρίζετε τυχόν προσωρινά αρχεία που δημιουργήθηκαν κατά τη διάρκεια της διαδικασίας.

#### Βήματα Υλοποίησης

1. **Λίστα προσωρινών αρχείων:**

   Διατηρήστε μια λίστα με διαδρομές για τα αρχεία που σκοπεύετε να διαγράψετε μετά τη χρήση:

   ```java
   List<String> files = List.of("YOUR_OUTPUT_DIRECTORY/svg/svg_image0.jpg");
   ```

2. **Διαγραφή προσωρινών αρχείων:**

   Επαναλάβετε τη λίστα αρχείων και αφαιρέστε το καθένα.

   ```java
   for (String file : files) {
       File tempFile = new File(file);
       if (tempFile.exists()) {
           boolean deleted = tempFile.delete();
           if (!deleted) {
               System.err.println("Failed to delete " + file);
           }
       }
   }
   ```
