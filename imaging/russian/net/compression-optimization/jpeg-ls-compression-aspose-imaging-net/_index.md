## Руководство по внедрению

Мы разделим нашу реализацию на два основных раздела: сжатие JPEG-LS с уменьшенным количеством бит на канал и преобразование JPEG-LS в PNG для визуального сравнения.

### Функция 1: Сжатие JPEG-LS с уменьшенным количеством бит на канал

#### Обзор
Эта функция демонстрирует сжатие изображения PNG с использованием формата JPEG-LS при уменьшении количества бит на канал, что оптимизирует дисковое пространство без существенной потери качества.

**Пошаговая реализация**

##### Шаг 1: Настройте пути к файлам и загрузите изображение
```csharp
// Определить входные и выходные пути
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string originPngFileName = System.IO.Path.Combine(dataDir, "lena_16g_lin.png");
string outputJpegFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.jls";

// Загрузите исходное изображение PNG
using (PngImage pngImage = (PngImage)Image.Load(originPngFileName))
{
    // Перейти к настройке сжатия JPEG-LS
}
```
**Объяснение:** Настройте пути к файлам и загрузите входное изображение PNG с помощью Aspose.Imaging. `Image.Load()` метод.

##### Шаг 2: Настройка параметров сжатия
```csharp
JpegOptions jpegOptions = new JpegOptions();
jpegOptions.BitsPerChannel = (byte)2; // Для демонстрации уменьшите до 2 бит на канал.
jpegOptions.CompressionType = JpegCompressionMode.JpegLs;
```
**Объяснение:** Инициализировать `JpegOptions` и настройте тип сжатия как JPEG-LS с уменьшенным количеством бит на канал.

##### Шаг 3: Сохраните сжатое изображение
```csharp
// Сохраните изображение в формате JPEG-LS
pngImage.Save(outputJpegFileName, jpegOptions);
```
**Объяснение:** Используйте `Save()` метод сохранения сжатого изображения в формате JPEG-LS. Этот шаг завершает процесс сжатия.

#### Советы по устранению неполадок
- Убедитесь, что путь к входному PNG-файлу указан правильно.
- Убедитесь, что библиотеки Aspose.Imaging правильно установлены и на них есть ссылки.

### Функция 2: Преобразование JPEG-LS в PNG для визуального сравнения

#### Обзор
После сжатия изображения его обратное преобразование в формат PNG позволяет сравнить визуальное качество до и после сжатия.

**Пошаговая реализация**

##### Шаг 1: Загрузите сжатое изображение
```csharp
string outputJpegFileName = "YOUR_DOCUMENT_DIRECTORY/lena24b 2-bit Gold.jls";
string outputPngFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.png";

// Загрузите сжатое изображение JPEG-LS
using (JpegImage jpegImage = (JpegImage)Image.Load(outputJpegFileName))
{
    // Перейти к настройке преобразования PNG
}
```
**Объяснение:** Загрузите ранее сохраненный файл JPEG-LS с помощью `Image.Load()`.

##### Шаг 2: Настройте параметры конвертации
```csharp
PngOptions pngOptions = new PngOptions();
```
**Объяснение:** Инициализировать `PngOptions` для сохранения в формате PNG.

##### Шаг 3: Сохранить как PNG
```csharp
// Конвертируйте и сохраните изображение обратно в формат PNG.
jpegImage.Save(outputPngFileName, pngOptions);
```
**Объяснение:** Использовать `Save()` для сохранения файла JPEG-LS обратно в формат PNG, что позволяет проводить визуальное сравнение.

#### Советы по устранению неполадок
- Убедитесь, что пути для входных и выходных файлов заданы правильно.
- Убедитесь, что Aspose.Imaging правильно настроен в вашем проекте.
