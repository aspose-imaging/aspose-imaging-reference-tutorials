## Руководство по внедрению
Мы разберем процесс преобразования на отдельные функции: несжатое преобразование DICOM, сжатие JPEG, сжатие JPEG2000 и сжатие RLE.

### Несжатое преобразование DICOM
Эта функция позволяет преобразовывать изображения без применения сжатия, сохраняя исходное качество.

#### Обзор
Преобразование изображения в несжатый формат DICOM идеально подходит, когда важно сохранить максимальную детализацию.

#### Этапы внедрения
1. **Загрузить изображение**: 
   ```csharp
   string inputFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "original.jpg");
   ```
2. **Установить параметры конвертации**:
   ```csharp
   var dicomOptions = new DicomOptions
   {
       ColorType = ColorType.Rgb24Bit,
       Compression = new Compression { Type = CompressionType.None }
   };
   ```
3. **Сохраните файл DICOM**:
   ```csharp
   string outputUncompressedDicom = Path.Combine("YOUR_OUTPUT_DIRECTORY", "original_Uncompressed.dcm");
   using (var inputImage = Image.Load(inputFile))
   {
       inputImage.Save(outputUncompressedDicom, dicomOptions);
   }
   ```

#### Основные параметры конфигурации
- **ЦветТип**: Установить на `Rgb24Bit` для высококачественного представления изображения.
- **Тип сжатия**: `None`, гарантируя отсутствие потери данных.

### Преобразование сжатого JPEG в DICOM
Сжатие JPEG значительно уменьшает размер файла, сохраняя при этом качество, что делает его пригодным для веб-приложений и оптимизации хранения.

#### Обзор
Этот метод сжимает изображение с использованием стандартов JPEG перед конвертацией в формат DICOM.

#### Этапы внедрения
1. **Установить параметры сжатия JPEG**:
   ```csharp
   var dicomOptions = new DicomOptions
   {
       ColorType = ColorType.Rgb24Bit,
       Compression = new Compression { Type = CompressionType.Jpeg }
   };
   ```
2. **Сохраните сжатый файл DICOM**:
   ```csharp
   string outputJpegDicom = Path.Combine("YOUR_OUTPUT_DIRECTORY", "original_JPEG.dcm");
   using (var inputImage = Image.Load(inputFile))
   {
       inputImage.Save(outputJpegDicom, dicomOptions);
   }
   ```

### Преобразование сжатого DICOM-файла JPEG2000
JPEG2000 обеспечивает более высокую эффективность сжатия и качество изображения по сравнению со стандартным JPEG, идеально подходит для изображений с высоким разрешением.

#### Обзор
Используйте расширенные возможности сжатия с такими опциями, как выбор кодека и необратимые настройки.

#### Этапы внедрения
1. **Настройте параметры JPEG2000**:
   ```csharp
   var dicomOptions = new DicomOptions
   {
       ColorType = ColorType.Rgb24Bit,
       Compression = new Compression
                      {
                          Type = CompressionType.Jpeg2000,
                          Jpeg2000 = new Jpeg2000Options
                                       {
                                           Codec = Jpeg2000Codec.Jp2,
                                           Irreversible = false
                                       }
                      }
   };
   ```
2. **Сохраните сжатый файл DICOM JPEG2000**:
   ```csharp
   string outputJpeg2000Dicom = Path.Combine("YOUR_OUTPUT_DIRECTORY", "original_JPEG2000.dcm");
   using (var inputImage = Image.Load(inputFile))
   {
       inputImage.Save(outputJpeg2000Dicom, dicomOptions);
   }
   ```

### Преобразование сжатого RLE DICOM
Кодирование длин серий (RLE) — это метод сжатия без потерь, который идеально подходит для медицинских изображений, где важна каждая деталь.

#### Обзор
Такое преобразование гарантирует отсутствие потери данных и оптимизирует хранение за счет эффективного кодирования.

#### Этапы внедрения
1. **Установить параметры сжатия RLE**:
   ```csharp
   var dicomOptions = new DicomOptions
   {
       ColorType = ColorType.Rgb24Bit,
       Compression = new Compression { Type = CompressionType.Rle }
   };
   ```
2. **Сохраните сжатый файл DICOM RLE**:
   ```csharp
   string outputRleDicom = Path.Combine("YOUR_OUTPUT_DIRECTORY", "original_RLE.dcm");
   using (var inputImage = Image.Load(inputFile))
   {
       inputImage.Save(outputRleDicom, dicomOptions);
   }
   ```
