## Руководство по внедрению
Теперь давайте сосредоточимся на реализации функции загрузки и сохранения изображений DICOM.
### Загрузка и сохранение изображения DICOM
#### Обзор
В этом разделе демонстрируется загрузка изображения DICOM из файла и сохранение его в формате PNG. Это упрощает обработку медицинских изображений для дальнейшей обработки или отображения в не-DICOM приложениях.
#### Загрузка изображения DICOM
Сначала вам необходимо загрузить изображение DICOM с помощью `Image.Load` метод:

```csharp
using Aspose.Imaging;
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
var inputPath = Path.Combine(dataDir, "input.dcm");

// Загрузите изображение DICOM по указанному пути.
using (var image = Image.Load(inputPath))
{
    // При необходимости продолжите обработку или сохранение.
}
```
**Объяснение:**  
- `Image.Load` используется для открытия и загрузки файла DICOM. Загруженный объект изображения затем может быть обработан или сохранен.
#### Сохранение в формате PNG
После загрузки вы можете сохранить изображение в другом формате, например PNG:

```csharp
string outputPath = Path.Combine(dataDir, "output.png");
image.Save(outputPath);
```
**Объяснение:**  
- `image.Save` Метод записывает загруженное изображение в файл. Здесь он сохраняет DICOM-изображение как PNG.
#### Очистка
При желании удалите сохраненный PNG-файл, если он больше не нужен:

```csharp
File.Delete(Path.Combine(dataDir, "output.png"));
```
### Советы по устранению неполадок
- **Отсутствующие библиотеки DLL:** Убедитесь, что все зависимости Aspose.Imaging указаны правильно.
- **Неверный путь к файлу:** Проверьте правильность и доступность пути к входному файлу DICOM.
- **Проблемы с разрешениями:** Убедитесь, что ваше приложение имеет необходимые разрешения на чтение/запись файлов в указанном каталоге.
