## Руководство по внедрению

### Особенность: Бинаризация с адаптивным порогом Брэдли
**Обзор:**
Эта функция преобразует изображение DICOM в двоичный формат с использованием адаптивного порога Брэдли, усиливая локальный контраст и улучшая результаты анализа.

#### Шаг 1: Откройте файл DICOM
Сначала откройте ваш DICOM-файл, указав путь к нему. Заменить `'YOUR_DOCUMENT_DIRECTORY'` с фактическим каталогом вашего документа.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string inputFile = Path.Combine(dataDir, "image.dcm");

using (var fileStream = new FileStream(inputFile, FileMode.Open, FileAccess.Read))
{
    // Перейти к шагу 2
}
```
*Почему?*Открытие файла в потоке обеспечивает эффективное чтение и обработку без загрузки всего содержимого в память.

#### Шаг 2: Загрузка изображения из потока с использованием класса DicomImage
Загрузите изображение с помощью `DicomImage`, который предоставляет методы, специфичные для файлов DICOM.

```csharp
using (DicomImage image = new DicomImage(fileStream))
{
    // Перейти к шагу 3
}
```
*Почему?*: На этом этапе данные DICOM подготавливаются к обработке, что позволяет выполнять различные преобразования и анализы.

#### Шаг 3: Примените адаптивный порог Брэдли для бинаризации изображения
Бинаризация повышает контрастность изображения, устанавливая пиксели выше порогового значения белыми, а пиксели ниже — черными. Параметр `10` настраивает этот процесс.

```csharp
image.BinarizeBradley(10);
```
*Почему?*: Метод Брэдли адаптируется к локальным изменениям яркости, что делает его идеальным для медицинских изображений с неравномерным освещением.

#### Шаг 4: Сохраните полученное двоичное изображение в формате BMP.
Наконец, сохраните обработанное изображение. Заменить `'YOUR_OUTPUT_DIRECTORY'` с указанием того, где вы хотите сохранить вывод.

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDir, "BinarizationWithBradleysAdaptiveThreshold_out.bmp");
image.Save(outputFile, new BmpOptions());
```
*Почему?*Формат BMP широко поддерживается и обеспечивает простой способ визуализации двоичных результатов.
