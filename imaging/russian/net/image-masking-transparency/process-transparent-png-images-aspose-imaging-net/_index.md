## Руководство по внедрению

Мы разделим процесс на две основные функции: загрузку PNG-изображения и создание нового изображения с прозрачностью.

### Загрузка и обработка изображения PNG

#### Обзор
Эта функция позволяет вам загружать существующий файл PNG, извлекать данные пикселей и сохранять их размеры. Это необходимо для операций, требующих прямого манипулирования пикселями изображения.

**Предпринимаемые шаги:**
##### Загрузите изображение PNG
1. **Загрузите изображение в объект RasterImage:**
   ```csharp
   string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
   using (RasterImage raster = (RasterImage)Image.Load(documentDirectory + "/aspose_logo.png"))
   {
       // Извлечение размеров изображения и данных о пикселях.
       int width = raster.Width;
       int height = raster.Height;
       Color[] pixels = raster.LoadPixels(new Rectangle(0, 0, width, height));
   }
   ```

##### Объяснение
- **Растровое изображение**: Этот класс представляет загруженное изображение и предоставляет методы для непосредственной работы с его содержимым.
- **Метод LoadPixels**: Извлекает данные пикселей в `Color` массив для дальнейшей обработки.

### Создание и сохранение изображения PNG с прозрачностью

#### Обзор
После обработки изображения вы можете захотеть сохранить его с определенными настройками прозрачности. Эта функция демонстрирует, как создать новое изображение PNG, применить прозрачность и сохранить его как файл JPEG.

**Предпринимаемые шаги:**
##### Инициализировать объект PngImage
1. **Создайте новое изображение PNG:**
   ```csharp
   string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
   using (PngImage png = new PngImage(width, height, PngColorType.TruecolorWithAlpha))
   {
       // Примените данные пикселей и настройки прозрачности.
       png.SavePixels(new Rectangle(0, 0, width, height), pixels);
       png.TransparentColor = Color.Black;
       png.HasTransparentColor = true;

       // Сохраните PNG как JPEG с информацией о прозрачности.
       png.Save(outputDirectory + "/SpecifyTransparencyforPNGImages_out.jpg");
   }
   ```

##### Объяснение
- **PNGImage**: Представляет новое создаваемое изображение. Поддерживает различные типы цветов, включая альфа для прозрачности.
- **Прозрачный цвет**: Устанавливает, какой цвет следует считать прозрачным на изображении.
