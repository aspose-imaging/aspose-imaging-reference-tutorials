## Руководство по внедрению

### Преобразование изображения

**Обзор:**
Конвертация изображений необходима для преобразования изображений в различные форматы и их оптимизации для различных платформ или приложений.

#### Пошаговый процесс:

**1. Загрузите изображение**
```csharp
using (var image = Image.Load("input.jpg"))
{
    // Продолжить обработку
}
```
*Почему?* Загрузка изображения инициализирует его в памяти для дальнейших манипуляций.

**2. Конвертировать и сохранить**
```csharp
image.Save("output.png", new PngOptions());
```
*Параметры и назначение:* The `Save` метод сохраняет изображение в указанном формате, в данном случае преобразуя в PNG с параметрами по умолчанию.

### Манипуляция с изображениями

**Обзор:**
Манипулируйте изображениями, изменяя их размер, обрезая или применяя фильтры.

#### Пошаговый процесс:

**1. Изменение размера изображения**
```csharp
using (var image = Image.Load("input.jpg"))
{
    image.Resize(200, 200);
    image.Save("resized.jpg");
}
```
*Объяснение:* Этот фрагмент изменяет размер изображения до 200x200 пикселей.

### Советы по устранению неполадок

- **Распространенная проблема:** Ошибки «Файл не найден» могут возникать, если пути указаны неверно.
- **Решение:** Еще раз проверьте пути к файлам и убедитесь, что изображения существуют в указанном каталоге.
