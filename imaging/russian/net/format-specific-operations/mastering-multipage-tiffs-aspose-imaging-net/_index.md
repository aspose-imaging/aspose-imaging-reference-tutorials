## Руководство по внедрению

### Загрузить несколько изображений (H2)
**Обзор:**
Загрузка нескольких изображений в память является основополагающим шагом во многих задачах обработки изображений. Эта функция позволяет выполнять пакетные операции, что упрощает одновременную обработку нескольких файлов.

#### Шаг 1: Определите пути к каталогам
```csharp
string baseFolder = "YOUR_DOCUMENT_DIRECTORY";
```
Это задает путь к каталогу, в котором находятся входные изображения.

#### Шаг 2: Список файлов и загрузка изображений
```csharp
string[] files = new string[]
{
    "33266.tif", "Animation.gif", "elephant.png",
    "MultiPage.cdr"
};

List<Image> images = new List<Image>();
foreach (var file in files)
{
    string filePath = Path.Combine(baseFolder, file);
    images.Add(Image.Load(filePath));
}
```
Здесь мы создаем список путей к изображениям и загружаем их с помощью `Image.Load()`. Этот метод эффективно считывает каждый файл в память.

### Создать многостраничное изображение TIFF (H2)
**Обзор:**
Создание многостраничного TIFF подразумевает компиляцию нескольких одностраничных изображений в один файл. Это особенно полезно для архивации документов или сокращения количества управляемых файлов.

#### Шаг 3: Объедините изображения
```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "MultipageImageCreateTest.tif");

using (Image image = Image.Create(images.ToArray(), true))
{
    image.Save(outputFilePath, new TiffOptions(TiffExpectedFormat.TiffJpegRgb));
}
```
Этот фрагмент кода создает многостраничный TIFF-файл с использованием загруженных изображений и сохраняет его в формате JPEG RGB.

### Советы по устранению неполадок
- Убедитесь, что пути к файлам указаны правильно, чтобы избежать `FileNotFoundException`.
- Убедитесь, что у вас есть права на запись в выходной каталог.
- Обрабатывать исключения при загрузке изображений с поврежденными файлами.
