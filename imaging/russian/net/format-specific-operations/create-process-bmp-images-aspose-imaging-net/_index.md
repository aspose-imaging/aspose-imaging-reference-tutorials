### Функция: Создание и обработка изображений

#### Обзор

Создание изображения BMP с определенными настройками позволяет вам адаптировать графику в соответствии с вашими требованиями. В этом руководстве показано, как использовать Aspose.Imaging для создания файла изображения, установки его значений пикселей и эффективного сохранения.

#### Шаг 1: Настройка проекта и создание изображения

Во-первых, убедитесь, что вы указали пути для каталога документов и выходного каталога:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputPath = @"YOUR_OUTPUT_DIRECTORY";

// Создайте путь для файла изображения.
string imageDataPath = System.IO.Path.Combine(documentDirectory, "sample.bmp");
```

#### Шаг 2: Создание изображения BMP с определенными параметрами

Начнем с создания экземпляра `BmpOptions` и настроив его на использование 32 бит на пиксель:

```csharp
using (System.IO.FileStream fileStream = System.IO.File.Create(imageDataPath))
{
    using (BmpOptions bmpOptions = new BmpOptions())
    {
        bmpOptions.BitsPerPixel = 32;
        bmpOptions.Source = new StreamSource(fileStream);
        
        // Создайте изображение с указанными размерами.
        using (RasterImage image = (RasterImage)Image.Create(bmpOptions, 10, 10))
        {
            Color[] pixels = new Color[4];
            for (int i = 0; i < 4; ++i)
            {
                // Установите цвет пикселя, используя значения ARGB.
                pixels[i] = Color.FromArgb(40, 30, 20, 10);
            }

            // Сохраните указанные пиксели в прямоугольной области изображения.
            image.SavePixels(new Rectangle(0, 0, 2, 2), pixels);

            // Сохраните обработанное изображение в выходной папке.
            string processedImagePath = System.IO.Path.Combine(outputPath, "processed_image.bmp");
            image.Save(processedImagePath);
        }
    }
}
```

#### Объяснение

- **BmpOptions**: Настраивает характеристики файла BMP, такие как глубина цвета. Здесь мы устанавливаем `BitsPerPixel` до 32 для более насыщенной цветопередачи.
  
- **Источник потока**используется как источник пиксельных данных, что позволяет нам работать напрямую с потоками.

- **Метод SavePixels**: Этот метод имеет решающее значение для установки определенных пикселей в пределах определенного прямоугольника на вашем изображении.

#### Шаг 3: Уборка

Убедитесь, что временные файлы удалены после обработки:

```csharp
finally

    System.IO.File.Delete(imageDataPath);
}
```
