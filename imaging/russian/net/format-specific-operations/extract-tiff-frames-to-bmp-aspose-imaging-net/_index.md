## Руководство по внедрению

### Извлечение кадров TIFF в файлы BMP

Эта функция позволяет извлекать каждый кадр из изображения TIFF и сохранять его как отдельный файл BMP. Давайте разберем процесс:

#### Загрузите изображение TIFF

Начните с загрузки многокадрового изображения TIFF в память.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (TiffImage multiImage = (TiffImage)Aspose.Imaging.Image.Load(Path.Combine(dataDir, "SampleTiff1.tiff")))
{
    // Далее следует код обработки...
}
```

#### Итерация по кадрам

Просмотрите каждый кадр изображения TIFF и обработайте его.

```csharp
int frameCounter = 0;
foreach (TiffFrame tiffFrame in multiImage.Frames)
{
    multiImage.ActiveFrame = tiffFrame;
    
    // Загрузка пикселей из TiffFrame в массив цветов
    Color[] pixels = multiImage.LoadPixels(tiffFrame.Bounds);
    
    // Логика настройки и сохранения следующая...
}
```

#### Настроить параметры BMP

Настройте параметры создания BMP-файла, указав количество бит на пиксель и выходной путь.

```csharp
BmpOptions bmpCreateOptions = new BmpOptions
{
    BitsPerPixel = 24,
    Source = new FileCreateSource(
        Path.Combine("YOUR_OUTPUT_DIRECTORY", string.Format("ExtractedFrame_out{0}.bmp", frameCounter)),
        false)
};
```

#### Создать и сохранить изображение BMP

Наконец, создайте новое изображение BMP для каждого кадра TIFF и сохраните его.

```csharp
using (BmpImage bmpImage = (BmpImage)Aspose.Imaging.Image.Create(bmpCreateOptions, tiffFrame.Width, tiffFrame.Height))
{
    // Сохраните пиксели из TiffFrame в изображение BMP
    bmpImage.SavePixels(tiffFrame.Bounds, pixels);
    
    // Сохранить файл BMP на диске
    bmpImage.Save();
}
frameCounter++;
```
