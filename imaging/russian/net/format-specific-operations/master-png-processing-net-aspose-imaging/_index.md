## Руководство по внедрению

### Загрузка изображения PNG с помощью Aspose.Imaging

**Обзор:**
Загрузка изображения — первый шаг к любой манипуляции. В этом разделе мы покажем вам, как легко загрузить файл PNG с помощью Aspose.Imaging.

#### Шаг 1: Загрузите изображение
```csharp
using System;
using Aspose.Imaging;

namespace CSharp.ModifyingAndConvertingImages.PNG
{
    public class LoadPngImage
    {
        private string dataDir = "YOUR_DOCUMENT_DIRECTORY/template.png";

        public void Run()
        {
            using (RasterImage image = (RasterImage)Image.Load(dataDir))
            {
                // Изображение теперь загружено и готово к обработке.
            }
        }
    }
}
```
**Объяснение:**
- `Image.Load`: Открывает указанный файл PNG, преобразуя его в `RasterImage` для дальнейших манипуляций.

### Сохранение изображения PNG с параметрами сжатия

**Обзор:**
После загрузки изображения его сохранение с определенными настройками, такими как уровень сжатия и тип цвета, может оптимизировать производительность и качество.

#### Шаг 2: Настройте и сохраните изображение
```csharp
using System;
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Png;

namespace CSharp.ModifyingAndConvertingImages.PNG
{
    public class SavePngWithCompressionOptions
    {
        private string dataDir = "YOUR_DOCUMENT_DIRECTORY/template.png";
        private string outputDir = "YOUR_OUTPUT_DIRECTORY/result.png";

        public void Run()
        {
            using (RasterImage image = (RasterImage)Image.Load(dataDir))
            {
                PngOptions options = new PngOptions
                {
                    CompressionLevel = 8, // Умеренный уровень компрессии.
                    ColorType = PngColorType.IndexedColor,
                    Palette = ColorPaletteHelper.GetCloseTransparentImagePalette(image, 256),
                    FilterType = PngFilterType.Avg,
                };

                image.Save(outputDir, options);
            }
        }
    }
}
```
**Объяснение:**
- **Уровень сжатия**: Установка этого значения `8` обеспечивает баланс между размером файла и качеством.
- **Тип цвета и палитра**: Использование индексированных цветов с прозрачностью помогает уменьшить размер файла, сохраняя при этом визуальную точность.
- **Тип фильтра**: Усредненный фильтр может помочь минимизировать размер файла без существенной потери качества изображения.

### Удаление файла

**Обзор:**
Иногда вам нужно удалить обработанные файлы из вашей системы. В этом разделе показано, как удалить выходной PNG с помощью .NET's `System.IO.File` методы.

#### Шаг 3: Удалить выходное изображение
```csharp
using System;
using System.IO;

namespace CSharp.ModifyingAndConvertingImages.PNG
{
    public class DeleteFile
    {
        private string outputPath = "YOUR_OUTPUT_DIRECTORY/result.png";

        public void Run()
        {
            if (File.Exists(outputPath))
            {
                File.Delete(outputPath);
            }
        }
    }
}
```
**Объяснение:**
- **Файл.Существует и Файл.Удалить**: Эти методы проверяют существование файла и удаляют его, гарантируя, что ваш каталог останется чистым.
