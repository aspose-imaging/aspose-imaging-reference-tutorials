## Руководство по внедрению

### Загрузка и сохранение изображений в формате JPEG Lossless CMYK

В этом разделе показано, как преобразовать JPEG в сжатое без потерь изображение CMYK с пользовательскими цветовыми профилями.

#### Шаг 1: Подготовьте среду
Загрузите необходимые файлы цветового профиля. Убедитесь, что они доступны по указанным вами путям.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
MemoryStream jpegStream = new MemoryStream();
FileStream rgbProfileStream = new FileStream("YOUR_DOCUMENT_DIRECTORY/eciRGB_v2.icc", FileMode.Open);
FileStream cmykProfileStream = new FileStream("YOUR_DOCUMENT_DIRECTORY/ISOcoated_v2_FullGamut4.icc", FileMode.Open);

Sources.StreamSource rgbColorProfile = new Sources.StreamSource(rgbProfileStream);
Sources.StreamSource cmykColorProfile = new Sources.StreamSource(cmykProfileStream);
```

#### Шаг 2: Загрузите и сохраните изображение
Загрузите изображение, примените цветовой режим CMYK со сжатием без потерь и сохраните его в потоке памяти.

```csharp
try
{
    using (JpegImage image = (JpegImage)Image.Load(dataDir + "/056.jpg"))
    {
        JpegOptions options = new JpegOptions();
        options.ColorType = JpegCompressionColorMode.Cmyk; // Установите цветовой режим CMYK.
        options.CompressionType = JpegCompressionMode.Lossless; // Использовать сжатие без потерь

        // Назначьте пользовательские профили RGB и CMYK.
        options.RgbColorProfile = rgbColorProfile;
        options.CmykColorProfile = cmykColorProfile;

        image.Save(jpegStream, options); // Сохраните измененное изображение в потоке памяти.
    }
}
finally
{
    jpegStream.Dispose(); // Утилизация потоков для освобождения ресурсов
    rgbProfileStream.Dispose();
    cmykProfileStream.Dispose();
}
```

#### Шаг 3: Загрузите и преобразуйте изображение
Сбросьте положение потоков и загрузите изображение JPEG без потерь CMYK из потока памяти, преобразуя его в формат PNG.

```csharp
jpegStream.Position = 0;

using (JpegImage image = (JpegImage)Image.Load(jpegStream))
{
    image.RgbColorProfile = rgbColorProfile; // Установить пользовательский профиль RGB для чтения
    image.CmykColorProfile = cmykColorProfile; // Установить пользовательский профиль CMYK для чтения

    image.Save("YOUR_OUTPUT_DIRECTORY/056_cmyk_custom_profiles.png", new PngOptions());
}
```
