## Руководство по внедрению

Вот как сохранить растровое изображение в виде файла TIFF с использованием сжатия AdobeDeflate:

### Шаг 1: Настройка параметров Tiff

Сначала создайте экземпляр `TiffOptions` и настройте его свойства:
```csharp
// Создайте экземпляр TiffOptions с форматом по умолчанию.
TiffOptions options = new TiffOptions(TiffExpectedFormat.Default);

// Установите количество бит на выборку для определения качества изображения (8 бит для RGB).
options.BitsPerSample = new ushort[] { 8, 8, 8 };

// Определим фотометрическую интерпретацию как RGB.
options.Photometric = TiffPhotometrics.Rgb;

// Установите разрешение в дюймах.
options.Xresolution = new TiffRational(72);
options.Yresolution = new TiffRational(72);

// Укажите единицу разрешения (дюймы).
options.ResolutionUnit = TiffResolutionUnits.Inch;

// Выберите непрерывную плоскую конфигурацию для хранения данных изображений.
options.PlanarConfiguration = TiffPlanarConfigs.Contiguous;
```

### Шаг 2: Примените сжатие AdobeDeflate

Установите метод сжатия AdobeDeflate:
```csharp
// Установите сжатие AdobeDeflate для эффективного уменьшения размера файла.
options.Compression = TiffCompressions.AdobeDeflate;
```

### Шаг 3: Загрузите и сохраните изображение.

Загрузите существующее растровое изображение и сохраните его в формате TIFF с указанными параметрами:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Замените на путь к каталогу вашего документа.
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Замените на желаемый путь к выходному каталогу.

// Загрузите существующее изображение.
using (RasterImage image = (RasterImage)Image.Load(dataDir + "SampleTiff1.tiff"))
{
    // Создайте TiffImage из RasterImage и сохраните его, используя параметры, настроенные выше.
    using (TiffImage tiffImage = new TiffImage(new TiffFrame(image)))
    {
        tiffImage.Save(outputDir + "SavingRasterImage_out.tiff", options);
    }
}
```

**Пояснение параметров:**
- `BitsPerSample`: Определяет качество изображения; 8 бит на канал — стандарт для RGB.
- `Photometric`: Указывает интерпретацию цвета (в данном случае RGB).
- `Compression`: выбирает AdobeDeflate для эффективного уменьшения размера файла.
