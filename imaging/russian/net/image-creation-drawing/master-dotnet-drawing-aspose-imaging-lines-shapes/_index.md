Улучшаете свои графические приложения с помощью .NET? Испытываете трудности с рисованием линий, фигур или сохранением их в универсальных форматах, таких как PDF? ## Руководство по внедрению

### Рисование линий с помощью EmfRecorderGraphics2D

В этом разделе мы рассмотрим, как рисовать линии с помощью `EmfRecorderGraphics2D`.

#### Обзор
Мы используем `EmfRecorderGraphics2D` для создания холста, на котором можно рисовать линии разных стилей. Эта функция позволяет настраивать свойства пера, такие как цвет и концевые наконечники.

##### Настройка графической среды

```csharp
using Aspose.Imaging.FileFormats.Emf.Graphics;
using System.Drawing;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputPath = dataDir + "DrawLines_output.emf";

// Инициализируйте EmfRecorderGraphics2D с указанным размером.
EmfRecorderGraphics2D graphics = new EmfRecorderGraphics2D(
    new Rectangle(0, 0, 1000, 1000), 
    new Size(1000, 1000),
    new Size(100, 100)
);
```

##### Рисование линий

###### Нарисуйте простую линию
Начните с создания ручки и рисования базовой линии.

```csharp
Pen pen = new Pen(Color.Bisque);

// Нарисуйте первую линию.
graphics.DrawLine(pen, 1, 1, 50, 50);
```

###### Настройте свойства пера для стильных линий
Измените свойства пера, чтобы рисовать линии разными стилями.

```csharp
pen = new Pen(Color.BlueViolet, 3) { EndCap = LineCap.Round };
graphics.DrawLine(pen, 15, 5, 50, 60);

// Отрегулируйте стиль заглушки.
pen.EndCap = LineCap.Square;
graphics.DrawLine(pen, 5, 10, 50, 10);
```

##### Сохраните свой рисунок

```csharp
graphics.EndRecording().Save(outputPath);
```

### Рисование фигур с помощью HatchBrush и Pen

Далее мы рассмотрим, как создавать фигуры с помощью `HatchBrush`.

#### Обзор
Использование `HatchBrush` позволяет использовать цветные и узорчатые заливки в нарисованных вами фигурах.

##### Инициализировать графическую среду

```csharp
string outputPath = dataDir + "DrawShapes_output.emf";

EmfRecorderGraphics2D graphics = new EmfRecorderGraphics2D(
    new Rectangle(0, 0, 1000, 1000), 
    new Size(1000, 1000),
    new Size(100, 100)
);
```

##### Использование HatchBrush для узоров

```csharp
HatchBrush hatchBrush = new HatchBrush
{
    BackgroundColor = Color.AliceBlue,
    ForegroundColor = Color.Red,
    HatchStyle = HatchStyle.Cross
};

Pen pen = new Pen(hatchBrush, 7);

// Нарисуйте прямоугольник с помощью штриховки.
graphics.DrawRectangle(pen, 50, 50, 20, 30);
```

##### Сохраните свой рисунок

```csharp
graphics.EndRecording().Save(outputPath);
```

### Рисование сложных фигур с помощью настроек пера

#### Обзор
В этом разделе демонстрируется рисование сложных фигур с использованием различных настроек пера.

##### Настраивать

```csharp
string outputPath = dataDir + "DrawComplexShapes_output.emf";

EmfRecorderGraphics2D graphics = new EmfRecorderGraphics2D(
    new Rectangle(0, 0, 1000, 1000), 
    new Size(1000, 1000),
    new Size(100, 100)
);
```

##### Рисование многоугольников и прямоугольников

```csharp
Pen polygonPen = new Pen(new SolidBrush(Color.Aqua), 3) { LineJoin = LineJoin.MiterClipped };

// Нарисуйте собственный многоугольник.
graphics.DrawPolygon(polygonPen, 
    new[] {
        new Point(10, 20),
        new Point(12, 45),
        new Point(22, 48),
        new Point(48, 36),
        new Point(30, 55)
    }
);
```

##### Сохраните свой рисунок

```csharp
graphics.EndRecording().Save(outputPath);
```

### Сохранение в формате PDF с помощью EmfRasterizationOptions

#### Обзор
Эта функция позволяет сохранять чертежи EMF в виде файлов PDF с использованием параметров растеризации.

##### Инициализировать графическую среду

```csharp
string outputPath = dataDir + "SaveAsPDF_output.pdf";

EmfRecorderGraphics2D graphics = new EmfRecorderGraphics2D(
    new Rectangle(0, 0, 1000, 1000), 
    new Size(1000, 1000),
    new Size(100, 100)
);
```

##### Сохранить как PDF

```csharp
using (EmfImage image = graphics.EndRecording())
{
    PdfOptions pdfOptions = new PdfOptions();
    EmfRasterizationOptions rasterizationOptions = new EmfRasterizationOptions { PageSize = image.Size };
    pdfOptions.VectorRasterizationOptions = rasterizationOptions;

    // Сохраните EMF как файл PDF.
    image.Save(outputPath, pdfOptions);
}
```
