## Руководство по внедрению

В этом разделе мы разобьем процесс на управляемые шаги, сосредоточившись на рисовании прямоугольников на изображении с помощью Aspose.Imaging для .NET.

### Создание и настройка изображения

Для начала давайте создадим новое растровое изображение, на котором мы сможем нарисовать наши прямоугольники:

```csharp
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SampleRectangle_out.bmp");

using (FileStream stream = new FileStream(dataDir, FileMode.Create))
{
    BmpOptions saveOptions = new BmpOptions();
    saveOptions.BitsPerPixel = 32; // Установите значение бит на пиксель 32 для получения высококачественных изображений.
    saveOptions.Source = new StreamSource(stream);
    
    using (Image image = Image.Create(saveOptions, 100, 100)) 
    {
        Graphics graphic = new Graphics(image);
        
        // Очистите поверхность желтым фоновым цветом.
        graphic.Clear(Color.Yellow);

        // На последующих этапах мы нарисуем прямоугольники.
    }
}
```

### Рисование прямоугольников

Теперь сосредоточимся на рисовании двух прямоугольников разных цветов на нашем изображении.

#### Красный прямоугольник

```csharp
// Нарисуйте красный прямоугольник в позиции (30, 10) шириной 40 и высотой 80.
graphic.DrawRectangle(new Pen(Color.Red), new Rectangle(30, 10, 40, 80));
```

Этот фрагмент кода создает красный контур для прямоугольника. `Pen` класс определяет цвет и стиль.

#### Прямоугольник с синей заливкой

```csharp
// Нарисуйте синий закрашенный прямоугольник в точке (10, 30) шириной 80 и высотой 40.
graphic.DrawRectangle(
    new Pen(new SolidBrush(Color.Blue)),
    new Rectangle(10, 30, 80, 40)
);
```

Здесь мы используем `SolidBrush` чтобы залить прямоугольник синим цветом.

### Сохранение изображения

После завершения всех рисунков сохраните изменения:

```csharp
image.Save();
```

На этом этапе окончательное изображение записывается в файловую систему, указанную в нашем пути потока.
