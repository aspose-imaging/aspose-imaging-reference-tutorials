### Создание изображения с использованием BmpOptions и Stream
#### Обзор
Мы покажем, как создать файл BMP, указав различные параметры, такие как количество бит на пиксель.

#### Пошаговая реализация:
**1. Установить каталоги:**
Начните с определения каталогов, в которых хранятся ваши документы и куда вы хотите сохранять выводимые данные.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**2. Настройте BmpOptions:**
Создать экземпляр `BmpOptions` и задайте его свойства, такие как количество бит на пиксель для глубины цвета.
```csharp
BmpOptions imageOptions = new BmpOptions();
imageOptions.BitsPerPixel = 24; // Конфигурация BMP с реальным цветом
```

**3. Определите поток для вывода:**
Используйте поток для управления выходным файлом, в котором будут сохранены ваши данные BMP.
```csharp
using (Stream stream = new FileStream(outputDir + "sample_out.bmp", FileMode.Create))
{
    // Добавьте здесь дополнительные сведения о реализации...
}
```

#### Объяснение
- **БитыНаПиксель:** Устанавливает глубину цвета изображения. Значение 24 используется для изображений с истинным цветом.
- **Файловый поток:** Управляет записью и чтением из файлов, обеспечивая правильное распределение ресурсов с помощью `using` заявление.

**Советы по устранению неполадок:**
- Перед запуском кода убедитесь, что каталоги существуют.
- Если у вас возникли проблемы с доступом, проверьте права доступа к файлам.
