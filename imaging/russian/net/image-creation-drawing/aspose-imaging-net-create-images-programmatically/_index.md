- Создавайте изображения программно

## Руководство по внедрению

В этом разделе вы узнаете, как создать изображение и рисовать на нем с помощью Aspose.Imaging для .NET.

### Создание изображения с определенными параметрами

**Обзор**: Начните с создания пустого изображения с определенными свойствами, такими как размер и глубина цвета.

```csharp
using Aspose.Imaging;
using Aspose.Imaging.ImageOptions;

// Определите выходной каталог.
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Настройте BmpOptions для параметров изображения.
BmpOptions imageOptions = new BmpOptions();
imageOptions.BitsPerPixel = 24; // Установите глубину цвета 24 бита на пиксель.

// Укажите путь к файлу и параметры источника.
string filePath = System.IO.Path.Combine(outputDir, "SampleImage_out.bmp");
imageOptions.Source = new FileCreateSource(filePath, false); // Если файл существует, перезапись не допускается.

using (var image = Image.Create(imageOptions, 500, 500)) // Создайте изображение размером 500x500 пикселей.
{
    // Продолжайте рисовать на этом экземпляре изображения.
}
```
**Объяснение**: Здесь мы настраиваем `BmpOptions` задать глубину цвета и указать путь к файлу для сохранения изображения. `Image.Create` метод инициализирует объект изображения указанных размеров.

### Рисование фигур и применение эффектов

**Обзор**: Рисуйте фигуры, такие как эллипсы, и применяйте эффекты градиента к многоугольникам с помощью Aspose.Imaging.

```csharp
using Aspose.Imaging;
using Aspose.Imaging.Brushes;
using System.Drawing;

using (var graphics = new Graphics(image)) // Получить графический объект.
{
    // Очистите фоновый цвет изображения до белого.
    graphics.Clear(Color.White);

    // Создайте ручку для рисования фигур, установите ее цвет на синий.
    var pen = new Pen(Color.Blue);

    // Нарисуйте на изображении эллипс.
    graphics.DrawEllipse(pen, new Rectangle(10, 10, 150, 100));

    // Нанесите линейный градиент кистью от красного к белому под углом 45 градусов.
    using (var linearGradientBrush = new LinearGradientBrush(image.Bounds, Color.Red, Color.White, 45f))
    {
        // Заполните многоугольник эффектом градиента.
        graphics.FillPolygon(
            linearGradientBrush,
            new[] { new Point(200, 200), new Point(400, 200), new Point(250, 350) });
    }
}
```
**Объяснение**Начинаем с очистки фона изображения, а затем рисуем эллипс. Далее, `LinearGradientBrush` используется для заливки многоугольника с эффектом градиента.

### Сохранение изображения

Наконец, сохраните созданное и измененное изображение:
```csharp
// Сохраните изменения, внесенные в изображение.
image.Save();
```
**Объяснение**: `Save()` Метод записывает все изменения в указанный путь к файлу.
