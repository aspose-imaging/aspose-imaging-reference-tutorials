### Функция: загрузка и доступ к кадрам из изображения WebP

#### Обзор

В этом разделе мы рассмотрим, как загрузить изображение WebP и получить доступ к его отдельным кадрам с помощью библиотеки Aspose.Imaging.

**Шаг 1: Определите каталог документов**
Начните с настройки пути к каталогу документов, в котором хранятся ваши изображения:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/";
```

#### Шаг 2: Загрузите изображение WebP
Используйте `WebPImage` класс для загрузки существующего файла изображения WebP. В этом примере предполагается, что у вас есть файл с именем `asposelogo.webp`.

```csharp
using (WebPImage image = new WebPImage(dataDir + "asposelogo.webp"))
{
    // Продолжайте доступ к кадрам.
}
```

#### Шаг 3: Проверьте наличие нескольких кадров
Определите, имеет ли изображение более одного кадра. Это важно для изображений с несколькими анимациями или последовательностями.

```csharp
if (image.Pages.Length > 2)
{
    // Для обработки доступно несколько кадров.
}
```

#### Шаг 4: Доступ к определенному фрейму
Вы можете получить доступ к любому фрейму по его индексу и преобразовать его в `RasterImage` для дальнейших операций:

```csharp
RasterImage block = (image.Pages[2] as RasterImage);
if (block != null)
{
    // Заполнитель для дополнительной обработки.
}
```
