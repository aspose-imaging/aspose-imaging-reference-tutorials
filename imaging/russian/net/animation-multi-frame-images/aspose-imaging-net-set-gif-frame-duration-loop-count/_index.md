### Установка длительности кадра GIF

#### Обзор
Настройка длительности кадра позволяет контролировать длительность показа каждого кадра в анимированном GIF-файле. Это может иметь решающее значение для синхронизации анимации с другими медиафайлами или достижения желаемых визуальных эффектов.

#### Этапы внедрения

**1. Загрузите GIF-изображение**
Начните с загрузки вашего GIF-изображения из указанного каталога:
```csharp
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "gif_images");
string filepath = Path.Combine(dataDir, "example.gif");

using (GifImage image = (GifImage)Image.Load(filepath))
{
    // Дальнейшая обработка...
}
```

**2. Установите длительность кадра**
Установите длительность всех кадров на 2000 миллисекунд и настройте индивидуальное время показа кадров:
```csharp
image.SetFrameTime(2000); // Устанавливает время кадра по умолчанию

// Настройте длительность первого кадра
((GifFrameBlock)image.Pages[0]).FrameTime = 200; // Устанавливает определенное время кадра
```

**Объяснение:**
- `SetFrameTime(2000)`: Настраивает все кадры на отображение в течение 2000 миллисекунд по умолчанию.
- `((GifFrameBlock)image.Pages[0]).FrameTime = 200;`: Регулирует длительность первого кадра, демонстрируя, как можно нацеливаться на определенные кадры.

### Настройка количества циклов GIF

#### Обзор
Управление количеством циклов определяет, сколько раз будет воспроизводиться ваш GIF. Эта функция полезна для создания анимаций, которые должны повторяться заданное количество раз.

#### Этапы внедрения

**1. Загрузите и сохраните GIF-файл**
Загрузите изображение и сохраните его с указанным количеством циклов:
```csharp
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.gif");

using (GifImage image = (GifImage)Image.Load(filepath))
{
    // Установите количество повторений цикла на 4 раза
    image.Save(outputPath, new GifOptions() { LoopsCount = 4 });
}
```

**Объяснение:**
- `LoopsCount = 4`: Настраивает GIF-файл на четыре воспроизведения перед остановкой.

### Удаление выходного файла

#### Обзор
Очистка после обработки гарантирует, что ваш выходной каталог останется организованным за счет удаления ненужных файлов.

#### Этапы внедрения

**1. Удалить указанный файл**
Проверьте наличие файла и при необходимости удалите:
```csharp
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.gif");

if (File.Exists(outputPath))
{
    File.Delete(outputPath);
}
```
