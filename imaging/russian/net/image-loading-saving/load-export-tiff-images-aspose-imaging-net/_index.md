## Руководство по внедрению

### Функция 1: Загрузка и отображение информации об изображении TIFF
Эта функция позволяет загружать изображение TIFF и получать доступ к его свойствам, таким как размеры и настройки разрешения.

#### Обзор
Вы узнаете, как:
- Загрузить файл TIFF
- Извлечение и отображение размеров изображения
- Доступ к информации о горизонтальном и вертикальном разрешении

#### Пошаговая реализация
**1. Подготовьте окружающую среду:**
Убедитесь, что библиотека Aspose.Imaging установлена, и настройте среду разработки, указав необходимые пути.

```csharp
using Aspose.Imaging;
using System;
using System.IO;

string filePath = "YOUR_DOCUMENT_DIRECTORY";
string fileName = Path.Combine(filePath, "AFREY-Original.tif");

if (!File.Exists(fileName))
{
    throw new FileNotFoundException("The specified TIFF file does not exist.");
}

using (TiffImage image = (TiffImage)Image.Load(fileName))
{
    // Отобразить размеры загруженного изображения TIFF
    Console.WriteLine($"Width: {image.Width}, Height: {image.Height}");
    
    // Доступ и отображение настроек разрешения изображения
    Console.WriteLine($"Horizontal Resolution: {image.HorizontalResolution} DPI, Vertical Resolution: {image.VerticalResolution} DPI");
}
```
**Объяснение:**
- `Image.Load(fileName)`: Загружает ваш файл TIFF.
- `TiffImage` cast гарантирует, что вы работаете со специфичным для TIFF классом для доступа к свойствам TIFF.
- На выходе консоли отображаются размеры и разрешение изображения.

### Функция 2: Экспорт изображения TIFF в PDF с определенными настройками DPI
Теперь давайте рассмотрим экспорт изображения TIFF в PDF с сохранением исходных настроек разрешения.

#### Обзор
В этом разделе рассматриваются:
- Подготовка параметров экспорта PDF на основе существующих настроек TIFF
- Сохранение TIFF как файла PDF

#### Пошаговая реализация
**1. Настройте параметры экспорта:**
Настройте параметры экспорта PDF, включая параметры DPI, и подготовьтесь к сохранению изображения.

```csharp
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Tiff;
using Aspose.Imaging.ImageOptions;
using System.IO;

string inputFilePath = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string fileName = Path.Combine(inputFilePath, "AFREY-Original.tif");

if (!File.Exists(fileName))
{
    throw new FileNotFoundException("The specified TIFF file does not exist.");
}

using (TiffImage image = (TiffImage)Image.Load(fileName))
{
    // Подготовьте параметры экспорта PDF с настройками разрешения из изображения TIFF
    PdfOptions pdfOptions = new PdfOptions()
    {
        ResolutionSettings = new ResolutionSetting(
            image.HorizontalResolution,
            image.VerticalResolution)
    };
    
    // Укажите выходной путь для экспортируемого PDF-файла.
    string outputPath = Path.Combine(outputDirectory, "result.pdf");
    
    // Сохраните TIFF как PDF с указанными настройками DPI.
    image.Save(outputPath, pdfOptions);
}
```
**Объяснение:**
- `PdfOptions`: Настраивает способ преобразования TIFF в PDF.
- `ResolutionSetting`: Устанавливает DPI на основе разрешения исходного TIFF.
- `image.Save(outputPath, pdfOptions)`: Сохраняет TIFF как PDF с указанными вами настройками.
