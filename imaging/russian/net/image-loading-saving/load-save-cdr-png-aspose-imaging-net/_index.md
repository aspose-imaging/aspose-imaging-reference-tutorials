## Руководство по внедрению

### Функция: загрузка и сохранение изображения в формате PNG

#### Обзор
Эта функция демонстрирует, как загрузить файл CDR с помощью Aspose.Imaging и сохранить его в формате PNG, применив определенные параметры растеризации.

#### Шаг 1: Определите пути и загрузите изображение

Сначала настройте входные и выходные пути. Заменить `"YOUR_DOCUMENT_DIRECTORY"` с фактическим путем к каталогу ваших документов.

```csharp
using System.IO;
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Cdr;
using Aspose.Imaging.ImageOptions;

public class ImageLoadingAndSaving
{
    public static void Run()
    {
        string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "CDR");
        string inputFileName = Path.Combine(dataDir, "test2.cdr");

        using (var image = (CdrImage)Image.Load(inputFileName))
        {
            // Изображение успешно загружено
        }
    }
}
```
*Почему*: `Image.Load` метод используется для загрузки CDR-файла в память для дальнейшей обработки. 

#### Шаг 2: Сохраните как PNG с параметрами растеризации

Затем настройте и сохраните изображение в формате PNG, используя определенные параметры растеризации.

```csharp
string outputFileName = Path.Combine(dataDir, "result.png");

image.Save(outputFileName, new PngOptions()
{
    VectorRasterizationOptions = new CdrRasterizationOptions
    {
        Positioning = PositioningTypes.Relative
    }
});
```
*Почему*: `PngOptions` позволяет настраивать вывод PNG. `VectorRasterizationOptions` убедитесь, что изображение сохраняет свои векторные свойства при сохранении.

### Функция: Удаление файла

#### Обзор
Узнайте, как удалить файл с помощью пространства имен System.IO .NET, гарантируя, что ваше приложение эффективно управляет ресурсами.

#### Шаг 1: Определите пути и удалите файл

Укажите путь к файлу, который вы хотите удалить:

```csharp
public class FileDeletion
{
    public static void Run()
    {
        string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "CDR");
        string outputFileName = Path.Combine(dataDir, "result.png");

        if (File.Exists(outputFileName))
        {
            File.Delete(outputFileName);
        }
    }
}
```
*Почему*: Всегда проверяйте существование файла перед попыткой его удаления, чтобы избежать исключений.
