## Руководство по внедрению

### Функция 1: Загрузка изображения WMF

**Обзор**: Эта функция демонстрирует загрузку изображения Windows Metafile (WMF) с помощью Aspose.Imaging для .NET.

#### Пошаговые инструкции

##### Загрузить существующее изображение WMF

Сначала укажите каталог, в котором хранятся ваши файлы WMF:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

Загрузите файл WMF и отобразите его размеры, чтобы убедиться, что он загружен правильно:

```csharp
Image image = Image.Load(dataDir + "/input.wmf");
Console.WriteLine($"Image Dimensions: Width={image.Width}, Height={image.Height}");
image.Dispose();  // Всегда утилизируйте ресурсы после использования.
```

### Функция 2: Создание и настройка параметров растеризации для изображения WMF

**Обзор**: Узнайте, как настроить параметры растеризации специально для преобразования изображения WMF.

#### Пошаговые инструкции

##### Рассчитать соотношение сторон

Сначала рассчитаем соотношение сторон, чтобы сохранить пропорции изображения при конвертации:

```csharp
double k = (image.Width * 1.00) / image.Height;
```

##### Установить параметры растеризации

Создать экземпляр `WmfRasterizationOptions` и настройте его свойства:

```csharp
WmfRasterizationOptions wmfRasterization = new WmfRasterizationOptions
{
    BackgroundColor = Color.WhiteSmoke,
    PageWidth = 400,
    PageHeight = (int)Math.Round(400 / k),
    BorderX = 5,
    BorderY = 10
};

Console.WriteLine($"Rasterization Options: Width={wmfRasterization.PageWidth}, Height={wmfRasterization.PageHeight}");
```

### Функция 3: Настройка параметров преобразования WebP и сохранение изображения

**Обзор**: Настройте преобразование изображения в формат WebP с использованием определенных параметров растеризации.

#### Пошаговые инструкции

##### Настройка параметров WebP для преобразования

Сначала укажите выходной каталог:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

Настроить `WebPOptions` и снова загрузите изображение WMF для конвертации:

```csharp
ImageOptionsBase webpOptions = new WebPOptions();
webpOptions.VectorRasterizationOptions = wmfRasterization;

using (Image imageToConvert = Image.Load(dataDir + "/input.wmf"))
{
    imageToConvert.Save(outputDir + "/ConvertedWebP_out.webp", webpOptions);
}

Console.WriteLine("WMF Image Converted to WebP format.");
```
