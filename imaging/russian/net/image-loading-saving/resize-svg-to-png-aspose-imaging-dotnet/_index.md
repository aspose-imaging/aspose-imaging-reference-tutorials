## Руководство по внедрению

В этом разделе мы разберем реализацию на две основные функции: изменение размера изображения SVG и сохранение его в виде файла PNG. 

### Функция 1: Изменение размера изображения SVG

#### Обзор

Изменение размера имеет решающее значение, когда вам нужно настроить размеры SVG для различных требований к отображению. Aspose.Imaging упрощает эту задачу.

#### Шаги:

**Шаг 1: Загрузите SVG**

Начните с загрузки вашего SVG-изображения с помощью `Image.Load` метод. Убедитесь, что путь к файлу указывает на место, где хранится ваш SVG.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (SvgImage image = (SvgImage)Image.Load(dataDir + "/aspose_logo.Svg"))
{
    // Продолжайте изменение размера...
}
```

**Шаг 2: Измените размер изображения**

Изменить размер, указав новые размеры. Здесь мы умножаем исходную ширину и высоту на коэффициенты, чтобы достичь желаемого размера.
```csharp
// Уменьшите ширину изображения на 10, а высоту на 15.
image.Resize(image.Width * 10, image.Height * 15);
```

**Шаг 3: Сохраните измененное изображение.**

После изменения размера сохраните изображение. Этот пример сохраняет его в формате PNG в указанный выходной каталог.
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/Logotype_10_15_out.png";
image.Save(outputPath, new PngOptions()
{
    VectorRasterizationOptions = new SvgRasterizationOptions()
});
```

### Функция 2: Сохранение SVG как PNG

#### Обзор

Конвертация файлов SVG в широко поддерживаемый формат PNG может улучшить совместимость. Aspose.Imaging упрощает этот процесс конвертации.

#### Шаги:

**Шаг 1: Определите параметры PNG**

Настройте свой `PngOptions` объект, задающий параметры растеризации для процесса преобразования.
```csharp
PngOptions pngOptions = new PngOptions()
{
    VectorRasterizationOptions = new SvgRasterizationOptions()
};
```

**Шаг 2: Сохранить как PNG**

Наконец, используйте эти параметры, чтобы сохранить изображение SVG как файл PNG.
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/Logotype_out.png";
image.Save(outputPath, pngOptions);
```
