## Руководство по внедрению

В этом разделе каждая функция разбита на этапы выполнения.

### Установить папку пользовательских шрифтов

#### Обзор
Создайте пользовательскую папку для шрифтов, чтобы стандартизировать способ отображения текста в вашем приложении.

**Этапы реализации:**

##### Определите каталог документа и путь к шрифту

Сначала укажите, где находятся каталог ваших документов и файлы шрифтов.
```csharp
using Aspose.Imaging;
using System.IO;

public static void SetCustomFontsFolder()
{
    string dataDir = "YOUR_DOCUMENT_DIRECTORY/";
    FontSettings.SetFontsFolder(Path.Combine(dataDir, "fonts"));
}
```
- **Параметры:** `dataDir` должен быть путем к каталогу ваших документов.
- **Цель:** Этот метод гарантирует, что Aspose.Imaging будет использовать только указанные вами шрифты.

##### Советы по устранению неполадок

- Убедитесь, что папка шрифтов существует и содержит файлы .ttf или .otf.
- Проверьте права доступа к файлу для приложения, чтобы получить доступ к этому каталогу.

### Отключить системные альтернативные шрифты

#### Обзор
Предотвращает использование альтернативных системных шрифтов, обеспечивая единообразную визуализацию текста на изображениях.

**Этапы реализации:**

##### Установить настройки шрифта

Отключите использование системных альтернативных шрифтов следующим образом:
```csharp
using Aspose.Imaging;

public static void DisableSystemAlternativeFont()
{
    FontSettings.GetSystemAlternativeFont = false;
}
```
- **Параметры:** Нет. Это глобальная настройка, влияющая на рендеринг всех шрифтов.
- **Цель:** Гарантирует, что текст будет отображаться именно так, как задумано, без использования системных шрифтов.

##### Советы по устранению неполадок

- Если вы заметили отсутствующие символы, убедитесь, что пользовательские шрифты включают необходимые глифы.
- Протестируйте с различными типами документов, чтобы подтвердить единообразие поведения.

### Экспорт векторной графики в PNG

#### Обзор
Конвертируйте векторную графику (такую как ODP или SVG) в высококачественный формат PNG, используя надежные функции Aspose.Imaging.

**Этапы реализации:**

##### Установить шрифт по умолчанию и загрузить документ

Настройте шрифт по умолчанию для рендеринга, затем загрузите документ:
```csharp
using Aspose.Imaging;
using Aspose.Imaging.ImageOptions;
using System.IO;

public static void ExportVectorToPng(string inputFilePath, string defaultFontName, string outputFilePath)
{
    FontSettings.DefaultFontName = defaultFontName;
    
    using (Aspose.Imaging.Image document = Aspose.Imaging.Image.Load(inputFilePath))
    {
        PngOptions saveOptions = new PngOptions();
        saveOptions.VectorRasterizationOptions = new OdgRasterizationOptions()
        {
            PageWidth = 1000,
            PageHeight = 1000
        };
        
        document.Save(outputFilePath, saveOptions);
    }
    
    File.Delete(outputFilePath); // Необязательно: удалить вывод после сохранения
}
```
- **Параметры:**
  - `inputFilePath`: Путь к векторному графическому файлу.
  - `defaultFontName`: Шрифт, используемый для отображения текста на изображении.
  - `outputFilePath`: Где будет сохранен полученный PNG-файл.
- **Цель:** Конвертирует векторные форматы в растровые изображения, сохраняя качество и гарантируя корректную отрисовку текста с указанными шрифтами.

##### Советы по устранению неполадок

- Убедитесь, что векторные файлы доступны и правильно отформатированы.
- Регулировать `PageWidth` и `PageHeight` исходя из ваших конкретных потребностей для правильного размещения контента.

## Раздел часто задаваемых вопросов

**1. Как мне справиться с отсутствующими шрифтами в пользовательских каталогах?**
- Убедитесь, что в указанной папке присутствуют все необходимые шрифты; в противном случае рендеринг текста может не выполняться должным образом.

**2. Можно ли экспортировать файлы SVG напрямую с помощью Aspose.Imaging?**
- Да, Aspose.Imaging поддерживает прямое преобразование из SVG в PNG и другие форматы.

**3. Что делать, если после конвертации мой выходной файл PNG выглядит искаженным?**
- Проверьте настройки векторной растеризации, такие как размеры страницы и разрешение; отрегулируйте их в соответствии с масштабом исходного файла.

**4. Можно ли использовать несколько пользовательских шрифтов в одном проекте?**
- Да, Aspose.Imaging позволяет указывать различные папки шрифтов для различных нужд вашего приложения.

**5. Что делать, если экспортированные PNG-файлы выглядят размытыми или пикселизированными?**
- Увеличьте настройки разрешения в `PngOptions` для повышения четкости изображения.
