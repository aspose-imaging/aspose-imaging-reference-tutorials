### Загрузка и отображение метаданных
Понимание того, как загрузить изображение и получить доступ к его метаданным, является основополагающим для любой задачи обработки изображений. Вот как вы можете добиться этого с помощью Aspose.Imaging:

#### Обзор
Эта функция демонстрирует загрузку файла изображения EMF+ с помощью Aspose.Imaging и запрос его метаданных.

#### Пошаговая реализация
**1. Задайте путь к изображению**
Определите каталог, содержащий файлы изображений:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
var path = dataDir + "TestEmfPlusFigures.emf";
```

**2. Загрузите файл изображения EMF+**
Используйте Aspose.Imaging для загрузки файла EMF+:
```csharp
using (var image = (MetaImage)Aspose.Imaging.Image.Load(path))
{
    // Теперь объект MetaImage загружен и может быть изменен или запрошен для получения метаданных.
}
```

**Объяснение:**
- `Aspose.Imaging.Image.Load`: Этот метод загружает указанный файл изображения в `MetaImage` объект, позволяющий получить доступ к его свойствам.

### Сохранить изображение EMF+ в файл
Сохранение изображений в их оригинальном формате при сохранении имеет решающее значение для поддержания качества. Вот как это можно сделать:

#### Обзор
В этой статье объясняется, как сохранить файл изображения EMF+ с помощью Aspose.Imaging с нужными параметрами.

#### Пошаговая реализация
**1. Установите входные и выходные пути**
Укажите, где будут находиться ваши входные и выходные файлы:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
var path = dataDir + "TestEmfPlusFigures.emf";
var outputPath = path + ".emf";
```

**2. Загрузите и сохраните изображение.**
Загрузите изображение и сохраните его с помощью `EmfOptions` для сохранения формата:
```csharp
using (var image = (MetaImage)Aspose.Imaging.Image.Load(path))
{
    // Сохраните загруженное изображение в новый файл с помощью EmfOptions, сохранив формат.
    image.Save(outputPath, new EmfOptions());
}
```

**Объяснение:**
- `EmfOptions`: Этот класс конфигурации обеспечивает сохранение формата EMF+ при сохранении.
