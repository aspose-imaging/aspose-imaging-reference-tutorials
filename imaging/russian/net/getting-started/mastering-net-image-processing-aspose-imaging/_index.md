## Руководство по внедрению
### Загрузка и обработка изображения
В этом разделе подробно описывается, как загрузить изображение и получить дату его последнего изменения с помощью Aspose.Imaging. Эта функция необходима для приложений, которым необходимо отслеживать изменения или обновлять изображения на основе их метаданных.

#### Шаг 1: Настройка каталогов
Сначала определите входные и выходные каталоги, в которых будут храниться ваши изображения:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Загрузите изображение
Использовать `Image.Load` для чтения файла изображения. Этот метод возвращает общий `Image` объект, который вы можете привести к `RasterImage` для более конкретных операций:

```csharp
using (RasterImage image = (RasterImage)Image.Load(dataDir + "/aspose-logo.jpg"))
{
    // Здесь располагается логика обработки изображений.
}
```

#### Шаг 3: Получите дату последнего изменения
Чтобы получить дату последнего изменения файла изображения, используйте `GetModifyDate` метод. Этот метод может учитывать метаданные XMP при необходимости:

```csharp
string modifyDateFile = image.GetModifyDate(true).ToString();
string modifyDateXmp = image.GetModifyDate(false).ToString();

Console.WriteLine("Last modified date from file: " + modifyDateFile);
Console.WriteLine("Last modified date from XMP metadata: " + modifyDateXmp);
```

**Объяснение**: 
- `true` в `GetModifyDate` метод учитывает метаданные файловой системы.
- `false` извлекает даты из метаданных XMP изображения, если они доступны.
