## Руководство по внедрению

### Добавление миниатюры в сегмент EXIF

Эта функция позволяет встраивать миниатюру непосредственно в данные EXIF файлов JPEG.

#### Шаг 1: Подготовка каталогов

Определите пути для входных и выходных каталогов:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```

#### Шаг 2: Создание миниатюрного изображения

Создайте новое изображение JPEG, которое будет использоваться в качестве миниатюры:
```csharp
JpegImage thumbnailImage = new JpegImage(100, 100);
```

#### Шаг 3: Подготовка основного изображения с данными EXIF

Создайте еще одно изображение JPEG и инициализируйте его данные EXIF:
```csharp
JpegImage image = new JpegImage(1000, 1000);
image.ExifData = new JpegExifData();
image.ExifData.Thumbnail = thumbnailImage;
```

#### Шаг 4: Сохраните изображение.

Сохраните измененное изображение со встроенной миниатюрой в выходной каталог:
```csharp
string outputPath = Path.Combine(outputDirectory, "thumbnail_out.jpg");
image.Save(outputPath);
```

### Обработка потока файлов с помощью MemoryStream

Обработка изображений с помощью `MemoryStream` может быть полезен для обработки в памяти без записи на диск.

#### Шаг 1: Инициализация MemoryStream

Создать экземпляр `MemoryStream`:
```csharp
using (MemoryStream stream = new MemoryStream())
{
    JpegImage jpegImage = new JpegImage(500, 500);
    
    // Операции с jpegImage можно выполнять здесь
    
    jpegImage.Save(stream);
}
```

В этом примере демонстрируется сохранение изображения JPEG в потоке памяти.
