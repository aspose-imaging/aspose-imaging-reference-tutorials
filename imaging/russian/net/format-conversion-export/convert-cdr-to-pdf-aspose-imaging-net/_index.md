## Руководство по внедрению

Мы рассмотрим, как конвертировать файлы CDR в PDF с помощью Aspose.Imaging для .NET. Учебник разделен на разделы в зависимости от функций.

### Создать параметры растеризации страницы

**Обзор:** Эта функция демонстрирует создание параметров растеризации для каждой страницы векторного изображения, что необходимо для многостраничных преобразований, таких как CDR в PDF.

#### Определите метод
Создайте массив `VectorRasterizationOptions` для каждой страницы вашего изображения:
```csharp
private static VectorRasterizationOptions[] CreatePageOptions<TOptions>(VectorMultipageImage image) where TOptions : VectorRasterizationOptions
{
    return image.Pages.Select(x => x.Size).Select(CreatePageOptions<TOptions>).ToArray();
}
```
**Объяснение:** Этот метод выполняет итерацию по каждой странице векторного изображения, создавая соответствующий параметр растеризации с помощью `CreatePageOptions` вспомогательный метод.

#### Создать параметры растеризации для размера страницы
Определите функцию, которая генерирует параметры растеризации:
```csharp
private static VectorRasterizationOptions CreatePageOptions<TOptions>(Size pageSize) where TOptions : VectorRasterizationOptions
{
    var options = Activator.CreateInstance<TOptions>();
    options.PageSize = pageSize;
    return options;
}
```
**Объяснение:** Этот метод использует отражение для создания экземпляра класса параметров растеризации и задает размер страницы, подготавливая ее к преобразованию.

### Конвертировать CDR в PDF

**Обзор:** Здесь мы преобразуем файл CorelDRAW (CDR) в многостраничный PDF-документ с помощью Aspose.Imaging для .NET.

#### Загрузите образ CDR
Начните с загрузки вашего CDR-файла:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string inputFileName = Path.Combine(dataDir, "MultiPage2.cdr");

using (var image = (VectorMultipageImage)Image.Load(inputFileName))
{
    // Продолжайте выполнять шаги преобразования...
}
```
**Объяснение:** Мы загружаем файл CDR в `VectorMultipageImage` объект для доступа к его страницам и свойствам.

#### Параметры растеризации страницы
Используйте ранее определенные методы для генерации параметров для каждой страницы:
```csharp
var pageOptions = CreatePageOptions<CdrRasterizationOptions>(image);
```
**Объяснение:** Этот шаг использует `CreatePageOptions` метод, адаптированный для растеризации CDR, обеспечивающий точную визуализацию PDF.

#### Настройте параметры экспорта PDF
Настройте конфигурации экспорта:
```csharp
var pdfOptions = new PdfOptions
{
    MultiPageOptions = new MultiPageOptions { PageRasterizationOptions = pageOptions }
};
```
**Объяснение:** The `PdfOptions` класс настроен на обработку многостраничного экспорта, связывая настройки растеризации каждой страницы.

#### Сохраните PDF-файл
Наконец, сохраните преобразованный файл:
```csharp
string outputFileName = Path.Combine(dataDir, "MultiPage2.cdr.pdf");
image.Save(outputFileName, pdfOptions);
```
**Объяснение:** The `Save` Метод записывает многостраничный PDF-файл, используя указанные параметры.
