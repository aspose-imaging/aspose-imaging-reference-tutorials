## Руководство по внедрению

### Конвертировать WebP в PDF

Эта функция позволяет вам эффективно конвертировать файл изображения WebP в документ PDF. Вот как:

#### Загрузите изображение WebP
```csharp
using System;
using System.IO;
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Pdf;
using Aspose.Imaging.ImageOptions;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string outputPath = @"YOUR_OUTPUT_DIRECTORY";

// Загрузите изображение WebP из указанного каталога.
string inputFile = Path.Combine(dataDir, "Animation.webp");
```

#### Параметры создания PDF-файла
```csharp
using (Aspose.Imaging.FileFormats.Webp.WebPImage image = 
    (Aspose.Imaging.FileFormats.Webp.WebPImage)Image.Load(inputFile))
{
    // Настройте выходной PDF-файл.
    PdfOptions options = new PdfOptions();
    options.PdfDocumentInfo = new PdfDocumentInfo();
}
```

#### Сохранить как PDF
```csharp
// Определите путь к выходному файлу и сохраните изображение в формате PDF.
string outputFile = Path.Combine(outputPath, "Animation.pdf");
image.Save(outputFile, options);
```
**Объяснение:** Этот фрагмент кода загружает изображение WebP, настраивает параметры преобразования PDF и сохраняет его как документ PDF. Настроить `dataDir` и `outputPath` для соответствия вашим каталогам.

### Удалить сгенерированный PDF

После конвертации вам, возможно, захочется очистить временные файлы:

#### Удалить PDF-файл
```csharp
string outputFile = Path.Combine(outputPath, "Animation.pdf");

// Перед удалением убедитесь, что файл существует.
if (File.Exists(outputFile))
{
    File.Delete(outputFile);
}
```
**Объяснение:** Этот код проверяет существование сгенерированного PDF-файла и удаляет его, чтобы освободить место.
