### Загрузка и сохранение изображения CDR в формате PNG

#### Обзор

Эта функция демонстрирует загрузку файла CDR с помощью библиотеки Aspose.Imaging и сохранение его в формате PNG. Это обеспечивает совместимость на разных платформах, которые могут изначально не поддерживать файлы CDR.

##### Шаг 1: Определите каталоги и загрузите файл

Сначала укажите входной каталог, в котором находится файл CDR, и выходной каталог для сохранения полученного изображения PNG.
```csharp
string dataDir = "/path/to/your/input/directory";
string outputDir = "/path/to/your/output/directory";
string inputFileName = dataDir + "SimpleShapes.cdr";

using (CdrImage image = (CdrImage)Image.Load(inputFileName))
{
    // Продолжить обработку...
}
```
##### Шаг 2: Настройте параметры PNG

Чтобы сохранить файл CDR как PNG, настройте `PngOptions`. Этот шаг имеет решающее значение для настройки таких свойств, как тип цвета и параметры растеризации.
```csharp
PngOptions options = new PngOptions();
options.ColorType = PngColorType.TruecolorWithAlpha; // Поддерживает прозрачность

// Установить параметры, специфичные для вектора
options.VectorRasterizationOptions = image.GetDefaultOptions(new object[] { Color.White, image.Width, image.Height }).VectorRasterizationOptions;

options.VectorRasterizationOptions.TextRenderingHint = TextRenderingHint.SingleBitPerPixel;
options.VectorRasterizationOptions.SmoothingMode = SmoothingMode.None;
```
##### Шаг 3: Сохраните изображение.

Наконец, сохраните загруженный файл CDR как PNG, используя заданные параметры.
```csharp
string outputFileName = outputDir + "SimpleShapes.png";
image.Save(outputFileName, options);
```
### Удаление выходного файла

#### Обзор

После обработки изображений вы можете захотеть очистить их, удалив выходные файлы. Эта функция показывает, как удалить файл PNG после его сохранения.

##### Шаг 1: Определите каталог и путь к файлу

Определите, где хранятся ваши выходные файлы, и укажите, какой файл следует удалить:
```csharp
string outputDir = "/path/to/your/output/directory";
string fileNameToRemove = "SimpleShapes.png";

string filePath = System.IO.Path.Combine(outputDir, fileNameToRemove);
```
##### Шаг 2: Проверьте наличие и удалите файл

Перед попыткой удаления убедитесь, что файл существует:
```csharp
if (System.IO.File.Exists(filePath))
{
    System.IO.File.Delete(filePath);
}
```
