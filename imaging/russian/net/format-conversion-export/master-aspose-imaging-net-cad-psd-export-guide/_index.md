## Руководство по внедрению

### Загрузка изображения САПР

#### Обзор
Загрузка файла САПР в ваше приложение .NET проста с Aspose.Imaging. Эта функция позволяет вам получать доступ и управлять содержимым таких файлов, как `.cdr`.

#### Пошаговая реализация
**Загрузите файл CAD**
```csharp
using Aspose.Imaging;
using System.IO;

string inputFileName = "YOUR_DOCUMENT_DIRECTORY/MultiPage.cdr"; // Заменить на ваш путь

// Загрузите изображение в объект Aspose.Imaging CdrImage
Aspose.Imaging.FileFormats.Cdr.CdrImage image = (Aspose.Imaging.FileFormats.Cdr.CdrImage)Image.Load(inputFileName);

image.Dispose(); // Очистить ресурсы после загрузки
```
**Объяснение**: 
- `Image.Load` читает ваш файл САПР, возвращая `CdrImage` объект для дальнейших манипуляций.
- Всегда помните, что нужно позвонить. `.Dispose()` для освобождения памяти.

### Экспорт многостраничного изображения в PSD с объединением слоев

#### Обзор
Экспорт многостраничных изображений САПР в виде однослойных PSD имеет решающее значение для упрощения сложных проектов. Эта функция объединяет все слои в один, делая изображение более управляемым.

#### Пошаговая реализация
**Настроить и сохранить как PSD**
```csharp
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Psd;
using Aspose.Imaging.ImageOptions;

string outputFileName = "YOUR_OUTPUT_DIRECTORY/MultiPageOut.psd"; // Заменить на ваш путь

Aspose.Imaging.FileFormats.Cdr.CdrImage image = (Aspose.Imaging.FileFormats.Cdr.CdrImage)Image.Load("YOUR_DOCUMENT_DIRECTORY/MultiPage.cdr");
try
{
    ImageOptionsBase options = new PsdOptions();

    // Объединить слои в один в файле PSD
    options.MultiPageOptions = new MultiPageOptions { MergeLayers = true };

    // Установите параметры растеризации для лучшего качества изображения
    options.VectorRasterizationOptions = image.GetDefaultOptions(new object[] { Color.White, image.Width, image.Height }).VectorRasterizationOptions;
    options.VectorRasterizationOptions.TextRenderingHint = TextRenderingHint.SingleBitPerPixel;
    options.VectorRasterizationOptions.SmoothingMode = SmoothingMode.None;

    // Сохраните CDR как PSD-файл с объединенными слоями.
    image.Save(outputFileName, options);
}
finally
{
    image.Dispose();
}
```
**Объяснение**: 
- `PsdOptions` настраивает способ сохранения изображений САПР в формате PSD.
- `MultiPageOptions.MergeLayers = true` обеспечивает объединение всех слоев исходного файла в один.

#### Обзор
После обработки важно управлять хранилищем, удаляя все временные файлы, созданные в ходе операций.
