## Руководство по внедрению

### Создать GIF из нескольких кадров

Создание GIF из нескольких кадров включает загрузку каждого кадра, настройку параметров GIF и сохранение конечного результата. Вот как этого добиться:

#### Рамы нагрузки

1. **Определить каталог фреймов**: Убедитесь, что все ваши кадры изображений хранятся в одном каталоге.

   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY/Animation frames";
   ```

2. **Рамы нагрузки**: Использовать `Iterable<RasterImage>` для загрузки каждого кадра из каталога.

   ```java
   Iterable<RasterImage> frames = loadFrames(dataDir);
   ```

#### Создание и добавление рамок

3. **Инициализировать GIF-изображение**:

   Начните с создания нового `GifImage` экземпляр с первым кадром, а затем повторите итерацию по последующим кадрам, чтобы добавить их.

   ```java
   GifImage image = null;

   for (RasterImage frame : frames) {
       if (image == null) {
           image = new GifImage(new GifFrameBlock(frame));
           continue;
       }
       // Добавьте дополнительные кадры здесь
   }
   ```

#### Сохранить GIF-файл

4. **Сохраните вывод**:

   После добавления всех кадров сохраните GIF-файл в указанном выходном каталоге.

   ```java
   String outDir = "YOUR_OUTPUT_DIRECTORY";
   image.save(outDir + "/output.gif");
   ```

### Объяснение основных шагов

- **GifFrameBlock**: Этот класс инкапсулирует индивидуальные настройки кадра. Поймите его параметры для пользовательских конфигураций.
- **Качество изображения и оптимизация**: Отрегулируйте настройки, чтобы сбалансировать качество и размер файла в соответствии с вашими потребностями.
