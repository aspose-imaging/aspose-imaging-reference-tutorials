## Руководство по внедрению

Мы разделим нашу реализацию на две основные функции:

### Загрузить и пересохранить изображение DICOM как BMP

#### Обзор

Эта функция демонстрирует, как загрузить изображение DICOM с диска и сохранить его в формате BMP, что делает его доступным для немедицинских приложений или систем, которым требуются базовые форматы изображений.

#### Пошаговая реализация

1. **Настройка каталогов**

   Определите входной и выходной каталоги, в которых будет находиться файл DICOM и куда вы хотите сохранить файл BMP.
   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY";
   String inputFile = dataDir + "image.dcm";
   String outputFile = "YOUR_OUTPUT_DIRECTORY" + "ResizedOutput.bmp";
   ```

2. **Загрузка и сохранение изображения DICOM**

   Использовать `DicomImage` из Aspose.Imaging, чтобы загрузить изображение, а затем сохранить его в формате BMP.
   ```java
   try (DicomImage image = (DicomImage) Image.load(inputFile)) {
       // Сохраните изображение как файл BMP.
       image.save(outputFile, new BmpOptions());
   }
   ```

3. **Объясните параметры**

   - `inputFile`: Путь к вашему DICOM-файлу.
   - `outputFile`: Путь назначения для выходных данных BMP.
   - `BmpOptions()`: Параметры конфигурации для формата BMP.

### Изменить высоту пропорционально

#### Обзор

Эта функция позволяет пропорционально изменять высоту изображения DICOM, сохраняя его соотношение сторон и сохраняя его как файл BMP.

#### Пошаговая реализация

1. **Загрузите изображение DICOM**

   Начните с загрузки изображения DICOM с помощью Aspose.Imaging.
   ```java
   String inputFile = dataDir + "image.dcm";
   String outputFile = "YOUR_OUTPUT_DIRECTORY" + "ResizeHeightProportionally_out.bmp";

   try (DicomImage image = (DicomImage) Image.load(inputFile)) {
       // Измените высоту пропорционально на 100 пикселей.
       image.resizeHeightProportionally(100, ResizeType.AdaptiveResample);
       
       // Сохраните измененное изображение в формате BMP.
       image.save(outputFile, new BmpOptions());
   }
   ```

2. **Параметры и методы**

   - `resizeHeightProportionally(100, ResizeType.AdaptiveResample)`: Этот метод изменяет высоту до 100 пикселей, сохраняя при этом соотношение сторон, используя адаптивную повторную выборку для повышения качества.
