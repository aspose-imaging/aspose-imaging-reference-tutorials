## Использование Aspose Imaging Java для конвертации PNG в JPEG‑LS (H2)

### Конвертация PNG в JPEG‑LS (H2)

Эта функция позволяет сжать PNG‑изображение с 8‑битным образцом в более эффективный формат JPEG‑LS с 2 битами на образец.

#### Обзор
JPEG‑LS известен своими возможностями сжатия без потерь и почти без потерь, что делает его идеальным для хранения изображений высокого качества при уменьшенном размере файлов. В этом разделе мы покажем, как конвертировать PNG в этот формат с помощью Aspose Imaging.

#### Шаги реализации

##### Загрузка оригинального PNG‑изображения (H3)
Начните с загрузки исходного PNG‑изображения:

```java
String originPngFileName = "YOUR_DOCUMENT_DIRECTORY/lena24b.png";
PngImage pngImage = (PngImage) Image.load(originPngFileName);
```
*Почему?*: Загрузка изображения позволяет получить к нему доступ и манипулировать им с помощью мощного API Aspose Imaging.

##### Настройка параметров JPEG (H3)
Установите необходимые параметры для конвертации:

```java
int bpp = 2; // Bits per sample
JpegOptions jpegOptions = new JpegOptions();
jpegOptions.setBitsPerChannel((byte) bpp);
jpegOptions.setCompressionType(JpegCompressionMode.JpegLs);
```
*Почему?*: Настройка этих параметров определяет тип сжатия и параметры качества для выходного JPEG‑LS изображения.

##### Сохранение изображения (H3)
Наконец, сохраните преобразованное изображение:

```java
String outputJpegFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.jls";
pngImage.save(outputJpegFileName, jpegOptions);
```
*Почему?*: Сохранение в конкретном формате завершает процесс конвертации.

##### Управление ресурсами (H3)
Убедитесь, что ресурсы освобождены после обработки:

```java
finally {
    pngImage.dispose();
}
```
*Почему?*: Освобождение объектов изображения помогает эффективно управлять использованием памяти.

#### Советы по устранению неполадок

- **Распространённая проблема**: При возникновении `FileNotFoundException` проверьте пути к файлам ввода и вывода.
- **Совет по решению**: Убедитесь, что ваша IDE имеет разрешения на чтение и запись в указанные каталоги.

### Конвертация JPEG‑LS в PNG (H2)

Эта функция позволяет вернуть JPEG‑LS изображение в формат PNG, облегчая визуальное сравнение или дальнейшую обработку.

#### Обзор
После конвертации изображения в JPEG‑LS для целей сжатия может потребоваться обратный процесс, чтобы оценить качество или использовать изображение в других целях. В этом разделе объясняется, как выполнить обратную конвертацию с помощью Aspose Imaging.

#### Шаги реализации

##### Загрузка JPEG‑LS изображения (H3)
Начните с загрузки вашего JPEG‑LS файла:

```java
String inputJpegFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.jls";
JpegImage jpegImage = (JpegImage) Image.load(inputJpegFileName);
```
*Почему?*: Загрузка изображения подготавливает его к конвертации в другой формат.

##### Сохранить как PNG (H3)
Преобразуйте и сохраните изображение в формате PNG:

```java
String outputPngFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.png";
jpegImage.save(outputPngFileName, new PngOptions());
```
*Почему?*: Этот шаг гарантирует, что ваш JPEG‑LS файл будет преобразован обратно в широко совместимый и без потерь PNG‑формат.

##### Управление ресурсами (H3)
Освободите ресурсы изображения после сохранения:

```java
finally {
    jpegImage.dispose();
}
```
*Почему?*: Правильное освобождение предотвращает утечки памяти, поддерживая производительность приложения.
