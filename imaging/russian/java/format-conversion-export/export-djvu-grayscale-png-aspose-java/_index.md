## Как загрузить изображение DjVu?
Загрузка файла DjVu проста с Aspose.Imaging. Используйте класс `Image` для открытия документа внутри блока `try‑with‑resources`, который автоматически освобождает файловые дескрипторы после обработки. Укажите путь к файлу .djvu, и библиотека разберёт его страницы, сделав их доступными для дальнейшей манипуляции.

`Image` — класс в Aspose.Imaging, представляющий файл изображения и предоставляющий методы для загрузки и обработки данных изображения.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.djvu.DjvuImage;

String dataDir = "YOUR_DOCUMENT_DIRECTORY/Sample.djvu";
try (DjvuImage image = (DjvuImage) Image.load(dataDir)) {
    // The image is now loaded and ready for further processing.
}
```

- **`dataDir`**: Путь к вашему файлу DjVu. Замените `YOUR_DOCUMENT_DIRECTORY` на фактический путь к каталогу.  
- **`try‑with‑resources`**: Автоматически управляет освобождением ресурсов после загрузки.

## Как настроить параметры экспорта PNG?
Чтобы получить градационный PNG, создайте объект `PngOptions` и установите его `colorType` в `PngColorType.Grayscale`. Эта конфигурация гарантирует, что выходное изображение будет использовать один канал интенсивности, уменьшая размер файла при сохранении визуальных деталей. Дополнительные параметры, такие как уровень сжатия, также могут быть скорректированы при необходимости.

`PngOptions` определяет настройки сохранения изображений в формате PNG, такие как тип цвета, уровень сжатия и чередование.  
`PngColorType` перечисляет поддерживаемые форматы цвета для PNG‑файлов, включая Grayscale, Truecolor и Indexed.

```java
import com.aspose.imaging.imageoptions.PngOptions;
import com.aspose.imaging.fileformats.png.PngColorType;

PngOptions exportOptions = new PngOptions();
exportOptions.setColorType(PngColorType.Grayscale);
```

- **`PngColorType.Grayscale`**: Преобразует изображение в градацию, идеально подходит для архивных целей или уменьшения размера файла.

## Как определить область экспорта на странице DjVu?
Укажите прямоугольную область, которую нужно извлечь со страницы DjVu, задав свойство `exportArea` у `DjvuMultiPageOptions`. Прямоугольник задаётся в пикселях и определяет часть страницы, которая будет отрисована и сохранена, позволяя сосредоточиться на интересующей области.

`DjvuMultiPageOptions` задаёт параметры экспорта страниц из документа DjVu, включая индекс страницы и область экспорта.

```java
import com.aspose.imaging.Rectangle;

Rectangle exportArea = new Rectangle(0, 0, 500, 500);
int exportPageIndex = 2; // Index starts from zero.
```

- **`exportArea`**: Определяет размеры прямоугольника для обрезки. Измените эти значения в соответствии с вашими потребностями.

## Как задать параметры экспорта многостраничного документа?
При работе с многостраничными файлами DjVu используйте `DjvuMultiPageOptions` для выбора нужной страницы через свойство `pageIndex` и при необходимости комбинируйте его с `exportArea`. Такой подход позволяет экспортировать конкретный регион любой страницы без обработки всего документа, оптимизируя производительность для больших файлов.

```java
import com.aspose.imaging.fileformats.djvu.DjvuMultiPageOptions;

exportOptions.setMultiPageOptions(new DjvuMultiPageOptions(exportPageIndex, exportArea));
```

- **`DjvuMultiPageOptions`**: Объединяет индекс страницы и область для экспорта определённых секций многостраничного файла DjVu.

## Как сохранить изображение?
После настройки параметров экспорта вызовите метод `save` у экземпляра `Image`, указав целевой путь к файлу и объект `PngOptions`. Убедитесь, что каталог вывода существует и имеет права записи; библиотека запишет градационный PNG на диск.

`save` — метод класса `Image`, записывающий данные изображения в указанный файл с использованием предоставленных параметров формата.

```java
import com.aspose.imaging.Image;

String outputPath = "YOUR_OUTPUT_DIRECTORY/ConvertSpecificPortionOfDjVuPage_out.png";
image.save(outputPath, exportOptions);
```

- **`outputPath`**: Укажите, куда сохранить ваш PNG‑файл. Убедитесь, что каталог существует перед запуском.
