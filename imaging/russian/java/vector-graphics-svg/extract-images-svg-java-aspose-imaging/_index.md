## Руководство по внедрению

В этом разделе мы разберем каждую функцию на простые шаги, которые помогут вам загрузить SVG-файлы, извлечь изображения, сохранить их и выполнить последующую очистку.

### Загрузка SVG и извлечение встроенных изображений

#### Обзор

Эта функция позволяет нам загружать определенный SVG-файл и получать доступ ко всем встроенным растровым изображениям, которые он содержит.

#### Этапы внедрения

1. **Определите входной путь:**

   Укажите путь к каталогу вашего SVG-файла в коде:

   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY/svg/";
   String fileName = dataDir + "test2.svg";
   ```

2. **Загрузите и транслируйте изображение:**

   Используйте Aspose.Imaging для загрузки SVG, а затем преобразуйте его в `VectorImage` тип.

   ```java
   try (Image image = Image.load(fileName)) {
       EmbeddedImage[] images = ((VectorImage)image).getEmbeddedImages();
   }
   ```

3. **Извлечение встроенных изображений:**

   The `getEmbeddedImages()` Метод возвращает массив встроенных растровых изображений, которые затем могут быть обработаны дополнительно.

### Итерация и сохранение встроенных изображений

#### Обзор

Эта функция включает в себя итерацию каждого извлеченного изображения, генерацию уникального имени файла и сохранение изображений в желаемом месте.

#### Этапы внедрения

1. **Настройте выходной путь:**

   Определите, где вы хотите сохранить извлеченные изображения:

   ```java
   String outputFolder = "YOUR_OUTPUT_DIRECTORY/svg/";
   ```

2. **Итерация по изображениям:**

   Пройдитесь по каждому `EmbeddedImage` объект и извлечь данные его изображения.

   ```java
   List<String> files = new ArrayList<>();
   int i = 0;
   
   for (EmbeddedImage im : images) {
       Image imImage = im.getImage();
       String outFileName = "svg_image" + i++ + getExtension(imImage.getFileFormat());
       String outFilePath = outputFolder + outFileName;
       files.add(outFilePath);
       
       // Сохранить изображение
       imImage.save(outFilePath);
   }
   ```

3. **Создать расширения файлов:**

   Используйте вспомогательный метод для определения расширения файла на основе формата изображения.

   ```java
   private static String getExtension(long format) {
       if (format == com.aspose.imaging.FileFormat.Jpeg) {
           return ".jpg";
       } else if (format == com.aspose.imaging.FileFormat.Png) {
           return ".png";
       } else if (format == com.aspose.imaging.FileFormat.Bmp) {
           return ".bmp";
       }
       return "." + com.aspose.imaging.FileFormat.toString(com.aspose.imaging.FileFormat.class, format);
   }
   ```

### Очистка после обработки изображения

#### Обзор

После обработки изображений рекомендуется удалить все временные файлы, созданные в ходе процесса.

#### Этапы внедрения

1. **Список временных файлов:**

   Ведите список путей к файлам, которые вы собираетесь удалить после использования:

   ```java
   List<String> files = List.of("YOUR_OUTPUT_DIRECTORY/svg/svg_image0.jpg");
   ```

2. **Удалить временные файлы:**

   Пройдитесь по списку файлов и удалите каждый из них.

   ```java
   for (String file : files) {
       File tempFile = new File(file);
       if (tempFile.exists()) {
           boolean deleted = tempFile.delete();
           if (!deleted) {
               System.err.println("Failed to delete " + file);
           }
       }
   }
   ```
