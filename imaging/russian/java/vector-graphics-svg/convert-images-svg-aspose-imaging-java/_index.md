## Руководство по внедрению

Мы разобьем урок на две основные части: загрузку изображений и преобразование их в SVG.

### Функция 1: Загрузка нескольких форматов изображений

#### Обзор
Эта функция демонстрирует, как загружать различные форматы изображений из каталога, подготавливая их к конвертации.

##### Пошаговая реализация

**H3. Определить пути**
Создайте массив, содержащий пути к различным файлам изображений:
```java
String[] paths = new String[]{
    "butterfly.gif",
    "33715-cmyk.jpeg",
    "3.JPG",
    "test.j2k",
    "Rings.png",
    "img4.TIF",
    "Lossy5.webp"
};
```

**H3. Загрузить изображения**
Переберите пути для загрузки каждого изображения:
```java
for (String path : paths) {
    Image image = Image.load("YOUR_DOCUMENT_DIRECTORY" + path);
    try {
        // Обработка будет выполнена в последующих функциях.
    } finally {
        image.dispose(); // Освобождение ресурсов после загрузки.
    }
}
```
**Объяснение**: `Image.load()` Метод считывает файл в память. Использование `try-finally` обеспечивает правильную утилизацию каждого изображения, эффективно управляя использованием ресурсов.

### Функция 2: Преобразование изображений в формат SVG

#### Обзор
Конвертируйте загруженные изображения в формат SVG, используя мощные возможности Aspose.Imaging для векторной растеризации.

##### Пошаговая реализация

**H3. Загрузка и подготовка изображения**
Загрузите пример изображения, чтобы продемонстрировать процесс конвертации:
```java
Image image = Image.load("YOUR_DOCUMENT_DIRECTORY" + "butterfly.gif");
```

**H3. Настройка параметров SVG**
Настройте параметры преобразования растровых изображений в формат SVG:
```java
SvgOptions svgOptions = new SvgOptions();
SvgRasterizationOptions svgRasterizationOptions = new SvgRasterizationOptions();

svgRasterizationOptions.setPageWidth(image.getWidth());
svgRasterizationOptions.setPageHeight(image.getHeight());

svgOptions.setVectorRasterizationOptions(svgRasterizationOptions);
```
**Объяснение**: `SvgRasterizationOptions` определить, как изображение растрируется в SVG. Установка ширины и высоты страницы в соответствии с исходным изображением гарантирует, что векторизованный вывод сохранит свои размеры.

**H3. Сохранить как SVG**
Определите путь назначения и сохраните преобразованное изображение:
```java
String destPath = "YOUR_OUTPUT_DIRECTORY" + "butterfly.svg";
image.save(destPath, svgOptions);
```
Наконец, удалите изображение, чтобы освободить ресурсы:
```java
finally {
    image.dispose();
}
```
