## Часто задаваемые вопросы

**Q: Можно ли конвертировать SVG в PNG без написания кода?**  
A: Да, онлайн‑демо Aspose.Imaging позволяет загрузить SVG и скачать PNG, но использование Java API обеспечивает полную автоматизацию.

**Q: Поддерживается ли “svg to png java” на Linux?**  
A: Абсолютно. Aspose.Imaging for Java независим от платформы и работает на Windows, Linux и macOS.

**Q: Как сохранить текст в виде выбираемых слоёв при конвертации?**  
A: Используйте `SvgExportOptions`, чтобы сохранить текст как векторные данные, либо растеризуйте с высоким DPI, если нужен растровый файл.

**Q: Какой DPI использовать для PNG высокого разрешения?**  
A: DPI 300–600 обычно используется для печати; для веба достаточно 72–96 DPI.

**Q: Поддерживает ли библиотека пакетную конвертацию?**  
A: Да — оберните логику загрузки‑и‑сохранения в цикл или используйте `Parallel.ForEach` для многопоточной обработки.
