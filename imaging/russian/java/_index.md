## Часто задаваемые вопросы

**Q: Можно ли использовать Aspose.Imaging в коммерческих Java‑проектах?**  
A: Да. Для продакшн‑использования требуется коммерческая лицензия, а бесплатная пробная версия доступна для оценки.

**Q: Поддерживает ли библиотека последние версии Java?**  
A: Библиотека полностью совместима с Java 8, 11, 17 и более новыми LTS‑выпусками.

**Q: Как пакетная обработка влияет на использование памяти?**  
A: Aspose.Imaging использует потоковую передачу и отложенную загрузку, чтобы держать объём памяти низким; также можно настроить пользовательские ограничения памяти для больших пакетов.

**Q: Можно ли конвертировать изображения без записи их на диск?**  
A: Абсолютно. Вы можете работать с потоками (например, `ByteArrayInputStream`/`ByteArrayOutputStream`), полностью конвертируя изображения в памяти.

**Q: Какие форматы поддерживаются для конвертации DICOM?**  
A: Поддерживаются все стандартные модальности DICOM, включая JPEG‑Lossless, JPEG‑2000 и необработанные пиксельные данные.
