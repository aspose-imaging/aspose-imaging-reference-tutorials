### Загрузка изображений TIFF из потока

**Обзор**: В этом разделе показано, как загрузить изображение TIFF из потока с помощью `FileInputStream`.

#### Шаг 1: Определите путь к изображению и загрузите изображение

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.tiff.TiffImage;
import java.io.FileInputStream;

public class LoadTIFFImagesFromStream {
    public static void main(String[] args) throws Exception {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        try (FileInputStream fileStream = new FileInputStream(dataDir + "TestDemo.tif")) {
            TiffImage image = (TiffImage) Image.load(fileStream);
            // Изображение теперь загружено и его можно изменять по мере необходимости.
        }
    }
}
```

- **Объяснение**: Начнем с указания пути к нашему файлу TIFF. Используя `FileInputStream`, мы загружаем изображение в `TiffImage` объект, что позволяет нам получить доступ к его кадрам для дальнейшей обработки.

### Копирование кадров TIFF из источника в место назначения

**Обзор**: Узнайте, как скопировать активный кадр из одного изображения TIFF и добавить его в другое.

#### Шаг 1: Загрузка исходных и конечных изображений

```java
import com.aspose.imaging.fileformats.tiff.TiffFrame;
import com.aspose.imaging.fileformats.tiff.TiffImage;
import java.io.FileInputStream;

public class CopyTIFFFrames {
    public static void main(String[] args) throws Exception {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        try (FileInputStream fileStream = new FileInputStream(dataDir + "TestDemo.tif")) {
            try (TiffImage destinationImage = (TiffImage) Image.load(fileStream)) {
                try (FileInputStream fileStream1 = new FileInputStream(dataDir + "sample1.tif")) {
                    try (TiffImage sourceImage = (TiffImage) Image.load(fileStream1)) {
                        TiffFrame copiedFrame = TiffFrame.copyFrame(sourceImage.getActiveFrame());
                        destinationImage.addFrame(copiedFrame);
                    }
                }
            }
        }
    }
}
```

- **Объяснение**: Здесь мы открываем как исходный, так и целевой файлы TIFF. Используя `TiffFrame.copyFrame()`активный кадр копируется из исходного изображения и добавляется в место назначения с помощью `addFrame()`.

### Сохранение объединенных изображений TIFF

**Обзор**: эта функция демонстрирует, как сохранить измененное изображение TIFF после добавления кадров или других изменений.

#### Шаг 1: Сохраните измененное изображение.

```java
import com.aspose.imaging.Image;
import java.io.IOException;

public class SaveConcatenatedTIFFImages {
    public static void main(String[] args) throws IOException {
        String outputDir = "YOUR_OUTPUT_DIRECTORY";
        
        // Предположим, что «изображение» — это объект TiffImage, который был изменен (например, добавлены кадры).
        Image image;  // Заполнитель для фактического кода обработки изображения
        
        image.save(outputDir + "ConcatenatedTIFFImage_out.tif");
    }
}
```

- **Объяснение**: После изменения изображения TIFF используйте `save()` метод для записи его обратно на диск. Этот шаг завершает любые изменения, сделанные во время обработки.
