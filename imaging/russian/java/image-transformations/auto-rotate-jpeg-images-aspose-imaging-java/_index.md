## Руководство по внедрению

### Функция 1: Загрузка изображения JPEG

#### Обзор
Загрузка изображения — первый шаг в любой задаче обработки изображений. Вот как загрузить изображение JPEG из указанного каталога.

##### Пошаговая реализация

**H3: Импорт библиотек**

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.jpeg.JpegImage;
```

**H3: Загрузите изображение**

Укажите путь к каталогу документа и загрузите изображение:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
try (JpegImage image = (JpegImage) Image.load(dataDir + "aspose-logo.jpg")) {
    // Изображение теперь загружено для обработки.
}
```

**Объяснение**: `Image.load()` метод считывает файл JPEG в `JpegImage` объект, позволяющий выполнять последующие операции.

### Функция 2: Автоматический поворот изображения JPEG на основе данных ориентации EXIF

#### Обзор
Автоматический поворот изображений на основе метаданных ориентации EXIF гарантирует их корректное отображение без ручной настройки.

##### Пошаговая реализация

**H3: Загрузка и автоматический поворот изображения**

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
try (JpegImage image = (JpegImage) Image.load(dataDir + "aspose-logo.jpg")) {
    // Выполнять автоматический поворот на основе ориентации EXIF
    image.autoRotate();
}
```

**Объяснение**: `autoRotate()` Метод корректирует ориентацию изображения в соответствии с встроенными данными EXIF, оптимизируя рабочий процесс.

### Функция 3: Сохранение изображения JPEG на диск

#### Обзор
После обработки изображения его сохранение обратно на диск имеет решающее значение для хранения или дальнейшего использования. Вот как можно сохранить обработанное изображение JPEG.

##### Пошаговая реализация

**H3: Загрузка, автоматический поворот и сохранение изображения**

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";

try (JpegImage image = (JpegImage) Image.load(dataDir + "aspose-logo.jpg")) {
    // Выполнять автоматический поворот на основе ориентации EXIF
    image.autoRotate();

    // Сохранить обработанное изображение на диск
    image.save(outDir + "AutoCorrectOrientationOfJPEGImages_out.jpg");
}
```

**Объяснение**: `save()` метод записывает обработанный JPEG обратно в указанный вами каталог, сохраняя все изменения.
