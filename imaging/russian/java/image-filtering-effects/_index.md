- **Что делает фильтр Gauss Wiener?** Он уменьшает шум, сохраняя границы, комбинируя гауссово сглаживание с вейнеровской деконволюцией.

## Что такое Gauss Wiener Filter Java?
Алгоритм **gauss wiener filter java** сочетает гауссово размытие с вейнеровским фильтром, интеллектуально сглаживая шумные области, при этом сохраняет важные детали резкими. Особенно полезен для фотографий, снятых при низкой освещённости или с размытием движения.

**Q: Является ли Gauss Wiener filter потокобезопасным?**
