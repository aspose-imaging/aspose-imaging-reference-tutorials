## Часто задаваемые вопросы

**В: Можно ли конвертировать несколько PNG‑файлов в один PDF одним вызовом?**  
О: Да, Aspose.Imaging позволяет добавить каждый PNG как страницу в PDF‑документ перед сохранением.

**В: Как внедрить XMP‑метаданные при создании PDF?**  
О: Используйте класс `ImageMetadata` для установки XMP‑свойств перед сохранением изображения или PDF.

**В: Есть ли способ мониторить длительные конвертации?**  
О: Абсолютно – библиотека предоставляет события прогресса, которые можно подключить для отображения статуса конвертации.

**В: Поддерживает ли библиотека конвертацию TIFF высокого разрешения в PDF?**  
О: Да, сохраняются DPI и цветовая информация, гарантируя, что PDF соответствует оригинальному TIFF по качеству.

**В: Какая лицензия требуется для продакшн‑использования?**  
О: Для продакшн‑использования требуется коммерческая лицензия Aspose.Imaging; бесплатная пробная версия доступна для оценки.
