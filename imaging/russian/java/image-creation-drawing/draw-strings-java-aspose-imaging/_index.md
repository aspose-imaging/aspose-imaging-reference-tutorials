### Рисование строк с различным выравниванием

Эта функция позволяет рисовать строки на изображении в разных выравниваниях: слева, по центру и справа. Она улучшает визуальное представление данных, размещая текст точно там, где это необходимо.

#### Обзор

Приведенный фрагмент кода демонстрирует, как создать изображение и нарисовать строки, используя различные шрифты и размеры, выровненные по вашему выбору.

#### Пошаговая реализация

##### Шаг 1: Инициализация PngOptions

Создать `PngOptions` объект и задать его свойства. Это настраивает формат выходного файла для вашего изображения.

```java
try (PngOptions pngOptions = new PngOptions()) {
    // Установите источник для PngOptions
    pngOptions.setSource(new FileCreateSource(outputFileName));
}
```

##### Шаг 2: Создайте изображение

Использовать `Image.create()` для инициализации нового изображения с указанными размерами.

```java
try (Image image = Image.create(pngOptions, width, height)) {
    Graphics graphics = new Graphics(image);
    graphics.beginUpdate();
    graphics.clear(Color.getWhite());
    // Продолжайте дальнейшие операции...
}
```

##### Шаг 3: Определите выравнивание строк

Установите выравнивание на основе ввода пользователя. Это определяет, как текст будет располагаться по горизонтали.

```java
int alignment;
switch (align) {
    case "Left":
        alignment = StringAlignment.Near;
        break;
    case "Center":
        alignment = StringAlignment.Center;
        break;
    case "Right":
        alignment = StringAlignment.Far;
        break;
    default:
        throw new IllegalArgumentException("Wrong alignment: " + align);
}
StringFormat stringFormat = new StringFormat(StringFormatFlags.ExactAlignment);
stringFormat.setAlignment(alignment);
```

##### Шаг 4: Нарисуйте нити

Пройдитесь по списку шрифтов и размеров, чтобы нарисовать строки на изображении. Используйте `graphics.drawString()` для рендеринга текста.

```java
for (String fontName : fontNames) {
    for (float fontSize : fontSizes) {
        Font font = new Font(fontName, fontSize);
        String text = String.format("This is font: %s, size:%d", fontName, (int) fontSize);
        
        SizeF s = graphics.measureString(text, font, SizeF.getEmpty(), null);
        graphics.drawString(text, font, brush, new RectangleF(x, y, w, s.getHeight()), stringFormat);
        
        // Перейти к следующей позиции строки
        y += s.getHeight();
    }
    
    // Нарисуйте горизонтальную линию после каждого набора струн.
    graphics.drawLine(pen, new Point((int) (x), (int) y), new Point((int) (x + w), (int) y));
}
```

##### Шаг 5: Завершите и сохраните

После того, как вы нарисуете ниточки, завершите свои операции с помощью `graphics.endUpdate()` и сохраните изображение.

```java
// Нарисуйте вертикальную линию, указывающую положение выравнивания.
graphics.drawLine(pen, new Point(lineX, 0), new Point(lineX, (int) y));
graphics.endUpdate();
image.save();
```
