## Руководство по внедрению

### Рисование линий с помощью EmfRecorderGraphics2D

В этом разделе рассматриваются основы рисования линий с помощью `EmfRecorderGraphics2D`, что позволяет настраивать цвет линии, толщину и стили концевой защелки.

#### Шаг 1: Инициализация графического объекта
Создать экземпляр `EmfRecorderGraphics2D` чтобы настроить свой холст:

```java
EmfRecorderGraphics2D graphics = new EmfRecorderGraphics2D(new Rectangle(0, 0, 1000, 1000),
        new Size(1000, 1000), new Size(100, 100));
```

#### Шаг 2: Нарисуйте основную линию

Используйте `Pen` класс для определения свойств линии:

```java
// Возьмите ручку цвета «бисквит» и нарисуйте линию.
Pen pen = new Pen(Color.getBisque());
graphics.drawLine(pen, 1, 1, 50, 50);
```

#### Шаг 3: Настройте стили пера

Измените цвет, толщину и стиль наконечника пера, чтобы добавить разнообразия:

```java
// Установите для параметра Pen значение BlueViolet, толщину 3 и закругленный наконечник.
pen = new Pen(Color.getBlueViolet(), 3);
pen.setEndCap(LineCap.Round);
graphics.drawLine(pen, 15, 5, 50, 60);

// Измените заглушку на квадратную и нарисуйте еще одну линию.
pen.setEndCap(LineCap.Square);
graphics.drawLine(pen, 5, 10, 50, 10);

// Используйте плоскую заглушку.
pen.setEndCap(LineCap.Flat);
graphics.drawLine(pen, new Point(5, 20), new Point(50, 20));
```

### Рисование фигур с помощью HatchBrush

Изучите рисование прямоугольников с помощью `HatchBrush` для заполнения узоров и настройки фоновых режимов.

#### Шаг 1: Создание HatchBrush
Определите стиль штриховки для узорчатой заливки:

```java
// Настройте HatchBrush с рисунком Cross.
HatchBrush hatchBrush = new HatchBrush();
hatchBrush.setBackgroundColor(Color.getAliceBlue());
hatchBrush.setForegroundColor(Color.getRed());
hatchBrush.setHatchStyle(HatchStyle.Cross);
```

#### Шаг 2: Нарисуйте узорчатый прямоугольник

Используйте `Pen` объект для рисования прямоугольников с заданными узорами:

```java
pen = new Pen(hatchBrush, 7);
graphics.drawRectangle(pen, 50, 50, 20, 30);

// Установите режим фона на НЕПРОЗРАЧНЫЙ для рисования еще одной линии.
graphics.setBackgroundMode(EmfBackgroundMode.OPAQUE);
graphics.drawLine(pen, 80, 50, 80, 80);
```

### Рисование многоугольников и прямоугольников с использованием стилей соединения линий

Эта функция демонстрирует, как рисовать многоугольники и прямоугольники с использованием различных `LineJoin` стили.

#### Шаг 1: Настройте Pen для Polygon
Настройте стиль соединения линий пера для рисования многоугольника:

```java
// Установите цвет пера «Акварель» с соединением MiterClipped.
pen = new Pen(new SolidBrush(Color.getAqua()), 3);
pen.setLineJoin(LineJoin.MiterClipped);
graphics.drawPolygon(pen, new Point[]{new Point(10, 20), new Point(12, 45),
        new Point(22, 48), new Point(48, 36), new Point(30, 55)});
```

#### Шаг 2: Нарисуйте прямоугольники с различными линиями соединения

Поэкспериментируйте с различными стилями соединения:

```java
// Измените соединение на Bevel и нарисуйте прямоугольник.
pen.setLineJoin(LineJoin.Bevel);
graphics.drawRectangle(pen, 50, 10, 10, 5);

// Установите параметр «Скруглить» для другого прямоугольника.
pen.setLineJoin(LineJoin.Round);
graphics.drawRectangle(pen, 65, 10, 10, 5);

// Для создания конечного прямоугольника используйте соединение под углом.
pen.setLineJoin(LineJoin.Miter);
graphics.drawRectangle(pen, 80, 10, 10, 5);
```

### Завершение и сохранение графики

Завершите сеанс рисования, сохранив графику в виде файла EMF или PDF:

```java
try (EmfImage image = graphics.endRecording()) {
    PdfOptions options = new PdfOptions();
    EmfRasterizationOptions rasterizationOptions = new EmfRasterizationOptions();
    options.setVectorRasterizationOptions(rasterizationOptions);
    rasterizationOptions.setPageSize(Size.to_SizeF(image.getSize()));
    
    String outPath = "YOUR_OUTPUT_DIRECTORY/Picture1.emf.pdf";
    image.save(outPath, options);
}
```
