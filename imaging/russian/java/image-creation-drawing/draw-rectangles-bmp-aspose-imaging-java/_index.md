### Создание BmpOptions и настройка свойств

**Обзор:**
The `BmpOptions` класс позволяет вам настраивать различные параметры для создания изображения BMP. Вот как задать его свойства:

1. **Инициализировать `BmpOptions`:**

   ```java
   import com.aspose.imaging.imageoptions.BmpOptions;
   import java.io.ByteArrayInputStream;

   try (BmpOptions bmpCreateOptions = new BmpOptions()) {
       // Установите количество бит на пиксель для изображения
       bmpCreateOptions.setBitsPerPixel(32);

       // Определить источник с помощью входного потока байтового массива
       bmpCreateOptions.setSource(new StreamSource(new ByteArrayInputStream(
           new byte[100 * 100 * 4])));
   }
   ```

   **Объяснение:**
   - `setBitsPerPixel(32)`: Настраивает изображение на использование 32 бит на пиксель, обеспечивая высококачественную глубину цвета.
   - Размер массива байтов `[100 * 100 * 4]` вычисляет необходимое количество байтов для изображения размером 100x100 с 32 битами (4 байта) на пиксель.

### Создание изображений и рисование прямоугольников

**Обзор:**
Эта функция демонстрирует, как создать экземпляр изображения и нарисовать прямоугольники, используя разные стили.

1. **Создайте экземпляр изображения:**

   ```java
   import com.aspose.imaging.Image;
   import com.aspose.imaging.Graphics;
   import com.aspose.imaging.Color;
   import com.aspose.imaging.Rectangle;
   import com.aspose.imaging.Pen;
   import com.aspose.imaging.brushes.SolidBrush;

   try (BmpOptions bmpCreateOptions = new BmpOptions()) {
       bmpCreateOptions.setBitsPerPixel(32);
       bmpCreateOptions.setSource(new StreamSource(new ByteArrayInputStream(
           new byte[100 * 100 * 4])));

       // Создайте изображение размером 100x100, используя настроенные параметры.
       try (Image image = Image.create(bmpCreateOptions, 100, 100)) {
           Graphics graphic = new Graphics(image);
           graphic.clear(Color.getYellow());

           // Нарисуйте пунктирный прямоугольник красной ручкой.
           Pen redPen = new Pen(Color.getRed());
           Rectangle rect1 = new Rectangle(30, 10, 40, 80);
           graphic.drawRectangle(redPen, rect1);

           // Нарисуйте непрерывный прямоугольник, используя синюю сплошную кисть.
           Pen bluePen = new Pen(new SolidBrush(Color.getBlue()));
           Rectangle rect2 = new Rectangle(10, 30, 80, 40);
           graphic.drawRectangle(bluePen, rect2);

           image.save("YOUR_OUTPUT_DIRECTORY/DrawingRectangle_out.bmp");
       }
   }
   ```

   **Объяснение:**
   - `Image.create(bmpCreateOptions, 100, 100)`: Инициализирует новое изображение с указанными размерами и параметрами.
   - `Graphics` объект: Используется для рисования на поверхности изображения. 
   - `drawRectangle()`: Рисует прямоугольники, используя указанные перья и кисти.
