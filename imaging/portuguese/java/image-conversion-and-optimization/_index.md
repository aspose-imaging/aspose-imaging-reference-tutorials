## O que é **convert raster to SVG**?
Converter imagens raster (como JPEG, PNG ou BMP) para SVG reescreve os dados visuais como caminhos matemáticos. Isso torna a imagem independente de resolução, perfeita para designs responsivos, mídia impressa e ícones de UI.
