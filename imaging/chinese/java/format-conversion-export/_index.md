## 如何在 Java 中执行 JPEG CMYK 转换？

使用 `Image.load` 加载 JPEG，切换其 `ColorType` 为 `CMYK`（或 `YCCK`），然后调用 `save` 写入结果——整个工作流可以用几行简洁的 Java 代码完成。  
`ColorType` 指定图像的颜色空间，例如 RGB、CMYK 或 YCCK。  
此方法保留图像质量，保留 EXIF 元数据，并且在 Windows 和 Linux 环境下均可工作，无需本机库。  

`Image` 类是 Aspose.Imaging 的核心对象，表示内存中的任何光栅图像。加载后，您可以查询或修改其属性，如颜色深度、DPI 和颜色空间，然后再持久化转换后的文件。
