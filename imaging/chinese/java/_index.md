## 常见问题

**问：可以在没有许可证的情况下生成动画 GIF 吗？**  
答：您可以使用免费试用版进行评估性 GIF 创建，但生产部署需要商业许可证。

**问：如何控制动画 GIF 中帧的延迟？**  
答：在将每帧添加到 `GifImage` 时，通过 `GifFrame` 的属性指定该帧的延迟（毫秒）。

**问：能否将一批 PNG 合并为动画 GIF？**  
答：可以——在循环中将 PNG 作为单独帧加入，设置所需延迟后保存生成的 `GifImage`。

**问：处理大图像时内存需要注意哪些问题？**  
答：使用基于流的加载，及时释放图像对象，并考虑分块处理图像，以保持 JVM 堆使用量低。

**问：Aspose.Imaging 是否支持其他动画格式，如 APNG？**  
答：库主要聚焦于 GIF 动画，但您可以将单帧导出为其他格式（如 APNG），并使用外部工具完成动画合成。
