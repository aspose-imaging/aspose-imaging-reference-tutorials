## 常见问题

**Q: 能否一次调用将多个 PNG 文件合并为一个 PDF？**  
A: 可以，Aspose.Imaging 允许您在保存 PDF 文档之前将每个 PNG 添加为页面。

**Q: 在创建 PDF 时如何嵌入 XMP 元数据？**  
A: 使用 `ImageMetadata` 类在保存图像或 PDF 之前设置 XMP 属性。

**Q: 是否有办法监控长时间运行的转换？**  
A: 当然可以——库提供进度事件，您可以挂钩这些事件以显示转换状态。

**Q: 库是否支持高分辨率 TIFF 到 PDF 的转换？**  
A: 支持，保留 DPI 和颜色信息，确保 PDF 与原始 TIFF 质量一致。

**Q: 生产环境需要什么许可证？**  
A: 生产环境需要商业版 Aspose.Imaging 许可证；免费试用可用于评估。
