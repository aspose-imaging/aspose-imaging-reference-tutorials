## 将 CDR 转换为 PNG – 快速概览
将 CorelDRAW (CDR) 文件转换为 PNG 是在需要从矢量源获取适用于网页的光栅图像时的常见任务。Aspose.Imaging for Java 在内部处理光栅化，因此你无需担心第三方工具或手动 DPI 计算。下面的专门教程将引导你加载 CDR 文件、配置光栅化选项，并将结果保存为高质量的 PNG。
