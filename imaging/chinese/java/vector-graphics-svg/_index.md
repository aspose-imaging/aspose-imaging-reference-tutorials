## 常见问题

**Q: 我可以在不编写任何代码的情况下将 SVG 转换为 PNG 吗？**  
A: 可以，Aspose.Imaging 在线演示允许您上传 SVG 并下载 PNG，但使用 Java API 可实现完整自动化。

**Q: “svg to png java” 在 Linux 上受支持吗？**  
A: 当然。Aspose.Imaging for Java 与平台无关，可在 Windows、Linux 和 macOS 上运行。

**Q: 转换时如何将文本保留为可选择的图层？**  
A: 使用 `SvgExportOptions` 将文本保留为矢量数据，或在需要位图时使用高 DPI 栅格化。

**Q: 高分辨率 PNG 应使用什么 DPI？**  
A: 打印质量通常使用 300–600 DPI；网页则 72–96 DPI 足够。

**Q: 该库支持批量转换吗？**  
A: 支持——将加载‑保存逻辑放入循环，或使用 `Parallel.ForEach` 进行多线程处理。
