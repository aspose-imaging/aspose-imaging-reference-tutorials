## 常见问题

**Q: 我可以在 Web 应用程序中使用这些绘图功能吗？**  
A: 可以，Aspose.Imaging 可与 ASP.NET、ASP.NET Core 以及其他 Web 框架无缝配合。

**Q: 绘制大量形状会影响性能吗？**  
A: 该库已针对批量操作进行优化；不过，将绘图命令批处理可以进一步提升速度。

**Q: 我需要手动管理资源吗？**  
A: 使用 `using` 语句或在 `Image` 和 `Graphics` 对象上调用 `Dispose` 以及时释放本机资源。

**Q: 我可以将最终图像导出为 PDF 吗？**  
A: 当然可以 – Aspose.Imaging 能保存为 PDF，或将位图交给 Aspose.PDF 进行进一步处理。

**Q: 如何处理高分辨率图像？**  
A: 创建图像时设置所需的 DPI，并在矢量模式下使用 `Graphics` 以保持质量。
