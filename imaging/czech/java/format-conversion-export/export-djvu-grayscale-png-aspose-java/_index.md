# Jak exportovat DjVu do odstínů šedi PNG pomocí Aspose.Imaging pro Java

## Co je **convert djvu to png**?
Fráze **convert djvu to png** popisuje proces převzetí dokumentu nebo obrázku uloženého ve formátu DjVu a vykreslení každé stránky jako souboru PNG. PNG je široce podporovaný rastrový formát, který zachovává bezztrátovou kvalitu obrazu, což jej činí vhodným pro webové zobrazení, archivní ukládání a další úlohy zpracování obrázků.

## Jak načíst DjVu obrázek?
Načtení souboru DjVu je s Aspose.Imaging jednoduché. Použijte třídu `Image` k otevření dokumentu v bloku try‑with‑resources, který po zpracování automaticky uvolní souborové handly. Zadejte cestu k souboru .djvu a knihovna rozparsuje jeho stránky, čímž je zpřístupní pro další manipulaci.

`Image` je třída v Aspose.Imaging, která představuje soubor obrázku a poskytuje metody pro načítání a manipulaci s daty obrázku.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.djvu.DjvuImage;

String dataDir = "YOUR_DOCUMENT_DIRECTORY/Sample.djvu";
try (DjvuImage image = (DjvuImage) Image.load(dataDir)) {
    // The image is now loaded and ready for further processing.
}
```

- `dataDir`: Cesta k vašemu souboru DjVu. Změňte `YOUR_DOCUMENT_DIRECTORY` na skutečnou cestu adresáře.  
- `try‑with‑resources`: Automaticky spravuje uvolnění zdrojů po načtení.

## Jak nakonfigurovat možnosti exportu PNG?
Pro vytvoření PNG v odstínech šedi vytvořte objekt `PngOptions` a nastavte jeho `colorType` na `PngColorType.Grayscale`. Toto nastavení zajišťuje, že výstupní obrázek používá jediný kanál intenzity, čímž snižuje velikost souboru při zachování vizuálních detailů. Další možnosti, jako úroveň komprese, lze také podle potřeby upravit.

`PngOptions` definuje nastavení pro ukládání obrázků ve formátu PNG, jako typ barvy, úroveň komprese a prokládání.  
`PngColorType` vyjmenovává podporované barevné formáty pro soubory PNG, včetně Grayscale, Truecolor a Indexed.

```java
import com.aspose.imaging.imageoptions.PngOptions;
import com.aspose.imaging.fileformats.png.PngColorType;

PngOptions exportOptions = new PngOptions();
exportOptions.setColorType(PngColorType.Grayscale);
```

- `PngColorType.Grayscale`: Převádí obrázek do odstínů šedi, ideální pro archivní účely nebo snížení velikosti souboru.

## Jak definovat oblast exportu na stránce DjVu?
Určete obdélníkovou oblast, kterou chcete extrahovat ze stránky DjVu, nastavením vlastnosti `exportArea` třídy `DjvuMultiPageOptions`. Obdélník je definován v pixelech a určuje část stránky, která bude vykreslena a uložena, což vám umožní zaměřit se na konkrétní oblast zájmu.

`DjvuMultiPageOptions` specifikuje možnosti pro export stránek z dokumentu DjVu, včetně indexu stránky a oblasti exportu.

```java
import com.aspose.imaging.Rectangle;

Rectangle exportArea = new Rectangle(0, 0, 500, 500);
int exportPageIndex = 2; // Index starts from zero.
```

- `exportArea`: Definuje rozměry obdélníku pro oříznutí. Upravte tyto hodnoty podle svých potřeb.

## Jak nastavit možnosti exportu více stránek?
Při práci s více‑stránkovými soubory DjVu použijte `DjvuMultiPageOptions` k výběru požadované stránky pomocí vlastnosti `pageIndex` a volitelně ji kombinujte s `exportArea`. Tento přístup vám umožní exportovat konkrétní oblast z libovolné stránky bez zpracování celého dokumentu, čímž optimalizuje výkon pro velké soubory.

```java
import com.aspose.imaging.fileformats.djvu.DjvuMultiPageOptions;

exportOptions.setMultiPageOptions(new DjvuMultiPageOptions(exportPageIndex, exportArea));
```

- `DjvuMultiPageOptions`: Kombinuje index stránky a oblast pro export konkrétních částí více‑stránkového souboru DjVu.

## Jak uložit obrázek?
Po nastavení možností exportu zavolejte metodu `save` na instanci `Image`, přičemž zadáte cílovou cestu souboru a objekt `PngOptions`. Ujistěte se, že výstupní adresář existuje a má oprávnění k zápisu; knihovna pak zapíše PNG v odstínech šedi na disk.

`save` je metoda třídy `Image`, která zapisuje data obrázku do určeného souboru pomocí poskytnutých formátovacích možností.

```java
import com.aspose.imaging.Image;

String outputPath = "YOUR_OUTPUT_DIRECTORY/ConvertSpecificPortionOfDjVuPage_out.png";
image.save(outputPath, exportOptions);
```

- `outputPath`: Zadejte, kam chcete uložit svůj soubor PNG. Před spuštěním se ujistěte, že adresář existuje.
