## Co je Gauss Wiener Filter Java?
Algoritmus **gauss wiener filter java** spojuje Gaussian blur s Wiener filtrem, aby inteligentně vyhladil šumové oblasti a zároveň zachoval důležité detaily. Je obzvláště užitečný pro fotografie pořízené za špatného osvětlení nebo s pohybovým rozmazáním.

**Q: Mohu upravit sílu Gauss Wiener filtru?**

**Q: Funguje filtr na všech formátech podporovaných Aspose.Imaging?**  
A: Rozhodně – API pracuje s JPEG, PNG, TIFF, BMP a mnoha dalšími formáty.

**Q: Je Gauss Wiener filtr thread‑safe?**  
A: Zpracovatelské metody jsou bezstavové, takže je můžete bezpečně volat z více vláken.

**Q: Jak se Gauss Wiener filtr liší od jednoduchého Gaussian blur?**
