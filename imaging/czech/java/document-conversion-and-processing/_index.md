## Často kladené otázky

**Q: Mohu převést více PNG souborů do jednoho PDF v jediném volání?**  
A: Ano, Aspose.Imaging vám umožní přidat každý PNG jako stránku do PDF dokumentu před uložením.

**Q: Jak vložit XMP metadata při vytváření PDF?**  
A: Použijte třídu `ImageMetadata` k nastavení XMP vlastností před uložením obrázku nebo PDF.

**Q: Existuje způsob, jak sledovat dlouho běžící konverze?**  
A: Rozhodně – knihovna poskytuje události průběhu, které můžete zachytit a zobrazit stav konverze.

**Q: Podporuje knihovna konverzi vysokého rozlišení TIFF do PDF?**  
A: Ano, zachovává DPI a informace o barvách, což zajišťuje, že PDF odpovídá původní kvalitě TIFF.

**Q: Jaká licence je potřeba pro produkční použití?**
