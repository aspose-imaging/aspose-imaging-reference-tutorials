## Převod CDR na PNG – Rychlý přehled
Převod souborů CorelDRAW (CDR) na PNG je běžný úkol, když potřebujete rastrové obrázky připravené pro web z vektorových zdrojů. Aspose.Imaging pro Java provádí rasterizaci interně, takže se nemusíte starat o nástroje třetích stran nebo ruční výpočty DPI. Vyhrazený tutoriál uvedený níže vás provede načtením souboru CDR, nastavením možností rasterizace a uložením výsledku jako vysoce kvalitního PNG.
