## Jak aplikovat Wienerův filtr v Javě
Níže je stručný přehled kroků, které budete při použití Aspose.Imaging pro aplikaci Wienerova filtru následovat:

1. **Načtěte obrázek** – Použijte `Image.load()` k načtení zdrojového souboru (podporuje JPEG, PNG, DICOM atd.).
2. **Vytvořte objekt `WienerFilterOptions`** – Nastavte parametry jako velikost jádra a varianci šumu.
3. **Aplikujte filtr** – Zavolejte `image.filter(new WienerFilter(options))`.
4. **Uložte výsledek** – Exportujte zpracovaný obrázek do požadovaného formátu.

> **Tip:** U velkých medicínských obrázků postupně zvyšujte velikost jádra a testujte výstup, abyste se vyhnuli nadměrnému vyhlazení.

## Časté problémy a řešení
- **Pokles výkonu u obrovských obrázků** – Aktivujte `image.setCacheSize()` nebo zpracovávejte obrázek po částech (tiles).
- **Posun barev po filtraci** – Ujistěte se, že používáte správný barevný prostor (`RGB` vs `CMYK`) při vytváření možností filtru.
- **Neočekávané artefakty** – Upravte parametr variance šumu; nižší hodnota zachovává více detailů.

**Q: Můžu použít Wienerův filtr na PNG obrázky?**

A: API je navrženo pro souběžné použití; jen se vyhněte sdílení stejné instance `Image` mezi vlákny.

**Q: Potřebuji speciální licenci pro medicínské formáty?**

**Q: Jak se Wienerův filtr liší od mediánového filtru?**  
A: Wienerův filtr se přizpůsobuje lokální statistice a lépe zachovává hrany, zatímco mediánový filtr je jednoduchá nelineární vyhlazovací technika.

**Q: Můžu kombinovat Wienerův filtr s dalšími filtry?**  
A: Rozhodně – můžete řetězit filtry (např. Wiener následovaný ostřením) pomocí metody `image.filter()` vícekrát.
