## คู่มือการใช้งาน

### กำลังโหลดภาพ DJVU

คุณสมบัตินี้ช่วยให้คุณโหลดไฟล์ DJVU ได้อย่างมีประสิทธิภาพด้วยตัวเลือกเฉพาะที่เหมาะกับความต้องการของคุณ

#### ภาพรวม

การโหลดภาพ DJVU เกี่ยวข้องกับการระบุเส้นทางไฟล์และตั้งค่าพารามิเตอร์เช่นคำแนะนำขนาดบัฟเฟอร์ ซึ่งสามารถเพิ่มประสิทธิภาพการโหลดได้

#### การดำเนินการแบบทีละขั้นตอน

1. **กำหนดเส้นทางไฟล์**: ระบุว่าไฟล์ DJVU ของคุณอยู่ที่ไหน
2. **โหลดภาพพร้อมตัวเลือก**: ใช้ `LoadOptions` เพื่อปรับแต่งวิธีโหลดภาพ เช่น การตั้งค่าคำแนะนำขนาดบัฟเฟอร์

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.djvu.DjvuImage;
import com.aspose.imaging.LoadOptions;

public class LoadDjvuImage {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/ConvertingImages/";
        
        try (DjvuImage image = (DjvuImage) Image.load(dataDir + "test.djvu", new LoadOptions() {
            { setBufferSizeHint(50); }
        })) {
            // ตอนนี้โหลดภาพแล้วและพร้อมสำหรับการประมวลผล
        } catch (Exception e) {
            System.out.println("Error loading DJVU file: " + e.getMessage());
        }
    }
}
```

**คำอธิบาย**- 
- **`LoadOptions`**:ช่วยให้คุณปรับพารามิเตอร์ต่างๆ เช่น ขนาดบัฟเฟอร์ ปรับปรุงเวลาในการโหลด โดยเฉพาะอย่างยิ่งสำหรับรูปภาพขนาดใหญ่
- **ลองด้วยทรัพยากร**:เพื่อให้แน่ใจว่าทรัพยากรภาพจะถูกปิดโดยอัตโนมัติหลังจากประมวลผล

#### การแก้ไขปัญหา

หากคุณประสบปัญหาในการโหลดไฟล์:
- ตรวจสอบเส้นทางไฟล์และการอนุญาตของคุณ
- ทำให้มั่นใจ `Aspose.Imaging` ไลบรารีได้รับการเพิ่มไปยังโครงการที่ต้องมีอย่างถูกต้องแล้ว

### การบันทึกหน้า DJVU เป็นรูปภาพ PNG

แปลงแต่ละหน้าของเอกสาร DJVU ให้เป็นไฟล์ PNG แต่ละไฟล์เพื่อให้สามารถแบ่งปันหรือแก้ไขได้อย่างง่ายดาย

#### ภาพรวม

ฟีเจอร์นี้จะวนซ้ำผ่านแต่ละหน้าในภาพ DJVU และบันทึกเป็นไฟล์ PNG แยกกัน

#### การดำเนินการแบบทีละขั้นตอน

1. **โหลดภาพ DJVU**:ใช้เทคนิคการโหลดแบบเดียวกันกับข้างต้น
2. **ทำซ้ำและบันทึกหน้า**:วนซ้ำหน้าต่างๆ และบันทึกแต่ละหน้าเป็นไฟล์ PNG

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.djvu.DjvuImage;
import com.aspose.imaging.imageoptions.PngOptions;

public class SaveDjvuPagesAsPng {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/ConvertingImages/";
        
        try (DjvuImage image = (DjvuImage) Image.load(dataDir + "test.djvu")) {
            int pageNum = 0;
            
            for (com.aspose.imaging.Image page : image.getPages()) {
                String outputPath = "YOUR_OUTPUT_DIRECTORY/page" + pageNum + ".png";
                page.save(outputPath, new PngOptions());
                pageNum++;
            }
        } catch (Exception e) {
            System.out.println("Error saving pages as PNG: " + e.getMessage());
        }
    }
}
```

**คำอธิบาย**- 
- **`PngOptions`**: กำหนดค่ารูปแบบผลลัพธ์ คุณสามารถตั้งค่าตัวเลือกเพิ่มเติมได้หากจำเป็น
- **การวนซ้ำหน้า**: แต่ละหน้าจะได้รับการปฏิบัติเป็น `Image`ซึ่งช่วยให้สามารถประมวลผลเป็นรายบุคคลได้

#### การแก้ไขปัญหา

ปัญหาทั่วไปได้แก่:
- เส้นทางไฟล์ไม่ถูกต้องหรือขาดการอนุญาตสำหรับไดเร็กทอรีเอาต์พุต
- หน่วยความจำไม่เพียงพอ โดยเฉพาะอย่างยิ่งเมื่อต้องจัดการกับรูปภาพที่มีความละเอียดสูง ให้พิจารณาเพิ่มขนาดฮีปของ JVM
