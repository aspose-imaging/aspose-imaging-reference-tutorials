### แปลง PNG เป็น JPEG‑LS (H2)

ฟีเจอร์นี้ช่วยให้คุณบีบอัดภาพ PNG ที่มี 8‑บิตต่อสแปลให้เป็นรูปแบบ JPEG‑LS ที่มีประสิทธิภาพกว่าโดยใช้ 2‑บิตต่อสแปล

#### ภาพรวม
JPEG‑LS มีชื่อเสียงในด้านความสามารถการบีบอัดแบบไม่มีการสูญเสียหรือเกือบไม่มีการสูญเสีย ทำให้เหมาะสำหรับการจัดเก็บภาพคุณภาพสูงโดยขนาดไฟล์ลดลง ส่วนนี้จะอธิบายขั้นตอนการแปลง PNG ไปยังรูปแบบนี้ด้วย Aspose Imaging

#### ขั้นตอนการดำเนินการ

##### โหลดภาพ PNG ดั้งเดิม (H3)
เริ่มต้นโดยการโหลดภาพ PNG ต้นฉบับของคุณ:

```java
String originPngFileName = "YOUR_DOCUMENT_DIRECTORY/lena24b.png";
PngImage pngImage = (PngImage) Image.load(originPngFileName);
```
*ทำไม?*: การโหลดภาพทำให้คุณสามารถเข้าถึงและจัดการได้ด้วย API ที่ทรงพลังของ Aspose Imaging.

##### กำหนดค่าตัวเลือก JPEG (H3)
ตั้งค่าตัวเลือกที่จำเป็นสำหรับการแปลง:

```java
int bpp = 2; // Bits per sample
JpegOptions jpegOptions = new JpegOptions();
jpegOptions.setBitsPerChannel((byte) bpp);
jpegOptions.setCompressionType(JpegCompressionMode.JpegLs);
```
*ทำไม?*: การกำหนดค่าตัวเลือกเหล่านี้ระบุประเภทการบีบอัดและพารามิเตอร์คุณภาพสำหรับภาพ JPEG‑LS ที่จะส่งออก

##### บันทึกภาพ (H3)
สุดท้าย ให้บันทึกภาพที่แปลงแล้ว:

```java
String outputJpegFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.jls";
pngImage.save(outputJpegFileName, jpegOptions);
```
*ทำไม?*: การบันทึกในรูปแบบเฉพาะทำให้กระบวนการแปลงเสร็จสมบูรณ์

##### การจัดการทรัพยากร (H3)
ตรวจสอบให้แน่ใจว่าทรัพยากรถูกปล่อยหลังการประมวลผล:

```java
finally {
    pngImage.dispose();
}
```
*ทำไม?*: การทำลายออบเจ็กต์ภาพช่วยจัดการการใช้หน่วยความจำได้อย่างมีประสิทธิภาพ

#### เคล็ดลับการแก้ปัญหา
- **ปัญหาทั่วไป**: หากคุณพบ `FileNotFoundException` ให้ตรวจสอบเส้นทางไฟล์ของไฟล์อินพุตและเอาต์พุตอีกครั้ง
- **เคล็ดลับการแก้ไข**: ตรวจสอบให้แน่ใจว่า IDE ของคุณมีสิทธิ์อ่านและเขียนในไดเรกทอรีที่ระบุ

### แปลง JPEG‑LS เป็น PNG (H2)

ฟีเจอร์นี้ช่วยให้คุณแปลงภาพ JPEG‑LS กลับเป็นรูปแบบ PNG เพื่ออำนวยความสะดวกในการเปรียบเทียบภาพหรือการประมวลผลต่อไป

#### ภาพรวม
หลังจากแปลงภาพเป็น JPEG‑LS เพื่อการบีบอัด อาจจำเป็นต้องย้อนกลับกระบวนการเพื่อประเมินคุณภาพหรือเพื่อการใช้งานอื่น ๆ ส่วนนี้อธิบายวิธีทำด้วย Aspose Imaging

#### ขั้นตอนการดำเนินการ

##### โหลดภาพ JPEG‑LS (H3)
เริ่มต้นโดยการโหลดไฟล์ JPEG‑LS ของคุณ:

```java
String inputJpegFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.jls";
JpegImage jpegImage = (JpegImage) Image.load(inputJpegFileName);
```
*ทำไม?*: การโหลดภาพทำให้พร้อมสำหรับการแปลงเป็นรูปแบบอื่น

##### บันทึกเป็น PNG (H3)
แปลงและบันทึกภาพในรูปแบบ PNG:

```java
String outputPngFileName = "YOUR_OUTPUT_DIRECTORY/lena24b 2-bit Gold.png";
jpegImage.save(outputPngFileName, new PngOptions());
```
*ทำไม?*: ขั้นตอนนี้ทำให้ไฟล์ JPEG‑LS ของคุณแปลงกลับเป็นรูปแบบ PNG ที่เข้ากันได้อย่างกว้างขวางและไม่มีการสูญเสีย

##### การจัดการทรัพยากร (H3)
ทำลายทรัพยากรภาพหลังจากบันทึก:

```java
finally {
    jpegImage.dispose();
}
```
*ทำไม?*: การทำลายอย่างเหมาะสมช่วยหลีกเลี่ยงการรั่วไหลของหน่วยความจำ รักษาประสิทธิภาพของแอปพลิเคชัน

- **การถ่ายภาพดิจิทัล** – ช่างภาพสามารถบีบอัดพอร์ตโฟลิโอโดยไม่ลดทอนความแม่นยำ
