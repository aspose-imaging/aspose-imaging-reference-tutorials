## คู่มือการใช้งาน

### การโหลดและการบันทึกภาพ TIFF

คุณสมบัตินี้ช่วยให้คุณโหลดภาพ TIFF จากดิสก์และบันทึกกลับด้วยตัวเลือกเฉพาะ

#### ขั้นตอนที่ 1: โหลดภาพ
แทนที่ `'YOUR_DOCUMENT_DIRECTORY'` ด้วยเส้นทางไฟล์ของคุณ:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/sample.tiff";
try (Image image = Image.load(dataDir)) {
    // ดำเนินการบันทึกรูปภาพ
}
```
โค้ดสั้นๆ นี้โหลดภาพ TIFF เพื่อเตรียมพร้อมสำหรับการจัดการหรือบันทึก

#### ขั้นตอนที่ 2: บันทึกภาพ
แทนที่ `'YOUR_OUTPUT_DIRECTORY'` ด้วยเส้นทางเอาท์พุตที่คุณต้องการ:
```java
image.save("YOUR_OUTPUT_DIRECTORY/out_sample.tiff");
```
การบันทึกภาพเกี่ยวข้องกับการระบุตำแหน่งและกำหนดค่าตัวเลือกในการบันทึก (จะกล่าวถึงต่อไป)

### การกำหนดค่าตัวเลือก TIFF สำหรับการบีบอัดและการวัดแสง

เพิ่มประสิทธิภาพภาพ TIFF ของคุณด้วยการตั้งค่าประเภทการบีบอัดและการตั้งค่าโฟโตเมตริกเฉพาะ

#### ภาพรวมของการกำหนดค่า
การกำหนดค่านี้ช่วยให้สามารถปรับแต่งวิธีการจัดเก็บภาพได้ ซึ่งส่งผลต่อขนาดและคุณภาพของไฟล์

#### ขั้นตอนที่ 1: ตั้งค่า TiffOptions
```java
import com.aspose.imaging.fileformats.tiff.enums.TiffCompressions;
import com.aspose.imaging.fileformats.tiff.enums.TiffExpectedFormat;
import com.aspose.imaging.fileformats.tiff.enums.TiffPhotometrics;
import com.aspose.imaging.imageoptions.TiffOptions;

TiffOptions outputSettings = new TiffOptions(TiffExpectedFormat.Default);
outputSettings.setBitsPerSample(new int[] { 4 });
outputSettings.setCompression(TiffCompressions.AdobeDeflate);
outputSettings.setPhotometric(TiffPhotometrics.Palette);
```
ที่นี่, `AdobeDeflate` การบีบอัดและการวัดแสงแบบใช้จานสีได้รับการตั้งค่าเพื่อการจัดเก็บที่มีประสิทธิภาพ

#### ขั้นตอนที่ 2: ใช้จานสีโทนสีเทา
```java
import com.aspose.imaging.ColorPaletteHelper;

outputSettings.setPalette(ColorPaletteHelper.create4BitGrayscale(false));
```
ขั้นตอนนี้จะกำหนดจานสีสีเทาให้กับภาพ TIFF ของคุณ ซึ่งจะช่วยเพิ่มความเข้ากันได้กับระบบการแสดงผลต่างๆ

### การประยุกต์ใช้งานจริง

- **การจัดเก็บถาวร**:ใช้การตั้งค่าเหล่านี้เพื่อสร้างรูปภาพเก็บถาวรแบบบีบอัดโดยไม่สูญเสียคุณภาพ
- **การเพิ่มประสิทธิภาพเว็บไซต์**: กำหนดค่าการบีบอัดและการวัดแสงเพื่อให้โหลดหน้าเว็บได้เร็วขึ้น
- **การถ่ายภาพทางวิทยาศาสตร์**:จานสีเทาเป็นสิ่งสำคัญในการแสดงภาพทางวิทยาศาสตร์เพื่อแสดงข้อมูลอย่างมีประสิทธิภาพ
