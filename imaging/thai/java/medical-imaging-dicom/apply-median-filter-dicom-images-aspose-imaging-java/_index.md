## คู่มือการใช้งาน

เราจะแนะนำคุณลักษณะแต่ละอย่างทีละขั้นตอน ตั้งแต่การโหลดภาพ DICOM การใช้ฟิลเตอร์มัธยฐาน และการบันทึกภาพที่กรองแล้วเป็น BMP

### การโหลดภาพ DICOM

การโหลดภาพอย่างถูกต้องถือเป็นพื้นฐานสำหรับการประมวลผลในขั้นถัดไป เริ่มต้นด้วยการโหลดไฟล์ DICOM:

#### ขั้นตอนที่ 1: นำเข้าคลาสที่จำเป็น
ตรวจสอบให้แน่ใจว่าคุณมีการนำเข้าเหล่านี้ในคลาส Java ของคุณ:
```java
import com.aspose.imaging.Image;
import java.io.File;
import java.io.FileInputStream;
import java.io.IOException;
```

#### ขั้นตอนที่ 2: โหลดภาพ

ใช้ `Image.load()` วิธีการอ่านไฟล์ DICOM ขั้นตอนนี้มีความสำคัญในการเตรียมข้อมูลภาพเพื่อประมวลผล
```java
String inputFile = "YOUR_DOCUMENT_DIRECTORY/image.dcm";
File file = new File(inputFile);
try (FileInputStream fis = new FileInputStream(file)) {
    com.aspose.imaging.fileformats.dicom.DicomImage image = 
        (com.aspose.imaging.fileformats.dicom.DicomImage) Image.load(fis);
} catch (IOException ex) {
    System.err.println(ex.getMessage());
}
```

### การใช้ตัวกรองค่ากลางกับภาพ DICOM

การใช้ฟิลเตอร์ค่ามัธยฐานเป็นวิธีที่ยอดเยี่ยมในการลดสัญญาณรบกวนในภาพ ซึ่งมีประโยชน์อย่างยิ่งในการถ่ายภาพทางการแพทย์

#### ขั้นตอนที่ 3: นำเข้าคลาสการกรอง
เพิ่มการนำเข้าที่จำเป็นสำหรับการกรอง:
```java
import com.aspose.imaging.fileformats.dicom.DicomImage;
import com.aspose.imaging.imagefilters.filteroptions.MedianFilterOptions;
```

#### ขั้นตอนที่ 4: ใช้ตัวกรอง

ที่นี่ เราใช้ฟิลเตอร์มัธยฐานที่มีรัศมี 8 เพื่อทำให้ภาพเรียบเนียน
```java
image.filter(image.getBounds(), new MedianFilterOptions(8));
```
การ `MedianFilterOptions` พารามิเตอร์กำหนดขนาดรัศมีสำหรับกระบวนการกรอง ปรับเปลี่ยนตามความต้องการในการลดสัญญาณรบกวนของคุณ

### การบันทึกภาพที่ผ่านการกรองเป็น BMP

เมื่อกรองแล้ว คุณอาจต้องการบันทึกรูปภาพที่ปรับปรุงแล้วในรูปแบบอื่น เช่น BMP:

#### ขั้นตอนที่ 5: นำเข้าคลาสตัวเลือก BMP
```java
import com.aspose.imaging.imageoptions.BmpOptions;
```

#### ขั้นตอนที่ 6: บันทึกภาพ
ระบุเส้นทางไฟล์เอาท์พุตและการใช้งาน `save()` วิธีการด้วย `BmpOptions`-
```java
String outputFile = "YOUR_OUTPUT_DIRECTORY/ApplyFilterOnDICOMImage_out.bmp";
image.save(outputFile, new BmpOptions());
```
ขั้นตอนนี้จะช่วยให้แน่ใจว่ารูปภาพที่คุณประมวลผลแล้วได้รับการเก็บรักษาในรูปแบบที่รองรับอย่างกว้างขวาง
