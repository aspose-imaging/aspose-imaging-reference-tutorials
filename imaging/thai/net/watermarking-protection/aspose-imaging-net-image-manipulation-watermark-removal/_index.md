## คู่มือการใช้งาน

### การโหลดและการแปลงรูปภาพ
ฟีเจอร์นี้ช่วยให้คุณโหลดรูปภาพ PNG จากไดเร็กทอรีโดยใช้ Aspose.Imaging และบันทึกลงในตำแหน่งอื่น

#### ขั้นตอนที่ 1: โหลดภาพ
เริ่มต้นด้วยการระบุเอกสารและไดเรกทอรีผลลัพธ์ของคุณ จากนั้นใช้ `Image.Load()` เพื่อโหลดไฟล์ PNG ของคุณ

```csharp
using Aspose.Imaging;
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
var imagePath = Path.Combine(dataDir, "ball.png");
Image image = Image.Load(imagePath);
var pngImage = (PngImage)image; // การหล่อเพื่อการดำเนินงานเฉพาะ
```

#### ขั้นตอนที่ 2: บันทึกภาพ
เมื่อโหลดแล้ว คุณสามารถบันทึกลงในไดเร็กทอรีเอาท์พุตได้

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
pngImage.Save(Path.Combine(outputDir, "loaded_image.png"));
```

### การสร้างและการใช้หน้ากากเส้นทางกราฟิก
การสร้างหน้ากากเส้นทางกราฟิกช่วยให้สามารถจัดการรูปภาพที่ซับซ้อนได้ เช่น การเพิ่มรูปร่างหรือเอฟเฟกต์

#### ขั้นตอนที่ 1: เริ่มต้น GraphicsPath
สร้างใหม่ `GraphicsPath` วัตถุที่จะกำหนดหน้ากากของคุณ

```csharp
using Aspose.Imaging.MagicWand.ImageMasks;

GraphicsPath mask = new GraphicsPath();
var firstFigure = new Figure();
```

#### ขั้นตอนที่ 2: เพิ่มรูปร่าง
เพิ่มรูปทรงเช่นวงรีให้กับรูปร่างของคุณ ซึ่งจะเป็นส่วนหนึ่งของหน้ากาก

```csharp
firstFigure.AddShape(new EllipseShape(new RectangleF(350, 170, 570 - 350, 400 - 170)));
masks.AddFigure(firstFigure);
```

### การลบลายน้ำออกจากรูปภาพ
ฟีเจอร์นี้สาธิตวิธีการลบลายน้ำที่ไม่ต้องการโดยใช้เครื่องมือลบลายน้ำของ Aspose.Imaging

#### ขั้นตอนที่ 1: กำหนดค่าตัวเลือก
ตั้งค่า `ContentAwareFillWatermarkOptions` ด้วยหน้ากากกราฟิกของคุณและกำหนดความพยายามในการวาดภาพ

```csharp
using Aspose.Imaging.Watermark;

var options = new ContentAwareFillWatermarkOptions(mask)
{
    MaxPaintingAttempts = 1 // จำนวนสูงสุดของความพยายามในการลบลายน้ำ
};
```

#### ขั้นตอนที่ 2: ลบลายน้ำ
ใช้ `WatermarkRemover` เพื่อใช้การกำหนดค่าของคุณและลบลายน้ำ

```csharp
var result = WatermarkRemover.PaintOver(pngImage, options);
result.Save(Path.Combine(outputDir, "watermark_removed.png"));
File.Delete(Path.Combine(dataDir, "ball.png")); // ล้างไฟล์ต้นฉบับหากจำเป็น
```
