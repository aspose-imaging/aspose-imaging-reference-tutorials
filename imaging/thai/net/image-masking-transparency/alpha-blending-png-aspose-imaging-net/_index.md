## คู่มือการใช้งาน: การผสมอัลฟ่าสำหรับรูปภาพ PNG

### ภาพรวมของการผสมอัลฟ่า

การผสมสีแบบอัลฟ่าช่วยให้คุณรวมภาพสองภาพเข้าด้วยกันโดยใช้ความโปร่งใส ซึ่งมีประโยชน์อย่างยิ่งเมื่อต้องซ้อนภาพ เช่น การเพิ่มโลโก้ลงในภาพอื่น

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีและโหลดรูปภาพ

เริ่มต้นด้วยการกำหนดเส้นทางสำหรับไดเร็กทอรีอินพุตและเอาต์พุตของคุณ:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

using (var background = Aspose.Imaging.Image.Load(Path.Combine(dataDir, "image0.png")) as RasterImage)
{
    using (var overlay = Image.Load(Path.Combine(dataDir, "aspose_logo.png")) as RasterImage)
```
ที่นี่, `background` และ `overlay` ถูกโหลดเป็น `RasterImage`ซึ่งรองรับการจัดการในระดับพิกเซล

#### ขั้นตอนที่ 2: คำนวณตำแหน่งศูนย์กลาง

คำนวณตำแหน่งที่จะวางภาพซ้อนบนพื้นหลัง:
```csharp
var center = new Point((background.Width - overlay.Width) / 2, (background.Height - overlay.Height) / 2);
```
สิ่งนี้จะทำให้ศูนย์กลาง `overlay` ภายใน `background`-

#### ขั้นตอนที่ 3: ดำเนินการผสมอัลฟ่า

ผสมภาพโดยใช้ค่าอัลฟ่าที่ระบุสำหรับความโปร่งใส:
```csharp
background.Blend(center, overlay, overlay.Bounds, 127); // ค่าอัลฟ่า 127
```
ค่าอัลฟาระหว่าง 0 (โปร่งใสเต็มที่) และ 255 (ทึบเต็มที่) จะควบคุมความเข้มข้นของการผสมผสาน

#### ขั้นตอนที่ 4: บันทึกภาพผสม

สุดท้ายให้บันทึกผลลัพธ์ของคุณด้วยการตั้งค่าความโปร่งใส:
```csharp
background.Save(Path.Combine(outputDir, "blended.png"), new PngOptions() { ColorType = Aspose.Imaging.FileFormats.Png.PngColorType.TruecolorWithAlpha });
```
นอกจากนี้ ให้ทำความสะอาดโดยการลบไฟล์ชั่วคราว
