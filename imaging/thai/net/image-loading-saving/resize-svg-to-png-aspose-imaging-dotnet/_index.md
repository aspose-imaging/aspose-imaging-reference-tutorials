## คู่มือการใช้งาน

ในส่วนนี้ เราจะแบ่งการใช้งานออกเป็นสองคุณสมบัติหลัก: การปรับขนาดภาพ SVG และการบันทึกเป็นไฟล์ PNG 

### คุณสมบัติ 1: การปรับขนาดภาพ SVG

#### ภาพรวม

การปรับขนาดเป็นสิ่งสำคัญเมื่อคุณต้องปรับขนาดของ SVG ให้เหมาะกับความต้องการในการแสดงผลที่แตกต่างกัน Aspose.Imaging ช่วยให้งานนี้ง่ายขึ้น

#### ขั้นตอน:

**ขั้นตอนที่ 1: โหลด SVG ของคุณ**

เริ่มต้นด้วยการโหลดภาพ SVG ของคุณโดยใช้ `Image.Load` วิธีการ ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ของคุณชี้ไปยังตำแหน่งที่เก็บ SVG ของคุณ
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (SvgImage image = (SvgImage)Image.Load(dataDir + "/aspose_logo.Svg"))
{
    // ดำเนินการปรับขนาด...
}
```

**ขั้นตอนที่ 2: ปรับขนาดภาพ**

ปรับขนาดโดยระบุขนาดใหม่ ที่นี่ เราคูณความกว้างและความสูงเดิมด้วยปัจจัยต่างๆ เพื่อให้ได้ขนาดที่ต้องการ
```csharp
// ปรับขนาดความกว้างของรูปภาพเป็น 10 และความสูงเป็น 15
image.Resize(image.Width * 10, image.Height * 15);
```

**ขั้นตอนที่ 3: บันทึกภาพที่ปรับขนาดแล้ว**

หลังจากปรับขนาดแล้ว ให้บันทึกรูปภาพของคุณ ตัวอย่างนี้จะบันทึกรูปภาพของคุณในรูปแบบ PNG ไปยังไดเร็กทอรีเอาต์พุตที่ระบุ
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/Logotype_10_15_out.png";
image.Save(outputPath, new PngOptions()
{
    VectorRasterizationOptions = new SvgRasterizationOptions()
});
```

### คุณสมบัติ 2: บันทึก SVG เป็น PNG

#### ภาพรวม

การแปลงไฟล์ SVG เป็นรูปแบบ PNG ที่ได้รับการสนับสนุนอย่างกว้างขวางสามารถปรับปรุงความเข้ากันได้ได้ Aspose.Imaging ทำให้กระบวนการแปลงนี้ง่ายขึ้น

#### ขั้นตอน:

**ขั้นตอนที่ 1: กำหนดตัวเลือก PNG**

ตั้งค่าของคุณ `PngOptions` วัตถุ ระบุการตั้งค่าการแรสเตอร์สำหรับกระบวนการแปลง
```csharp
PngOptions pngOptions = new PngOptions()
{
    VectorRasterizationOptions = new SvgRasterizationOptions()
};
```

**ขั้นตอนที่ 2: บันทึกเป็น PNG**

สุดท้ายให้ใช้ตัวเลือกเหล่านี้เพื่อบันทึกภาพ SVG ของคุณเป็นไฟล์ PNG
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/Logotype_out.png";
image.Save(outputPath, pngOptions);
```

4. **การตลาดดิจิตอล:** เพิ่มประสิทธิภาพกราฟิกโซเชียลมีเดียให้เหมาะกับแพลตฟอร์มต่างๆ
