## การแนะนำ
คุณกำลังมองหาวิธีโหลดและส่งออกรูปภาพ TIFF อย่างมีประสิทธิภาพภายในแอปพลิเคชัน .NET ของคุณอยู่หรือไม่ การจัดการไฟล์ TIFF อาจมีความซับซ้อนเนื่องจากมีข้อมูลเมตาที่หลากหลาย คู่มือนี้จะแนะนำคุณเกี่ยวกับขั้นตอนการโหลดรูปภาพ TIFF การเข้าถึงคุณสมบัติของรูปภาพ และการส่งออกไปยัง PDF ในขณะที่รักษาการตั้งค่า DPI เฉพาะโดยใช้ Aspose.Imaging สำหรับ .NET

**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน

### คุณสมบัติ 1: โหลดและแสดงข้อมูลภาพ TIFF
คุณสมบัตินี้ช่วยให้คุณโหลดภาพ TIFF และเข้าถึงคุณสมบัติต่างๆ เช่น ขนาดและการตั้งค่าความละเอียด

#### ภาพรวม
คุณจะได้เรียนรู้วิธีการ:
- โหลดไฟล์ TIFF
- ดึงข้อมูลและแสดงขนาดภาพ
- เข้าถึงข้อมูลความละเอียดแนวนอนและแนวตั้ง

#### การดำเนินการแบบทีละขั้นตอน
**1. เตรียมสภาพแวดล้อมของคุณ:**
ตรวจสอบให้แน่ใจว่าได้ติดตั้งไลบรารี Aspose.Imaging แล้ว และตั้งค่าสภาพแวดล้อมการพัฒนาของคุณด้วยเส้นทางที่จำเป็น

```csharp
using Aspose.Imaging;
using System;
using System.IO;

string filePath = "YOUR_DOCUMENT_DIRECTORY";
string fileName = Path.Combine(filePath, "AFREY-Original.tif");

if (!File.Exists(fileName))
{
    throw new FileNotFoundException("The specified TIFF file does not exist.");
}

using (TiffImage image = (TiffImage)Image.Load(fileName))
{
    // แสดงขนาดของภาพ TIFF ที่โหลด
    Console.WriteLine($"Width: {image.Width}, Height: {image.Height}");
    
    // เข้าถึงและแสดงการตั้งค่าความละเอียดของภาพ
    Console.WriteLine($"Horizontal Resolution: {image.HorizontalResolution} DPI, Vertical Resolution: {image.VerticalResolution} DPI");
}
```
**คำอธิบาย:**
- `Image.Load(fileName)`:โหลดไฟล์ TIFF ของคุณ
- `TiffImage` cast ช่วยให้แน่ใจว่าคุณกำลังทำงานกับคลาสเฉพาะ TIFF เพื่อเข้าถึงคุณสมบัติ TIFF
- เอาท์พุตของคอนโซลจะแสดงขนาดและความละเอียดของภาพ

### คุณสมบัติ 2: ส่งออกภาพ TIFF ไปยัง PDF ด้วยการตั้งค่า DPI เฉพาะ
ตอนนี้เราลองมาสำรวจการส่งออกภาพ TIFF ไปยัง PDF โดยยังคงการตั้งค่าความละเอียดเดิมเอาไว้

#### ภาพรวม
ส่วนนี้ครอบคลุมถึง:
- เตรียมตัวเลือกการส่งออก PDF ตามการตั้งค่า TIFF ที่มีอยู่
- การบันทึก TIFF เป็นไฟล์ PDF

#### การดำเนินการแบบทีละขั้นตอน
**1. ตั้งค่าตัวเลือกการส่งออก:**
กำหนดค่าตัวเลือกการส่งออก PDF ของคุณ รวมถึงการตั้งค่า DPI และเตรียมบันทึกภาพ

```csharp
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Tiff;
using Aspose.Imaging.ImageOptions;
using System.IO;

string inputFilePath = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string fileName = Path.Combine(inputFilePath, "AFREY-Original.tif");

if (!File.Exists(fileName))
{
    throw new FileNotFoundException("The specified TIFF file does not exist.");
}

using (TiffImage image = (TiffImage)Image.Load(fileName))
{
    // เตรียมตัวเลือกการส่งออก PDF ด้วยการตั้งค่าความละเอียดจากภาพ TIFF
    PdfOptions pdfOptions = new PdfOptions()
    {
        ResolutionSettings = new ResolutionSetting(
            image.HorizontalResolution,
            image.VerticalResolution)
    };
    
    // ตั้งค่าเส้นทางเอาท์พุตสำหรับไฟล์ PDF ที่ส่งออก
    string outputPath = Path.Combine(outputDirectory, "result.pdf");
    
    // บันทึก TIFF เป็น PDF พร้อมการตั้งค่า DPI ที่ระบุ
    image.Save(outputPath, pdfOptions);
}
```
**คำอธิบาย:**
- `PdfOptions`:กำหนดค่าวิธีการแปลงไฟล์ TIFF ของคุณเป็น PDF
- `ResolutionSetting`:ตั้งค่า DPI ตามความละเอียด TIFF ดั้งเดิม
- `image.Save(outputPath, pdfOptions)`:บันทึก TIFF เป็น PDF โดยใช้การตั้งค่าที่คุณระบุ

### เคล็ดลับการแก้ไขปัญหา
หากคุณพบปัญหา:
- ตรวจสอบให้แน่ใจว่าเส้นทางได้รับการตั้งค่าอย่างถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่า Aspose.Imaging ได้รับการติดตั้งและได้รับอนุญาตอย่างถูกต้อง
- ตรวจสอบข้อยกเว้นใด ๆ ที่เกิดขึ้นระหว่างการดำเนินการไฟล์
