## คู่มือการใช้งาน

ในส่วนนี้จะแบ่งคุณลักษณะแต่ละอย่างออกเป็นขั้นตอนที่สามารถดำเนินการได้

### ตั้งค่าโฟลเดอร์แบบอักษรที่กำหนดเอง

#### ภาพรวม
ตั้งค่าโฟลเดอร์ที่กำหนดเองสำหรับแบบอักษรเพื่อสร้างมาตรฐานการแสดงข้อความในแอปพลิเคชันของคุณ

**ขั้นตอนการดำเนินการ:**

##### กำหนดไดเรกทอรีเอกสารและเส้นทางแบบอักษร

ก่อนอื่น ระบุว่าไดเรกทอรีเอกสารและไฟล์แบบอักษรของคุณอยู่ที่ไหน
```csharp
using Aspose.Imaging;
using System.IO;

public static void SetCustomFontsFolder()
{
    string dataDir = "YOUR_DOCUMENT_DIRECTORY/";
    FontSettings.SetFontsFolder(Path.Combine(dataDir, "fonts"));
}
```
- **พารามิเตอร์:** `dataDir` ควรเป็นเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
- **วัตถุประสงค์:** วิธีนี้จะช่วยให้แน่ใจว่า Aspose.Imaging จะใช้เฉพาะแบบอักษรที่คุณระบุเท่านั้น

##### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบว่าโฟลเดอร์แบบอักษรมีอยู่และมีไฟล์ .ttf หรือ .otf
- ตรวจสอบสิทธิ์ไฟล์สำหรับแอปพลิเคชันเพื่อเข้าถึงไดเร็กทอรีนี้

### ปิดใช้งานแบบอักษรทางเลือกของระบบ

#### ภาพรวม
ป้องกันไม่ให้มีการใช้แบบอักษรทางเลือกของระบบ ช่วยให้การแสดงข้อความในรูปภาพของคุณมีความสอดคล้องกัน

**ขั้นตอนการดำเนินการ:**

##### ตั้งค่าฟอนต์

ปิดการใช้งานแบบอักษรทางเลือกของระบบดังนี้:
```csharp
using Aspose.Imaging;

public static void DisableSystemAlternativeFont()
{
    FontSettings.GetSystemAlternativeFont = false;
}
```
- **พารามิเตอร์:** ไม่มี นี่เป็นการตั้งค่าทั่วไปที่ส่งผลต่อการแสดงผลแบบอักษรทั้งหมด
- **วัตถุประสงค์:** ทำให้แน่ใจว่าข้อความจะปรากฏตรงตามที่ต้องการโดยไม่ต้องย้อนกลับไปใช้แบบอักษรของระบบ

##### เคล็ดลับการแก้ไขปัญหา

- หากคุณสังเกตเห็นว่ามีอักขระที่หายไป โปรดตรวจสอบว่าแบบอักษรที่กำหนดเองมีรูปสัญลักษณ์ที่จำเป็น
- ทดสอบกับชนิดเอกสารที่แตกต่างกันเพื่อยืนยันพฤติกรรมที่สอดคล้องกัน

### ส่งออกกราฟิกเวกเตอร์เป็น PNG

#### ภาพรวม
แปลงกราฟิกเวกเตอร์ (เช่น ODP หรือ SVG) เป็นรูปแบบ PNG คุณภาพสูงโดยใช้คุณสมบัติอันแข็งแกร่งของ Aspose.Imaging

**ขั้นตอนการดำเนินการ:**

##### ตั้งค่าฟอนต์เริ่มต้นและโหลดเอกสาร

กำหนดค่าแบบอักษรเริ่มต้นสำหรับการเรนเดอร์ จากนั้นโหลดเอกสารของคุณ:
```csharp
using Aspose.Imaging;
using Aspose.Imaging.ImageOptions;
using System.IO;

public static void ExportVectorToPng(string inputFilePath, string defaultFontName, string outputFilePath)
{
    FontSettings.DefaultFontName = defaultFontName;
    
    using (Aspose.Imaging.Image document = Aspose.Imaging.Image.Load(inputFilePath))
    {
        PngOptions saveOptions = new PngOptions();
        saveOptions.VectorRasterizationOptions = new OdgRasterizationOptions()
        {
            PageWidth = 1000,
            PageHeight = 1000
        };
        
        document.Save(outputFilePath, saveOptions);
    }
    
    File.Delete(outputFilePath); // ตัวเลือก: ลบเอาท์พุตหลังจากบันทึก
}
```
- **พารามิเตอร์:**
  - `inputFilePath`: เส้นทางไปยังไฟล์กราฟิกเวกเตอร์
  - `defaultFontName`:แบบอักษรที่ใช้ในการแสดงข้อความในภาพ
  - `outputFilePath`: โดยไฟล์ PNG ที่ได้จะถูกบันทึกไว้
- **วัตถุประสงค์:** แปลงรูปแบบเวกเตอร์เป็นภาพแรสเตอร์โดยยังคงคุณภาพและรับรองการแสดงข้อความที่ถูกต้องด้วยแบบอักษรที่ระบุ

##### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าไฟล์เวกเตอร์สามารถเข้าถึงได้และมีรูปแบบที่ถูกต้อง
- ปรับ `PageWidth` และ `PageHeight` ตามความต้องการเฉพาะของคุณเพื่อให้พอดีกับเนื้อหาอย่างเหมาะสม

## การประยุกต์ใช้งานจริง

Aspose.Imaging สำหรับ .NET มีความยืดหยุ่นและเหมาะกับเวิร์กโฟลว์ต่างๆ มากมาย ต่อไปนี้คือกรณีการใช้งานบางส่วน:
1. **การประมวลผลเอกสาร:** แปลงสไลด์การนำเสนอหรือแผนผังเป็นภาพ PNG สำหรับการแสดงบนเว็บโดยอัตโนมัติ
2. **การสร้างแบรนด์ที่กำหนดเอง:** รักษาความสอดคล้องของแบรนด์ด้วยการใช้แบบอักษรเฉพาะบริษัทในเอกสารและกราฟิกที่ส่งออกทั้งหมด
3. **การจัดเก็บถาวร:** แปลงรูปแบบเวกเตอร์ดั้งเดิมให้เป็นไฟล์ PNG ที่สามารถเข้าถึงได้ทั่วโลกมากขึ้น
4. **ความเข้ากันได้ข้ามแพลตฟอร์ม:** ทำให้แน่ใจว่าข้อความแสดงผลอย่างถูกต้องเมื่อแชร์ภาพระหว่างระบบปฏิบัติการที่แตกต่างกัน
