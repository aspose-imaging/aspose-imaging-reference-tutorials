## คู่มือการใช้งาน

### แปลง WebP เป็น PDF

ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ภาพ WebP เป็นเอกสาร PDF ได้อย่างมีประสิทธิภาพ ดังต่อไปนี้:

#### โหลดภาพ WebP
```csharp
using System;
using System.IO;
using Aspose.Imaging;
using Aspose.Imaging.FileFormats.Pdf;
using Aspose.Imaging.ImageOptions;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string outputPath = @"YOUR_OUTPUT_DIRECTORY";

// โหลดภาพ WebP จากไดเร็กทอรีที่ระบุ
string inputFile = Path.Combine(dataDir, "Animation.webp");
```

#### สร้างตัวเลือก PDF
```csharp
using (Aspose.Imaging.FileFormats.Webp.WebPImage image = 
    (Aspose.Imaging.FileFormats.Webp.WebPImage)Image.Load(inputFile))
{
    // กำหนดค่าเอาท์พุต PDF
    PdfOptions options = new PdfOptions();
    options.PdfDocumentInfo = new PdfDocumentInfo();
}
```

#### บันทึกเป็น PDF
```csharp
// กำหนดเส้นทางไฟล์เอาต์พุตและบันทึกรูปภาพเป็น PDF
string outputFile = Path.Combine(outputPath, "Animation.pdf");
image.Save(outputFile, options);
```
**คำอธิบาย:** โค้ดสั้นๆ นี้โหลดภาพ WebP ตั้งค่าตัวเลือกการแปลง PDF และบันทึกเป็นเอกสาร PDF ปรับแต่ง `dataDir` และ `outputPath` เพื่อให้ตรงกับไดเร็กทอรีของคุณ

### ลบ PDF ที่สร้างขึ้น

หลังการแปลง คุณอาจต้องการล้างไฟล์ชั่วคราว:

#### ลบไฟล์ PDF
```csharp
string outputFile = Path.Combine(outputPath, "Animation.pdf");

// ตรวจสอบให้แน่ใจว่าไฟล์มีอยู่ก่อนที่จะลบ
if (File.Exists(outputFile))
{
    File.Delete(outputFile);
}
```
**คำอธิบาย:** รหัสนี้จะตรวจสอบการมีอยู่ของ PDF ที่สร้างขึ้นและลบออกเพื่อเพิ่มพื้นที่ว่าง
