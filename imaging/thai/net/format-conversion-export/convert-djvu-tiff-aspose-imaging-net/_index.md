## คู่มือการใช้งาน

คู่มือนี้ครอบคลุมคุณสมบัติหลักๆ ได้แก่ การโหลดภาพ DjVu การกำหนดค่า TiffOptions และการบันทึกช่วงหน้าที่ระบุเป็นไฟล์ TIFF

### โหลดภาพ DjVu โดยใช้ Aspose.Imaging

#### ภาพรวม
การโหลดภาพ DjVu เป็นขั้นตอนแรกในการประมวลผลเพื่อการแปลง ในส่วนนี้จะแสดงวิธีการโหลดไฟล์ภาพโดยใช้ Aspose.Imaging

#### การดำเนินการแบบทีละขั้นตอน

**1. โหลดภาพ DjVu**

เริ่มต้นโดยระบุเส้นทางของไฟล์ DjVu ของคุณและโหลดมัน:

```csharp
using Aspose.Imaging.FileFormats.Djvu;
using System.IO;

string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Sample.djvu");

// โหลดภาพ DjVu โดยใช้ Aspose.Imaging
DjvuImage image = (DjvuImage)Image.Load(dataDir);

// ให้แน่ใจว่าจะปล่อยทรัพยากรหลังจากโหลด
image.Dispose();
```

**คำอธิบาย:**
- การ `Image.Load` วิธีการโหลดไฟล์ DjVu ของคุณลงใน `DjvuImage` วัตถุ.
- อย่าลืมกำจัดทรัพยากรรูปภาพเพื่อเพิ่มหน่วยความจำ

### สร้าง TiffOptions ด้วยช่วงหน้า

#### ภาพรวม
การกำหนดค่า TiffOptions เป็นสิ่งสำคัญในการระบุหน้าที่คุณต้องการส่งออกจากเอกสาร DjVu ของคุณ

#### การดำเนินการแบบทีละขั้นตอน

**1. กำหนดช่วงหน้า**

ระบุช่วงของหน้าที่คุณต้องการแปลง:

```csharp
using Aspose.Imaging.FileFormats.Tiff.Enums;
using Aspose.Imaging.ImageOptions;

// กำหนดช่วงหน้า (เช่น สามหน้าแรก)
IntRange range = new IntRange(0, 2);
```

**2. กำหนดค่า TiffOptions**

สร้างอินสแตนซ์ของ `TiffOptions` และตั้งค่าช่วงหน้า:

```csharp
TiffOptions tiffOptions = new TiffOptions(TiffExpectedFormat.TiffDeflateBw);
tiffOptions.MultiPageOptions = new DjvuMultiPageOptions(range);
```

**คำอธิบาย:**
- `IntRange(0, 2)` ระบุหน้าจากดัชนี 0 ถึง 2
- `TiffOptions` ได้รับการกำหนดค่าด้วยรูปแบบ TIFF และช่วงหน้าตามต้องการ

### บันทึกภาพ DjVu เป็น TIFF โดยใช้ตัวเลือกช่วงหน้า

#### ภาพรวม
ในที่สุด เราจะสาธิตวิธีการบันทึกช่วงที่ระบุของหน้า DjVu เป็นภาพ TIFF

#### การดำเนินการแบบทีละขั้นตอน

**1. โหลดและกำหนดค่าตัวเลือกการส่งออก**

โหลดไฟล์ DjVu ของคุณและกำหนดค่าตัวเลือกการส่งออกโดยใช้ TiffOptions ที่กำหนดไว้ก่อนหน้านี้:

```csharp
string outputDir = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertRangeOfDjVuPages_out.tiff");

using (DjvuImage image = (DjvuImage)Image.Load(dataDir))
{
    TiffOptions exportOptions = new TiffOptions(TiffExpectedFormat.TiffDeflateBw);
    IntRange range = new IntRange(0, 2);
    exportOptions.MultiPageOptions = new DjvuMultiPageOptions(range);

    // บันทึกภาพเป็นไฟล์ TIFF
    image.Save(outputDir, exportOptions);
}
```

**คำอธิบาย:**
- การ `image.Save` วิธีการส่งออกหน้าที่ระบุเป็นไฟล์ TIFF โดยใช้ตัวเลือกที่กำหนดค่าไว้
