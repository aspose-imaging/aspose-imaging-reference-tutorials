## คู่มือการใช้งาน
### การโหลดภาพ DjVu
#### ภาพรวม
ขั้นตอนแรกคือการโหลดภาพ DjVu ของคุณโดยใช้ Aspose.Imaging สำหรับ .NET
#### ทีละขั้นตอน
**1. กำหนดเส้นทางไฟล์ของคุณ**
ตรวจสอบให้แน่ใจว่าคุณมีไฟล์ DjVu ของคุณพร้อมแล้ว:
```csharp
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Sample.djvu");
```
**2. โหลดภาพ**
โหลดภาพเข้าสู่หน่วยความจำ:
```csharp
DjvuImage image = (DjvuImage)Image.Load(dataDir);
```
### การส่งออกส่วนที่เจาะจงไปยังรูปแบบ PNG
#### ภาพรวม
หัวข้อนี้มุ่งเน้นที่การส่งออกส่วนเฉพาะของหน้า DjVu ในรูปแบบภาพ PNG ระดับสีเทา
#### ทีละขั้นตอน
**1. ตั้งค่าไดเร็กทอรีเอาท์พุต**
กำหนดว่ารูปภาพที่คุณส่งออกจะถูกบันทึกที่ไหน:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```
**2. กำหนดค่าตัวเลือกการส่งออก**
สร้างอินสแตนซ์ของ `PngOptions` และตั้งค่าเป็นโทนสีเทา:
```csharp
PngOptions exportOptions = new PngOptions();
exportOptions.ColorType = PngColorType.Grayscale;
```
**3. กำหนดพื้นที่ส่งออก**
ใช้ `Rectangle` เพื่อระบุส่วนที่คุณต้องการส่งออก:
```csharp
Rectangle exportArea = new Rectangle(0, 0, 500, 500);
```
**4. ระบุดัชนีหน้า**
เลือกหน้า DjVu ที่ต้องการส่งออก:
```csharp
int exportPageIndex = 2;
exportOptions.MultiPageOptions = new DjvuMultiPageOptions(exportPageIndex, exportArea);
```
**5. บันทึกภาพที่ส่งออก**
บันทึกภาพของคุณเป็นไฟล์ PNG ในไดเร็กทอรีที่ระบุ:
```csharp
image.Save(Path.Combine(outputDir, "ConvertSpecificPortionOfDjVuPage_out.png"), exportOptions);
```
#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีเอาต์พุตอยู่ก่อนที่จะบันทึกไฟล์
- ตรวจสอบซ้ำ `Rectangle` ขนาดให้พอดีกับขนาดหน้าของคุณ
