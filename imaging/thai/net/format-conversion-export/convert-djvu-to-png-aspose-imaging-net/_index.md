## คู่มือการใช้งาน
ในส่วนนี้จะอธิบายขั้นตอนในการโหลดภาพ DJVU และบันทึกหน้าเป็นไฟล์ PNG

### การโหลดภาพ DJVU
การโหลดไฟล์ DJVU เกี่ยวข้องกับการอ่านไฟล์ลงในหน่วยความจำเพื่อการจัดการ Aspose.Imaging ทำให้ขั้นตอนนี้ง่ายขึ้นด้วยวิธีการที่มีประสิทธิภาพซึ่งปรับแต่งให้เหมาะกับรูปแบบต่างๆ รวมถึง DJVU

#### ขั้นตอนที่ 1: ตั้งค่าเส้นทางไฟล์
```csharp
using System.IO;
using Aspose.Imaging.FileFormats.Djvu;

// แทนที่ YOUR_DOCUMENT_DIRECTORY ด้วยเส้นทางไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
การ `dataDir` ตัวแปรระบุตำแหน่งไฟล์ DJVU ของคุณ

#### ขั้นตอนที่ 2: โหลดภาพ
```csharp
DjvuImage djvuImage = (DjvuImage)Image.Load(Path.Combine(dataDir, "test.djvu"), new LoadOptions { BufferSizeHint = 50 });
```
การ `Image.Load` วิธีการอ่านไฟล์ DJVU ของคุณ การปรับ `BufferSizeHint` เพิ่มประสิทธิภาพการใช้หน่วยความจำ

### การบันทึกหน้า DJVU เป็นรูปภาพ PNG
การแปลงหน้าเฉพาะเป็นรูปแบบ PNG ช่วยให้การแชร์และการดูข้ามแพลตฟอร์มเป็นไปได้สะดวกยิ่งขึ้น

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีผลลัพธ์
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```
ทำให้มั่นใจ `outputDir` ชี้ไปยังตำแหน่งที่คุณต้องการบันทึกไฟล์ PNG

#### ขั้นตอนที่ 2: บันทึกหน้าเฉพาะ
```csharp
int pageNumber = 2; // จำนวนหน้าที่ต้องการบันทึก
DjvuImage image = (DjvuImage)Image.Load(Path.Combine(dataDir, "test.djvu"));

for (int pageNum = 0; pageNum < pageNumber; pageNum++) {
    string outputFilePath = Path.Combine(outputDir, $"page{pageNum}.png");
    image.Pages[pageNum].Save(outputFilePath, new PngOptions());
}
```
ลูปจะบันทึกแต่ละหน้าที่ระบุเป็นไฟล์ PNG `PngOptions` สามารถปรับแต่งเพิ่มเติมได้หากจำเป็น

### เคล็ดลับการแก้ไขปัญหา
- **ไม่พบไฟล์**: ตรวจสอบเส้นทาง (`dataDir`- `outputDir`) ถูกต้องและสามารถเข้าถึงได้
- **ปัญหาการอนุญาต**:ให้แน่ใจว่าไดเร็กทอรีที่เกี่ยวข้องมีสิทธิ์การอ่าน/เขียน
- **ประสิทธิภาพการทำงานล่าช้า**: ปรับ `BufferSizeHint` เพื่อความสมดุลระหว่างการใช้หน่วยความจำและประสิทธิภาพการทำงาน
