## คู่มือการใช้งาน

### คุณสมบัติ 1: การแปลง EMF เป็น WMF

#### ภาพรวม
ฟีเจอร์นี้สาธิตวิธีการแปลง Enhanced Metafile (EMF) เป็น Windows Metafile (WMF) เพื่อให้มั่นใจถึงความเข้ากันได้กับสภาพแวดล้อมการประมวลผลกราฟิกที่แตกต่างกัน

**การดำเนินการแบบทีละขั้นตอน**

##### กำลังโหลดภาพ EMF
ขั้นแรก ให้แน่ใจว่าไฟล์ EMF ของคุณอยู่ในไดเร็กทอรี วิธีโหลดไฟล์มีดังนี้
```csharp
using Aspose.Imaging;
using Aspose.Imaging.ImageOptions;
using Aspose.Imaging.FileFormats.Emf;

string path = "YOUR_DOCUMENT_DIRECTORY"; // ไดเร็กทอรีที่ประกอบด้วยไฟล์ EMF
string[] files = new string[] { "TestEmfRotatedText.emf", "TestEmfPlusFigures.emf", "TestEmfBezier.emf" };

foreach (string file in files)
{
    string filePath = System.IO.Path.Combine(path, file);
    
    using (MetaImage image = (MetaImage)Image.Load(filePath))
    {
        // บันทึกภาพที่โหลดเป็น WMF
        image.Save(System.IO.Path.Combine("YOUR_OUTPUT_DIRECTORY", file + "_out.wmf"), new WmfOptions());
    }
}
```
##### คำอธิบาย
- **`MetaImage`-** คลาสเฉพาะสำหรับการจัดการไฟล์ EMF
- **`WmfOptions`-** ระบุตัวเลือกขณะบันทึกเป็นรูปแบบ WMF ช่วยให้ปรับแต่งได้

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าไดเร็กทอรีอินพุตและชื่อไฟล์ได้รับการระบุอย่างถูกต้อง
- ตรวจสอบว่าคุณมีสิทธิ์การเขียนสำหรับไดเร็กทอรีเอาต์พุตหรือไม่

### คุณสมบัติ 2: การโหลดและบันทึกรูปภาพ

#### ภาพรวม
ฟีเจอร์นี้จะแสดงการโหลดรูปภาพจากเส้นทางและการบันทึกด้วยตัวเลือกเฉพาะ ซึ่งเป็นตัวอย่างความคล่องตัวของ Aspose.Imaging ในการจัดการรูปแบบรูปภาพที่แตกต่างกัน

**การดำเนินการแบบทีละขั้นตอน**

##### การโหลดและการประมวลผลภาพ
```csharp
using Aspose.Imaging;
using Aspose.Imaging.ImageOptions;

string inputPath = "YOUR_DOCUMENT_DIRECTORY";
string outputPath = "YOUR_OUTPUT_DIRECTORY";
string imageName = "example.emf";

string fullPath = System.IO.Path.Combine(inputPath, imageName);

using (Image image = Image.Load(fullPath))
{
    image.Save(System.IO.Path.Combine(outputPath, imageName + "_processed.wmf"), new WmfOptions());
}
```
##### คำอธิบาย
- **`Image.Load`-** โหลดไฟล์ที่ระบุลงในหน่วยความจำ
- **`image.Save`-** บันทึกรูปภาพที่ประมวลผลแล้วด้วยตัวเลือก WMF

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบว่า `imageName` มีอยู่ในไดเร็กทอรีอินพุตของคุณ
- ตรวจสอบให้แน่ใจว่าไม่มีข้อขัดแย้งในการตั้งชื่อในไดเร็กทอรีเอาต์พุต
