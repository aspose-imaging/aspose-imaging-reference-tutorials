## คู่มือการใช้งาน

### แปลง JPEG เป็น DICOM

คุณลักษณะนี้สาธิตวิธีการแปลงภาพ JPEG หน้าเดียวเป็นรูปแบบ DICOM

#### ขั้นตอนที่ 1: โหลดภาพ JPEG

ระบุไดเรกทอรีและชื่อไฟล์ของ JPEG ต้นฉบับของคุณ:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string fileName = "sample.jpg"; // ระบุชื่อไฟล์ JPEG ต้นฉบับของคุณที่นี่
```

โหลด JPEG โดยใช้ Aspose.Imaging `Image` ระดับ:
```csharp
using (var image = Image.Load(Path.Combine(dataDir, fileName)))
{
    // ดำเนินการบันทึกในรูปแบบ DICOM
}
```

#### ขั้นตอนที่ 2: บันทึกเป็น DICOM

ใช้ `DicomOptions` การแปลงและบันทึกภาพของคุณเป็นไฟล์ DICOM:
```csharp
string outputFileNameSingleDcm = Path.Combine(dataDir, "output.dcm");
image.Save(outputFileNameSingleDcm, new DicomOptions());
```

### แปลงไฟล์ TIFF หลายหน้าเป็น DICOM

ขั้นตอนต่อไป คือการแปลงภาพ TIFF หลายหน้าเป็นรูปแบบไฟล์ DICOM

#### ขั้นตอนที่ 1: โหลดภาพ TIFF หลายหน้า

ระบุไฟล์ TIFF ต้นฉบับของคุณ:
```csharp
string inputFileNameMultipage = Path.Combine(dataDir, "multipage.tif");
```

โหลดโดยใช้ Aspose.Imaging `Image` ระดับ:
```csharp
using (var image = Image.Load(inputFileNameMultipage))
{
    // ดำเนินการบันทึกในรูปแบบ DICOM
}
```

#### ขั้นตอนที่ 2: บันทึกเป็น DICOM หลายหน้า

คล้ายกับการแปลง JPEG ใช้ `DicomOptions` เพื่อการบันทึก:
```csharp
string outputFileNameMultipageDcm = Path.Combine(dataDir, "outputMultipage.dcm");
image.Save(outputFileNameMultipageDcm, new DicomOptions());
```
