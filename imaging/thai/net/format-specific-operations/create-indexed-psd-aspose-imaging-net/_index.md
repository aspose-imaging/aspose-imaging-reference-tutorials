### การสร้างไฟล์ PSD ใหม่ด้วยสีที่สร้างดัชนี
คุณสมบัตินี้ช่วยให้คุณสร้างไฟล์ PSD โดยใช้โหมดสี RGB แต่มีจานสีดัชนีเพื่อประสิทธิภาพที่เพิ่มขึ้นและขนาดไฟล์ที่เล็กลง

#### ขั้นตอนที่ 1: กำหนดค่า PsdOptions
```csharp
using Aspose.Imaging.FileFormats.Psd;
using Aspose.Imaging.ImageOptions;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

var createOptions = new PsdOptions();
createOptions.Source = new FileCreateSource(outputDir + "/Newsample_out.psd", false);
createOptions.ColorMode = ColorModes.Rgb;
createOptions.Version = 5; // รับรองความเข้ากันได้กับ PSD เวอร์ชัน 5

// กำหนดจานสีสำหรับไฟล์ PSD
Color[] palette = { Color.Red, Color.Green, Color.Blue };
createOptions.Palette = new PsdColorPalette(palette);

createOptions.CompressionMethod = CompressionMethod.RLE; // ใช้การบีบอัด RLE เพื่อลดขนาดไฟล์
```

#### ขั้นตอนที่ 2: สร้างและวาดบนไฟล์ PSD
```csharp
using (var psd = Image.Create(createOptions, 500, 500))
{
    var graphics = new Graphics(psd);
    
    // ล้างภาพด้วยพื้นหลังสีขาว
    graphics.Clear(Color.White);

    // วาดรูปวงรีในไฟล์ PSD โดยใช้สีตามจานสีที่กำหนดไว้
    graphics.DrawEllipse(new Pen(Color.Red, 6), new Rectangle(0, 0, 400, 400));

    // บันทึกเอาท์พุต
    psd.Save(outputDir + "/CreateIndexedPSDFiles_out.psd");
}
```
#### คำอธิบายพารามิเตอร์และวัตถุประสงค์ของวิธีการ
- **ตัวเลือก PSD:** กำหนดค่าการตั้งค่าสำหรับการสร้างไฟล์ PSD
- **โหมดสี:** ตั้งค่าโหมดสีเป็น RGB ซึ่งอนุญาตให้สร้างดัชนีสีผ่านทางจานสี
- **จานสี:** กำหนดสีเฉพาะที่ใช้ในภาพเพื่อเพิ่มประสิทธิภาพการใช้หน่วยความจำ
- **วิธีการบีบอัด:** ใช้การบีบอัด RLE เพื่อลดขนาดไฟล์อย่างมีประสิทธิภาพ
