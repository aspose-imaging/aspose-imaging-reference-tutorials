## Najczęściej zadawane pytania

**Q: Czy mogę konwertować wiele plików PNG do jednego PDF w jednym wywołaniu?**  
A: Tak, Aspose.Imaging pozwala dodać każdy PNG jako stronę w dokumencie PDF przed zapisaniem.

**Q: Jak osadzić metadane XMP podczas tworzenia PDF?**  
A: Use the `ImageMetadata` class to set XMP properties before saving the image or PDF.

**Q: Czy istnieje sposób na monitorowanie długotrwałych konwersji?**  
A: Zdecydowanie – biblioteka udostępnia zdarzenia postępu, które możesz podłączyć, aby wyświetlać status konwersji.

**Q: Czy biblioteka obsługuje konwersję TIFF wysokiej rozdzielczości do PDF?**  
A: Tak, zachowuje DPI i informacje o kolorach, zapewniając, że PDF odpowiada jakości oryginalnego TIFF.
