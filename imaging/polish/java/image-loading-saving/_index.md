# Samouczki ładowania i zapisywania obrazów w Javie dla Aspose.Imaging

Jeśli pracujesz w Javie i potrzebujesz **konwertować CDR na PNG**, trafiłeś we właściwe miejsce. ## Konwertowanie CDR na PNG — szybki przegląd
Konwertowanie plików CorelDRAW (CDR) na PNG jest powszechnym zadaniem, gdy potrzebujesz obrazów rastrowych gotowych do użycia w sieci pochodzących ze źródeł wektorowych. Aspose.Imaging for Java obsługuje rasteryzację wewnętrznie, więc nie musisz martwić się o narzędzia firm trzecich ani ręczne obliczenia DPI. Dedykowany samouczek wymieniony poniżej przeprowadzi Cię przez ładowanie pliku CDR, konfigurowanie opcji rasteryzacji oraz zapisywanie wyniku jako wysokiej jakości PNG.
