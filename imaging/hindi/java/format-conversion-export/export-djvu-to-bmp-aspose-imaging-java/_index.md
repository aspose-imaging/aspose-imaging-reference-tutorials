# Aspose.Imaging Java का उपयोग करके DjVu छवियों को BMP में कैसे निर्यात करें: एक चरण-दर-चरण मार्गदर्शिका

**आप क्या सीखेंगे:**
- Java के लिए Aspose.Imaging कैसे सेट करें
- Aspose.Imaging के साथ DjVu छवि लोड करना
- उच्च गुणवत्ता वाले आउटपुट के लिए BMP विकल्पों को कॉन्फ़िगर करना

- DjVu दस्तावेज़ के विशिष्ट पृष्ठों को BMP फ़ाइलों के रूप में सहेजना

## कार्यान्वयन मार्गदर्शिका

यह अनुभाग रूपांतरण प्रक्रिया की प्रत्येक विशेषता के लिए तार्किक चरणों में विभाजित है।

### DjVu छवि लोड करें

#### अवलोकन
DjVu इमेज को लोड करना उसे BMP में बदलने का पहला चरण है। Aspose.Imaging इस कार्य को पूरा करने के लिए सरल तरीके प्रदान करता है।

**चरण 1: आवश्यक पैकेज आयात करें**
```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.djvu.DjvuImage;
```

**चरण 2: DjVu छवि लोड करें**
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "Sample.djvu"; 
try (DjvuImage image = (DjvuImage) Image.load(dataDir)) {
    // छवि लोड हो गई है और प्रसंस्करण के लिए तैयार है
} catch (Exception e) {
    e.printStackTrace();
}
```
यहाँ, हम आपकी DjVu फ़ाइल का पथ परिभाषित करते हैं और इसका उपयोग करके इसे लोड करते हैं `Image.load()`सुनिश्चित करें कि आपकी निर्देशिका और फ़ाइल नाम सही ढंग से निर्दिष्ट हैं।

### BmpOptions कॉन्फ़िगर करें

#### अवलोकन
वांछित आउटपुट गुणवत्ता और प्रारूप विनिर्देशों को सेट करने के लिए BMP विकल्पों को कॉन्फ़िगर करना महत्वपूर्ण है।

**चरण 1: BmpOptions का एक इंस्टेंस बनाएँ**
```java
import com.aspose.imaging.imageoptions.BmpOptions;

BmpOptions exportOptions = new BmpOptions();
```

**चरण 2: प्रति पिक्सेल बिट्स सेट करें**
```java
exportOptions.setBitsPerPixel(32);
```
प्रति पिक्सेल बिट्स को 32 पर सेट करने से वास्तविक रंग समर्थन के साथ उच्च गुणवत्ता वाला आउटपुट सुनिश्चित होता है।

### निर्यात के लिए पृष्ठ श्रेणी निर्धारित करें

#### अवलोकन
मल्टी-पेज DjVu दस्तावेज़ों के साथ काम करते समय, आप केवल विशिष्ट पृष्ठों को ही निर्यात करना चाह सकते हैं। यह चरण दर्शाता है कि पृष्ठ श्रेणी को कैसे परिभाषित किया जाए।

**चरण 1: एक IntRange इंस्टेंस बनाएँ**
```java
import com.aspose.imaging.IntRange;

IntRange range = new IntRange(0, 2); // पहले दो पृष्ठ निर्यात करें
```
The `IntRange` ऑब्जेक्ट उन पृष्ठ अनुक्रमणिकाओं के प्रारंभ और अंत को निर्दिष्ट करता है जिन्हें आप निर्यात करना चाहते हैं।

### DjVu पृष्ठों को BMP छवियों के रूप में सहेजें

#### अवलोकन
अंत में, हम कॉन्फ़िगर किए गए विकल्पों का उपयोग करके DjVu दस्तावेज़ से निर्दिष्ट पृष्ठों को अलग BMP फ़ाइलों में सहेजेंगे।

**चरण 1: छवि लोड करें और निर्यात विकल्प सेटअप करें**
```java
try (DjvuImage image = (DjvuImage) Image.load(dataDir + "Sample.djvu")) {
    BmpOptions exportOptions = new BmpOptions();
    exportOptions.setBitsPerPixel(32);
    IntRange range = new IntRange(0, 2); // निर्यात किये जाने वाले पृष्ठों को परिभाषित करें
```

**चरण 2: पृष्ठ निर्यात करें**
```java
for (int i : range.getArray()) {
    exportOptions.setMultiPageOptions(new DjvuMultiPageOptions(range.getArrayOneItemFromIndex(i)));
    String output = "YOUR_OUTPUT_DIRECTORY" + "/ConvertDjvuPagesToImages_out" + i + ".bmp";
    image.save(output, exportOptions);
}
} catch (Exception e) {
    e.printStackTrace();
}
```
निर्धारित सीमा के भीतर प्रत्येक पृष्ठ को BMP फ़ाइल के रूप में सहेजा जाता है `image.save()`, प्रत्येक के लिए आउटपुट पथ निर्दिष्ट किए गए हैं।
