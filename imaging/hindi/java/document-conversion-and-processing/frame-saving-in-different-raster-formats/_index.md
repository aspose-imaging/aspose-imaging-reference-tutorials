# विभिन्न रास्टर प्रारूपों में फ़्रेम सहेजना

सॉफ़्टवेयर डेवलपमेंट की गतिशील दुनिया में, ऐसे उपकरण और लाइब्रेरी होना ज़रूरी है जो आपको विभिन्न फ़ाइल फ़ॉर्मेट के साथ आसानी से काम करने की अनुमति दें। Aspose.Imaging for Java एक ऐसी बहुमुखी लाइब्रेरी है जो डेवलपर्स को आसानी से विभिन्न रास्टर फ़ॉर्मेट में छवियों को हेरफेर करने, परिवर्तित करने और सहेजने की शक्ति देती है। इस चरण-दर-चरण ट्यूटोरियल में, हम यह पता लगाएंगे कि विभिन्न रास्टर फ़ॉर्मेट में TIFF छवि से फ़्रेम को सहेजने के लिए Aspose.Imaging for Java का उपयोग कैसे करें।

3. नमूना TIFF छवि: इस ट्यूटोरियल के लिए, सुनिश्चित करें कि आपके पास एक नमूना TIFF छवि है जिसे आप संसाधित करना चाहते हैं और इसके फ़्रेम को विभिन्न रास्टर प्रारूपों में सहेजना चाहते हैं।

अब जब आपके पास सभी पूर्वापेक्षाएँ मौजूद हैं, तो चलिए चरण-दर-चरण मार्गदर्शिका के साथ आगे बढ़ते हैं।

## पैकेज आयात करें

सबसे पहले, आपको लाइब्रेरी के साथ प्रभावी ढंग से काम करने के लिए Aspose.Imaging for Java से आवश्यक पैकेज आयात करने की आवश्यकता है।

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.imageoptions.PngOptions;
import com.aspose.imaging.fileformats.tiff.TiffImage;
import com.aspose.imaging.fileformats.tiff.TiffFrame;
```

अपनी डेटा निर्देशिका को आरंभीकृत करके और उस TIFF छवि को लोड करके आरंभ करें जिसे आप संसाधित करना चाहते हैं।

```java
String dataDir = "Your Document Directory" + "ModifyingImages/";

```

## चरण 2: फ़्रेम को विभिन्न रास्टर प्रारूपों में सहेजें

इसके बाद, हम TIFF छवि के फ़्रेम पर पुनरावृति करेंगे और प्रत्येक फ़्रेम को वांछित रास्टर प्रारूप में सहेजेंगे। इस उदाहरण में, हम फ़्रेम को PNG प्रारूप में सहेजेंगे।

```java
    int i = 0;
    for (TiffFrame tiffFrame : multiImage.getFrames()) {
        tiffFrame.save("Your Document Directory" + i + "_out.png", new PngOptions());
        i++;
    }
}
```

प्रत्येक फ़्रेम के लिए इन चरणों को दोहराएँ जिसे आप अलग-अलग रास्टर फ़ॉर्मेट में सहेजना चाहते हैं। आप आउटपुट फ़ॉर्मेट विकल्पों को बदलकर JPEG, BMP, या GIF जैसे विभिन्न फ़ॉर्मेट में फ़्रेम सहेजने के लिए इस कोड को आसानी से अनुकूलित कर सकते हैं।
