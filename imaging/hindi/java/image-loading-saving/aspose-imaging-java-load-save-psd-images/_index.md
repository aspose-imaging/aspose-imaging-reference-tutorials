## कार्यान्वयन मार्गदर्शिका

### छवि लोड करना

#### अवलोकन:
किसी भी प्रोसेसिंग वर्कफ़्लो में इमेज लोड करना आपका पहला कदम है। यह सुविधा आपको Aspose.Imaging का उपयोग करके निर्दिष्ट निर्देशिका से इमेज लोड करने की अनुमति देती है।

#### चरण-दर-चरण कार्यान्वयन:

**1. अपना वातावरण तैयार करें:**
सुनिश्चित करें कि लाइब्रेरी ऊपर बताए अनुसार सही ढंग से स्थापित की गई है।

**2. आवश्यक कक्षाएं आयात करें:**
```java
import com.aspose.imaging.Image;
```

**3. छवि लोड करें:**
यहां बताया गया है कि आप अपनी निर्देशिका से BMP छवि कैसे लोड करते हैं:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // अपने दस्तावेज़ निर्देशिका का पथ सेट करें

try (Image image = Image.load(dataDir + "sample.bmp")) {
    // छवि अब लोड हो गई है और प्रसंस्करण के लिए तैयार है
}
```
- **पैरामीटर्स की व्याख्या:** 
  - `dataDir`: वह निर्देशिका जहाँ आपके स्रोत चित्र संग्रहीत हैं.
  - `"sample.bmp"`: उस छवि का फ़ाइल नाम जिसे आप लोड करना चाहते हैं.

### PSD सेव विकल्प कॉन्फ़िगर करना

#### अवलोकन:
सहेजने से पहले, कॉन्फ़िगर करें कि आपकी छवि PSD फ़ाइल के रूप में कैसे सहेजी जाएगी। इसमें संपीड़न और रंग मोड सेट करना शामिल है।

#### चरण-दर-चरण कार्यान्वयन:

**1. आवश्यक कक्षाएं आयात करें:**
```java
import com.aspose.imaging.imageoptions.PsdOptions;
import com.aspose.imaging.fileformats.psd.CompressionMethod;
import com.aspose.imaging.fileformats.psd.ColorModes;
```

**2. सहेजें विकल्प कॉन्फ़िगर करें:**
छवि को कैसे सहेजा जाएगा यह निर्धारित करने के लिए PSD विकल्प सेट करें:
```java
PsdOptions saveOptions = new PsdOptions();
saveOptions.setCompressionMethod(CompressionMethod.Raw); // रॉ या आरएलई (कोई संपीड़न नहीं) के बीच चुनें
saveOptions.setColorMode(ColorModes.Rgb); // विकल्पों में बिटमैप और ग्रेस्केल शामिल हैं
```
- **मुख्य विन्यास:**
  - `CompressionMethod`: PSD डेटा को संपीड़ित करने के लिए प्रयुक्त विधि निर्धारित करता है।
  - `ColorMode`: छवि का रंग प्रारूप परिभाषित करता है.

### छवि को PSD के रूप में सहेजना

#### अवलोकन:
अंत में, कॉन्फ़िगर किए गए विकल्पों का उपयोग करके अपनी संसाधित छवि को PSD प्रारूप में सहेजें।

#### चरण-दर-चरण कार्यान्वयन:

**1. आवश्यक कक्षाएं आयात करें:**
```java
import java.io.IOException;
```

**2. छवि सहेजें:**
छवि को पूर्वनिर्धारित सेटिंग्स के साथ निर्दिष्ट निर्देशिका में सहेजें:
```java
try {
    String outputDir = "YOUR_OUTPUT_DIRECTORY"; // अपना आउटपुट निर्देशिका पथ सेट करें
    image.save(outputDir + "/ExportImageToPSD_out.psd", saveOptions); // कॉन्फ़िगर किए गए विकल्पों का उपयोग करके सहेजता है
} catch (IOException e) {
    e.printStackTrace();
}
```
- **पैरामीटर्स की व्याख्या:**
  - `outputDir`: वह निर्देशिका जहाँ आप PSD फ़ाइल सहेजना चाहते हैं.
  - `"ExportImageToPSD_out.psd"`: सहेजी गई छवि का फ़ाइल नाम.
