## कार्यान्वयन मार्गदर्शिका

अब, हम मुख्य कार्यक्षमता पर ध्यान केंद्रित करेंगे: शिफ्ट मानों द्वारा DICOM छवि को क्रॉप करना और इसे BMP के रूप में सहेजना।

### DICOM छवि को लोड करना और क्रॉप करना

#### अवलोकन

हम अपने एप्लिकेशन में DICOM फ़ाइल लोड करने से शुरू करते हैं। फिर, Aspose.Imaging के शक्तिशाली API का उपयोग करके, हम निर्दिष्ट शिफ्ट मानों (बाएं, दाएं, ऊपर, नीचे) के आधार पर छवि को क्रॉप करेंगे।

#### चरण-दर-चरण कार्यान्वयन

**1. DICOM फ़ाइल लोड करें**

सुनिश्चित करें कि आपकी DICOM फ़ाइल आपके दस्तावेज़ निर्देशिका में तैयार है:

```csharp
using System.IO;
using Aspose.Imaging.FileFormats.Dicom;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // अपने दस्तावेज़ निर्देशिका पथ से बदलें

// DICOM फ़ाइल पढ़ने के लिए स्ट्रीम खोलें
using (var fileStream = new FileStream(dataDir + "/file.dcm", FileMode.Open, FileAccess.Read))
{
    using (DicomImage image = new DicomImage(fileStream))
    {
        // फसल काटने के लिए आगे बढ़ें
```

**2. छवि को क्रॉप करें**

छवि के प्रत्येक पक्ष से आप कितना क्रॉप करना चाहते हैं, यह निर्धारित करने के लिए शिफ्ट मानों का उपयोग करें:

```csharp
// शिफ्ट परिभाषित करें: बाएं, दाएं, ऊपर, नीचे
int leftShift = 1;
int rightShift = 1;
int topShift = 1;
int bottomShift = 1;

// DICOM छवि को क्रॉप करें
crop(image, leftShift, rightShift, topShift, bottomShift);
```

**3. BMP के रूप में सहेजें**

अंत में, अपनी क्रॉप की गई छवि को BMP प्रारूप में सहेजें:

```csharp
        string outputDir = "YOUR_OUTPUT_DIRECTORY";    // अपने आउटपुट निर्देशिका पथ से बदलें

        // आउटपुट फ़ाइल पथ निर्धारित करें और सहेजें
        string outputPath = Path.Combine(outputDir, "DICOMCroppingByShifts_out.bmp");
saveAsBMP(image, outputPath);
    }
}
```

- **त्रुटि प्रबंधन:** अपवादों को सुचारू रूप से संभालने के लिए फ़ाइल संचालन के आसपास try-catch ब्लॉकों को लागू करें।

3. **अनुकूलित रिपोर्टिंग उपकरण:** ऐसे उपकरण बनाना जो मुख्य निष्कर्षों को उजागर करने के लिए काटे गए चित्रों के साथ रिपोर्ट तैयार करें।
