### डिकॉम इमेज पर चित्र बनाना

DICOM छवियों पर आयतों और दीर्घवृत्तों जैसे वेक्टर ग्राफ़िक्स बनाना चिकित्सा निदान में विशिष्ट क्षेत्रों को उजागर करने के लिए महत्वपूर्ण हो सकता है। Aspose.Imaging के साथ इसे प्राप्त करने का तरीका यहां बताया गया है:

#### अवलोकन
हम इसका एक उदाहरण बनाएंगे `DicomImage`, ग्राफ़िक्स ऑब्जेक्ट को आरंभ करें, और उस पर आकृतियाँ बनाएँ।

#### चरण:

##### 1. एक नया DicomImage इंस्टेंस बनाएँ

अपनी DICOM छवि को आरंभीकृत करके प्रारंभ करें:
```csharp
using (DicomImage image = (DicomImage)Image.Create(
    new DicomOptions() { Source = new StreamSource(new MemoryStream()) },
    100, 100))
{
    // आपका ड्राइंग कोड यहाँ जाएगा
}
```

##### 2. ग्राफ़िक्स ऑब्जेक्ट को प्रारंभ करें

The `Graphics` ऑब्जेक्ट आपको छवि पर चित्र बनाने की अनुमति देता है:
```csharp
Graphics graphics = new Graphics(image);
```

##### 3. आकृतियाँ बनाएँ

यहां बताया गया है कि आप विभिन्न रंगों से आयत और दीर्घवृत्त कैसे बना सकते हैं:
- **आयत** संपूर्ण सीमा को कवर करते हुए:
  
  ```csharp
ग्राफिक्स.FillRectangle(नया सॉलिडब्रश(रंग.BlueViolet), छवि.Bounds);
```

- **Aqua Rectangle** at a specific location:
  
  ```csharp
graphics.FillRectangle(new SolidBrush(Color.Aqua), 10, 20, 50, 20);
```

- **नारंगी दीर्घवृत्त** एक बिंदु पर केन्द्रित:
  
  ```csharp
ग्राफिक्स.FillEllipse(नया सॉलिडब्रश(रंग.नारंगी), 30, 50, 70, 30);
```

### Saving Pixel Data to DicomPage Instances

To save drawn image pixels across multiple DICOM pages:

#### Overview
We'll load pixel data from the initial page and replicate it across new pages, adjusting brightness as needed.

#### Steps:

##### 1. Load ARGB32 Pixel Data

First, extract pixel data from the image:
```csharp
int[] pixels = image.LoadArgb32Pixels(image.Bounds);
```

##### 2. नए पेज जोड़ें और चमक समायोजित करें

पृष्ठों को जोड़ने और उनकी चमक को संशोधित करने के लिए लूप करें:
```csharp
for (int i = 1; i < 5; i++)
{
    DicomPage page = image.AddPage();
    page.SaveArgb32Pixels(page.Bounds, pixels);
    page.AdjustBrightness(i * 30);
}
```

इसी प्रकार, पृष्ठों को आरंभ में डालें और उनकी चमक को विपरीत दिशा में समायोजित करें:
```csharp
for (int i = 1; i < 5; i++)
{
    DicomPage page = image.InsertPage(0);
    page.SaveArgb32Pixels(page.Bounds, pixels);
    page.AdjustBrightness(-i * 30);
}
```
