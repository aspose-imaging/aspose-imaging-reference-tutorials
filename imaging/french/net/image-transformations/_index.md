# Tutoriels de transformation d'images .NET pour Aspose.Imaging

Nos tutoriels de transformation d'images fournissent des instructions détaillées pour manipuler la géométrie et les dimensions des images avec .NET. Suivez ces guides pratiques pour apprendre à redimensionner les images proportionnellement, à recadrer des zones spécifiques, à faire pivoter les images selon des angles précis et à mettre en œuvre des transformations géométriques complexes. Ces tutoriels incluent des exemples complets de code C# pour les opérations de transformation courantes, vous aidant à créer des applications permettant de contrôler précisément l'apparence et les dimensions des images.
