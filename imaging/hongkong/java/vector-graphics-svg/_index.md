## 常見問題

**Q: 我可以在不寫任何程式碼的情況下將 SVG 轉換為 PNG 嗎？**  
A: 可以，Aspose.Imaging 線上示範允許您上傳 SVG 並下載 PNG，但使用 Java API 可實現完整自動化。

**Q: “svg to png java” 是否支援 Linux？**  
A: 當然支援。Aspose.Imaging for Java 為跨平台，能在 Windows、Linux 與 macOS 上運行。

**Q: 轉換時如何保留文字為可選取的圖層？**  
A: 使用 `SvgExportOptions` 以保留文字為向量資料，或在需要點陣圖時以高 DPI 進行點陣化。

**Q: 高解析度 PNG 應使用多少 DPI？**  
A: 列印品質通常使用 300–600 DPI；網頁則 72–96 DPI 即可。

**Q: 此函式庫支援批次轉換嗎？**  
A: 支援——將載入與儲存的邏輯包在迴圈中，或使用 `Parallel.ForEach` 進行多執行緒處理。
