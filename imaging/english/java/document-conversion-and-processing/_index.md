## Step‑by‑Step Guide

### Step 1: Set up the project
Add the Aspose.Imaging dependency to your `pom.xml` (Maven) or `build.gradle` (Gradle). This makes the conversion classes available in your project.

### Step 2: Load the PNG image
Create an `Image` object from the PNG file. The library automatically detects the format.

### Step 3: Convert to PDF
Instantiate a `PdfOptions` object (if you need custom PDF settings) and call `save` with the target format.

### Step 4: (Optional) Crop the image before conversion
If you need to **how to crop image java**, use the `crop` method on the `Image` object before saving it as PDF. This ensures only the desired region appears in the final document.

### Step 5: Save and verify
Write the resulting PDF to disk and open it to confirm that the conversion succeeded and the image appears as expected.
