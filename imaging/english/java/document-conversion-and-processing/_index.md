## Frequently Asked Questions

**Q: Can I convert multiple PNG files to a single PDF in one call?**  
A: Yes, Aspose.Imaging lets you add each PNG as a page in the PDF document before saving.

**Q: How do I embed XMP metadata while creating the PDF?**  
A: Use the `ImageMetadata` class to set XMP properties before saving the image or PDF.

**Q: Is there a way to monitor long‑running conversions?**  
A: Absolutely – the library provides progress events that you can hook into to display conversion status.

**Q: Does the library support high‑resolution TIFF to PDF conversion?**  
A: Yes, it retains DPI and color information, ensuring the PDF matches the original TIFF quality.
