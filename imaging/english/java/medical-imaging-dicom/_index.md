## Frequently Asked Questions

**Q: Can I adjust brightness without losing DICOM metadata?**  
A: Yes. Aspose.Imaging lets you modify pixel data while preserving all original DICOM tags, so the file remains fully compliant.

**Q: What brightness range should I use?**  
A: The library accepts a float value where 1.0 leaves the image unchanged. Values >1.0 brighten, <1.0 darken. Typical adjustments are between 0.5 and 2.0.

**Q: Is the operation thread‑safe?**  
A: Loading and processing separate DICOM instances in parallel is safe. Do not share the same `Image` object across threads without synchronization.

**Q: How do I test the changes visually?**  
A: After saving, open the resulting DICOM file in any DICOM viewer (e.g., RadiAnt, OsiriX) to verify the brightness adjustment.

**Q: Does this work on compressed DICOM (JPEG2000, RLE)?**  
A: Yes. Aspose.Imaging automatically decompresses, applies the brightness change, and re‑compresses using the original compression scheme.
