## Convertir CDR a PNG – Visión general rápida
Convertir archivos CorelDRAW (CDR) a PNG es una tarea común cuando necesitas imágenes rasterizadas listas para la web a partir de fuentes vectoriales. Aspose.Imaging para Java maneja la rasterización internamente, por lo que no tienes que preocuparte por herramientas de terceros o cálculos manuales de DPI. El tutorial dedicado que se muestra a continuación te guía a través de la carga de un archivo CDR, la configuración de opciones de rasterización y el guardado del resultado como un PNG de alta calidad.
