## دليل التنفيذ

### الميزة: التثنية باستخدام عتبة برادلي التكيفية
**ملخص:**
تعمل هذه الميزة على تحويل صورة DICOM إلى تنسيق ثنائي باستخدام عتبة برادلي التكيفية، مما يعزز التباين المحلي ويحسن نتائج التحليل.

#### الخطوة 1: افتح ملف DICOM
أولاً، افتح ملف DICOM بتحديد مساره. استبدل `'YOUR_DOCUMENT_DIRECTORY'` مع الدليل الفعلي للمستند الخاص بك.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string inputFile = Path.Combine(dataDir, "image.dcm");

using (var fileStream = new FileStream(inputFile, FileMode.Open, FileAccess.Read))
{
    // انتقل إلى الخطوة 2
}
```
*لماذا؟*:يسمح فتح الملف في مجرى القراءة والمعالجة بكفاءة دون تحميل المحتوى بأكمله في الذاكرة.

#### الخطوة 2: تحميل الصورة من الدفق باستخدام فئة DicomImage
قم بتحميل الصورة باستخدام `DicomImage`، الذي يوفر طرقًا خاصة بملفات DICOM.

```csharp
using (DicomImage image = new DicomImage(fileStream))
{
    // انتقل إلى الخطوة 3
}
```
*لماذا؟*:تعمل هذه الخطوة على تحضير بيانات DICOM الخاصة بك للمعالجة، مما يتيح إجراء العديد من التحويلات والتحليلات.

#### الخطوة 3: تطبيق عتبة برادلي التكيفية لتحويل الصورة إلى صورة ثنائية
تعمل عملية التثليث على تحسين تباين الصورة عن طريق ضبط البكسلات فوق عتبة معينة إلى اللون الأبيض، وتلك التي تحتها إلى اللون الأسود. المعلمة `10` ضبط هذه العملية بشكل دقيق.

```csharp
image.BinarizeBradley(10);
```
*لماذا؟*تتكيف طريقة برادلي مع الاختلافات المحلية في السطوع، مما يجعلها مثالية للصور الطبية ذات الإضاءة غير المتساوية.

#### الخطوة 4: احفظ الصورة الثنائية الناتجة بتنسيق BMP
أخيرًا، احفظ الصورة التي تمت معالجتها. استبدلها `'YOUR_OUTPUT_DIRECTORY'` حيث تريد حفظ الناتج.

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
string outputFile = Path.Combine(outputDir, "BinarizationWithBradleysAdaptiveThreshold_out.bmp");
image.Save(outputFile, new BmpOptions());
```
*لماذا؟*:يحظى تنسيق BMP بدعم واسع النطاق ويوفر طريقة مباشرة لتوضيح النتائج الثنائية.
