## ضبط وحفظ صور DICOM بتنسيق BMP باستخدام Aspose.Imaging لـ .NET

بمجرد تثبيت الحزمة اللازمة ومعالجة الترخيص، دعنا ننفذ وظائفنا الأساسية.

### ضبط سطوع صورة DICOM

**ملخص**
تتيح لك هذه الميزة تعديل مستوى سطوع صورة DICOM بمقدار محدد، مما يجعلها أكثر وضوحًا أو أكثر ملاءمة لمزيد من التحليل.

#### التنفيذ خطوة بخطوة
1. **افتح ملف DICOM وقم بتحميله**:ابدأ بفتح ملف DICOM الخاص بك باستخدام `FileStream`. وهذا يضمن أن Aspose.Imaging يمكنه قراءة البيانات بشكل صحيح.
   
   ```csharp
   string dataDir = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "file.dcm");
   using (var fileStream = new FileStream(dataDir, FileMode.Open, FileAccess.Read))
   {
       // تحميل الصورة إلى كائن DicomImage
       using (DicomImage image = new DicomImage(fileStream))
       {
           // انتقل إلى ضبط السطوع
       }
   }
   ```

2. **ضبط السطوع**: يستخدم `image.AdjustBrightness(int value)` أين `value` هو عدد الوحدات التي تريد زيادة أو تقليل السطوع بها.

   ```csharp
   image.AdjustBrightness(50);  // زيادة السطوع بمقدار 50 وحدة
   ```

3. **حفظ كملف BMP**:قم بتكوين صورة DICOM المعدلة وحفظها بتنسيق BMP باستخدام `BmpOptions`.

   ```csharp
   var options = new BmpOptions();
   string outputPath = Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "AdjustBrightnessDICOM_out.bmp");
   image.Save(outputPath, options);
   ```

**نصائح استكشاف الأخطاء وإصلاحها**
- تأكد من تعيين مسارات الملفات الخاصة بدليل الإدخال والإخراج بشكل صحيح.
- تأكد من إمكانية الوصول إلى ملف DICOM وأنه غير تالف.

### حفظ صورة DICOM بتنسيق BMP

**ملخص**
يؤدي تحويل صورة DICOM إلى تنسيق BMP إلى تعزيز التوافق عبر الأنظمة الأساسية التي قد لا تدعم DICOM بشكل أصلي.

#### التنفيذ خطوة بخطوة
1. **افتح ملف DICOM وقم بتحميله**:على غرار ضبط السطوع، ابدأ بتحميل ملف DICOM الخاص بك.
   
   ```csharp
   string dataDir = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "file.dcm");
   using (var fileStream = new FileStream(dataDir, FileMode.Open, FileAccess.Read))
   {
       // تحميل الصورة إلى كائن DicomImage
       using (DicomImage image = new DicomImage(fileStream))
       {
           // انتقل إلى الحفظ بتنسيق BMP
       }
   }
   ```

2. **حفظ كملف BMP**: يستخدم `BmpOptions` لتحديد كيفية حفظ صورتك.

   ```csharp
   var options = new BmpOptions();
   string outputPath = Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "SaveDICOMAsBMP_out.bmp");
   image.Save(outputPath, options);
   ```

**نصائح استكشاف الأخطاء وإصلاحها**
- التحقق من أذونات الكتابة في دليل الإخراج.
- يضمن `BmpOptions` تم تكوينه بشكل صحيح إذا كنت بحاجة إلى إعدادات BMP محددة.
