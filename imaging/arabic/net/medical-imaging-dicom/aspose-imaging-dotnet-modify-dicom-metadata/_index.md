- حفظ التغييرات والتحقق من التحديثات في البيانات الوصفية.
- تنظيف الملفات المؤقتة بعد المعالجة.

## دليل التنفيذ

سنقوم بتقسيم تنفيذنا إلى ثلاث ميزات رئيسية: تحميل صور DICOM وحفظها باستخدام البيانات الوصفية المعدلة، والتحقق من هذه التغييرات، وتنظيف الملفات المؤقتة.

### الميزة 1: تحميل صور DICOM وحفظها

**ملخص**
توضح هذه الميزة كيفية تحميل صورة DICOM وتعديل بياناتها الوصفية باستخدام علامات XMP وحفظ الملف المحدث والتأكد من تطبيق التعديلات بشكل صحيح.

#### التنفيذ خطوة بخطوة

##### تحميل صورة DICOM

```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";
using (DicomImage image = (DicomImage)Image.Load(Path.Combine(dataDir, "file.dcm")))
```
*لماذا؟*:يعتبر تحميل ملف DICOM الخاص بك هو الخطوة الأولى للوصول إلى بياناته الوصفية وتعديلها.

##### تعديل البيانات الوصفية باستخدام علامات XMP

```csharp
XmpPacketWrapper xmpPacketWrapper = new XmpPacketWrapper();
DicomPackage dicomPackage = new DicomPackage();

// تعيين سمات DICOM المختلفة باستخدام الحزمة
dicomPackage.SetEquipmentInstitution("Test Institution");
dicomPackage.SetPatientName("Test Name");

xmpPacketWrapper.AddPackage(dicomPackage);
```
*لماذا؟*:يعد تعديل البيانات الوصفية أمرًا ضروريًا لتخصيص وتحديث تفاصيل المريض أو المعدات.

##### حفظ الصورة المعدلة

```csharp
image.Save(Path.Combine(dataDir, "output.dcm"), new DicomOptions() { XmpData = xmpPacketWrapper });
```
*لماذا؟*:يضمن الحفظ كتابة جميع التغييرات مرة أخرى في ملف DICOM جديد.

### الميزة 2: التحقق من التغييرات في بيانات DICOM الوصفية

**ملخص**
تتمثل هذه الميزة في التحقق من التعديلات التي تم إجراؤها عن طريق مقارنة البيانات الوصفية قبل وبعد حفظ الصورة.

#### التنفيذ خطوة بخطوة

##### تحميل الصورة المعدلة واسترداد البيانات الوصفية

```csharp
using (DicomImage imageSaved = (DicomImage)Image.Load(Path.Combine(dataDir, "output.dcm")))
{
    ReadOnlyCollection<string> savedMetadata = imageSaved.FileInfo.DicomInfo;
}
```
*لماذا؟*:يسمح لك تحميل الملف المعدل بالتحقق من انعكاس التغييرات بشكل دقيق.

##### مقارنة البيانات الوصفية

```csharp
int tagsCountDiff = Math.Abs(savedMetadata.Count - originalMetadata.Count);
```
*لماذا؟*:تساعد مقارنة أعداد البيانات الوصفية على ضمان تطبيق جميع التعديلات المقصودة بشكل صحيح.

### الميزة 3: تنظيف ملفات الإخراج

**ملخص**
توضح هذه الميزة كيفية حذف الملفات المؤقتة التي تم إنشاؤها أثناء سير عمل معالجة DICOM.

#### التنفيذ خطوة بخطوة

##### حذف الملفات المؤقتة

```csharp
if (File.Exists(Path.Combine(dataDir, "output.dcm")))
{
    File.Delete(Path.Combine(dataDir, "output.dcm"));
}
```
*لماذا؟*:يساعد التنظيف على منع الاستخدام غير الضروري للتخزين والحفاظ على بيئتك مرتبة.
