- تعليمات خطوة بخطوة لضبط تباين صورة DICOM
- إعداد البيئة الخاصة بك باستخدام Aspose.Imaging

## دليل التنفيذ
سنغطي كيفية تحميل صورة DICOM وضبط تباينها.

### تحميل وعرض صورة DICOM (H2)
**ملخص**:يعتبر تحميل ملف DICOM الخطوة الأولى قبل أي معالجة، مثل ضبط التباين.

#### الخطوة 1: تحديد مسارات الملفات
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

#### الخطوة 2: فتح FileStream
إنشاء دفق لقراءة ملف DICOM للتعامل بكفاءة مع الملفات الكبيرة النموذجية في التصوير الطبي:

```csharp
using (var fileStream = new FileStream(dataDir + "/file.dcm", FileMode.Open, FileAccess.Read))
{
    var image = new DicomImage(fileStream);
    // الصورة الآن جاهزة للتلاعب.
}
```

### ضبط تباين صورة DICOM (H2)
**ملخص**:يساعد تعزيز التباين على الكشف عن الميزات في الصور الطبية، مما يساعد على إجراء تحليل أفضل.

#### الخطوة 1: تحديد دليل الإخراج
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: تحميل الصورة وتعديلها
استخدام `DicomImage`افتح ملفك واضبط تباينه. هنا، سنزيده بمقدار 50 وحدة، وهي قيمة يمكنك تعديلها حسب الحاجة.

```csharp
using (var fileStream = new FileStream(dataDir + "/file.dcm", FileMode.Open, FileAccess.Read))
using (DicomImage image = new DicomImage(fileStream))
{
    image.AdjustContrast(50);
}
```

#### الخطوة 3: حفظ الصورة المعدلة
بعد التعديل، احفظ صورتك بالتنسيق المفضل لديك مثل BMP:

```csharp
var options = new BmpOptions();
image.Save(outputDir + "/AdjustContrastDICOM_out.bmp", options);
```
