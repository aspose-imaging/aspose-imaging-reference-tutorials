## دليل التنفيذ

الآن بعد أن قمت بإعداد كل شيء، دعنا ننتقل إلى عملية تحويل صورة CMX إلى PDF.

### تحميل وحفظ صورة CMX إلى PDF

توضح هذه الميزة تحميل ملف صورة CMX وحفظه كملف PDF. سنقسم العملية إلى خطوات سهلة.

#### الخطوة 1: تعيين أدلة الإدخال والإخراج
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string inputFile = Path.Combine(dataDir, "MultiPage.cmx");
```
**توضيح**: يستبدل `YOUR_DOCUMENT_DIRECTORY` مع مسار الدليل الفعلي. هذا يُهيئ مسار ملف الإدخال للتحميل.

#### الخطوة 2: تحميل ملف صورة CMX
```csharp
using (CmxImage image = (CmxImage)Image.Load(inputFile)) {
    // سيتم نقل خطوات التكوين والحفظ هنا.
}
```
**توضيح**:نقوم بتحميل صورة CMX باستخدام Aspose.Imaging's `Image.Load` الطريقة، التأكد من إرسال الملف بشكل صحيح إلى `CmxImage`.

#### الخطوة 3: تكوين خيارات PDF
```csharp
PdfOptions options = new PdfOptions();
options.PdfDocumentInfo = new Aspose.Imaging.FileFormats.Pdf.PdfDocumentInfo();

// تعيين خيارات التحويل النقطي لعرض المتجهات
options.VectorRasterizationOptions = image.GetDefaultOptions(new object[] { Color.White, image.Width, image.Height }).VectorRasterizationOptions;
options.VectorRasterizationOptions.TextRenderingHint = TextRenderingHint.SingleBitPerPixel;
options.VectorRasterizationOptions.SmoothingMode = SmoothingMode.None;
```
**توضيح**:قم بتكوين خيارات PDF لضمان جودة عرض عالية. قمنا بضبط `TextRenderingHint` و `SmoothingMode` للحصول على أفضل النتائج.

#### الخطوة 4: حفظ صورة CMX بتنسيق PDF
```csharp
string outputPdfPath = Path.Combine(dataDir, "MultiPage.pdf");
image.Save(outputPdfPath, options);
```
**توضيح**أخيرًا، احفظ الصورة في المسار المحدد باستخدام إعدادات PDF المُعدّة. تُحوّل هذه الخطوة ملف CMX وتُخزّنه كملف PDF.

#### الخطوة 5: التنظيف (اختياري)
```csharp
File.Delete(Path.Combine(dataDir, "MultiPage.pdf"));
```
**توضيح**:اختياريًا، يمكنك حذف ملف PDF الناتج إذا كنت بحاجة إليه مؤقتًا فقط.
