## دليل التنفيذ

### تغيير حجم وتحويل WMF إلى PNG

#### ملخص
تُركز هذه الميزة على تغيير حجم صورة WMF مع الحفاظ على نسبة العرض إلى الارتفاع، ثم تحويلها إلى صيغة PNG عالية الجودة. سنستكشف كيفية إعداد خيارات التصغير النقطي وتكوينها لتحقيق أفضل النتائج.

#### الخطوة 1: تحميل صورة WMF
ابدأ بتحميل ملف صورتك إلى التطبيق:
```csharp
using (Image image = Image.Load("@YOUR_DOCUMENT_DIRECTORY/input.wmf"))
{
    // سيتم اتباع خطوات المعالجة الإضافية هنا
}
```
هنا، `Image.Load` يقرأ ملف WMF الخاص بك. تأكد من استبدال `@YOUR_DOCUMENT_DIRECTORY/input.wmf` مع مسار الملف الفعلي الخاص بك.

#### الخطوة 2: تغيير حجم الصورة
حدد الأبعاد المطلوبة مع الحفاظ على نسبة العرض إلى الارتفاع:
```csharp
// تغيير الحجم إلى 100 × 100 بكسل
double k = (image.Width * 1.00) / image.Height;
image.Resize(100, 100);
```
يضمن هذا النهج أن الصورة التي تم تغيير حجمها تحتفظ بنسبها الأصلية.

#### الخطوة 3: إعداد خيارات التنقيح
تكوين إعدادات التحويل النقطي لتحويل WMF إلى PNG:
```csharp
WmfRasterizationOptions emfRasterization = new WmfRasterizationOptions
{
    BackgroundColor = Color.WhiteSmoke,
    PageWidth = 100,
    PageHeight = (int)Math.Round(100 / k),
    BorderX = 5,
    BorderY = 10
};
```
تحدد هذه الخيارات أبعاد الإخراج ولون الخلفية والحدود.

#### الخطوة 4: احفظ بتنسيق PNG
احفظ صورتك التي تم تغيير حجمها بتنسيق PNG:
```csharp
ImageOptionsBase imageOptions = new PngOptions
{
    VectorRasterizationOptions = emfRasterization
};
image.Save("@YOUR_OUTPUT_DIRECTORY/CreateEMFMetaFileImage_out.png", imageOptions);
```
تستخدم هذه الخطوة خيارات التحويل النقطي المُهيأة لإنشاء ملف PNG عالي الجودة.
