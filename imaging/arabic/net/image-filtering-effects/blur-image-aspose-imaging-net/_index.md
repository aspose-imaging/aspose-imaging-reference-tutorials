للبدء، ستحتاج إلى دمج مكتبة Aspose.Imaging في مشروعك. إليك الطريقة:

### خيارات التثبيت

**استخدام .NET CLI:**
```bash
dotnet add package Aspose.Imaging
```

## دليل التنفيذ: تشويش الصورة باستخدام مرشح غاوسي

دعونا نوضح كيفية تنفيذ هذه الوظيفة خطوة بخطوة:

### تحميل الصورة

ابدأ بتحديد مجلدات صور المصدر والإخراج. تأكد من وجود ملف باسم `asposelogo.gif` في دليل المستندات الخاص بك.

```csharp
using Aspose.Imaging;
using Aspose.Imaging.ImageFilters.FilterOptions;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

using (Image image = Image.Load(dataDir + "/asposelogo.gif"))
{
    // تتبع خطوات التحويل والمعالجة هنا
}
```

### تحويل إلى صورة نقطية

لتطبيق المرشحات، قم بتحويل الصورة المحملة إلى `RasterImage`.

```csharp
RasterImage rasterImage = (RasterImage)image;
// متابعة عمليات التصفية
```

### تطبيق Gaussian Blur

استخدم `GaussianBlurFilterOptions` لتمويه صورتك. هنا، يُطبّق نصف قطر 5×5 على كامل حدود الصورة.

```csharp
rasterImage.Filter(rasterImage.Bounds, new GaussianBlurFilterOptions(5, 5));
```

**توضيح**: 
- ال **نصف القطر** (5، 5) يحدد قوة تأثير التمويه. القيم الأكبر تُنتج تمويهات أكثر وضوحًا.
- **الحدود**:يضمن تطبيق الفلتر على الصورة بأكملها.

### احفظ الصورة الضبابية

بعد المعالجة، احفظ صورتك الضبابية في دليل الإخراج المخصص.

```csharp
rasterImage.Save(outputDir + "/BlurAnImage_out.gif");
```
