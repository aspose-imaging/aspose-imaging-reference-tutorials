## دليل التنفيذ: تحميل إطارات TIFF وحفظها

يوفر هذا القسم دليلاً خطوة بخطوة لاستخدام Aspose.Imaging لـ .NET لمعالجة صور TIFF متعددة الإطارات.

### نظرة عامة على الميزة

تتيح الميزة الموضحة في هذا البرنامج التعليمي للمطورين تحميل صورة TIFF متعددة الإطارات، وتكرارها، وحفظ كل إطار كملف مستقل. يُعد هذا مفيدًا عند التعامل مع مجموعات بيانات كبيرة تتطلب معالجة أو تحليلًا مستقلًا للإطارات.

#### الخطوة 1: تحديد الدلائل للإدخال والإخراج

أولاً، حدد الدلائل لملفات الإدخال والإخراج الخاصة بك:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: تحميل صورة TIFF متعددة الإطارات

استخدم Aspose.Imaging's `TiffImage` الفئة لتحميل صورة TIFF متعددة الإطارات من القرص:

```csharp
using (var multiImage = (TiffImage)Image.Load(dataDir + "/SampleTiff1.tiff"))
{
    // متابعة معالجة كل إطار.
}
```

#### الخطوة 3: التكرار على كل إطار

قم بالتنقل بين الإطارات في الصورة المحملة. `Frames` ممتلكات `TiffImage` يوفر الوصول إلى كل إطار:

```csharp
int i = 0;
foreach (var tiffFrame in multiImage.Frames)
{
    // احفظ كل إطار كملف TIFF فردي.
}
```

#### الخطوة 4: احفظ كل إطار

احفظ كل إطار بتنسيق JPEG RGB. توضح هذه الطريقة كيفية تخصيص إعدادات الإخراج:

```csharp
tiffFrame.Save(outputDir + $"/{i}_out.tiff", new TiffOptions(TiffExpectedFormat.TiffJpegRgb));
i++;
```
