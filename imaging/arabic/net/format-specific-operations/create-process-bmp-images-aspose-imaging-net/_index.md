### الميزة: إنشاء الصور ومعالجتها

#### ملخص

إنشاء صورة BMP بإعدادات محددة يُمكّنك من تخصيص الرسومات وفقًا لاحتياجاتك. يوضح هذا البرنامج التعليمي كيفية استخدام Aspose.Imaging لإنشاء ملف صورة، وضبط قيم البكسل، وحفظه بكفاءة.

#### الخطوة 1: إعداد مشروعك وإنشاء الصورة

أولاً، تأكد من أنك قمت بتحديد مسارات لدليل المستند ودليل الإخراج:

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputPath = @"YOUR_OUTPUT_DIRECTORY";

// إنشاء مسار لملف صورتك.
string imageDataPath = System.IO.Path.Combine(documentDirectory, "sample.bmp");
```

#### الخطوة 2: إنشاء صورة BMP باستخدام خيارات محددة

سنبدأ بإنشاء مثيل لـ `BmpOptions` وضبطه لاستخدام 32 بت لكل بكسل:

```csharp
using (System.IO.FileStream fileStream = System.IO.File.Create(imageDataPath))
{
    using (BmpOptions bmpOptions = new BmpOptions())
    {
        bmpOptions.BitsPerPixel = 32;
        bmpOptions.Source = new StreamSource(fileStream);
        
        // إنشاء صورة بالأبعاد المحددة.
        using (RasterImage image = (RasterImage)Image.Create(bmpOptions, 10, 10))
        {
            Color[] pixels = new Color[4];
            for (int i = 0; i < 4; ++i)
            {
                // تعيين لون البكسل باستخدام قيم ARGB.
                pixels[i] = Color.FromArgb(40, 30, 20, 10);
            }

            // احفظ وحدات البكسل المحددة في منطقة مستطيلة في الصورة.
            image.SavePixels(new Rectangle(0, 0, 2, 2), pixels);

            // احفظ الصورة المعالجة في مسار الإخراج.
            string processedImagePath = System.IO.Path.Combine(outputPath, "processed_image.bmp");
            image.Save(processedImagePath);
        }
    }
}
```

#### توضيح

- **خيارات Bmp**: يُهيئ مواصفات ملف BMP، مثل عمق البت. هنا، نضبط `BitsPerPixel` إلى 32 للحصول على تمثيل لوني أكثر ثراءً.
  
- **مصدر التدفق**:يتم استخدامه كمصدر لبيانات البكسل، مما يسمح لنا بالعمل مباشرة مع التدفقات.

- **طريقة حفظ البكسل**:تعتبر هذه الطريقة ضرورية لتعيين وحدات بكسل محددة داخل مستطيل محدد في صورتك.
