## دليل التنفيذ

سنقوم بتقسيم العملية إلى ميزتين رئيسيتين: تحميل صورة PNG وإنشاء صورة جديدة ذات شفافية.

### تحميل ومعالجة صورة PNG

#### ملخص
تتيح لك هذه الميزة تحميل ملف PNG موجود، واستخراج بيانات البكسل، وتخزين أبعاده. تُعد هذه الميزة ضرورية للعمليات التي تتطلب معالجة مباشرة لبكسلات الصورة.

**الخطوات المتبعة:**
##### تحميل صورة PNG
1. **تحميل الصورة إلى كائن RasterImage:**
   ```csharp
   string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
   using (RasterImage raster = (RasterImage)Image.Load(documentDirectory + "/aspose_logo.png"))
   {
       // استرداد أبعاد الصورة وبيانات البكسل.
       int width = raster.Width;
       int height = raster.Height;
       Color[] pixels = raster.LoadPixels(new Rectangle(0, 0, width, height));
   }
   ```

##### توضيح
- **صورة نقطية**:تمثل هذه الفئة الصورة المحملة وتوفر طرقًا للعمل مع محتواها بشكل مباشر.
- **طريقة LoadPixels**:يستخرج بيانات البكسل إلى `Color` مصفوفة لمزيد من المعالجة.

### إنشاء صورة PNG وحفظها مع الشفافية

#### ملخص
بعد معالجة الصورة، قد ترغب في حفظها بإعدادات شفافية محددة. توضح هذه الميزة كيفية إنشاء صورة PNG جديدة، وتطبيق الشفافية عليها، وحفظها كملف JPEG.

**الخطوات المتبعة:**
##### تهيئة كائن PngImage
1. **إنشاء صورة PNG جديدة:**
   ```csharp
   string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
   using (PngImage png = new PngImage(width, height, PngColorType.TruecolorWithAlpha))
   {
       // تطبيق بيانات البكسل وإعدادات الشفافية.
       png.SavePixels(new Rectangle(0, 0, width, height), pixels);
       png.TransparentColor = Color.Black;
       png.HasTransparentColor = true;

       // احفظ ملف PNG بصيغة JPEG مع معلومات الشفافية.
       png.Save(outputDirectory + "/SpecifyTransparencyforPNGImages_out.jpg");
   }
   ```

##### توضيح
- **صورة PNG**: يُمثل الصورة الجديدة التي يتم إنشاؤها. يدعم أنواعًا مختلفة من الألوان، بما في ذلك لون ألفا للشفافية.
- **لون شفاف**:يحدد اللون الذي يجب اعتباره شفافًا في الصورة.
