## دليل التنفيذ
### تحميل الصورة ومعالجتها
يوضح هذا القسم كيفية تحميل صورة واسترجاع تاريخ آخر تعديل لها باستخدام Aspose.Imaging. تُعد هذه الميزة أساسية للتطبيقات التي تحتاج إلى تتبع التغييرات أو تحديث الصور بناءً على بياناتها الوصفية.

#### الخطوة 1: إعداد الدلائل
أولاً، قم بتحديد أدلة الإدخال والإخراج التي سيتم تخزين صورك فيها:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: تحميل صورة
يستخدم `Image.Load` لقراءة ملف صورة. تُرجع هذه الطريقة قيمة عامة `Image` الكائن الذي يمكنك إرساله إلى `RasterImage` لمزيد من العمليات المحددة:

```csharp
using (RasterImage image = (RasterImage)Image.Load(dataDir + "/aspose-logo.jpg"))
{
    // منطق معالجة الصورة يذهب هنا.
}
```

#### الخطوة 3: استرداد تاريخ آخر تعديل
للحصول على تاريخ آخر تعديل لملف صورة، استخدم `GetModifyDate` هذه الطريقة يمكنها أخذ بيانات تعريف XMP في الاعتبار عند الحاجة:

```csharp
string modifyDateFile = image.GetModifyDate(true).ToString();
string modifyDateXmp = image.GetModifyDate(false).ToString();

Console.WriteLine("Last modified date from file: " + modifyDateFile);
Console.WriteLine("Last modified date from XMP metadata: " + modifyDateXmp);
```

**توضيح**: 
- `true` في `GetModifyDate` تأخذ الطريقة بعين الاعتبار بيانات تعريف نظام الملفات.
- `false` يسترجع التواريخ من بيانات XMP الخاصة بالصورة، إذا كانت متاحة.
