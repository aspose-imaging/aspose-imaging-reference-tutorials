## دليل التنفيذ

### الميزة 1: تحميل الصورة وتخزينها مؤقتًا

#### ملخص
يُعد تحميل الصور بكفاءة أمرًا بالغ الأهمية لتحسين الأداء. بتخزين الصور مؤقتًا عند تحميلها، تُسرّع العمليات اللاحقة بشكل ملحوظ، إذ تعتمد على البيانات المخزنة مؤقتًا بدلًا من القراءة المتكررة من القرص.

**التنفيذ خطوة بخطوة**

**الخطوة 1:** تحديد مسارات الدليل.
```csharp
string dataDir = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_DOCUMENT_DIRECTORY");
```

**الخطوة 2:** تحميل الصورة وتخزينها مؤقتًا.
```csharp
using Aspose.Imaging;
using System.IO;

Image.Load(dataDir + "/aspose-logo.jpg").Dispose();
// يتم تخزين الصورة مؤقتًا تلقائيًا، مما يعزز الأداء للعمليات المستقبلية.
```
*توضيح:* تحميل صورة مع `Image.Load()` يقوم بتخزينها في الذاكرة، مما يؤدي إلى تسريع الوصول المتكرر.

### الميزة 2: ضبط تباين الصورة

#### ملخص
يُمكن لضبط التباين أن يُحسّن من جاذبية الصورة. تُوضّح هذه الميزة كيفية تعديل الصور وحفظها باستخدام Aspose.Imaging لـ .NET.

**التنفيذ خطوة بخطوة**

**الخطوة 1:** تحديد أدلة الإدخال والإخراج.
```csharp
string dataDir = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_DOCUMENT_DIRECTORY");
string outputDir = Path.Combine(Directory.GetCurrentDirectory(), "YOUR_OUTPUT_DIRECTORY");
```

**الخطوة 2:** قم بتحميل الصورة كصورة RasterImage للتلاعب بها.
```csharp
using Aspose.Imaging;
using System.IO;

using (Image image = Image.Load(dataDir + "/aspose-logo.jpg"))
{
    RasterImage rasterImage = (RasterImage)image;
    
    // ضمان التخزين المؤقت لتحسين الأداء
    if (!rasterImage.IsCached)
    {
        rasterImage.CacheData();
    }
```

**الخطوة 3:** ضبط التباين وحفظ الصورة.
```csharp
    // ضبط مستوى التباين ضمن نطاق من -100 إلى 100
    rasterImage.AdjustContrast(10);
    
    TiffOptions tiffOptions = new TiffOptions(TiffExpectedFormat.Default)
    {
        BitsPerSample = new ushort[] { 8, 8, 8 },
        Photometric = TiffPhotometrics.Rgb
    };
    
    // حفظ الصورة المعدلة كملف TIFF
    rasterImage.Save(Path.Combine(outputDir, "AdjustContrast_out.tiff"), tiffOptions);
}
```
*توضيح:* يؤدي ضبط التباين إلى تعديل كثافة البكسل لتعزيز أو تقليل الاختلافات. الحفظ باستخدام أدوات محددة `TiffOptions` ضمان الجودة وتناسق التنسيق.
