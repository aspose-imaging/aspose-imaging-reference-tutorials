## دليل التنفيذ

سوف يرشدك هذا القسم خلال تنفيذ الميزات المختلفة باستخدام Aspose.Imaging لـ .NET.

### تحميل الصورة وتطبيعها

#### ملخص
تحميل صورة إلى تطبيقك هو الخطوة الأولى في معالجتها. بعد التحميل، يضمن تطبيع الهيستوجرام توزيعًا أفضل للألوان.

#### الخطوة 1: تحميل صورة JPEG
لتحميل صورة، حدد المسار إلى ملفك:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/sample.jpg";
using (RasterImage image = (RasterImage)Image.Load(dataDir))
{
    // متابعة المعالجة...
}
```

#### الخطوة 2: تطبيع الهيستوجرام
تعمل التطبيع على ضبط توازن الألوان في الصورة، مما يجعلها تبدو أكثر حيوية.

```csharp
// تطبيع الهيستوجرام لتحسين توزيع الألوان
image.NormalizeHistogram();
```

### ضبط تباين الصورة
يمكن لضبط التباين أن يُبرز الصورة من خلال زيادة عمقها البصري. إليك كيفية القيام بذلك:

#### الخطوة 1: حفظ الصورة الطبيعية
قبل التعديل، احفظ الصورة الطبيعية:

```csharp
string outputFilePath = "YOUR_OUTPUT_DIRECTORY" + "/result.png";
image.Save(outputFilePath);
```

#### الخطوة 2: ضبط التباين والحفظ مرة أخرى
زيادة أو تقليل التباين باستخدام `AdjustContrast`:

```csharp
// زيادة التباين بمقدار 30 وحدة
image.AdjustContrast(30);

// حفظ الصورة المعدلة في ملف آخر
string outputFilePath2 = "YOUR_OUTPUT_DIRECTORY" + "/result2.png";
image.Save(outputFilePath2);
```

### التنظيف: حذف الملفات المحفوظة
بعد المعالجة، قم بالتنظيف عن طريق حذف الملفات المؤقتة:

```csharp
File.Delete(outputFilePath);
File.Delete(outputFilePath2);
```
