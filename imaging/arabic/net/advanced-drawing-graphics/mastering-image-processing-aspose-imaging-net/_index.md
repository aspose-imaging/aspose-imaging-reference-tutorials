## دليل التنفيذ

### تحويل الصور

**ملخص:**
يعد تحويل الصور أمرًا ضروريًا لتحويل الصور إلى تنسيقات مختلفة، وتحسينها للمنصات أو التطبيقات المختلفة.

#### عملية خطوة بخطوة:

**1. تحميل الصورة**
```csharp
using (var image = Image.Load("input.jpg"))
{
    // المضي قدما في المعالجة
}
```
*لماذا؟* يؤدي تحميل الصورة إلى تهيئتها في الذاكرة للتعامل معها بشكل أكبر.

**2. التحويل والحفظ**
```csharp
image.Save("output.png", new PngOptions());
```
*المعايير والغرض:* ال `Save` تحفظ الطريقة الصورة بتنسيق محدد، وهنا يتم تحويلها إلى PNG مع الخيارات الافتراضية.

### معالجة الصور

**ملخص:**
قم بالتلاعب بالصور عن طريق تغيير حجمها أو قصها أو تطبيق المرشحات.

#### عملية خطوة بخطوة:

**1. تغيير حجم الصورة**
```csharp
using (var image = Image.Load("input.jpg"))
{
    image.Resize(200, 200);
    image.Save("resized.jpg");
}
```
*توضيح:* يقوم هذا المقطع بتغيير حجم الصورة إلى 200 × 200 بكسل.

### نصائح استكشاف الأخطاء وإصلاحها

- **مشكلة شائعة:** قد تحدث أخطاء عدم العثور على الملف إذا كانت المسارات غير صحيحة.
- **حل:** تأكد من مسارات الملفات وتأكد من وجود الصور في الدليل المحدد.
