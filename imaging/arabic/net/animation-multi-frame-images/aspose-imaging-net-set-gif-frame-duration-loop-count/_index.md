## دليل التنفيذ

تم تقسيم هذا الدليل إلى أقسام حسب الميزة، مما يضمن لك اكتساب المعرفة الشاملة لكل جانب.

### ضبط مدة إطار GIF

#### ملخص
يتيح لك ضبط مدة الإطار التحكم في مدة ظهور كل إطار في صورة GIF المتحركة. يُعد هذا أمرًا بالغ الأهمية لمزامنة الرسوم المتحركة مع الوسائط الأخرى أو لتحقيق التأثيرات المرئية المطلوبة.

#### خطوات التنفيذ

**1. قم بتحميل صورة GIF**
ابدأ بتحميل صورة GIF الخاصة بك من دليل محدد:
```csharp
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "gif_images");
string filepath = Path.Combine(dataDir, "example.gif");

using (GifImage image = (GifImage)Image.Load(filepath))
{
    // معالجة إضافية...
}
```

**2. تعيين مدة الإطار**
قم بتعيين مدة جميع الإطارات إلى 2000 مللي ثانية وقم بتخصيص توقيتات الإطارات الفردية:
```csharp
image.SetFrameTime(2000); // تعيين وقت الإطار الافتراضي

// تخصيص مدة الإطار الأول
((GifFrameBlock)image.Pages[0]).FrameTime = 200; // تعيين وقت إطار محدد
```

**توضيح:**
- `SetFrameTime(2000)`:يتم تكوين كافة الإطارات لعرضها لمدة 2000 مللي ثانية بشكل افتراضي.
- `((GifFrameBlock)image.Pages[0]).FrameTime = 200;`:ضبط مدة الإطار الأول، مع توضيح كيفية استهداف إطارات محددة.

### ضبط عدد حلقات GIF

#### ملخص
يُحدد التحكم في عدد مرات تشغيل ملف GIF عدد مرات تشغيله. هذه الميزة مفيدة لإنشاء رسوم متحركة تحتاج إلى تكرار عدد محدد من المرات.

#### خطوات التنفيذ

**1. قم بتحميل ملف GIF وحفظه**
قم بتحميل الصورة وحفظها بعدد محدد من الحلقات:
```csharp
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.gif");

using (GifImage image = (GifImage)Image.Load(filepath))
{
    // ضبط عدد الحلقات إلى 4 مرات
    image.Save(outputPath, new GifOptions() { LoopsCount = 4 });
}
```

**توضيح:**
- `LoopsCount = 4`:يتم تكوين GIF ليتم تشغيله أربع مرات قبل التوقف.

### حذف ملف الإخراج

#### ملخص
يضمن التنظيف بعد المعالجة أن يظل دليل الإخراج الخاص بك منظمًا عن طريق إزالة الملفات غير الضرورية.

#### خطوات التنفيذ

**1. حذف الملف المحدد**
التحقق من وجود الملف وحذفه إذا لزم الأمر:
```csharp
string outputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.gif");

if (File.Exists(outputPath))
{
    File.Delete(outputPath);
}
```
