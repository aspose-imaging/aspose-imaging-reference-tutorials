### دليل التنفيذ

#### نظرة عامة على الميزة
تتيح لك هذه الميزة إنشاء صور GIF من خلال الجمع بين إطارات متعددة، مما يجعلها مثالية للرسوم المتحركة أو تسلسلات الصور.

##### الخطوة 1: تحديد الدلائل
ابدأ بتحديد مستندك ومجلدات الإخراج:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

##### الخطوة 2: تحميل الإطارات
قم بتحميل جميع إطارات الصور التي ترغب في تضمينها في ملف GIF الخاص بك:

```csharp
private static IEnumerable<RasterImage> LoadFrames(string directory)
{
    foreach (var filePath in Directory.GetFiles(directory))
    {
        yield return (RasterImage)Image.Load(filePath);
    }
}
```
هنا، نقوم بتكرار الملفات في دليل محدد وتحميلها كـ `RasterImage` أشياء.

##### الخطوة 3: إنشاء ملف GIF
قم بتهيئة ملف GIF الخاص بك بالإطار الأول، ثم أضف إطارات إضافية:

```csharp
var frames = LoadFrames(Path.Combine(dataDir, "Animation frames")).ToArray();

using (var image = new GifImage(new GifFrameBlock(frames[0])))
{
    for (var index = 1; index < frames.Length; index++)
    {
        image.AddPage(frames[index]);
    }
    
    image.Save(Path.Combine(outputDir, "Multipage.gif"));
}
```
يقوم هذا المقطع بتحميل كل إطار وإضافته بشكل تسلسلي إلى ملف GIF. `GifImage` يقوم المنشئ بتهيئة GIF باستخدام الإطار الأول.
