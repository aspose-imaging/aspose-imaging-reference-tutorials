## دليل التنفيذ

الآن، دعونا نقسم العملية إلى خطوات قابلة للإدارة.

### الميزة 1: إنشاء الرسوم المتحركة من صورة متعددة الصفحات

تتيح لك هذه الميزة تحويل صورة TIFF متعددة الصفحات إلى ملف PNG متحرك. إليك الطريقة:

#### الخطوة 1: تحميل صورة TIFF المدخلة

ابدأ بتحميل صورة TIFF الخاصة بك باستخدام Aspose.Imaging `Image.Load` طريقة.

```csharp
using Aspose.Imaging;
using System.IO;

string inputFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "img4.tif");
```

#### الخطوة 2: تحديد مسار الإخراج لملف PNG المتحرك

قم بتعيين مسار الإخراج حيث سيتم حفظ ملف PNG المتحرك الخاص بك:

```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "img4.tif.500ms.png");
```

#### الخطوة 3: تحويل TIFF إلى APNG

يستخدم `Image.Save` الطريقة مع `ApngOptions` لإنشاء صورة متحركة بصيغة PNG. مدة الإطار ٥٠٠ ميلي ثانية.

```csharp
using (Image image = Image.Load(inputFilePath))
{
    image.Save(outputFilePath, new ApngOptions() { DefaultFrameTime = 500 });
}
```

#### الخطوة 4: التنظيف

بعد المعالجة، قد ترغب في حذف ملف الإخراج. هذا اختياري، ويمكنك القيام به كما يلي:

```csharp
File.Delete(outputFilePath);
```

### الميزة 2: إدارة مسار الدليل

إن إدارة مسارات الدليل بكفاءة تضمن وضع ملفات الإدخال والإخراج الخاصة بك في مكانها الصحيح.

#### إنشاء مسارات الملفات

قم بتحديد مستندك ومجلدات الإخراج باستخدام العناصر النائبة، ثم قم بدمجها مع أسماء الملفات لإنشاء مسارات ملفات كاملة.

```csharp
string docDir = "YOUR_DOCUMENT_DIRECTORY";
string outDir = "YOUR_OUTPUT_DIRECTORY";

string inputFile = Path.Combine(docDir, "img4.tif");
string outputFile = Path.Combine(outDir, "img4.tif.500ms.png");
```
