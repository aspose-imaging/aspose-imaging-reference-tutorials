## دليل التنفيذ

### إضافة صورة مصغرة إلى مقطع EXIF

تتيح لك هذه الميزة تضمين صورة مصغرة مباشرة في بيانات EXIF الخاصة بملفات JPEG.

#### الخطوة 1: إعداد الدلائل

قم بتحديد المسارات الخاصة بمجلدات الإدخال والإخراج الخاصة بك:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY/";
```

#### الخطوة 2: إنشاء صورة مصغرة

إنشاء صورة JPEG جديدة لتكون بمثابة الصورة المصغرة:
```csharp
JpegImage thumbnailImage = new JpegImage(100, 100);
```

#### الخطوة 3: تحضير الصورة الرئيسية باستخدام بيانات EXIF

قم بإنشاء صورة JPEG أخرى وقم بتهيئة بيانات EXIF الخاصة بها:
```csharp
JpegImage image = new JpegImage(1000, 1000);
image.ExifData = new JpegExifData();
image.ExifData.Thumbnail = thumbnailImage;
```

#### الخطوة 4: حفظ الصورة

احفظ الصورة المعدلة مع الصورة المصغرة المضمنة في دليل الإخراج الخاص بك:
```csharp
string outputPath = Path.Combine(outputDirectory, "thumbnail_out.jpg");
image.Save(outputPath);
```

### معالجة تدفق الملفات باستخدام MemoryStream

التعامل مع الصور باستخدام `MemoryStream` يمكن أن يكون مفيدًا للمعالجة في الذاكرة دون الكتابة على القرص.

#### الخطوة 1: تهيئة MemoryStream

إنشاء مثيل لـ `MemoryStream`:
```csharp
using (MemoryStream stream = new MemoryStream())
{
    JpegImage jpegImage = new JpegImage(500, 500);
    
    // يمكن إجراء العمليات على jpegImage هنا
    
    jpegImage.Save(stream);
}
```

يوضح هذا المثال كيفية حفظ صورة JPEG في مجرى الذاكرة.
