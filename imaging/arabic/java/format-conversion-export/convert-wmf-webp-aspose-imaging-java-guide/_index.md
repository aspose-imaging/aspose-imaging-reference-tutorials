## كيفية تحويل WMF إلى WebP باستخدام Aspose.Imaging؟

حمّل ملف WMF الخاص بك باستخدام `Image.load()` واستدعِ `save()` باستخدام كائن `WebPOptions`—هذا النمط ذو الخطوتين يكمل التحويل. تقوم المكتبة تلقائيًا برستر ملف WMF المتجه، وتطبيق خيارات الرستر التي تحددها، وتكتب ملف WebP يحافظ على جودة الصورة الأصلية. `Image.load()` يقرأ ملف WMF من القرص ويعيد كائن Image جاهز للمعالجة.

## دليل التنفيذ

### الميزة 1: تحميل صورة WMF

#### نظرة عامة:
تظهر هذه الميزة كيفية تحميل صورة WMF من دليل محدد باستخدام Aspose.Imaging للـ Java.

#### مرساة التعريف
تمثل الفئة `Image` أي صيغة صورة مدعومة وتوفر طرقًا ثابتة للتحميل والحفظ.

#### تنفيذ خطوة بخطوة

##### استيراد الفئات المطلوبة
```java
import com.aspose.imaging.Image;
```  

##### تحميل صورة WMF
حدد دليل المستند الخاص بك واستخدم طريقة `Image.load()`:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Replace with your path
try (Image image = Image.load(dataDir + "/input.wmf")) {
    // The WMF image is now loaded and ready for manipulation.
}
```  

### الميزة 2: إنشاء WmfRasterizationOptions

#### نظرة عامة:
توضح هذه الميزة كيفية تكوين خيارات الرستر لتخصيص طريقة معالجة صورة WMF.

#### مرساة التعريف
`WmfRasterizationOptions` يحدد كيفية رستر محتوى WMF المتجه، بما في ذلك DPI، لون الخلفية، وأبعاد الصفحة.

#### تنفيذ خطوة بخطوة

##### استيراد الفئات المطلوبة
```java
import com.aspose.imaging.Color;
import com.aspose.imaging.imageoptions.WmfRasterizationOptions;
```  

##### إعداد خيارات الرستر
إنشاء وتكوين `WmfRasterizationOptions`:
```java
double k = (image.getWidth() * 1.00) / image.getHeight();
WmfRasterizationOptions wmfRasterization = new WmfRasterizationOptions() {
    {
        setBackgroundColor(Color.getWhiteSmoke()); // Background color: white smoke
        setPageWidth(400); // Set page width to 400 units
        setPageHeight((int) Math.round(400 / k)); // Maintain aspect ratio for height
        setBorderX(5); // Horizontal border size
        setBorderY(10); // Vertical border size
    }
};
```  

### الميزة 3: تكوين WebPOptions لحفظ كـ WebP

#### نظرة عامة:
إعداد الخيارات لحفظ صورة WMF بصيغة WebP باستخدام إعدادات الرستر التي تم تكوينها مسبقًا.

#### مرساة التعريف
`WebPOptions` يضم إعدادات خاصة بـ WebP مثل الضغط بدون فقدان، الجودة، وما إذا كان سيتم استخدام الشفافية.

#### تنفيذ خطوة بخطوة

##### استيراد الفئات المطلوبة
```java
import com.aspose.imaging.imageoptions.WebPOptions;
import com.aspose.imaging.ImageOptionsBase;
```  

##### تكوين WebPOptions
تعيين خيارات الرستر إلى `WebPOptions`:
```java
ImageOptionsBase imageOptions = new WebPOptions();
imageOptions.setVectorRasterizationOptions(wmfRasterization);
```  

### الميزة 4: حفظ الصورة كـ WebP

#### نظرة عامة:
حفظ صورة WMF التي تم تحميلها بصيغة WebP باستخدام Aspose.Imaging للـ Java.

#### مرساة التعريف
استدعاء `save()` على كائن `Image` مع كائن `WebPOptions` يكتب ملف الإخراج بصيغة WebP.

#### تنفيذ خطوة بخطوة

##### استيراد الفئات المطلوبة
```java
import com.aspose.imaging.examples.Utils; // Ensure you have this or similar utility class if needed
```  

##### حفظ كـ WebP
حدد دليل الإخراج الخاص بك واحفظ الصورة:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // Replace with your path
image.save(outputDir + "/ConvertWMFToWebp_out.webp", imageOptions);
```

- **رسومات متعددة المنصات:** تقديم أصل WebP واحد للمتصفحات التي تدعمه، مع الرجوع إلى PNG للعملاء القدامى.

- **تحسين استخدام الذاكرة:** استخدم try‑with‑resources أو استدعِ `dispose()` صراحةً على كائنات الصورة لتحرير الذاكرة الأصلية بسرعة.  
- **اختيار صيغ كفء:** يقدم WebP نسبة ضغط‑جودة متفوقة؛ يفضَّل على PNG عندما لا تكون الجودة بدون فقدان إلزامية.
