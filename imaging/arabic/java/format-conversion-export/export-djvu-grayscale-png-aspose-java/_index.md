## كيفية تحميل صورة DjVu؟
تحميل ملف DjVu سهل مع Aspose.Imaging. استخدم الفئة `Image` لفتح المستند داخل كتلة `try‑with‑resources`، التي تُطلق تلقائيًا موارد الملفات بعد المعالجة. قدم مسار ملف .djvu، وستقوم المكتبة بتحليل صفحاته وجعلها متاحة للمزيد من المعالجة.

`Image` هي فئة في Aspose.Imaging تمثل ملف صورة وتوفر طرقًا لتحميل ومعالجة بيانات الصورة.

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.djvu.DjvuImage;

String dataDir = "YOUR_DOCUMENT_DIRECTORY/Sample.djvu";
try (DjvuImage image = (DjvuImage) Image.load(dataDir)) {
    // The image is now loaded and ready for further processing.
}
```

- **`dataDir`**: مسار ملف DjVu الخاص بك. غيّر `YOUR_DOCUMENT_DIRECTORY` إلى المسار الفعلي للمجلد.  
- **`try‑with‑resources`**: يدير تحرير الموارد تلقائيًا بعد التحميل.

## كيفية تكوين خيارات تصدير PNG؟
لإنشاء PNG بتدرج رمادي، أنشئ كائن `PngOptions` واضبط خاصية `colorType` إلى `PngColorType.Grayscale`. يضمن هذا الإعداد أن الصورة الناتجة تستخدم قناة شدة واحدة، مما يقلل حجم الملف مع الحفاظ على التفاصيل البصرية. يمكن أيضًا تعديل خيارات إضافية مثل مستوى الضغط حسب الحاجة.

`PngOptions` يحدد إعدادات حفظ الصور بصيغة PNG، مثل نوع اللون، مستوى الضغط، والتداخل.  
`PngColorType` يعدد صيغ اللون المدعومة لملفات PNG، بما في ذلك Grayscale، Truecolor، وIndexed.

```java
import com.aspose.imaging.imageoptions.PngOptions;
import com.aspose.imaging.fileformats.png.PngColorType;

PngOptions exportOptions = new PngOptions();
exportOptions.setColorType(PngColorType.Grayscale);
```

- **`PngColorType.Grayscale`**: يحول الصورة إلى تدرج رمادي، مثالي للأرشفة أو تقليل حجم الملف.

## كيفية تحديد منطقة التصدير في صفحة DjVu؟
حدد منطقة مستطيلة لاستخلاصها من صفحة DjVu عن طريق ضبط خاصية `exportArea` في `DjvuMultiPageOptions`. تُعرّف المستطيل بالبكسل وتحدد الجزء من الصفحة الذي سيُرسم ويُحفظ، مما يتيح لك التركيز على منطقة معينة من الاهتمام.

`DjvuMultiPageOptions` يحدد خيارات تصدير الصفحات من مستند DjVu، بما في ذلك فهرس الصفحة ومنطقة التصدير.

```java
import com.aspose.imaging.Rectangle;

Rectangle exportArea = new Rectangle(0, 0, 500, 500);
int exportPageIndex = 2; // Index starts from zero.
```

- **`exportArea`**: يحدد أبعاد المستطيل للقص. عدّل هذه القيم وفقًا لاحتياجاتك.

## كيفية تعيين خيارات التصدير المتعدد الصفحات؟
عند التعامل مع ملفات DjVu متعددة الصفحات، استخدم `DjvuMultiPageOptions` لاختيار الصفحة المطلوبة عبر خاصية `pageIndex` وربما دمجها مع `exportArea`. يتيح لك هذا النهج تصدير منطقة محددة من أي صفحة دون معالجة المستند بالكامل، مما يحسن الأداء للملفات الكبيرة.

```java
import com.aspose.imaging.fileformats.djvu.DjvuMultiPageOptions;

exportOptions.setMultiPageOptions(new DjvuMultiPageOptions(exportPageIndex, exportArea));
```

- **`DjvuMultiPageOptions`**: يجمع بين فهرس الصفحة والمنطقة لتصدير أقسام محددة من ملف DjVu متعدد الصفحات.

## كيفية حفظ الصورة؟
بعد تكوين خيارات التصدير، استدعِ طريقة `save` على كائن `Image`، مع توفير مسار الملف الهدف وكائن `PngOptions`. تأكد من وجود دليل الإخراج وصلاحيات الكتابة؛ ستقوم المكتبة بعد ذلك بكتابة PNG بتدرج رمادي إلى القرص.

`save` هي طريقة في فئة `Image` تكتب بيانات الصورة إلى ملف محدد باستخدام خيارات الصيغة المقدمة.

```java
import com.aspose.imaging.Image;

String outputPath = "YOUR_OUTPUT_DIRECTORY/ConvertSpecificPortionOfDjVuPage_out.png";
image.save(outputPath, exportOptions);
```

- **`outputPath`**: حدد المكان الذي تريد حفظ ملف PNG فيه. تأكد من وجود الدليل قبل التنفيذ.
