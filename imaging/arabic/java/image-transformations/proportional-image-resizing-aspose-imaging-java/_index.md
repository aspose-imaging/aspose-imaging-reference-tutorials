# إتقان تغيير حجم الصور بشكل متناسب باستخدام Aspose.Imaging لـ Java

## مقدمة

هل سئمت من تغيير حجم الصور يدويًا والحفاظ على نسب أبعادها؟ سواء كنت مطورًا تعمل على تطبيقات معالجة الصور أو ترغب ببساطة في أتمتة سير عملك، فإن الأدوات المناسبة تُحدث فرقًا كبيرًا. في هذا الدليل الشامل، سنستكشف كيفية استخدام Aspose.Imaging لجافا لتحميل الصور وتغيير حجمها بشكل متناسب وحفظها بسهولة. 

**ما سوف تتعلمه:**
- كيفية تحميل صورة من دليل محدد.
- تقنيات لتغيير حجم الصورة بشكل متناسب حسب العرض أو الارتفاع باستخدام Aspose.Imaging.
- طرق تخزين الصور مؤقتًا لتحسين الأداء.
- خطوات لحفظ الصورة المعالجة بكفاءة.

دعنا نتعمق في هذا البرنامج التعليمي، ولكن أولاً، دعنا نتأكد من أن لديك كل ما تحتاجه للبدء!

### المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك:
- **مجموعة تطوير Java (JDK):** تأكد من تثبيت JDK على جهازك. يُنصح باستخدام الإصدار 8 أو أعلى.
- **مكتبة Aspose.Imaging لـ Java:** سوف تحتاج إلى هذه المكتبة لمتابعة الأمثلة المقدمة.
- **IDE/محرر النصوص:** أي بيئة تطوير متكاملة تدعم Java، مثل IntelliJ IDEA أو Eclipse، ستعمل بشكل مثالي.

## إعداد Aspose.Imaging لـ Java

لبدء استخدام Aspose.Imaging في مشاريعك، لديك عدة خيارات تثبيت. اختر واحدًا بناءً على أداة بناء مشروعك:

### مافن
أضف التبعية التالية إلى ملفك `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-imaging</artifactId>
    <version>25.5</version>
</dependency>
```

### جرادل
قم بتضمين هذا السطر في `build.gradle` ملف:
```gradle
compile(group: 'com.aspose', name: 'aspose-imaging', version: '25.5')
```

### التحميل المباشر
بدلاً من ذلك، قم بتنزيل أحدث ملف JAR من [إصدارات Aspose.Imaging لـ Java](https://releases.aspose.com/imaging/java/).

**خطوات الحصول على الترخيص:**
- **نسخة تجريبية مجانية:** ابدأ بإصدار تجريبي مجاني لاستكشاف إمكانيات المكتبة.
- **رخصة مؤقتة:** احصل على ترخيص مؤقت للاستخدام الموسع دون قيود.
- **شراء:** بالنسبة للمشاريع طويلة الأمد، فكر في شراء ترخيص كامل.

**التهيئة والإعداد الأساسي:**

## دليل التنفيذ

الآن بعد أن قمت بإعداد كل شيء، دعنا نتعمق في تنفيذ ميزات تغيير حجم الصورة باستخدام Aspose.Imaging لـ Java.

### تحميل الصورة

**ملخص:** تحميل الصورة هو الخطوة الأولى قبل أي معالجة. سنشرح كيفية تحميل صورة من مجلد والتأكد من تخزينها مؤقتًا لتحسين الأداء.

#### التنفيذ خطوة بخطوة:

1. **تعيين دليل المستندات الخاص بك:**
   ```java
   String dataDir = Utils.getSharedDataDir() + "ModifyingImages/";
   ```

2. **تحميل الصورة:**
   يستخدم `Image.load()` لقراءة ملف صورة.
   ```java
   try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
       if (!image.isCached()) {
           image.cacheData(); // تخزين الصورة مؤقتًا للحصول على أداء أفضل
       }
   }
   ```

### تغيير حجم الصورة بشكل متناسب حسب العرض

**ملخص:** تتيح لك هذه الميزة تغيير حجم الصورة استنادًا إلى العرض الجديد مع الحفاظ على نسبة العرض إلى الارتفاع.

#### التنفيذ خطوة بخطوة:

1. **حساب العرض الجديد:**
   ```java
   int newWidth = image.getWidth() / 2;
   ```

2. **تغيير حجم الصورة بشكل متناسب:**
   اضبط الارتفاع بشكل متناسب باستخدام `resizeHeightProportionally`.
   ```java
   try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
       if (!image.isCached()) {
           image.cacheData();
       }
       image.resizeHeightProportionally(newWidth);
   }
   ```

### تغيير حجم الصورة بشكل متناسب مع الارتفاع

**ملخص:** على غرار تغيير الحجم حسب العرض، يمكنك أيضًا تغيير حجم الصورة استنادًا إلى ارتفاع جديد.

#### التنفيذ خطوة بخطوة:

1. **حساب الارتفاع الجديد:**
   ```java
   int newHeight = image.getHeight() / 2;
   ```

2. **تغيير حجم الصورة بشكل متناسب:**
   اضبط العرض بشكل متناسب باستخدام `resizeWidthProportionally`.
   ```java
   try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
       if (!image.isCached()) {
           image.cacheData();
       }
       image.resizeWidthProportionally(newHeight);
   }
   ```

### حفظ الصورة

**ملخص:** بعد المعالجة، يعد حفظ الصورة في دليل الإخراج المحدد أمرًا بالغ الأهمية.

#### التنفيذ خطوة بخطوة:

1. **حفظ الصورة المعالجة:**
   ```java
   try (Image image = Image.load(dataDir + "aspose-logo.jpg")) {
       if (!image.isCached()) {
           image.cacheData();
       }
       // حفظ الصورة المعالجة
       image.save(Utils.getOutDir() + "ResizeImageProportionally_out.png");
   }
   ```
