- طرق حفظ الصور المعدلة على القرص.

## دليل التنفيذ

### الميزة 1: تحميل صورة JPEG

#### ملخص
تحميل الصورة هو الخطوة الأولى في أي عملية معالجة صور. إليك كيفية تحميل صورة JPEG من مجلد محدد.

##### التنفيذ خطوة بخطوة

**H3: استيراد المكتبات**

```java
import com.aspose.imaging.Image;
import com.aspose.imaging.fileformats.jpeg.JpegImage;
```

**H3: تحميل الصورة**

قم بتعيين مسار دليل المستند الخاص بك وتحميل الصورة:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
try (JpegImage image = (JpegImage) Image.load(dataDir + "aspose-logo.jpg")) {
    // تم تحميل الصورة الآن للتلاعب بها
}
```

**توضيح**: ال `Image.load()` تقرأ الطريقة ملف JPEG في `JpegImage` الكائن، مما يتيح العمليات اللاحقة.

### الميزة 2: التدوير التلقائي لصورة JPEG استنادًا إلى بيانات اتجاه EXIF

#### ملخص
يضمن تدوير الصور تلقائيًا استنادًا إلى بيانات اتجاه EXIF الخاصة بها عرضها بشكل صحيح دون الحاجة إلى تعديلات يدوية.

##### التنفيذ خطوة بخطوة

**H3: تحميل الصورة وتدويرها تلقائيًا**

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
try (JpegImage image = (JpegImage) Image.load(dataDir + "aspose-logo.jpg")) {
    // تنفيذ التدوير التلقائي استنادًا إلى اتجاه EXIF
    image.autoRotate();
}
```

**توضيح**: ال `autoRotate()` تعمل الطريقة على ضبط اتجاه الصورة وفقًا لبيانات EXIF المضمنة فيها، مما يؤدي إلى تبسيط سير عملك.

### الميزة 3: حفظ صورة JPEG على القرص

#### ملخص
بعد معالجة الصورة، يُعدّ حفظها على القرص أمرًا بالغ الأهمية للتخزين أو الاستخدام لاحقًا. إليك كيفية حفظ صورة JPEG مُعدّلة.

##### التنفيذ خطوة بخطوة

**H3: تحميل الصورة وتدويرها تلقائيًا وحفظها**

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";

try (JpegImage image = (JpegImage) Image.load(dataDir + "aspose-logo.jpg")) {
    // تنفيذ التدوير التلقائي استنادًا إلى اتجاه EXIF
    image.autoRotate();

    // حفظ الصورة المعالجة على القرص
    image.save(outDir + "AutoCorrectOrientationOfJPEGImages_out.jpg");
}
```

**توضيح**: ال `save()` تكتب الطريقة ملف JPEG الذي تمت معالجته مرة أخرى إلى الدليل المحدد، مع الحفاظ على جميع التعديلات.
